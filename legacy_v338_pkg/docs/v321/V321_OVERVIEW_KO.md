# V321 개요

V321은 V319의 “현장 운영형” 구조를 유지하면서,
실무에서 가장 많이 발생하는 문제 3가지를 직접 해결합니다.

1) 부분 매칭이 편하지만 오탐이 걱정된다  
→ **정규식 룰/가중치/화이트리스트 + 최소 점수(threshold)**로 오탐 감소

2) Drill-down 테이블이 있어도 눈으로 찾아야 한다  
→ 클릭 기반 필터로 **원인 추적 속도** 개선

3) 데이터가 커지면 부분 매칭이 느려진다  
→ prefix/token 인덱싱으로 **대규모 데이터에서도 빠르게**


## V321 추가 기능
- **커버리지(coverage)**: 전체 전환/매출 대비 귀속된 비율을 제공해, 귀속 품질을 빠르게 점검합니다.
- **진단(Diagnostics)**: 미귀속인데 광고비가 큰 항목을 상위로 보여줘 룰 튜닝 포인트를 쉽게 찾습니다.
- **CSV 내보내기**: 대시보드에서 주요 테이블을 CSV로 다운로드할 수 있습니다.

# What is GENIE_ROI (V312) – Simple Explanation

## One-sentence explanation
GENIE_ROI is a **Growth Operating System** that helps a company run **commerce + ads + influencers** in one place,
with **safety controls (policies/approvals/roles)** so teams can automate safely.

## What problems it solves (in plain words)
When you sell on many channels and run ads, you usually face:
- data scattered across many dashboards
- manual Excel work
- over-selling inventory
- ad spend increasing without control
- teams stepping on each other (marketing vs finance vs ops)

GENIE_ROI brings these into **one governed workflow**:
1) collect data (orders, inventory, ad reports, influencer metrics)
2) analyze results (by product/SKU, channel, campaign, creator)
3) recommend actions (scale/hold/stop)
4) execute actions with guardrails (policy checks + approvals + audit log)

## Who uses it
- Brand/Manufacturer (own products)
- Distributor/Reseller
- Agency (performance marketing)
- Global teams (KR + US + SEA etc.)

## Core idea
Automation is powerful but risky.
GENIE_ROI does **controlled automation**:
- checks rules (budget caps, margin guardrails)
- asks approval when needed
- shows a preview/diff before applying changes
- logs everything for auditing

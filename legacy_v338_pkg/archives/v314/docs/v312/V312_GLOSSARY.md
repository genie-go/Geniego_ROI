# Glossary

- Tenant: one customer/company inside the SaaS (multi-tenant)
- RBAC: Role-Based Access Control (role decides what pages/actions you can access)
- Policy: rules that prevent risky automation (e.g., budget cap)
- Approval: human checkpoint (e.g., finance approves budget increase)
- Audit log: history of who did what/when
- Bulk action: select many rows and run one operation (pause campaigns, request price change, etc.)
- Webhook: provider sends events to us (real-time signals)
- Rate limit: API call limit by provider; we throttle/retry safely

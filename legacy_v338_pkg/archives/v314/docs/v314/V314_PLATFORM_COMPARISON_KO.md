# V314 플랫폼 비교(요약) - 광고 중심 & 인플루언서/ROI 운영 관점

> 이 문서는 '기능 관점' 비교이며, 실제 선택은 조직의 예산/팀 역량/데이터 요구에 따라 달라질 수 있습니다.

## 1) 마케팅 자동화/CRM 계열
- HubSpot: CRM 기반 리드/이메일 자동화, 캠페인 관리, 리포팅 강점
- Salesforce Marketing Cloud: 고객 여정(Journey) 기반 메시징/세그먼트, 엔터프라이즈 운영 강점

**V314 차별점**
- 광고/커머스 매출 데이터와 연결된 **ROI 중심 운영 루프**를 템플릿/대시보드로 빠르게 구축
- 인플루언서 포함/미포함을 선택형으로 운영(Feature Toggle)

## 2) 광고 운영(Ads Manager) 계열
- Google Ads / Meta Ads Manager 등: 채널 내부 최적화/집행 기능이 강함(입찰/타겟/소재)
- 하지만 여러 채널을 동시에 운영하면, '전사 ROI' 관점으로 비교/집계가 어려워짐

**V314 차별점**
- 채널이 달라도 공통 스키마로 묶어 **채널/캠페인별 ROAS/CPA**를 같은 기준으로 비교
- 인플루언서를 쓰는 경우에도 동일한 ROI 지표로 기여도 비교 가능

## 3) 인플루언서 전용 플랫폼
- Upfluence / CreatorIQ 등: 인플루언서 탐색/관리/성과 측정, 사기 탐지, 캠페인 운영을 제공

**V314 차별점**
- 인플루언서 운영을 하더라도, 광고/커머스/운영 데이터를 묶어 '돈' 기준으로 의사결정
- '광고만 운영'으로 시작했다가, 필요 시 인플루언서 모듈을 켜는 확장성

## 4) 추천 선택 가이드(빠른 결론)
- **광고만 빠르게 최적화**가 목적 → 채널 Ads Manager + V314 ROI 리포트/대시보드 조합
- **CRM/리드 너처링 중심** → HubSpot/SFMC + V314(커머스/광고 ROI 레이어)
- **대규모 인플루언서 운영** → Upfluence/CreatorIQ + V314(전사 ROI/매출 연결 레이어)

# V314: 광고만 운영(인플루언서 미사용) 모드 상세 설명

## 1) 왜 필요한가?
- 업종/브랜드 전략에 따라 **인플루언서를 쓰지 않고 광고만**으로 성장을 만드는 경우가 많습니다.
- 또한 인플루언서 운영은 검증/계약/콘텐츠 제작 리드타임이 있어, 빠른 테스트는 광고가 유리할 수 있습니다.

V314는 인플루언서 모듈을 **선택적으로 끄고**도(또는 나중에 켜고) 동일한 ROI 운영 체계를 적용할 수 있게 설계했습니다.

---

## 2) 광고만 모드에서 제공하는 기능(설계)
### A. 캠페인 성과 수집
- 채널별 캠페인 지표: 노출, 클릭, CTR, CPC, 광고비(Spend)
- 최소 공통 스키마로 정규화해 저장/집계
- (선택) 플랫폼 API 연동 또는 CSV 업로드 방식 지원

### B. 전환/매출(또는 리드) 연결
- UTM, 전환ID, 쿠폰코드 등으로 전환/매출을 캠페인에 연결
- 핵심 산출:
  - **ROAS = 매출 / 광고비**
  - **CPA = 광고비 / 전환수**
  - 리드 기반 사업이라면 CPL(리드당 비용) 제공

### C. 성과 최적화 지원(운영 기능)
- 채널/캠페인/소재(크리에이티브) 단위의 효율 비교
- 예산 재배분(수동/추천) 의사결정에 필요한 리포트
- A/B 테스트 기록(소재/카피/랜딩)과 결과(CTR/CVR/ROAS) 연결

---

## 3) 인플루언서 모드와의 차이
| 구분 | 인플루언서 포함 | 광고만 운영 |
|---|---|---|
| 핵심 데이터 | 콘텐츠 성과 + 전환/매출 | 캠페인 성과 + 전환/매출 |
| 리드타임 | 상대적으로 김(제작/계약) | 빠름(즉시 집행/중단) |
| 리스크 | 가짜 팔로워/브랜드 세이프티 | 플랫폼 정책/어뷰징 |
| 최적화 레버 | 크리에이터/콘텐츠/오디언스 | 타겟/입찰/소재/랜딩 |

---

## 4) 초보자용 체크리스트(광고만 운영)
1. 목표 정의: 구매/리드/앱설치 등
2. 추적 설계: UTM 규칙, 픽셀/SDK, 전환 이벤트 정의
3. 최소 실험 단위: 캠페인 2~3개, 소재 3~5개로 시작
4. 판단 지표: CTR/CVR/CPA/ROAS 중 무엇을 1순위로 볼지 결정
5. 리포트 루틴: 주 2회(또는 일 1회) 예산/소재/타겟 조정

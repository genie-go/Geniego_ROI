# GENIE_ROI V314 업그레이드 개요 (한국어)

V314는 V313(인플루언서 운영/검증 체계화)에 더해 다음을 **시스템 레벨로 확장**한 버전입니다.

## 1) 핵심 추가 기능
1. **인플루언서 평가 자동 리포트**
   - 엑셀/구글시트용 템플릿 제공
   - 캠페인/콘텐츠/전환 데이터를 입력하면 대시보드가 자동 집계(ROAS, CTR, 평균 조회/참여율 등)

2. **광고만 운영(인플루언서 미사용) 모드**
   - 인플루언서 없이도 광고 캠페인 성과(구글/메타/틱톡/네이버 등)를 수집·집계
   - 전환/매출(또는 리드)과 연결해 ROI(ROAS/CPA)를 산출
   - 선택형 기능(Feature Toggle)로 인플루언서 모듈 없이도 운영 가능

3. **대시보드 구조(구현 가이드)**
   - BI(예: Looker Studio/Metabase/Grafana) 또는 내부 대시보드에 적용 가능한 지표·차트 사양 제공

## 2) 추천 운영 흐름
- (A) 인플루언서 포함 운영: 인플루언서 선정 → 콘텐츠 성과 → 전환/매출 연결 → 인플루언서별 ROAS/기여도 비교
- (B) 광고만 운영: 광고 캠페인 성과 → 전환/매출 연결 → 채널/캠페인별 ROAS/CPA 최적화

## 3) 폴더 안내
- `spreadsheets/` : V314 자동 리포트 템플릿(엑셀)
- `templates/v314/` : 구글시트/CSV 템플릿, 입력 가이드
- `dashboards/v314/` : 대시보드 지표/차트 구조(명세)
- `db/migrations/0030_v314_ads_and_reporting.sql` : 광고 모듈 + 리포팅 테이블 예시
- `docs/v314/` : V314 설명, 광고만 운영 모드, 비교 문서

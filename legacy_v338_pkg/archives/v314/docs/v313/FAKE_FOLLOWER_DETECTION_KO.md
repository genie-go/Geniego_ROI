# 가짜 팔로워(팔로워 사기) 구별 방법 (V313)

가짜 팔로워/봇은 **팔로워 수는 크지만 실제 구매/상담 전환이 거의 없는** 형태로 나타납니다.
아래는 실무에서 많이 쓰는 확인 방법입니다.

## A. 빠르게 거르는 “레드 플래그” 10가지
1) 팔로워가 짧은 기간에 급증/급감(스파이크)  
2) 좋아요/댓글이 팔로워 대비 너무 낮거나, 반대로 비정상적으로 높음  
3) 댓글이 반복 문구/이모지 위주(대화가 없음)  
4) 게시물별 반응 편차가 지나치게 큼(특정 게시물만 과하게 높음)  
5) 팔로워 계정이 프로필 사진 없음/게시물 0/팔로잉 과다  
6) 오디언스 지역이 타겟과 무관하게 특정 국가에 과도하게 쏠림  
7) 조회수는 높은데 클릭/저장/공유/댓글이 매우 낮음  
8) 협찬이 과도하고 콘텐츠가 “광고만”인 느낌  
9) 최근 콘텐츠 업로드가 불규칙하거나, 갑자기 활동이 멈춤  
10) 외부 링크 클릭이 거의 없음(측정 가능 시)

## B. 숫자로 확인하는 방법 (권장)
- **참여율(ER)** = (좋아요 + 댓글 + 공유) / 팔로워  
- 최근 10개 게시물 기준으로 평균을 내고, 업계 평균과 비교합니다.
- 가능하면 **스토리/릴스/쇼츠**의 “저장/공유/완시율” 지표를 요청합니다.

## C. 플랫폼/툴로 교차검증
- 크리에이터가 제공하는 **인사이트 스크린샷(지역/연령/성별/도달)** 요청
- 서드파티 툴(예: Upfluence 등)의 **fake follower 추정치** 확인
- 브랜드 측 트래킹(UTM/쿠폰/제휴링크)으로 **실제 전환**을 검증

## D. V313에서의 표준 “Fraud Signals”
V313에서는 아래 신호를 표준화해 체크리스트/점수표에 반영합니다.
- Growth Spike Score (팔로워 증가 이상치)
- Comment Authenticity Score (댓글 자연스러움)
- Audience Match Score (타겟 국가/언어 적합)
- Engagement Consistency Score (콘텐츠별 반응 안정성)
- Conversion Reality Check (쿠폰/링크 전환 검증)

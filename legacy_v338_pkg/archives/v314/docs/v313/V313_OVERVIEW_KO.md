# GENIE_ROI V313 개요 (초보자용)

## 한 문장 설명
GENIE_ROI V313는 **커머스 + 광고 + 인플루언서** 운영 데이터를 한곳에 모으고, **ROI 중심으로 분석/추천/실행**까지 연결해 주는 “성장 운영 시스템(Growth OS)”입니다.

## V312 대비 V313에서 강화된 포인트 (인플루언서 운영)
V313은 특히 **인플루언서 선정/검증/성과측정**을 더 “시스템화”했습니다.

- **인플루언서 선정 체크리스트 + 점수화(Scoring)**
- **가짜 팔로워/비정상 트래픽 탐지 신호(Fraud Signals) 표준화**
- **산업별(뷰티/패션/식품/전자/제조/B2B 등) 캠페인 플레이북 제공**
- **링크/쿠폰/UTM 기반 기여도(ROAS/CPA/매출) 추적 가이드 강화**

> 주의: 실제로 수집 가능한 ‘세부 오디언스(연령/지역 등)’ 범위는 각 플랫폼(인스타/틱톡/유튜브) 권한, API 정책, 크리에이터 제공 인사이트 여부에 따라 달라질 수 있습니다.

## V313이 제공하는 핵심 서비스
1) **데이터 수집(Collect)**: 주문/재고/광고 리포트 + 인플루언서 게시물/성과 지표를 연결  
2) **분석(Analyze)**: SKU/채널/캠페인/크리에이터 단위 성과(노출·클릭·전환·매출) 분석  
3) **추천(Recommend)**: 확장/유지/중단 + 예산/제품/크리에이티브 방향 추천  
4) **실행(Execute)**: 승인(roles/approvals) 기반으로 안전하게 자동화 실행

## 누가 쓰면 좋은가?
- **에이전시**: 고객사별 캠페인/인플루언서 성과를 표준 리포트로 제공
- **유통사/브랜드**: 제품별 매출 기여도가 높은 크리에이터를 반복적으로 발굴/확장
- **제조사(B2B 포함)**: 리드/견적/도입 문의까지 퍼널을 만들고 성과를 추적

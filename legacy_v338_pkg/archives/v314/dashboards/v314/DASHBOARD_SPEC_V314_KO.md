# GENIE_ROI V314 대시보드 구조(명세)

이 문서는 Looker Studio/Metabase/Grafana/내부 대시보드 어디에나 적용 가능한 **지표·차트 설계도**입니다.

## 1) 공통 원칙
- 날짜 기준: 일/주/월로 집계 가능
- 비교 단위:
  - 광고만 모드: 채널 → 캠페인 → 소재
  - 인플루언서 모드: 인플루언서 → 콘텐츠 → 캠페인(연결된 경우)
- 핵심 KPI: ROAS, CPA(CPL), CTR, CVR, 매출, 광고비, 전환

---

## 2) 광고만 운영 대시보드(권장 위젯)
### A. KPI 카드(상단)
- 총 광고비, 총 매출, 총 전환, ROAS, CPA

### B. 트렌드 차트
- (라인) 일별 광고비 vs 매출
- (라인) 일별 ROAS
- (라인) 일별 CPA

### C. 채널 비교(바/테이블)
- 채널별 광고비/매출/ROAS/CPA
- 채널별 CTR/CVR

### D. 캠페인 TOP 리스트(테이블)
- ROAS 상위/하위 10개 캠페인
- CPA 상위/하위 10개 캠페인

### E. 소재(크리에이티브) 분석(선택)
- 소재별 CTR, CVR, CPA
- A/B 테스트 결과(실험명, 가설, 결과, 다음 액션)

---

## 3) 인플루언서 포함 대시보드(권장 위젯)
### A. 인플루언서별 요약
- 평균 조회, 평균 참여율, 링크 클릭 합, 매출/ROAS(연결 가능 시)

### B. 콘텐츠 퍼포먼스
- 포맷별(릴스/쇼츠/피드) 성과 비교
- 게시물별 상위 콘텐츠(조회/참여율/클릭 기준)

### C. 리스크(가짜 팔로워) 신호 패널
- 성장 스파이크, 댓글 진정성, 오디언스 매치, 참여 일관성

---

## 4) 데이터 모델(권장 최소 테이블)
- `ads_campaign_daily` : 일별 광고 성과(채널/캠페인 단위)
- `conversions_daily` : 일별 전환/매출(UTM/캠페인 단위)
- `influencers`, `influencer_content_daily`, `influencer_campaign` : 인플루언서 모듈

---

## 5) 데이터 품질 체크(필수)
- 날짜 누락, spend=0인데 매출만 있는 케이스 확인
- UTM 규칙/캠페인 ID 규칙이 통일되어야 정확한 연결 가능
- 전환 정의가 바뀌면(이벤트 변경) 기간을 나눠 비교

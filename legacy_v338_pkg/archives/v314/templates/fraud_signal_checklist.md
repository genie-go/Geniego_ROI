# Fraud Signal Checklist (템플릿)

## Growth
- [ ] 최근 30일 팔로워 증가가 비정상적 스파이크인가?
- [ ] 팔로워 증가와 콘텐츠 성과(조회/댓글)가 같이 움직이나?

## Engagement
- [ ] 댓글이 반복 문구/이모지 위주인가?
- [ ] 게시물별 반응 편차가 지나치게 큰가?

## Audience
- [ ] 오디언스 국가/언어가 타겟과 맞는가?
- [ ] 신규 팔로워 계정 품질(게시물 0, 프로필 없음)이 과도한가?

## Conversion
- [ ] 링크/쿠폰 기반 전환이 실제로 발생하는가?
- [ ] 클릭 대비 구매/가입 전환이 지나치게 낮은가?

결과: PASS / REVIEW / FAIL

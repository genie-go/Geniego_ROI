# V316 개요(운영형)

V316은 V315에서 제안했던 요소를 **실제로 운영 가능한 형태**로 확장했습니다.

## 무엇이 운영형인가?
- 팀/대행사가 매일/매주 내려받는 CSV를 ‘폴더에 넣는 방식’으로 표준화
- 자동 적재 → KPI 자동 산출 → JSON 생성 → 로컬 대시보드 확인
- 인플루언서 운영은 외부 입력(오디언스/사기 추정치)을 정규 스키마로 받아
  Risk/Fit을 분리해서 운영 가능

## 모드
- Ads-only(광고만 운영)
- Influencer+Ads(인플루언서 포함)
- 둘은 DB 스키마를 공유하고, 실행 흐름만 선택합니다.

# V313 vs 다른 플랫폼 비교 (개념 비교)

> 이 문서는 “기능 범주” 관점의 비교입니다. 실제 기능/가격/연동 범위는 각 벤더의 요금제와 API 정책에 따라 달라집니다.

## 1) HubSpot / Salesforce 같은 CRM·마케팅 자동화와 비교
- HubSpot/Marketing Cloud: CRM 중심으로 **이메일/워크플로우/고객 여정**에 강점
- GENIE_ROI: **커머스 운영(주문·재고·SKU) + 광고 + 인플루언서**를 “ROI/거버넌스” 관점으로 묶는 것에 초점

## 2) CreatorIQ/Upfluence 같은 인플루언서 플랫폼과 비교
- 전용 인플루언서 플랫폼: **인플루언서 탐색/데이터 품질/사기 탐지/캠페인 관리**에 강점
- GENIE_ROI: 인플루언서 데이터를 **SKU 매출·광고비·재고**와 함께 봐서 “운영 의사결정(확장/중단/재고/딜)”에 연결하는 것에 초점

## 3) 전자상거래 운영툴/BI와 비교
- BI 도구: 데이터 시각화/리포팅은 강하지만, “실행 워크플로우/승인/정책”은 별도 구성 필요
- GENIE_ROI: 데이터→추천→실행(승인 포함)을 한 흐름으로 설계

## 권장 조합
- CRM이 필요하면: HubSpot/Salesforce + GENIE_ROI(운영/ROI)
- 인플루언서 탐색이 핵심이면: CreatorIQ/Upfluence + GENIE_ROI(매출/운영 연결)

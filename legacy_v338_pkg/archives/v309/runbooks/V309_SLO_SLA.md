
# V309 SLO/SLA Template

## Core SLOs
- Commerce job success rate: >= 99.5%
- Ads control command success rate: >= 99.0%
- Webhook ingestion latency p95: <= 2s
- Inventory inconsistency rate: <= 0.1%
- API availability: >= 99.9% (Enterprise tier)

## Error Budgets
- Monthly error budget for availability: 0.1%
- Alert when 50% budget consumed in 15 days

## Key SLIs
- Job completion latency p95
- Queue backlog depth
- Provider error ratio (429/5xx)
- Token refresh failures

# GENIE_ROI V311 Sales Demo SaaS Frontend (React)

이 프로젝트는 V311에서 추가된 **세일즈 데모용 SaaS 프론트**입니다.
- React + Router 기반
- 권한(RBAC) 기반 메뉴 노출
- 테넌트 전환
- 검색(/) 단축키 + 필터/검색 테이블
- 대량 작업(Bulk actions) + 승인함 워크플로 데모
- 가이드 투어(클릭 시나리오)

## 실행(로컬)
```bash
cd demo/saas_frontend_react
npm install
npm run dev
```
브라우저에서 http://localhost:5173

## 데이터
- public/demo/sample_data/*.json (V309 샘플 데이터 복사본)

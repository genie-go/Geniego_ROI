# GENIE_ROI V309 Demo Pack

## 목적
- 실연동 없이 고객이 플랫폼을 '가상 체험'하도록 하는 데모 패키지입니다.
- 샘플 테넌트/상품/주문/광고캠페인/인플루언서 데이터를 로컬 JSON으로 제공합니다.

## 실행 방법
1) demo/ui/index.html 을 브라우저로 열기
2) 각 메뉴(Overview/Commerce/Ads/Influencer/Policies/Analytics)로 이동해 샘플 데이터를 확인

# GENIE ROI V299 Beginner Guide (10-minute start)

This guide is written for first-time users. You can follow it without technical background.

## 1) What this platform is
GENIE ROI is a Growth Operating System:
- Run Journeys/Campaigns across channels (email/SMS/ads/CRM)
- Measure impact (cohort, ordered funnel, segments, experiments)
- Decide (deal recommendations with guardrails and uncertainty)
- Expand (marketplace apps/connectors with scopes + review + billing)

## 2) Login / Access
This demo uses API keys.
1. Open the Dashboard UI
2. Set:
- Tenant ID
- API Key
(Your admin can also enable SSO later.)

## 3) Create your first Journey (2 minutes)
1. Go to **Journey Builder**
2. Click **New Journey**
3. Pick a preset (Onboarding / Cart Abandon)
4. Press **Publish**
5. Trigger an event (use Quickstart scripts)

## 4) Check results (3 minutes)
1. Go to **Analytics**
2. Use **Ordered Funnel** to ensure true step order
3. Use **Cohort** to compare new users week by week

## 5) Experiments (2 minutes)
1. Go to **Experiment Pro**
2. Create an A/B test
3. Turn on **Sequential** and set looks (e.g., 5)
4. Watch the recommendation: CONTINUE / STOP_SUCCESS / STOP_FUTILE

## 6) Marketplace & Connectors (3 minutes)
1. Go to **Marketplace Pro**
2. Open **Publisher Console**
3. Create an App / Connector draft
4. Request scopes → upload evidence → run Auto Security Review
5. Submit → Approve → Publish (admin flow)

## 7) Billing (1 minute)
1. Go to **Billing**
2. Pick a plan and activate subscription
3. Watch usage counters and invoice preview

If you're stuck, open the **Help** tab inside the UI.

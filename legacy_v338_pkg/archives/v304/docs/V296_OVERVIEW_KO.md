# GENIE ROI V296 개요 (Final-mile-of-final-mile)

V296는 '실행 → 검증 → 의사결정 → 확장(마켓플레이스)' 루프를 톱티어급으로 완성하는 릴리스입니다.

## 핵심 업그레이드
- 실험/딜 추천: 정식 샘플사이즈 산정 + 가드레일 자동화 + 베이지안 업데이트(몬테카를로 샘플링 기반 credible interval)
- 앱스토어: Scope 승인 플로우 + 버전 배포(상태 전이) + 리뷰 가중치 랭킹 + 퍼블리셔 콘솔

## 서비스 정의
이 플랫폼은 마케팅/광고/파트너 운영을 하나의 운영 루프로 묶습니다.
- Orchestration: Journey/캠페인 상시 실행
- Measurement: 코호트/퍼널/세그먼트 + 실험(인크리멘털) + 딜 성과
- Decisioning: 신뢰구간/확률 기반 추천(리스크 포함)
- Expansion: 앱스토어/마켓플레이스로 채널/파트너 확장

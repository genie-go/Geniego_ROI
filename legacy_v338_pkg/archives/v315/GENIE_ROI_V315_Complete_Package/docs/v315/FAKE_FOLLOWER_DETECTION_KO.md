# 가짜 팔로워/비정상 지표 구별 방법(V315 요약)

## 대표 레드 플래그(빠른 체크)
- [ ] 팔로워가 짧은 기간에 급증/급감(스파이크)
- [ ] 좋아요/댓글이 팔로워 대비 비정상적으로 낮거나, 반대로 과도하게 높음
- [ ] 댓글이 'nice', 'cool' 같은 반복 패턴/이모지 반복 위주
- [ ] 조회수는 높은데 클릭/전환이 거의 없음(풀퍼널 불일치)
- [ ] 팔로워 지역이 타겟과 크게 불일치(가능한 경우 확인)

## 현업에서 가장 강력한 검증(추천)
1) **링크(UTM) 클릭**과 **실제 전환/매출**로 검증  
2) 동일 비용으로 여러 인플루언서를 소규모 테스트(파일럿) 후 확대  
3) 상위/하위 성과의 패턴을 “지표 + 콘텐츠 품질”로 함께 확인

> V315의 점수 엔진은 외부 입력(오디언스 분포, 사기 추정치 등)이 들어오면
> risk 점수에 반영하도록 확장하는 형태입니다.

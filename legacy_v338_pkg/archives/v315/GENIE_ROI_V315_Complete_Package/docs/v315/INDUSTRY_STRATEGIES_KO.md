# 산업별 인플루언서/광고 전략(V315 요약)

## 1) 뷰티/패션(B2C)
- 인플루언서: 제품 사용감/전후 비교/루틴 콘텐츠가 강점
- 광고: UGC(사용자 콘텐츠) 기반 리타겟팅/카탈로그 광고와 궁합
- KPI: 뷰→클릭→구매 전환까지 풀퍼널, 재구매/반복구매도 중요

## 2) 식품/리빙
- 인플루언서: 레시피/사용 장면/가족 단위 리뷰
- 광고: 프로모션 시즌(명절/행사) 예산 집중, 번들/세트 구성 테스트
- KPI: 객단가(AOV)와 재구매(반복 구매) 추적

## 3) IT/가전
- 인플루언서: 기능 비교/리뷰/장단점 정리(유튜브 장문 리뷰 강세)
- 광고: 검색/리뷰 콘텐츠 리타겟팅, 상세페이지 전환 최적화
- KPI: 리드(상담) vs 직접구매를 분리해 ROAS/CPA를 따로 본다

## 4) 앱/구독
- 인플루언서: 사용법/튜토리얼/전후 변화(생산성, 공부, 운동)
- 광고: 앱 설치(Install)와 결제(Purchase) 이벤트를 분리해 최적화
- KPI: CAC, LTV, 결제 전환율, 유지율

## 5) 제조/B2B
- 인플루언서: 업계 전문가(테크/엔지니어) 또는 전시회/현장 콘텐츠
- 광고: 리드(문의/견적) 중심 퍼널 + 리마케팅 + 세일즈 파이프라인 연결
- KPI: MQL/SQL, 리드당 비용(CPL), 파이프라인 기여 매출

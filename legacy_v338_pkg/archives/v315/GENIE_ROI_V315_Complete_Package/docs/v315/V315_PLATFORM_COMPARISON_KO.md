# V315 vs 다른 플랫폼 비교(요약)

## 1) HubSpot/CRM 기반 마케팅 자동화
- 강점: CRM 데이터 기반 워크플로우/이메일 자동화/리드 스코어링 등 “고객 여정 자동화”에 초점
- V315 관점: CRM 메시징 자동화는 강하지 않을 수 있으나,
  **광고비·전환·매출을 한 기준으로 통합**하고 “전사 ROI 운영 루프”를 만드는 데 초점

## 2) Salesforce Marketing Cloud (Journey Builder)
- 강점: Journey Builder로 이벤트 기반 캠페인/여정 자동화
- V315 관점: 여정 자동화 제품군과 병행할 수 있고,
  V315는 **퍼포먼스 KPI 통합/리포팅 레이어**로 붙이기 좋음

## 3) 광고 플랫폼 자체(구글/메타 등)
- 강점: 채널 내부 최적화(자동입찰/타겟팅/소재 테스트)에 매우 강함
- 한계: 여러 채널을 동시에 운영할 때 “한 화면에서 ROAS/CPA 비교”가 어렵고,
  전환 정의/귀속 규칙이 채널마다 달라 보고가 흔들릴 수 있음
- V315 관점: 채널별 CSV를 통합 스키마로 변환해 **동일 KPI로 비교**하도록 설계

## 4) 인플루언서 전용 플랫폼(Upfluence 등)
- 강점: 인플루언서 발굴/캠페인/수익 귀속/가짜 팔로워 추정 등 특화
- V315 관점: 인플루언서를 쓰든 안 쓰든 “광고·전환·매출의 공통 ROI 레이어”를 유지하면서
  인플루언서 성과를 같은 프레임으로 비교하도록 설계

## 추천 조합(현실적인 운영)
- 광고 최적화는 각 채널에서 수행(구글/메타/틱톡)
- 고객 여정/CRM 자동화는 HubSpot/Salesforce 등에서 수행
- 전사 ROI 통합·리포팅과 인플루언서 성과 합산은 V315 레이어로 통합

# V315 광고만 운영(인플루언서 미사용) 모드 상세

## 언제 필요한가요?
- 인플루언서 없이 **퍼포먼스 마케팅(검색/디스플레이/리타겟팅)만** 운영하는 경우
- 채널이 여러 개라서 “한 기준으로 ROAS/CPA를 비교”해야 하는 경우
- 대행사/유통사/제조사가 내부 보고를 위해 **통합 리포트**가 필요한 경우

## 무엇을 수집하나요?
### 광고 성과(채널 CSV)
- 노출(Impressions), 클릭(Clicks), 비용(Spend)
- 캠페인/광고그룹/소재 ID(가능한 범위)

### 전환/매출(웹/앱/오프라인)
- 전환수(Conversions), 매출(Revenue)
- 캠페인ID 또는 UTM(utm_campaign)로 귀속

## 무엇을 분석/제공하나요?
- CTR = 클릭/노출
- CVR = 전환/클릭
- ROAS = 매출/광고비
- CPA = 광고비/전환
- 채널/캠페인 TOP & Bottom, 추세(일별/주별)

## 초보자 운영 체크리스트
1) “전환 정의”를 먼저 고정합니다(구매, 리드, 회원가입 등).  
2) 광고 CSV와 전환/매출 CSV의 연결 키를 하나로 통일합니다(캠페인ID 또는 UTM).  
3) ROAS/CPA 기준선을 정하고, 캠페인 중단/확대 규칙을 문서화합니다.  
4) 채널 내부 최적화(예: 자동입찰)는 하되, “전사 비교”는 V315 통합 리포트로 수행합니다.

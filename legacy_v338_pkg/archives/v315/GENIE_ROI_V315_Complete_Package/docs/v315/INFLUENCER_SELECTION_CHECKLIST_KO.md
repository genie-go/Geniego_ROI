# 인플루언서 선정 체크리스트(V315 요약)

## A. 브랜드/타겟 적합도(Fit)
- [ ] 카테고리/제품군이 브랜드와 일치한다
- [ ] 팔로워의 주요 국가/언어가 타겟과 일치한다(가능한 범위에서 확인)
- [ ] 기존 콘텐츠 톤&매너가 브랜드와 충돌하지 않는다(브랜드 세이프티)

## B. 오디언스 품질(Quality)
- [ ] 참여율(ER)이 업계 평균 대비 과도하게 낮지 않다
- [ ] 최근 30~90일 팔로워 증감이 자연스럽다(급증/급감 스파이크 없음)
- [ ] 댓글이 반복/봇 패턴이 아니다(짧은 이모지 반복 과다 등)

## C. 성과 가능성(Performance)
- [ ] 최근 10개 콘텐츠의 조회수/저장/공유가 꾸준하다
- [ ] 링크 클릭/전환 데이터를 제공할 수 있다(UTM/쿠폰/제휴링크)
- [ ] 협업 콘텐츠의 광고 표기 준수 및 가이드 이행이 가능하다

## D. 운영/계약/컴플라이언스
- [ ] 계약 조건(게시물 수, 일정, 리소스)이 명확하다
- [ ] 2차 활용(광고 소재) 허용 범위가 명확하다
- [ ] 광고 표기(예: #광고 등) 준수 계획이 있다

> V315에서는 위 항목을 `weights.json` 가중치로 점수화할 수 있도록 설계되어 있습니다.

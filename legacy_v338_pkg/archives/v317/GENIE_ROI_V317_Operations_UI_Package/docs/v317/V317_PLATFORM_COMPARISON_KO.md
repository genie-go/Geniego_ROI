# 다른 플랫폼과 비교(실무 관점) - V317

## 1) 광고 채널 관리자(구글/메타/틱톡/네이버 등)
- 강점: 채널 내부 최적화(입찰/타겟/소재)와 실시간 지표
- 한계: 채널이 늘어날수록 통합 리포팅/공통 KPI 비교가 번거로움
- V317: CSV 업로드 기반으로 여러 채널 데이터를 공통 스키마로 정규화 → ROAS/CPA로 통합 비교

## 2) CRM/마케팅 자동화(예: HubSpot 계열)
- 강점: 고객 여정/리드 육성/이메일 자동화
- V317 포지션: CRM 대체가 아니라 “광고비-전환-매출” ROI 레이어로 보완 (같이 쓰기 좋음)

## 3) 인플루언서 전용 플랫폼
- 강점: 크리에이터 탐색/캠페인 운영/성과 분석
- V317: 인플루언서를 쓰든 안 쓰든 동일한 ROI 운영 루프(통합 KPI)에 붙일 수 있고,
  향후 외부 입력(Risk/Fit) 스키마와 함께 운영 체계를 구성 가능

## 결론
V317은 ‘대형 SaaS’가 아니라, **CSV 기반 운영 현실**에 맞춘 경량 운영형 시스템으로,
- “업로드 한 번”
- “귀속 룰 선택”
- “통합 KPI 대시보드 확인”
까지의 시간을 크게 줄이는 것에 초점이 있습니다.

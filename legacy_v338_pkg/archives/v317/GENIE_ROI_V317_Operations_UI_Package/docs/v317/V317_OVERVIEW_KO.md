# V317 개요

V317은 “CSV 기반 현실 운영”을 유지하면서, **업로드/귀속/통화/세금**을 운영자(초보자) 관점에서 더 쉽게 만들었습니다.

## 운영 흐름
1) 웹 UI에서 광고 CSV 업로드(채널 선택)
2) 웹 UI에서 전환 CSV 업로드
3) 귀속 룰 선택(캠페인ID / UTM / 소재ID)
4) 자동으로 KPI JSON 생성 → 대시보드 갱신

## 무엇이 달라졌나
- (V316) 폴더 드롭 → (V317) 업로드 버튼 기반
- (V316) 고정 조인(캠페인ID) → (V317) 선택형 귀속 룰
- (V316) 비용 숫자만 → (V317) 통화/세금 고려한 “기준 통화 비용” 산출

# V319 매핑 UI: 자동 추천 + 템플릿 내보내기/가져오기

## 1) 자동 추천(유사도)
- CSV 헤더 목록과 표준 필드명/키워드를 비교해 가장 가능성이 높은 헤더를 자동 선택합니다.
- 운영자는 추천 결과를 확인하고 필요하면 수정합니다.

## 2) 템플릿 내보내기/가져오기
- 대행사/브랜드/팀 간에 같은 형식의 CSV를 반복 사용합니다.
- 매핑을 한 번 만든 뒤, JSON 템플릿으로 내보내고 다른 환경에서 가져올 수 있습니다.

### 내보내기
- UI에서 [템플릿 내보내기] → 현재 채널 또는 전체 채널 매핑을 다운로드

### 가져오기
- UI에서 [템플릿 가져오기] → JSON 업로드
- 서버가 DB(custom_mappers)에 업서트하고, templates/v319/custom_mappers.json도 동기화합니다.

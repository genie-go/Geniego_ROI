YouTube API Key
申請手冊

這是一份即使是新手也能照著操作的實戰指南，整理了在 Google Cloud Console 中啟用 YouTube Data API v3 並建立 API Key 的完整流程。

預計耗時約 5–10 分鐘
所需帳號Google 帳號
使用的 APIYouTube Data API v3
成果AIzaSy... 格式的 API Key

1. API Key 與 OAuth 的差異

這是剛開始時最容易讓人混淆的部分。

API Key

用於查詢公開資料

用於取得公開的 YouTube 資料，例如搜尋影片、查詢頻道資訊、取得播放清單以及讀取留言等。

OAuth 2.0

用於需要你帳號權限的作業

用於需要使用者授權的作業，例如上傳影片、管理自己的頻道以及存取私人資料等。

重要: 若要自動上傳影片，僅靠 API Key 是不夠的，還需要 OAuth 2.0 認證。

2. 啟用 YouTube Data API v3

在建立 API Key 之前，必須先在專案中開啟你要使用的 API。

1

存取 Google Cloud Console

登入 Google Cloud Console。

console.cloud.google.com
2

選擇或建立專案

在頂部的專案選擇區域中選擇現有專案，或建立一個新專案。

例如：Geniego ROI
3

進入 API 程式庫

在左側選單中進入 API 與服務 → 程式庫。

API 與服務 → 程式庫
4

搜尋 YouTube Data API v3 並啟用

在搜尋框中輸入 YouTube Data API v3，選擇該 API，然後點擊啟用按鈕。

YouTube Data API v3 → 啟用
如果畫面上顯示 狀態：已啟用，表示已正常啟用。

3. 建立 API Key

啟用 YouTube Data API v3 後，即可在憑證畫面建立 API Key。

1

進入憑證選單

在左側選單中進入 API 與服務 → 憑證。

API 與服務 → 憑證
2

點擊建立憑證

點擊頂部的 + 建立憑證按鈕。

+ 建立憑證
3

選擇 API 金鑰

在彈出的選單中選擇 API 金鑰。

API 金鑰
4

選擇可存取的 API

在 API 限制中確認已選擇 YouTube Data API v3。

已選擇的 API：YouTube Data API v3
5

點擊建立按鈕

點擊底部的建立按鈕即可產生 API Key。

建立
產生的金鑰通常以 AIzaSy... 開頭。這串字元就是你的 YouTube API Key。

4. 在建立畫面應該選擇哪些選項？

項目新手推薦設定說明
名稱YouTube API Key輸入一個日後便於辨識的名稱。
API 限制YouTube Data API v3限制此金鑰僅用於 YouTube API。
服務帳戶認證不勾選建立 YouTube Data API Key 通常不需要。
應用程式限制首次測試設為無確認正常運作後，建議設定網站或 IP 位址限制。
部署到正式服務時，請務必設定金鑰限制。如果在沒有限制的情況下公開，他人可能會使用你的金鑰。

5. API Key 使用範例

將產生的金鑰填入 YOUR_API_KEY 部分即可進行測試。

影片搜尋範例https://www.googleapis.com/youtube/v3/search?part=snippet&q=korea&type=video&key=YOUR_API_KEY
頻道資訊查詢範例https://www.googleapis.com/youtube/v3/channels?part=snippet&id=CHANNEL_ID&key=YOUR_API_KEY
在瀏覽器網址列輸入上述網址後，如果出現 JSON 格式的結果，表示 API Key 運作正常。

6. 常見問題排解

問題原因解決方法
沒有選擇任何 APIYouTube Data API v3 尚未啟用，或在 API 限制中未選擇在 API 程式庫中啟用 YouTube Data API v3，然後重新選擇。
沒有可顯示的 API 金鑰尚未建立 API Key點擊建立憑證 → API 金鑰。
403 錯誤API 限制、配額超出或權限問題檢查 API 限制、配額和啟用狀態。
無法上傳僅使用了 API Key影片上傳必須使用 OAuth 2.0 認證。

7. 最終檢查清單

  • 已選擇 Google Cloud 專案。
  • 已啟用 YouTube Data API v3。
  • 已在憑證中建立 API Key。
  • 已確認產生的 API Key 以 AIzaSy 開頭。
  • 已將 API 限制設定為 YouTube Data API v3。
  • 在部署到正式服務前設定網站或 IP 限制。

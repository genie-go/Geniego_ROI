Shopify
API Key 申請手冊

本手冊介紹在 Shopify 管理後台（Admin）中建立自訂應用程式以申請 Admin API 存取權杖，並將其與 Shop 網域一併註冊到 GenieGo 整合中心，從而連接 Shopify 商店的整個流程。

預計耗時約 5~15 分鐘
所需帳戶Shopify 商店管理員權限
申請金鑰Shop 網域 · Admin API 存取權杖
成果Admin API 憑證

1. 開始之前

只有擁有商店管理員權限才能建立 Shopify 自訂應用程式。

必要

商店管理員權限

您需要擁有 Shopify 商店的管理員（Admin）權限，才能建立自訂應用程式並申請權杖。

申請金鑰

Shop 網域 · Admin API 權杖

您將取得 xxx.myshopify.com 格式的 Shop 網域，以及安裝應用程式後顯示的 Admin API 存取權杖。

注意： Admin API 存取權杖僅在安裝後立即顯示一次，因此顯示時必須立即複製。

2. 申請步驟

請依以下順序申請並註冊到 GenieGo。

1

前往應用程式與銷售管道

在 Shopify 管理後台（Admin）中，前往 [設定] > [應用程式與銷售管道]。

https://www.shopify.com/admin/settings/apps
2

建立自訂應用程式

點擊 [開發應用程式] > [建立應用程式] 以建立自訂應用程式。

開發應用程式 → 建立應用程式
3

選擇範圍 · 安裝應用程式

選擇 Admin API 範圍（scope）並安裝應用程式。

4

複製存取權杖

立即複製安裝後顯示的 Admin API 存取權杖（僅顯示一次）。

安裝應用程式後顯示 Admin API 存取權杖即表示申請成功。由於僅顯示一次，請立即複製。

3. 需在 GenieGo 註冊的資訊

欄位說明
shop_domainxxx.myshopify.com 格式的 Shop 網域
access_token已申請的 Admin API 存取權杖（請妥善保管）
在 GenieGo [註冊] 中輸入您的 shop_domain（xxx.myshopify.com）和 access_token 並儲存。憑證將以 AES-256-GCM 加密儲存，並在讀取時進行遮罩處理。

4. 常見問題

問題原因解決方法
無法再次檢視權杖Admin API 存取權杖僅顯示一次如果遺失了權杖，請在自訂應用程式中重新申請權杖並立即複製。
權限不足錯誤未選擇 Admin API 範圍（scope）選擇所需的 Admin API 範圍（scope），然後重新安裝應用程式。
網域錯誤Shop 網域格式填寫錯誤請準確輸入 xxx.myshopify.com 格式的 Shop 網域。

最終檢查清單

  • 已使用 Shopify 商店管理員權限登入。
  • 已在 設定 > 應用程式與銷售管道 > 開發應用程式 中建立自訂應用程式。
  • 已選擇 Admin API 範圍（scope）並安裝應用程式。
  • 已立即複製安裝後顯示的 Admin API 存取權杖。
  • 已在 GenieGo [註冊] 中輸入 shop_domain·access_token 並儲存。

TikTok Shop
API Key 申請手冊

本手冊手把手指導新手在 TikTok Shop Partner Center 中建立應用程式(App Key·App Secret·Service ID),透過商店授權簽發 access_token,並註冊到 GenieGo 整合中心。

預計時間約 10~15 分鐘
所需帳戶TikTok Shop 賣家帳戶
簽發金鑰App Key · App Secret · Access Token
成果API 憑證

1. 開始之前

只有在 Partner Center 中建立應用程式並授權商店後才會簽發權杖。

必需

TikTok Shop 賣家帳戶

您需要一個 TikTok Shop 賣家帳戶,並在 partner.tiktokshop.com(Partner Center)註冊為開發者。

簽發金鑰

App Key · App Secret · Access Token

建立應用程式後可取得 App Key·App Secret·Service ID,並透過商店授權取得 access_token。

注意: 切換至正式環境時需要進行應用程式審查。

2. 簽發步驟

依以下順序簽發並在 GenieGo 中註冊。

1

在 Partner Center 註冊為開發者

在 partner.tiktokshop.com(Partner Center)註冊為開發者。

https://partner.tiktokshop.com
2

建立應用程式

透過 [App & Service] > [Create App] 建立應用程式,並查看 App Key、App Secret、Service ID。

App & Service → Create App
3

啟用 API 權限

在應用程式設定中啟用所需的 API 權限(Shop Authorized Information、Product、Order)。

4

商店授權 → 簽發 access_token

在 [Get shop authorization] 中選擇地區(Region)以授權賣家商店,即可簽發 access_token。

Get shop authorization
5

確認金鑰

確認 App Key·App Secret 和簽發的 access_token。如有需要,一併確認 Shop Cipher。

商店授權後簽發 access_token,即表示已正常處理。

3. 需在 GenieGo 中註冊的資訊

欄位說明
app_key簽發的 App Key
app_secret簽發的 App Secret(請妥善保管)
access_token透過商店授權簽發的存取權杖
shop_cipherShop Cipher(選填)
在 GenieGo 整合中心的 [註冊] 中輸入 app_key·app_secret·access_token 並儲存後,憑證將以 AES-256-GCM 加密儲存,並在查詢時進行遮罩處理。

4. 常見問題

問題原因解決方法
未簽發 access_token商店授權(Get shop authorization)未完成,或未選擇地區(Region)在 [Get shop authorization] 中選擇地區以授權賣家商店。
權限錯誤未啟用所需的 API 權限在應用程式設定中啟用 Shop Authorized Information·Product·Order 權限。
正式呼叫遭拒未通過正式切換審查完成切換至正式環境所需的應用程式審查。

最終檢查清單

  • 已在 partner.tiktokshop.com 註冊為開發者。
  • 已透過 App & Service > Create App 查看 App Key·App Secret·Service ID。
  • 已啟用所需的 API 權限(Shop/Product/Order)。
  • 已透過商店授權(Get shop authorization)簽發 access_token。
  • 已在 GenieGo 的 [註冊] 中輸入 app_key·app_secret·access_token 並儲存。

Smart Tracker (Unified Tracking)
API Key 申請手冊

本手冊手把手指導新手在 SweetTracker(tracking.sweettracker.co.kr)註冊，簽發全貨運公司統一追蹤金鑰(t_key),並註冊到 GenieGo 整合中心。一個 t_key 即可接入所有貨運公司的物流追蹤。

預計時間約 5~15 分鐘
所需帳戶SweetTracker 註冊帳戶
簽發金鑰t_key (全貨運公司統一追蹤金鑰)
成果統一物流追蹤 API 憑證

1. 開始之前

智慧貨運(統一追蹤)是一個無需另外簽約、僅註冊即可使用的自助簽發渠道。

必需

SweetTracker 註冊帳戶

您只需在 tracking.sweettracker.co.kr 註冊一個帳戶即可。無需另外簽約，僅註冊即可簽發。

簽發金鑰

t_key (統一追蹤金鑰)

簽發一個全貨運公司統一追蹤金鑰 t_key，即可接入所有貨運公司的物流追蹤。

注意: 在 GenieGo 註冊時，將 t_key 填入 api_key 欄位。

2. t_key 申請步驟

依以下順序簽發並在 GenieGo 中註冊。

1

註冊並登入 SweetTracker

前往 tracking.sweettracker.co.kr，註冊並登入。

https://tracking.sweettracker.co.kr
2

申請簽發 t_key

申請簽發 t_key(全貨運公司統一追蹤金鑰)。

3

確認 t_key

確認已簽發的 t_key。

4

在 GenieGo 中註冊

在 GenieGo [註冊] 中填入 t_key(api_key),即可接入所有貨運公司的物流追蹤。

t_key 簽發後即表示已正確簽發，註冊後立即接入所有貨運公司的物流追蹤。

3. 需在 GenieGo 中註冊的資訊

欄位說明
api_key已簽發的 t_key (全貨運公司統一追蹤金鑰)
在 GenieGo 整合中心 [註冊] 中填入並儲存後，憑證將以 AES-256-GCM 加密儲存，並在查詢時進行遮罩處理。註冊後立即接入所有貨運公司的物流追蹤。

4. 常見問題

問題原因解決方法
找不到 t_key 簽發選單未註冊或未登入 SweetTracker在 tracking.sweettracker.co.kr 註冊並登入後，再申請簽發 t_key。
物流追蹤未接入t_key 填寫錯誤或未填寫請重新確認是否已將簽發的 t_key 準確填入 api_key 欄位。

最終檢查清單

  • 已在 tracking.sweettracker.co.kr 註冊並登入。
  • 已申請簽發 t_key(全貨運公司統一追蹤金鑰)。
  • 已確認簽發的 t_key。
  • 已在 GenieGo [註冊] 中將 t_key 填入 api_key 欄位並儲存。

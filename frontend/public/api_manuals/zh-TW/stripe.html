Stripe
API 金鑰簽發手冊

一份適合初學者的逐步指南，協助您從 Stripe 儀表板的 Developers 選單簽發 API 金鑰（Publishable Key·Secret Key），並將其註冊到 GenieGo 整合中心。

預估時間約 5–10 分鐘
所需帳戶Stripe 帳戶
簽發金鑰Publishable Key · Secret Key
結果API 憑證

1. 開始之前

Stripe API 金鑰是從您的 Stripe 帳戶儀表板簽發的。

必要

Stripe 帳戶

您必須能夠使用 Stripe 帳戶登入 dashboard.stripe.com。

簽發金鑰

Publishable Key · Secret Key

在 Developers > API keys 畫面查看您的 Publishable key 與 Secret key。

注意：進行實際付款整合時，請使用正式金鑰，而非測試金鑰。

2. 簽發步驟

請依下列順序簽發並在 GenieGo 中註冊。

1

登入 Stripe

使用您準備好的 Stripe 帳戶登入儀表板。

dashboard.stripe.com
2

前往 API keys 選單

前往 [Developers] > [API keys] 選單。

Developers → API keys
3

查看金鑰

查看您的 Publishable key 與 Secret key。請使用正式金鑰。

https://dashboard.stripe.com/apikeys
當 Publishable key 與 Secret key 皆已確認，即表示已正確簽發。

3. 需在 GenieGo 中註冊的資訊

欄位說明
publishable_key已簽發的 Publishable Key
secret_key已簽發的 Secret Key（請妥善保管）
在 GenieGo 整合中心的[註冊]中輸入 publishable_key·secret_key 並儲存。憑證會以 AES-256-GCM 靜態加密儲存，並於讀取時進行遮罩處理。

4. 常見問題

問題原因解決方法
驗證失敗輸入了測試金鑰請確認您是否正在使用正式金鑰（Publishable·Secret）。
看不到 Secret key該金鑰僅顯示一次或處於隱藏狀態請在 Developers > API keys 畫面再次查看，或重新簽發。

最終檢查清單

  • 已使用 Stripe 帳戶登入 dashboard.stripe.com。
  • 已前往 Developers > API keys。
  • 已確認正式的 Publishable Key·Secret Key。
  • 已在 GenieGo [註冊] 中輸入 publishable_key·secret_key 並儲存。

Kakao Moment
API 金鑰申請手冊

本手冊指導您申請 Kakao Moment 開放API權限，取得存取權杖與廣告帳戶ID，並在 GenieGo 整合中心進行註冊。開放API權限僅授予 Kakao 官方廣告代理商。

1. 開始之前

Kakao Moment 開放API 並非自助申請，而須經過權限申請與審核流程。

重要: Kakao Moment 開放API權限僅授予 Kakao 官方廣告代理商。一般廣告主無法自助申請。
必需

廣告帳戶 + 商業頻道

您需要一個 Kakao Moment 廣告帳戶和一個 Kakao 商業頻道。

申請的金鑰

存取權杖 · 廣告帳戶ID

權限核准後，透過 OAuth 授權取得存取權杖並確認您的廣告帳戶ID。

2. 申請流程

依以下順序進行：開放API權限申請 → 審核/核准 → 權杖申請 → 註冊。

1

建立應用程式並切換為商業應用程式

在 developers.kakao.com 建立應用程式，並將其切換為[商業應用程式]。

developers.kakao.com
2

申請開放API權限

提交 Kakao Moment 開放API[附加功能申請]。在審核您的官方代理商資格後授予權限。

附加功能申請
3

申請權杖

權限核准後，透過 OAuth 授權取得存取權杖並確認您的廣告帳戶ID。您可以使用 GenieGo 整合中心的一鍵 OAuth。

權限核准且存取權杖與廣告帳戶ID確認後，您即可進行註冊。

3. 需在 GenieGo 註冊的資訊

欄位說明
access_token透過 OAuth 授權申請的存取權杖(請謹慎處理)
ad_account_idKakao Moment 廣告帳戶ID
在 GenieGo 整合中心[註冊]中輸入並儲存，憑證將以 AES-256-GCM 靜態加密儲存，並在讀取時進行遮罩處理。

4. 常見問題

問題原因解決方法
無法取得開放API權限並非官方廣告代理商開放API權限僅授予 Kakao 官方廣告代理商。請透過代理商申請。
無法進行[附加功能申請]應用程式尚未切換為商業應用程式請先將應用程式切換為[商業應用程式]，然後再申請附加功能。

最終檢查清單

  • 已準備好 Kakao Moment 廣告帳戶和 Kakao 商業頻道。
  • 已將應用程式切換為[商業應用程式]並提交了開放API[附加功能申請]。
  • 權限核准後，已透過 OAuth 授權確認存取權杖與廣告帳戶ID。
  • 已在 GenieGo[註冊]中輸入 access_token·ad_account_id 並儲存。

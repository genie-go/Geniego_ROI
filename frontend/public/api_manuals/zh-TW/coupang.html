Coupang Wing
API Key 發放手冊

本手冊詳細介紹如何在 Coupang WING 賣家控制台發放 Open API 認證金鑰（Access Key·Secret Key）並將其註冊到 GenieGo 整合中心，初學者也能照著操作完成。

預計耗時約5〜10分鐘
所需帳戶Coupang WING 賣家帳戶
發放的金鑰Access Key · Secret Key
成果Open API 認證憑證

1. 開始之前

Coupang Open API 只能從賣家帳戶發放。

必需

WING 賣家帳戶

您需要一個已完成企業認證的 Coupang WING 賣家帳戶。一般買家會員帳戶無法發放。

發放的金鑰

Access Key · Secret Key

在發放完成頁面查看 Access Key 和 Secret Key，並一併確認賣家（vendor）ID。

提示：發放時，您必須填寫公司名稱、首頁／服務URL（如無則填 wing.coupang.com）以及存取IP（最多10個）。

2. Open API Key 發放步驟

按以下順序發放並註冊到 GenieGo。

1

登入 WING

使用賣家帳戶登入。

wing.coupang.com
2

進入 Open API 選單

進入頂部／設定區域的 [Open API] 或 [API 認證金鑰管理] 選單。

Open API → API 認證金鑰管理
3

發放 Open API Key

點擊 [發放 Open API Key]，並填寫公司名稱、首頁URL以及存取IP（最多10個）。

發放 Open API Key
4

確認金鑰

在發放完成頁面查看 Access Key·Secret Key 和賣家（vendor）ID。

當發放完成頁面顯示 Access Key·Secret Key 時，即表示已正常發放。

3. 需要註冊到 GenieGo 的資訊

欄位說明
access_key已發放的 Access Key
secret_key已發放的 Secret Key（請妥善保管）
vendor_id賣家（vendor）ID
在 GenieGo 整合中心的 [註冊] 中填寫並儲存。憑證將以 AES-256-GCM 加密儲存，讀取時進行遮罩處理。註冊後立即開始訂單同步。

4. 常見問題

問題原因解決
沒有發放選單一般會員帳戶，或企業認證未完成請使用已完成企業認證的 WING 賣家帳戶登入。
呼叫時出現401錯誤存取IP未註冊，或 Secret Key 填寫錯誤請重新確認發放時填寫的存取IP和 Secret Key。

最終檢查清單

  • 已使用完成企業認證的 WING 賣家帳戶登入。
  • 已發放 Open API Key 並註冊了存取IP。
  • 已確認 Access Key·Secret Key·Vendor ID。
  • 已在 GenieGo [註冊] 中填寫 access_key·secret_key·vendor_id 並儲存。

Qoo10
API Key 申請手冊

本手冊手把手指導新手在 Qoo10 QSM(賣家控制台)中簽發 API 認證金鑰，並註冊到 GenieGo 整合中心。

預計時間約 5~10 分鐘
所需帳戶Qoo10 賣家(QSM)帳戶
簽發金鑰QSM API 金鑰
成果API 憑證

1. 開始之前

Qoo10 API 金鑰只能從賣家(QSM)帳戶中簽發。

必需

Qoo10 賣家(QSM)帳戶

您需要一個 Qoo10 賣家(QSM)帳戶。一般購買會員帳戶無法簽發 API 金鑰。

簽發金鑰

QSM API 金鑰

在 QSM 的 API 管理選單中簽發 API 認證金鑰，並同時確認您本人的賣家 ID。

2. 簽發步驟

依以下順序簽發並在 GenieGo 中註冊。

1

登入 QSM

使用 Qoo10 賣家(QSM)帳戶登入。

https://qsm.qoo10.jp
2

進入 API 管理選單

在 QSM 中進入 API 管理選單。

API 管理
3

簽發 API 認證金鑰

在 API 管理選單中簽發 API 認證金鑰。

4

確認金鑰

確認簽發的 QSM API 金鑰和您本人的賣家 ID。

當 API 認證金鑰顯示在 API 管理畫面上時，即表示已正確簽發。

3. 需在 GenieGo 中註冊的資訊

欄位說明
api_key簽發的 QSM API 金鑰(請妥善保管)
seller_id您本人的賣家 ID
在 GenieGo 整合中心的 [註冊] 中輸入 api_key·seller_id 並儲存後，憑證將以 AES-256-GCM 加密儲存，並在查詢時進行遮罩處理。

4. 常見問題

問題原因解決方法
沒有 API 管理選單不是賣家(QSM)帳戶，或權限不足使用 Qoo10 賣家(QSM)帳戶登入並查看 API 管理選單。
認證失敗API 金鑰或賣家 ID 填寫錯誤重新確認 QSM 中簽發的 API 金鑰和賣家 ID 並輸入。

最終檢查清單

  • 已使用 Qoo10 賣家(QSM)帳戶登入。
  • 已在 API 管理選單中簽發 API 認證金鑰。
  • 已確認本人的賣家 ID。
  • 已在 GenieGo [註冊] 中輸入 api_key·seller_id 並儲存。

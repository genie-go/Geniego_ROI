Auction (옥션)
API 簽發手冊

一份任何人都能照做的逐步指南，協助您簽發 Auction (옥션) 憑證（API 金鑰/權杖）並在 GenieGo 整合中心註冊。

預估時間約 5–15 分鐘
所需帳戶Auction (옥션) 帳戶
步驟數6 步
成果API 憑證

簽發步驟

依下列步驟操作即可簽發並註冊憑證。

1

準備事項：您需要 Auction 與 G마켓（Gmarket）各自的賣家帳號，以及 ESM Plus 主帳號 ID。

2

在 Auction 與 Gmarket 分別註冊為賣家後，登入 ESM Plus（esmplus.com）建立主帳號 ID。

3

前往 etapi.gmarket.com（ESM Trading API）申請 API 使用。（採核准制 — 可能被拒絕。）

4

申請時須提交 Auction 賣家 ID、Gmarket 賣家 ID、ESM 主帳號 ID 這 3 項。

5

核准後即會發行 API 金鑰。（洽詢：etapihelp@gmail.com）

6

在 GenieGo〔註冊〕中輸入所發行的 api_key 與 seller_id 並儲存。

Shopee
API Key 申請手冊

本手冊手把手指導新手在 Shopee Open Platform 完成開發者註冊與審查後建立應用程式以簽發 Partner ID 和 Partner Key，經商店授權後註冊到 GenieGo 整合中心的整個流程。

預計時間約 10~20 分鐘(審查等待另計)
所需帳戶Shopee 賣家帳戶
簽發金鑰Partner ID · Partner Key · Shop ID
成果API 憑證

1. 開始之前

Shopee Open Platform 需要開發者註冊與審查，以及商店授權步驟。

必需

Shopee 賣家帳戶

若要進行簽發，您需要一個 Shopee 賣家帳戶。使用該帳戶在 Shopee Open Platform 上註冊為開發者。

簽發金鑰

Partner ID · Partner Key

建立應用程式後會簽發 Partner ID 和 Partner Key。隨後經商店授權並取得 access_token。

注意: 在 Shopee Open Platform 完成開發者註冊後需要等待審查，且簽發的 access_token 每 4 小時更新一次。

2. 簽發步驟

依以下順序簽發並在 GenieGo 中註冊。

1

開發者註冊與審查

在 Shopee Open Platform(open.shopee.com)上註冊為開發者並等待審查。

https://open.shopee.com
2

建立應用程式

在 [App Management] > [App List] 中建立應用程式，即可簽發 Partner ID 和 Partner Key。

App Management → App List
3

商店授權

產生商店授權(authorization)連結並授權賣家商店，隨後會簽發 code。

4

簽發 access_token

使用 code 取得 access_token。access_token 每 4 小時更新一次。

當簽發了 Partner ID 和 Partner Key，並透過商店授權取得 access_token 時，即表示已正確簽發。

3. 需在 GenieGo 中註冊的資訊

欄位說明
partner_id簽發的 Partner ID
partner_key簽發的 Partner Key(請妥善保管)
shop_id已授權賣家商店的 Shop ID
在 GenieGo 整合中心 [註冊] 中輸入 partner_id、partner_key、shop_id 並儲存後，憑證將以 AES-256-GCM 加密儲存，並在查詢時進行遮罩處理。

4. 常見問題

問題原因解決方法
無法建立應用程式開發者註冊尚未通過審查在 Shopee Open Platform 註冊為開發者，等待審查通過後再建立應用程式。
認證/呼叫失敗商店授權未完成，或 access_token 已過期透過商店授權連結授權商店，並每 4 小時更新一次 access_token。

最終檢查清單

  • 已使用 Shopee 賣家帳戶完成 Open Platform 開發者註冊與審查。
  • 已在 App Management > App List 中建立應用程式並簽發 Partner ID 和 Partner Key。
  • 已透過商店授權連結授權商店並取得 access_token。
  • 已在 GenieGo [註冊] 中輸入 partner_id、partner_key、shop_id 並儲存。

Walmart Marketplace
API 발급 매뉴얼

Walmart Marketplace 자격증명(API 키/토큰)을 발급해 GenieGo 연동허브에 등록하는 과정을, 초보자도 그대로 따라 할 수 있도록 단계별로 정리했습니다.

예상 소요약 5~15분
필요 계정Walmart Marketplace 계정
단계 수4단계
결과물API 자격증명

발급 절차 (따라하기)

아래 순서를 그대로 따라 하면 자격증명을 발급받아 등록할 수 있습니다.

1

준비물: Walmart 마켓플레이스 셀러 계정이 필요합니다.

2

Walmart 셀러센터 > Developer Portal에 접속합니다.

3

API Key(client_id)와 Client Secret을 생성합니다.

4

GenieGo [등록]에 client_id, client_secret을 입력해 저장합니다.

발급 안내: Walmart 마켓플레이스 셀러센터(개발자)에서 client_id/secret을 발급합니다.

최종 체크리스트

  • 준비물: Walmart 마켓플레이스 셀러 계정이 필요합니다.
  • Walmart 셀러센터 > Developer Portal에 접속합니다.
  • API Key(client_id)와 Client Secret을 생성합니다.
  • GenieGo [등록]에 client_id, client_secret을 입력해 저장합니다.
발급한 키는 GenieGo 연동허브 [등록]에 입력하면 AES-256-GCM으로 암호화 저장되며, 조회 시 마스킹됩니다.
발급한 키/시크릿은 외부에 노출되지 않도록 안전하게 보관하세요. 노출이 의심되면 즉시 재발급(회전)하세요.

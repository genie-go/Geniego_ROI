TNT Express
Anleitung zur API-Key-Ausstellung

TNT Express wurde von FedEx übernommen und verwendet den im FedEx Developer Portal (developer.fedex.com) ausgestellten API Key·Secret zusammen mit Ihrer FedEx-Kontonummer. Diese Anleitung führt Sie durch die Registrierung dieser Anmeldedaten im GenieGo Integration Hub.

Geschätzte ZeitEtwa 5–15 Min.
Benötigtes KontoFedEx-Konto (Kontonummer)
Ausgestellte SchlüsselAPI Key · API Secret
ErgebnisAPI-Anmeldedaten · Kontonummer

1. Bevor Sie beginnen

TNT wurde in FedEx integriert und hat kein eigenes TNT-Entwicklerportal; es verwendet die Anmeldedaten des FedEx Developer Portals.

Hinweis: TNT wurde von FedEx übernommen und stellt Anmeldedaten (API Key/Secret) im FedEx Developer Portal (developer.fedex.com) aus. Es gibt kein eigenes TNT-Entwicklerportal, und eine FedEx-Kontonummer ist erforderlich.
Voraussetzungen

FedEx-Kontonummer

Sie benötigen eine FedEx-Kontonummer. Die Kontonummer wird bei der Registrierung in GenieGo gemeinsam eingegeben.

Ausgestellte Schlüssel

API Key · API Secret

Wenn Sie im FedEx Developer Portal ein Projekt erstellen, werden ein API Key und ein Secret ausgestellt.

2. Ausstellungsschritte

Stellen Sie Ihre Anmeldedaten im FedEx Developer Portal aus und registrieren Sie sie dann in GenieGo.

1

Im FedEx Developer Portal registrieren und ein Projekt erstellen

Registrieren Sie sich im FedEx Developer Portal und erstellen Sie ein Projekt.

https://developer.fedex.com
2

API Key·Secret ausstellen

Wenn Sie ein Projekt erstellen, werden der API Key/Secret ausgestellt.

Projekt erstellen
3

Kontonummer prüfen

Prüfen Sie Ihre FedEx-Kontonummer.

Sobald der API Key·Secret im FedEx Developer Portal ausgestellt wurde, ist alles in Ordnung.

3. In GenieGo zu registrierende Informationen

FeldBeschreibung
api_keyDer ausgestellte API Key (Client ID)
api_secretDas ausgestellte API Secret (Client Secret, vertraulich behandeln)
account_numberFedEx-Kontonummer
Geben Sie sie im GenieGo Integration Hub [Registrieren] ein und speichern Sie sie. Die Anmeldedaten werden im Ruhezustand mit AES-256-GCM verschlüsselt und beim Lesen maskiert.

4. Häufige Probleme

ProblemUrsacheLösung
Kein TNT-spezifisches Entwicklerportal zu findenTNT wurde in FedEx integriert, daher gibt es kein eigenes PortalErstellen Sie ein Projekt auf developer.fedex.com, um den API Key/Secret auszustellen.
Authentifizierungsfehler bei AufrufenFalsch eingegebenes Secret oder fehlende FedEx-KontonummerÜberprüfen Sie das api_secret und die Kontonummer (account_number) erneut.

Abschließende Checkliste

  • Bestätigt, dass TNT die Anmeldedaten von FedEx verwendet.
  • Auf developer.fedex.com registriert und ein Projekt erstellt.
  • Den API Key·Secret und die FedEx-Kontonummer geprüft.
  • api_key·api_secret·account_number in GenieGo [Registrieren] eingegeben und gespeichert.

PayPal
Anleitung zur API-Key-Ausstellung

Eine einsteigerfreundliche Schritt-für-Schritt-Anleitung zum Erstellen einer App im PayPal Developer Portal, um eine Client ID und ein Secret auszustellen und diese im GenieGo Integration Hub zu registrieren.

Geschätzte ZeitEtwa 5–10 Min.
Benötigtes KontoPayPal-Geschäftskonto
Ausgestellte SchlüsselClient ID · Secret
ErgebnisAPI-Zugangsdaten

1. Bevor Sie beginnen

PayPal-API-Zugangsdaten werden ausgestellt, indem Sie mit einem Geschäftskonto eine App im Developer Portal erstellen.

Erforderlich

PayPal-Geschäftskonto

Sie benötigen ein PayPal-Geschäftskonto, mit dem Sie sich bei developer.paypal.com anmelden können.

Ausgestellte Schlüssel

Client ID · Secret

Beim Erstellen einer Live-App werden eine Client ID und ein Secret ausgestellt.

Hinweis: Erstellen Sie die App für die tatsächliche Integration auf dem Tab [Live], nicht in der Sandbox.

2. Ausstellungsschritte

Stellen Sie in der folgenden Reihenfolge aus und registrieren Sie in GenieGo.

1

Beim Developer Portal anmelden

Melden Sie sich mit Ihrem PayPal-Geschäftskonto an und gehen Sie dann zu [Apps & Credentials].

developer.paypal.com → Apps & Credentials
2

Eine Live-App erstellen

Klicken Sie auf dem Tab [Live] auf [Create App], um die App zu erstellen.

Live → Create App
3

Zugangsdaten prüfen

Prüfen Sie die ausgestellte Client ID und das Secret.

Sobald die Client ID und das Secret angezeigt werden, wurden sie korrekt ausgestellt.

3. In GenieGo zu registrierende Informationen

FeldBeschreibung
client_idDie ausgestellte Client ID
client_secretDas ausgestellte Secret (sorgfältig behandeln)
Geben Sie im GenieGo Integration Hub unter [Registrieren] Ihre client_id·client_secret ein und speichern Sie. Zugangsdaten werden mit AES-256-GCM verschlüsselt gespeichert und beim Lesen maskiert.

4. Häufige Probleme

ProblemUrsacheLösung
Authentifizierung fehlgeschlagenVerwendung von Zugangsdaten einer Sandbox-AppVerwenden Sie die Client ID·Secret einer auf dem Tab [Live] erstellten App.
App kann nicht erstellt werdenMit einem regulären (privaten) Konto angemeldetMelden Sie sich mit einem PayPal-Geschäftskonto an.

Abschließende Checkliste

  • Mit einem PayPal-Geschäftskonto bei developer.paypal.com angemeldet.
  • In [Apps & Credentials] zum Tab [Live] gewechselt.
  • Über [Create App] eine App erstellt und die Client ID·Secret geprüft.
  • client_id·client_secret in GenieGo [Registrieren] eingegeben und gespeichert.

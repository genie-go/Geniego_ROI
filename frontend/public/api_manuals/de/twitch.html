1. Die Twitch-Kontostruktur verstehen

Anders als Instagram bietet Twitch keine getrennte Auswahl von Unternehmens-/Creator-Konten.

Offizieller Unternehmenskanal

Für den Markenbetrieb

Erstellen Sie ein normales Twitch-Konto unter Ihrem Firmen- oder Markennamen und betreiben Sie es wie einen offiziellen Kanal. Gut geeignet für Produktvorstellungen, Live-Commerce und Event-Übertragungen.

Streamer-Wachstumsstufe

Affiliate / Partner

Sobald Sie die Anforderungen an Stream-Leistung, Follower und Sehzeit erfüllen, können Sie am Affiliate- oder Partner-Programm teilnehmen.

Kernpunkt: Auch Unternehmen sollten zuerst ein normales Twitch-Konto erstellen und dann Kanal-Branding, Zwei-Faktor-Authentifizierung, OBS-Verbindung und die App-Registrierung in der Developer Console durchführen.

2. Was Sie benötigen

  • Geschäftliche E-Mail-Adresse
  • Marken- oder Firmenname
  • Kandidat für den Twitch-Benutzernamen
  • Firmenlogo und Kanal-Bannerbild
  • Firmenbeschreibungstext und Website-URL
  • PC oder Laptop für das Streaming
  • OBS Studio oder Streaming-Software
  • OAuth Redirect URL für die API-Integration

3. Ein Twitch-Konto erstellen

1

Twitch öffnen

Öffnen Sie die offizielle Twitch-Website in Ihrem Browser.

https://www.twitch.tv
2

Auf Registrieren klicken

Klicken Sie oben rechts auf die Schaltfläche Sign Up oder Registrieren.

Sign Up
3

Mit Firmendaten registrieren

Geben Sie die Firmen-E-Mail, den Benutzernamen, das Passwort und das Geburtsdatum ein.

Email / Username / Password
4

E-Mail verifizieren

Öffnen Sie die von Twitch gesendete Bestätigungs-E-Mail und schließen Sie Verify Email ab.

Verify Email

4. Einen Markenkontonamen wählen

Ihr Twitch-Benutzername wird zur Kanaladresse, daher ist ein kurzer, einprägsamer Name am besten.

6. Zwei-Faktor-Authentifizierung einrichten

Die 2FA-Einrichtung ist sehr wichtig, um Stream- und Sicherheitseinstellungen auf Twitch zuverlässig zu nutzen.

1

Das Sicherheitsmenü öffnen

Öffnen Sie das Menü Security and Privacy in den Einstellungen.

Settings → Security and Privacy
2

2FA aktivieren

Aktivieren Sie Two-Factor Authentication.

Set Up Two-Factor Authentication
3

Ein Telefon oder eine Authenticator-App verbinden

Verbinden Sie die SMS-Verifizierung oder eine Authenticator-App wie Google Authenticator oder Authy.

Authenticator App
Achtung: Unternehmenskonten können von mehreren Personen genutzt werden, verwalten Sie daher unbedingt die Administratorberechtigungen und die Wiederherstellungs-E-Mail.

7. Den Stream-Key prüfen & OBS verbinden

1

Das Creator Dashboard öffnen

Gehen Sie über das Profilmenü zum Creator Dashboard.

Creator Dashboard
2

Das Stream-Menü öffnen

Wählen Sie den Punkt Stream im Einstellungsmenü.

Settings → Stream
3

Den Primary Stream Key prüfen

Kopieren Sie den primären Stream-Key. Dieser Key muss wie ein Passwort aufbewahrt werden.

Primary Stream Key
4

In OBS verbinden

Stellen Sie in OBS Studio den Dienst auf Twitch und geben Sie den kopierten Wert als Stream-Key ein.

OBS → Settings → Stream → Twitch
Wenn Ihr Stream-Key offengelegt wird, kann jemand anderes auf Ihrem Kanal senden. Veröffentlichen Sie ihn niemals.

9. Registrierung in der Twitch Developer Console

Um Chatbots, Zuschaueranalysen, Stream-Statusabfragen und Automatisierungsfunktionen zu erstellen, müssen Sie eine Twitch-Developer-App registrieren.

1

Twitch Developer öffnen

Öffnen Sie die Twitch-Entwicklerseite.

https://dev.twitch.tv
2

Zur Console gehen

Klicken Sie im oberen Menü auf Console oder Developer Console.

Developer Console
3

Das Menü Applications auswählen

Wählen Sie Applications im linken oder oberen Menü.

Applications
4

Auf Register Your Application klicken

Klicken Sie auf die Schaltfläche zum Registrieren einer neuen Anwendung.

Register Your Application

10. Registrierungsdaten der Twitch-API-App eingeben

EingabefeldEingabebeispielBeschreibung
NameGeniego Twitch AppDer App-Name. Geben Sie ihn anhand Ihres Servicenamens ein.
OAuth Redirect URLshttps://yourdomain.com/auth/twitch/callbackDie Serveradresse, zu der nach dem OAuth-Login zurückgekehrt wird.
CategoryWebsite Integration / Chat BotWählen Sie die Kategorie, die dem Zweck Ihrer App entspricht.
Client TypeConfidentialGeeignet für allgemeine Dienste, die das Secret sicher auf dem Server speichern.
Sobald die Registrierung abgeschlossen ist, wird die Client ID angezeigt, und Sie können mit der Schaltfläche New Secret ein Client Secret generieren.

11. Die Client ID / das Client Secret prüfen

1

Die registrierte App auswählen

Klicken Sie in der Applications-Liste der Developer Console auf die gerade erstellte App.

Applications → App auswählen
2

Die Client ID kopieren

Kopieren Sie die angezeigte Client ID und speichern Sie sie in Ihren Servereinstellungen.

Client ID
3

Das Client Secret generieren

Klicken Sie auf die Schaltfläche New Secret, um ein Client Secret zu generieren, und speichern Sie es sicher.

New Secret
Das Client Secret kann oft nur einmal eingesehen werden. Speichern Sie es unmittelbar nach der Generierung an einem sicheren Ort.

12. Authentifizierungsstruktur der Twitch-API

Twitch Channel │ ▼ Developer Console │ ▼ Application │ ▼ Client ID / Client Secret │ ▼ OAuth Token │ ▼ Twitch Helix API │ ▼ Kanalinfos, Stream-Status, Chatbot, Zuschaueranalyse

13. Ablauf der OAuth-Token-Ausstellung

1) Die Twitch-OAuth-Autorisierungs-URL generieren 2) Der Benutzer meldet sich mit seinem Twitch-Konto an 3) Den authorization code an der Redirect URL empfangen 4) Mit dem authorization code + Client Secret ein Access Token anfordern 5) Mit dem Access Token die Twitch-API aufrufen 6) Bei Bedarf mit dem Refresh Token aktualisieren
Einfache Abfragen öffentlicher Daten sind mit einem App Access Token möglich, während Vorgänge, die Benutzerberechtigungen erfordern, ein User Access Token benötigen.

15. Abschließende Checkliste

  • Ein Twitch-Konto mit einer geschäftlichen E-Mail erstellt.
  • Einen Benutzernamen festgelegt, der mit dem Markennamen übereinstimmt.
  • Die E-Mail-Verifizierung abgeschlossen.
  • Profilbild, Banner, Bio-Text und Links festgelegt.
  • Die Zwei-Faktor-Authentifizierung aktiviert.
  • Den Stream Key im Creator Dashboard geprüft.
  • OBS Studio mit dem Twitch-Kanal verbunden.
  • Die App in der Twitch Developer Console registriert.
  • Die Client ID und das Client Secret erhalten.
  • Die OAuth Redirect URL und den Token-Ausstellungsablauf vorbereitet.

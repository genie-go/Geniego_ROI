Magento (Adobe Commerce)
Anleitung zur API-Key-Ausstellung

Eine einsteigerfreundliche Schritt-für-Schritt-Anleitung zum Erstellen einer Integration in der Magento-(Adobe Commerce)-Verwaltung, um ein Zugriffstoken auszustellen und es im GenieGo Integration Hub zu registrieren.

Geschätzte ZeitEtwa 5–15 Min.
Benötigtes KontoMagento-(Adobe Commerce)-Administratorrechte
Ausgestellte SchlüsselIntegration-Zugriffstoken
ErgebnisREST-API-Anmeldedaten

1. Bevor Sie beginnen

Eine Integration kann nur mit Administratorrechten erstellt werden.

Erforderlich

Magento-Administratorrechte

Sie benötigen Magento-(Adobe Commerce)-Administratorrechte. Mit normalen Benutzerrechten kann keine Integration erstellt werden.

Ausgestellte Schlüssel

Integration-Zugriffstoken

Wenn Sie eine Integration erstellen und auf Activate klicken, wird ein Zugriffstoken ausgestellt.

2. Ausstellungsschritte

Stellen Sie in der unten angegebenen Reihenfolge aus und registrieren Sie in GenieGo.

1

Zum Integrations-Menü gehen

Gehen Sie in der Magento-Verwaltung zu [System] > [Integrationen].

Verwaltung → System → Integrationen
2

Integration erstellen

Erstellen Sie eine Integration über [Add New Integration] und konfigurieren Sie die Ressourcenberechtigungen.

Add New Integration
3

Token über Activate ausstellen

Klicken Sie auf [Activate], um das Zugriffstoken zu erhalten.

Activate
4

Informationen überprüfen

Überprüfen Sie das ausgestellte Zugriffstoken und Ihre Store-URL (base_url).

Sobald das Zugriffstoken nach Activate angezeigt wird, wurde es korrekt ausgestellt.

3. In GenieGo zu registrierende Informationen

FeldBeschreibung
base_urlStore-URL
access_tokenDas ausgestellte Integration-Zugriffstoken (vertraulich behandeln)
Geben Sie im GenieGo Integration Hub unter [Registrieren] base_url·access_token ein und speichern Sie. Anmeldedaten werden im Ruhezustand mit AES-256-GCM verschlüsselt und beim Lesen maskiert.

4. Häufige Probleme

ProblemUrsacheLösung
Das Integrations-Menü fehltUnzureichende AdministratorrechteMelden Sie sich mit einem Administratorkonto an, das die Berechtigung zum Erstellen von Integrationen hat.
Berechtigungsfehler bei API-AufrufenRessourcenberechtigungen der Integration nicht gesetzt oder Activate nicht abgeschlossenLegen Sie die Ressourcenberechtigungen der Integration fest und schließen Sie Activate ab, um das Token auszustellen.

Abschließende Checkliste

  • Mit Magento-(Adobe Commerce)-Administratorrechten angemeldet.
  • Eine Integration über Add New Integration unter System > Integrationen erstellt.
  • Ressourcenberechtigungen festgelegt und das Zugriffstoken über Activate ausgestellt.
  • base_url·access_token in GenieGo [Registrieren] eingegeben und gespeichert.

Klarna
Anleitung zur API-Key-Ausstellung

Eine einsteigerfreundliche Schritt-für-Schritt-Anleitung zum Erstellen von API-Anmeldedaten (username und password) im Klarna Merchant Portal und zum Registrieren zusammen mit Ihrer Region im GenieGo Integration Hub.

Geschätzte ZeitEtwa 5–10 Min.
Benötigtes KontoKlarna-Händlerkonto (BNPL, Vertrag erforderlich)
Ausgestellte SchlüsselAPI username · password · Region
ErgebnisAPI-Anmeldedaten

1. Bevor Sie beginnen

Klarna-Anmeldedaten können erst nach Abschluss Ihres Händlervertrags ausgestellt werden.

Erforderlich

Klarna-Händlerkonto

Sie benötigen zunächst einen Klarna-(BNPL-)Händlervertrag. Erst nach Abschluss des Vertrags können Sie API-Anmeldedaten im Merchant Portal ausstellen.

Ausgestellte Schlüssel

API username · password · Region

Erstellen Sie im Merchant Portal unter Settings > Klarna API credentials das API username und password und bestätigen Sie Ihre Region (eu/na/oc).

Hinweis: Die Region ist eine von eu / na / oc und muss bei der Registrierung mit angegeben werden.

2. Ausstellungsschritte

Stellen Sie nach Ihrem Händlervertrag in der unten angegebenen Reihenfolge aus und registrieren Sie in GenieGo.

1

Händlervertrag

Schließen Sie zunächst Ihren Klarna-(BNPL-)Händlervertrag ab. Erst nach Abschluss des Vertrags können Sie zum nächsten Schritt fortfahren.

2

Beim Merchant Portal anmelden

Melden Sie sich beim Klarna Merchant Portal an.

portal.klarna.com
3

Zu API credentials wechseln

Wechseln Sie zum Menü [Settings] > [Klarna API credentials].

Settings → Klarna API credentials
4

Anmeldedaten erstellen und bestätigen

Erstellen Sie das API username und password und bestätigen Sie Ihre Region (eu/na/oc).

Wenn unter Settings > Klarna API credentials das API username und password erstellt wurden und Sie die Region bestätigt haben, ist alles in Ordnung.

3. In GenieGo zu registrierende Informationen

FeldBeschreibung
usernameAPI Username (MID/PID)
passwordAPI Password (vertraulich behandeln)
regionRegion (eu / na / oc)
Geben Sie im GenieGo Integration Hub unter [Registrieren] username, password und region ein und speichern Sie. Anmeldedaten werden im Ruhezustand mit AES-256-GCM verschlüsselt und beim Lesen maskiert.

4. Häufige Probleme

ProblemUrsacheLösung
Keine Ausstellungsoption im Menü API credentialsHändlervertrag nicht abgeschlossenSchließen Sie zunächst Ihren Klarna-Händlervertrag ab.
AuthentifizierungsfehlerRegion (eu/na/oc) stimmt nicht überein oder password falsch eingegebenÜberprüfen Sie die Region sowie username und password erneut, die Sie im Merchant Portal bestätigt haben.

Abschließende Checkliste

  • Den Klarna-(BNPL-)Händlervertrag abgeschlossen.
  • Anmeldedaten unter Settings > Klarna API credentials im Merchant Portal erstellt.
  • API username, password und Region (eu/na/oc) bestätigt.
  • username, password und region in GenieGo [Registrieren] eingegeben und gespeichert.

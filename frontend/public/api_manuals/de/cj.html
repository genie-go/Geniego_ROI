CJ Logistics
Anleitung zur API-Key-Ausstellung

Die dedizierte API von CJ Logistics wird auf Basis eines Vertrags-(Kunden-)Codes ausgestellt. Diese Anleitung führt Sie durch das Erhalten des Vertrags-(Kunden-)Codes und API-Schlüssels von Ihrem Vertriebsmitarbeiter und deren Registrierung im GenieGo Integration Hub.

Geschätzte ZeitMehrere Tage bis Wochen, je nach Vertragsfortschritt
Benötigtes KontoCJ-Logistics-Vertrag (Firmenkunde)
Ausgestellte SchlüsselAPI-Schlüssel · Kunden-(Vertrags-)Code
ErgebnisAPI-Anmeldedaten

1. Bevor Sie beginnen

Die dedizierte API von CJ Logistics ist ein vertragsbasierter Kanal und wird nicht sofort über eine Self-Service-Konsole ausgestellt.

Erforderlich

CJ-Logistics-Vertrags-(Kunden-)Code

Sie müssen über einen Vertrags-(Kunden-)Code verfügen, der durch einen Vertrag mit CJ Logistics ausgestellt wird. Er wird nicht direkt über eine separate Konsole ausgestellt.

Empfohlen

Smart Parcel (einheitliches Tracking)

Wenn Sie nur Sendungsverfolgung benötigen, können Sie über den Kanal [Smart Parcel (einheitliches Tracking)] mit einem einzigen t_key alle Versanddienstleister verbinden.

Hinweis: Wenn Ihr Ziel die Sendungsverfolgung ist, empfehlen wir die Verbindung über den einheitlichen Smart-Parcel-(SweetTracker-)Schlüssel ohne dedizierten API-Vertrag.

2. Antragsverfahren für die Ausstellung

Beantragen und registrieren Sie in der unten angegebenen Reihenfolge.

1

Vertrags-(Kunden-)Code vorbereiten

Bereiten Sie Ihren CJ-Logistics-Vertrags-(Kunden-)Code vor. Die dedizierte API wird auf Vertragsbasis bereitgestellt.

2

Entscheiden, ob Sie einheitliches Tracking benötigen

Wenn Sie nur Sendungsverfolgung benötigen, können Sie statt des dedizierten CJ-Logistics-Kanals über den Kanal [Smart Parcel (einheitliches Tracking)] mit einem einzigen t_key alle Versanddienstleister verbinden.

3

Dedizierte API beantragen

Wenn Sie die dedizierte API von CJ Logistics benötigen, erhalten Sie den Vertrags-(Kunden-)Code und den API-Schlüssel von Ihrem Vertriebsmitarbeiter.

4

In GenieGo registrieren

Geben Sie api_key und cust_code im GenieGo Integration Hub [Registrieren] ein und speichern Sie.

Sobald Sie den API-Schlüssel und den Vertrags-(Kunden-)Code von Ihrem Vertriebsmitarbeiter erhalten haben, sind Sie zur Registrierung bereit.

3. In GenieGo zu registrierende Informationen

FeldBeschreibung

Wenn Sie ihn im GenieGo Integration Hub [Registrieren] eingeben und speichern, werden Anmeldedaten im Ruhezustand mit AES-256-GCM verschlüsselt und beim Lesen maskiert.

Anleitung zur API-Key-Ausstellung

Eine einsteigerfreundliche Schritt-für-Schritt-Anleitung zum Ausstellen eines Open-API-Schlüssels im 11Street Open API Center (openapi.11st.co.kr) und zum Registrieren im GenieGo Integration Hub.

Geschätzte ZeitEtwa 5–15 Min.
Benötigtes Konto11Street-Verkäuferkonto
Ausgestellte SchlüsselOpen-API-Schlüssel · Verkäufer-ID
ErgebnisOpen-API-Anmeldedaten

1. Bevor Sie beginnen

Open-API-Schlüssel von 11Street können nur über ein Verkäuferkonto ausgestellt werden.

Erforderlich

11Street-Verkäuferkonto

Sie benötigen ein 11Street-Verkäuferkonto. Melden Sie sich mit Ihrer Verkäufer-ID am Open API Center an.

Ausgestellte Schlüssel

Open-API-Schlüssel · Verkäufer-ID

Nach Abschluss der Dienstregistrierung und der Zugriffs-IP-Registrierung prüfen Sie den ausgestellten Open-API-Schlüssel zusammen mit Ihrer Verkäufer-ID.

Hinweis: Sie müssen eine Server-IP registrieren, die auf die API zugreifen darf, bevor Sie den Schlüssel verwenden können.

2. Ausstellungsschritte

Stellen Sie in der unten angegebenen Reihenfolge aus und registrieren Sie in GenieGo.

1

Beim Open API Center anmelden

Melden Sie sich mit Ihrer Verkäufer-ID am 11Street Open API Center an.

https://openapi.11st.co.kr/openapi/OpenApiFrontMain.tmall
2

Dienst registrieren

Klicken Sie im Menü auf [Dienstregistrierung & Prüfung], geben Sie Ihre Verkäuferdaten ein, stimmen Sie den Bedingungen zu und registrieren Sie.

Dienstregistrierung & Prüfung
3

Zugriffs-IP registrieren

Registrieren Sie unter [Seller API Info] > [Bearbeiten] die Server-IP, die auf die API zugreifen darf.

Seller API Info → Bearbeiten
4

API Key kopieren

Kopieren Sie den ausgestellten API Key.

Sobald die Dienstregistrierung und die Zugriffs-IP-Registrierung abgeschlossen sind, wird der API Key ausgestellt.

3. In GenieGo zu registrierende Informationen

FeldBeschreibung
api_keyDer ausgestellte Open-API-Schlüssel
seller_idVerkäufer-ID
Geben Sie ihn im GenieGo Integration Hub [Registrieren] ein und speichern Sie. Anmeldedaten werden im Ruhezustand mit AES-256-GCM verschlüsselt und beim Lesen maskiert.

4. Häufige Probleme

ProblemUrsacheLösung
API Key wird nicht ausgestelltDienstregistrierung oder Zugriffs-IP-Registrierung nicht abgeschlossenSchließen Sie [Dienstregistrierung & Prüfung] und die Zugriffs-IP-Registrierung unter [Seller API Info] ab.
Authentifizierungsfehler bei AufrufenZulässige Server-IP nicht registriertPrüfen Sie die zulässige Server-IP unter [Seller API Info] > [Bearbeiten] erneut.

5. Vor der Produktregistrierung (wichtig)

Auch mit gültigem API-Schlüssel lehnt 11Street die Produktregistrierung ab, solange die folgenden drei Punkte fehlen.

  • Anzeigekategorie (dispCtgrNo) – in GenieGo unter [Katalog-Sync] > [Kategoriezuordnung] oder im Panel „Kategorie & Marke erforderlich" festlegen
  • Marke – in GenieGo unter [Katalog-Sync] > [Markenverwaltung] anlegen und dem Produkt zuweisen (der Markenname genügt, kein Markencode nötig)
  • Verkaufsart – GenieGo sendet automatisch Festpreis (01); keine Eingabe erforderlich

Abschließende Checkliste

  • Mit einem 11Street-Verkäuferkonto am Open API Center angemeldet.
  • Verkäuferdaten eingegeben und den Bedingungen unter [Dienstregistrierung & Prüfung] zugestimmt.
  • Die zulässige Server-IP unter [Seller API Info] registriert.
  • Den ausgestellten Open-API-Schlüssel und die Verkäufer-ID bestätigt.
  • api_key·seller_id in GenieGo [Registrieren] eingegeben und gespeichert.

Shopify
API-Key-Ausstellungshandbuch

Eine Schritt-für-Schritt-Anleitung zum Erstellen einer benutzerdefinierten App in Ihrem Shopify-Admin, um ein Admin API access token auszustellen und es zusammen mit Ihrer Shop-Domain im GenieGo Integration Hub zu registrieren, um Ihren Shopify-Store zu verbinden.

Geschätzte ZeitCa. 5–15 Min.
Benötigtes KontoShopify-Store-Adminberechtigung
Ausgestellte SchlüsselShop-Domain · Admin API access token
ErgebnisAdmin API-Anmeldedaten

1. Bevor Sie beginnen

Eine benutzerdefinierte Shopify-App kann nur mit Store-Adminberechtigung erstellt werden.

Erforderlich

Store-Adminberechtigung

Sie benötigen die Admin-Berechtigung für den Shopify-Store, um eine benutzerdefinierte App zu erstellen und ein Token auszustellen.

Ausgestellte Schlüssel

Shop-Domain · Admin API token

Sie erhalten eine Shop-Domain im Format xxx.myshopify.com und das Admin API access token, das nach der Installation der App angezeigt wird.

Hinweis: Das Admin API access token wird unmittelbar nach der Installation nur einmal angezeigt, daher müssen Sie es sofort kopieren, sobald es erscheint.

2. Ausstellungsschritte

Stellen Sie in der unten angegebenen Reihenfolge aus und registrieren Sie in GenieGo.

1

Zu Apps und Vertriebskanälen wechseln

Gehen Sie in Ihrem Shopify-Admin zu [Einstellungen] > [Apps und Vertriebskanäle].

https://www.shopify.com/admin/settings/apps
2

Eine benutzerdefinierte App erstellen

Klicken Sie auf [Apps entwickeln] > [App erstellen], um eine benutzerdefinierte App zu erstellen.

Apps entwickeln → App erstellen
3

Scopes auswählen · App installieren

Wählen Sie die Admin API scopes aus und installieren Sie die App.

4

Das access token kopieren

Kopieren Sie sofort das nach der Installation angezeigte Admin API access token (wird nur einmal angezeigt).

Sobald das Admin API access token nach der Installation der App erscheint, wurde es korrekt ausgestellt. Es wird nur einmal angezeigt, kopieren Sie es daher sofort.

3. In GenieGo zu registrierende Informationen

FeldBeschreibung
shop_domainShop-Domain im Format xxx.myshopify.com
access_tokenDas ausgestellte Admin API access token (mit Sorgfalt behandeln)
Geben Sie in GenieGo [Registrieren] Ihre shop_domain (xxx.myshopify.com) und Ihr access_token ein und speichern Sie. Anmeldedaten werden im Ruhezustand mit AES-256-GCM verschlüsselt und beim Lesen maskiert.

4. Häufige Probleme

ProblemUrsacheLösung
Token kann nicht erneut angezeigt werdenDas Admin API access token wird nur einmal angezeigtWenn Sie das Token verloren haben, stellen Sie es in der benutzerdefinierten App erneut aus und kopieren Sie es sofort.
Fehler wegen unzureichender BerechtigungAdmin API scopes nicht ausgewähltWählen Sie die erforderlichen Admin API scopes aus und installieren Sie dann die App erneut.
Domain-FehlerShop-Domain-Format falsch eingegebenGeben Sie die Shop-Domain genau im Format xxx.myshopify.com ein.

Abschließende Checkliste

  • Mit Shopify-Store-Adminberechtigung angemeldet.
  • Eine benutzerdefinierte App unter Einstellungen > Apps und Vertriebskanäle > Apps entwickeln erstellt.
  • Die Admin API scopes ausgewählt und die App installiert.
  • Das nach der Installation angezeigte Admin API access token sofort kopiert.
  • shop_domain·access_token in GenieGo [Registrieren] eingegeben und gespeichert.

Ausstellungsschritte

Folgen Sie den Schritten unten, um Ihre Zugangsdaten auszustellen und zu registrieren.

1

Voraussetzung: Sie benötigen ein persönliches Facebook-Konto sowie ein Meta-Werbekonto, über das bereits Anzeigen geschaltet werden.

2

Rufen Sie im Browser business.facebook.com auf, melden Sie sich an, erstellen Sie über [Unternehmen erstellen] ein Unternehmenskonto und schließen Sie die E-Mail-Verifizierung ab.

3

Gehen Sie zu developers.facebook.com → oben rechts [Meine Apps] → [App erstellen] und wählen Sie als Typ „Business“, um die App zu erstellen.

4

Suchen Sie im App-Dashboard die Karte „Marketing API“ und klicken Sie auf [Einrichten], um sie als Produkt hinzuzufügen.

5

Unternehmenseinstellungen (business.facebook.com/settings) → [Benutzer] > [Systembenutzer] → [Hinzufügen], um einen Systembenutzer zu erstellen, und weisen Sie ihm unter [Assets zuweisen] die Berechtigung für das Werbekonto zu.

6

Klicken Sie im selben Bildschirm auf [Token generieren], wählen Sie die App aus, haken Sie die Berechtigungen (ads_read, ads_management) an, generieren Sie das Token und kopieren Sie dessen Wert.

7

Ermitteln Sie im Werbeanzeigenmanager (adsmanager.facebook.com) die Werbekonto-ID (eine Zahl, die mit act_ beginnt).

8

Klicken Sie im GenieGo-Integrationshub beim betreffenden Kanal auf [Registrieren], fügen Sie das „Zugriffstoken“ und die „Werbekonto-ID (act_)“ ein und speichern Sie – damit ist die Integration abgeschlossen.

Im GenieGo Integration Hub [Register] eingegebene Schlüssel werden mit AES-256-GCM verschlüsselt gespeichert und beim Lesen maskiert.
Bewahren Sie Ihre Schlüssel/Secrets sicher auf und geben Sie sie niemals öffentlich preis. Bei Verdacht auf ein Leck sofort neu ausstellen.

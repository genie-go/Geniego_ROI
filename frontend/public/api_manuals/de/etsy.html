Anleitung zur API-Key-Ausstellung

Eine einsteigerfreundliche Schritt-für-Schritt-Anleitung zur Registrierung einer App im Etsy-Entwicklerportal, um einen API Key (keystring) auszustellen, ein access token per OAuth-Autorisierung zu erhalten und es im GenieGo Integration Hub zu registrieren.

Geschätzte ZeitEtwa 5–15 Min.
Benötigtes KontoEtsy-Verkäuferkonto
Ausgestellte SchlüsselAPI Key (keystring) · Shop ID
ErgebnisAPI-Anmeldedaten

1. Bevor Sie beginnen

Der Etsy API Key wird durch die Registrierung einer App unter Ihrem Verkäuferkonto ausgestellt.

Erforderlich

Etsy-Verkäuferkonto

Für die Ausstellung benötigen Sie ein Etsy-Verkäuferkonto. Mit diesem Konto greifen Sie auf das Etsy-Entwicklerportal zu.

Ausgestellte Schlüssel

API Key (keystring)

Sobald Sie Ihre App-Informationen eingeben, wird ein API Key (keystring) ausgestellt. Anschließend erhalten Sie über die OAuth-Autorisierung ein access token.

Hinweis: Die GenieGo-Registrierung erfordert den ausgestellten api_key zusammen mit der shop_id, die Ihren Shop identifiziert.

2. Ausstellungsschritte

Stellen Sie in der unten angegebenen Reihenfolge aus und registrieren Sie in GenieGo.

1

Neue App erstellen

Gehen Sie zu etsy.com/developers und klicken Sie auf [Create a New App].

etsy.com/developers
2

App-Informationen eingeben und Schlüssel ausstellen

Sobald Sie Ihre App-Informationen eingeben, wird ein API Key (keystring) ausgestellt.

https://www.etsy.com/developers/your-apps
3

OAuth-Autorisierung

Erhalten Sie über die OAuth-Autorisierung ein access token.

4

Registrierungsinformationen bestätigen

Überprüfen Sie den ausgestellten api_key und die shop_id Ihres Shops.

Sobald der API Key (keystring) ausgestellt wurde und Sie über die OAuth-Autorisierung ein access token erhalten haben, wurde es korrekt ausgestellt.

3. In GenieGo zu registrierende Informationen

FeldBeschreibung
api_keyDer ausgestellte API Key (keystring)
shop_idDie Shop ID des Etsy-Shops, den Sie verbinden möchten
Geben Sie im GenieGo Integration Hub unter [Registrieren] Ihren api_key und Ihre shop_id ein und speichern Sie. Anmeldedaten werden im Ruhezustand mit AES-256-GCM verschlüsselt und beim Lesen maskiert.

4. Häufige Probleme

ProblemUrsacheLösung
App kann nicht erstellt werdenKein Etsy-Verkäuferkonto oder kein Zugriff auf das EntwicklerportalGreifen Sie mit Ihrem Etsy-Verkäuferkonto auf etsy.com/developers zu und fahren Sie mit [Create a New App] fort.
Authentifizierung fehlgeschlagenOAuth-Autorisierung nicht abgeschlossen oder access token nicht ausgestelltErhalten Sie nach der Ausstellung des API Key (keystring) über die OAuth-Autorisierung ein access token.

Abschließende Checkliste

  • Mit meinem Etsy-Verkäuferkonto auf etsy.com/developers zugegriffen.
  • Über [Create a New App] einen API Key (keystring) ausgestellt.
  • Über die OAuth-Autorisierung ein access token erhalten.
  • api_key und shop_id in GenieGo [Registrieren] eingegeben und gespeichert.

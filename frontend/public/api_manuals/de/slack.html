Slack Webhook
API-Ausstellungshandbuch

Eine Schritt-für-Schritt-Anleitung, der jeder folgen kann, um Slack Webhook-Zugangsdaten (API-Schlüssel/Token) auszustellen und im GenieGo Integration Hub zu registrieren.

Dauerca. 5–15 Min.
Benötigtes KontoSlack Webhook-Konto
Schritte5 Schritte
ErgebnisAPI-Zugangsdaten

Ausstellungsschritte

Folgen Sie den Schritten unten, um Ihre Zugangsdaten auszustellen und zu registrieren.

1

Voraussetzung: Sie benötigen Verwaltungsrechte für den Slack-Workspace.

2

Klicken Sie auf api.slack.com/apps auf [Create New App].

3

Aktivieren Sie [Incoming Webhooks] und klicken Sie auf [Add New Webhook to Workspace].

4

Wenn Sie einen Kanal auswählen, wird eine Webhook URL generiert.

5

Geben Sie bei GenieGo unter [Registrieren] webhook_url ein und speichern Sie.

Abschluss-Checkliste

  • Voraussetzung: Sie benötigen Verwaltungsrechte für den Slack-Workspace.
  • Klicken Sie auf api.slack.com/apps auf [Create New App].
  • Aktivieren Sie [Incoming Webhooks] und klicken Sie auf [Add New Webhook to Workspace].
  • Wenn Sie einen Kanal auswählen, wird eine Webhook URL generiert.
  • Geben Sie bei GenieGo unter [Registrieren] webhook_url ein und speichern Sie.
Im GenieGo Integration Hub [Register] eingegebene Schlüssel werden mit AES-256-GCM verschlüsselt gespeichert und beim Lesen maskiert.
Bewahren Sie Ihre Schlüssel/Secrets sicher auf und geben Sie sie niemals öffentlich preis. Bei Verdacht auf ein Leck sofort neu ausstellen (rotieren).

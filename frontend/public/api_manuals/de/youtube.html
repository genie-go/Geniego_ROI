YouTube API Key
Ausstellungshandbuch

Eine praktische Schritt-für-Schritt-Anleitung, der selbst Anfänger folgen können, um die YouTube Data API v3 in der Google Cloud Console zu aktivieren und einen API Key zu erstellen.

Geschätzte ZeitEtwa 5–10 Min.
Benötigtes KontoGoogle-Konto
Verwendete APIYouTube Data API v3
ErgebnisEin API Key im Format AIzaSy...

1. API Key vs. OAuth

Das ist der Teil, der beim Einstieg am meisten Verwirrung stiftet.

API Key

Zum Abfragen öffentlicher Daten

Wird verwendet, um öffentliche YouTube-Daten abzurufen, etwa zum Suchen von Videos, Abrufen von Kanalinformationen, Laden von Playlists und Lesen von Kommentaren.

OAuth 2.0

Für Aufgaben, die die Berechtigungen deines Kontos erfordern

Wird für Aufgaben verwendet, die die Zustimmung des Nutzers erfordern, etwa das Hochladen von Videos, das Verwalten des eigenen Kanals und den Zugriff auf private Daten.

Wichtig: Um Videos automatisch hochzuladen, reicht ein API Key allein nicht aus; eine OAuth-2.0-Authentifizierung ist erforderlich.

2. YouTube Data API v3 aktivieren

Bevor du einen API Key erstellst, musst du zunächst die gewünschte API in deinem Projekt aktivieren.

1

Auf die Google Cloud Console zugreifen

Melde dich in der Google Cloud Console an.

console.cloud.google.com
2

Projekt auswählen oder erstellen

Wähle in der Projektauswahl oben ein bestehendes Projekt aus oder erstelle ein neues.

z. B. Geniego ROI
3

Zur API-Bibliothek wechseln

Gehe im linken Menü zu APIs und Dienste → Bibliothek.

APIs und Dienste → Bibliothek
4

Nach der YouTube Data API v3 suchen und aktivieren

Gib YouTube Data API v3 in das Suchfeld ein, wähle diese API aus und klicke auf die Schaltfläche Aktivieren.

YouTube Data API v3 → Aktivieren
Wenn auf dem Bildschirm Status: Aktiviert angezeigt wird, wurde sie korrekt aktiviert.

3. Einen API Key erstellen

Sobald die YouTube Data API v3 aktiviert ist, kannst du auf dem Bildschirm Anmeldedaten einen API Key erstellen.

1

Zum Menü Anmeldedaten wechseln

Gehe im linken Menü zu APIs und Dienste → Anmeldedaten.

APIs und Dienste → Anmeldedaten
2

Auf Anmeldedaten erstellen klicken

Klicke oben auf die Schaltfläche + Anmeldedaten erstellen.

+ Anmeldedaten erstellen
3

API-Schlüssel auswählen

Wähle im angezeigten Menü API-Schlüssel aus.

API-Schlüssel
4

Die zugängliche API auswählen

Bestätige unter API-Einschränkungen, dass die YouTube Data API v3 ausgewählt ist.

Ausgewählte API: YouTube Data API v3
5

Auf die Schaltfläche Erstellen klicken

Klicke unten auf die Schaltfläche Erstellen, um den API Key zu generieren.

Erstellen
Der generierte Schlüssel beginnt in der Regel mit AIzaSy.... Diese Zeichenfolge ist dein YouTube API Key.

4. Welche Optionen solltest du im Erstellungsbildschirm wählen?

ElementEmpfohlene Einstellung für AnfängerBeschreibung
NameYouTube API KeyGib einen Namen ein, der später leicht zu erkennen ist.
API-EinschränkungenYouTube Data API v3Beschränke diesen Schlüssel so, dass er nur mit der YouTube API verwendet wird.
Dienstkonto-AuthentifizierungNicht aktivierenFür die Erstellung eines YouTube Data API Key in der Regel nicht erforderlich.
AnwendungseinschränkungenFür den ersten Test keineNachdem du bestätigt hast, dass es korrekt funktioniert, empfehlen wir, Website- oder IP-Adress-Einschränkungen festzulegen.
Wenn du in einen echten Dienst bereitstellst, lege unbedingt Schlüsseleinschränkungen fest. Wird er ohne Einschränkungen offengelegt, können andere deinen Schlüssel verwenden.

5. Beispiele für die Verwendung des API Key

Du kannst testen, indem du deinen generierten Schlüssel in den Teil YOUR_API_KEY einfügst.

Beispiel für Videosuchehttps://www.googleapis.com/youtube/v3/search?part=snippet&q=korea&type=video&key=YOUR_API_KEY
Beispiel für Kanalinfo-Abfragehttps://www.googleapis.com/youtube/v3/channels?part=snippet&id=CHANNEL_ID&key=YOUR_API_KEY
Wenn beim Eingeben der obigen URL in die Adressleiste deines Browsers ein JSON-Ergebnis erscheint, funktioniert der API Key korrekt.

6. Häufige Probleme beheben

ProblemUrsacheLösung
Keine API ausgewähltDie YouTube Data API v3 wurde noch nicht aktiviert oder unter API-Einschränkungen nicht ausgewähltAktiviere die YouTube Data API v3 in der API-Bibliothek und wähle sie dann erneut aus.
Keine API-Schlüssel zum AnzeigenDu hast noch keinen API Key erstelltKlicke auf Anmeldedaten erstellen → API-Schlüssel.
403-FehlerAPI-Einschränkungen, überschrittenes Kontingent oder BerechtigungsproblemeÜberprüfe die API-Einschränkungen, das Kontingent und den Aktivierungsstatus.
Uploads funktionieren nichtEs wird nur ein API Key verwendetVideo-Uploads müssen eine OAuth-2.0-Authentifizierung verwenden.

7. Abschließende Checkliste

  • Ein Google-Cloud-Projekt ausgewählt.
  • Die YouTube Data API v3 aktiviert.
  • Unter Anmeldedaten einen API Key erstellt.
  • Bestätigt, dass der generierte API Key mit AIzaSy beginnt.
  • Die API-Einschränkungen auf die YouTube Data API v3 gesetzt.
  • Website- oder IP-Einschränkungen vor der Bereitstellung in einem echten Dienst festgelegt.

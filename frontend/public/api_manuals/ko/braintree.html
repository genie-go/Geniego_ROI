Braintree
API Key 발급 매뉴얼

Braintree(PayPal) Control Panel에서 Production API 자격증명(Merchant ID·Public Key·Private Key)을 확인하고 GenieGo 연동 허브에 등록하는 과정을 초보자도 그대로 따라 할 수 있도록 정리한 매뉴얼입니다.

소요 시간약 5~10분
필요 계정Braintree(PayPal) 가맹 계정(승인 필요)
발급 키Merchant ID · Public Key · Private Key
결과물Production API 자격증명

1. 시작하기 전에

Braintree(PayPal) 자격증명은 가맹 계정이 승인된 후 확인할 수 있습니다.

필요

Braintree(PayPal) 가맹 계정

먼저 Braintree(PayPal) 가맹 계정 승인이 필요합니다. 가맹 계정 승인이 완료되어야 Production 자격증명을 확인할 수 있습니다.

발급 키

Merchant ID · Public/Private Key

Control Panel의 Settings > API Keys에서 Production 환경의 Merchant ID·Public Key·Private Key를 확인합니다.

참고: 반드시 Production 환경의 자격증명을 확인합니다. Sandbox 키와 혼동하지 않도록 주의합니다.

2. 발급 신청 절차

가맹 계정 승인 후 아래 순서대로 자격증명을 확인하고 GenieGo에 등록합니다.

1

가맹 계정 승인

Braintree(PayPal) 가맹 계정 승인을 먼저 완료합니다. 승인이 완료되어야 다음 단계로 진행할 수 있습니다.

2

Control Panel 로그인

Braintree Control Panel에 로그인합니다.

braintreegateway.com
3

API Keys 이동

[Settings] > [API Keys] 메뉴로 이동합니다.

Settings → API Keys
4

자격증명 확인

Production 환경의 Merchant ID·Public Key·Private Key를 확인합니다.

Settings > API Keys에서 Production의 Merchant ID·Public Key·Private Key가 보이면 정상입니다.

3. GenieGo에 등록할 정보

필드설명
merchant_idProduction Merchant ID
public_keyProduction Public Key
private_keyProduction Private Key(노출 주의)
GenieGo 연동 허브 [등록]에 merchant_id·public_key·private_key를 입력해 저장하면 자격증명은 AES-256-GCM으로 암호화 저장되고 조회 시 마스킹됩니다.

4. 자주 발생하는 문제

문제원인해결
API Keys 메뉴에 자격증명이 없음가맹 계정 승인 미완료Braintree(PayPal) 가맹 계정 승인을 먼저 완료합니다.
인증 실패Sandbox 키를 사용 또는 Private Key 오기재Production 환경의 Merchant ID·Public Key·Private Key를 다시 확인합니다.

최종 체크리스트

  • Braintree(PayPal) 가맹 계정 승인이 완료되었다.
  • Control Panel의 Settings > API Keys에서 Production 자격증명을 확인했다.
  • Merchant ID·Public Key·Private Key를 확인했다.
  • GenieGo [등록]에 merchant_id·public_key·private_key를 입력해 저장했다.

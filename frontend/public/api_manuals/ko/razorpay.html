Razorpay
API Key 발급 매뉴얼

Razorpay Dashboard에서 Live 모드 API 키(Key ID·Key Secret)를 생성·확인하고 GenieGo 연동 허브에 등록하는 과정을 초보자도 그대로 따라 할 수 있도록 정리한 매뉴얼입니다.

소요 시간약 5~10분
필요 계정Razorpay 계정(인도 중심)
발급 키Key ID(rzp_live_) · Key Secret
결과물Live API 자격증명

1. 시작하기 전에

Razorpay API 키는 Dashboard의 Live 모드에서 생성합니다.

필요

Razorpay 계정

Razorpay 계정이 필요합니다(인도 중심). 계정으로 Dashboard에 로그인할 수 있어야 합니다.

발급 키

Key ID · Key Secret

Dashboard의 Settings > API Keys에서 Live 모드의 Key ID와 Key Secret을 생성·확인합니다.

참고: 반드시 Live 모드에서 키를 생성합니다. Key ID는 rzp_live_ 로 시작합니다.

2. 발급 단계

아래 순서대로 발급하고 GenieGo에 등록합니다.

1

Dashboard 접속

Razorpay Dashboard에 로그인합니다.

dashboard.razorpay.com
2

API Keys 이동

[Settings] > [API Keys]로 이동합니다.

Settings → API Keys
3

키 생성·확인

Live 모드에서 Key ID와 Key Secret을 생성·확인합니다.

Settings > API Keys에서 Live 모드의 Key ID(rzp_live_)와 Key Secret이 보이면 정상 생성된 것입니다.

3. GenieGo에 등록할 정보

필드설명
key_idLive 모드 Key ID(rzp_live_로 시작)
key_secretLive 모드 Key Secret(노출 주의)
GenieGo 연동 허브 [등록]에 key_id·key_secret을 입력해 저장하면 자격증명은 AES-256-GCM으로 암호화 저장되고 조회 시 마스킹됩니다.

4. 자주 발생하는 문제

문제원인해결
인증 실패Test 모드 키를 사용Live 모드에서 생성한 Key ID(rzp_live_)·Key Secret인지 확인합니다.
Key Secret을 다시 볼 수 없음Key Secret은 생성 시 1회만 표시생성 시 Key Secret을 안전하게 보관하고, 분실 시 새로 생성합니다.

최종 체크리스트

  • Razorpay 계정으로 dashboard.razorpay.com에 로그인했다.
  • Settings > API Keys에서 Live 모드 키를 생성했다.
  • Key ID(rzp_live_)·Key Secret을 확인했다.
  • GenieGo [등록]에 key_id·key_secret을 입력해 저장했다.

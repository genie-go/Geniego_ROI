EMS (우체국 국제특송)
API 발급 매뉴얼

EMS (우체국 국제특송) 자격증명(API 키/토큰)을 발급해 GenieGo 연동허브에 등록하는 과정을, 초보자도 그대로 따라 할 수 있도록 단계별로 정리했습니다.

예상 소요약 5~15분
필요 계정EMS (우체국 국제특송) 계정
단계 수4단계
결과물API 자격증명

발급 절차 (따라하기)

아래 순서를 그대로 따라 하면 자격증명을 발급받아 등록할 수 있습니다.

1

준비물: 우체국 국제특송(EMS) 계약(고객) 코드가 필요합니다(계약형).

2

인터넷우체국/계약창구에서 EMS 계약을 진행합니다.

3

계약 완료 후 고객(계약) 코드와 우체국 API 키를 발급받습니다.

4

GenieGo [등록]에 api_key, cust_code 를 입력해 저장합니다.

최종 체크리스트

  • 준비물: 우체국 국제특송(EMS) 계약(고객) 코드가 필요합니다(계약형).
  • 인터넷우체국/계약창구에서 EMS 계약을 진행합니다.
  • 계약 완료 후 고객(계약) 코드와 우체국 API 키를 발급받습니다.
  • GenieGo [등록]에 api_key, cust_code 를 입력해 저장합니다.
발급한 키는 GenieGo 연동허브 [등록]에 입력하면 AES-256-GCM으로 암호화 저장되며, 조회 시 마스킹됩니다.
발급한 키/시크릿은 외부에 노출되지 않도록 안전하게 보관하세요. 노출이 의심되면 즉시 재발급(회전)하세요.

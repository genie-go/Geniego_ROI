PayPal
API Key 발급 매뉴얼

PayPal 개발자 포털에서 앱(App)을 생성해 Client ID·Secret을 발급하고 GenieGo 연동 허브에 등록하는 과정을 초보자도 그대로 따라 할 수 있도록 정리한 매뉴얼입니다.

소요 시간약 5~10분
필요 계정PayPal 비즈니스 계정
발급 키Client ID · Secret
결과물API 자격증명

1. 시작하기 전에

PayPal API 자격증명은 비즈니스 계정으로 개발자 포털에서 앱을 만들어 발급합니다.

필요

PayPal 비즈니스 계정

developer.paypal.com 에 로그인할 수 있는 PayPal 비즈니스 계정이 있어야 합니다.

발급 키

Client ID · Secret

Live 앱을 생성하면 Client ID와 Secret이 발급됩니다.

참고: 실제 연동에는 Sandbox가 아닌 [Live] 탭에서 앱을 생성합니다.

2. 발급 단계

아래 순서대로 발급하고 GenieGo에 등록합니다.

1

개발자 포털 로그인

PayPal 비즈니스 계정으로 로그인한 뒤 [Apps & Credentials]로 이동합니다.

developer.paypal.com → Apps & Credentials
2

Live 앱 생성

[Live] 탭에서 [Create App]을 눌러 앱을 만듭니다.

Live → Create App
3

자격증명 확인

발급된 Client ID와 Secret을 확인합니다.

Client ID와 Secret이 표시되면 정상적으로 발급된 것입니다.

3. GenieGo에 등록할 정보

필드설명
client_id발급된 Client ID
client_secret발급된 Secret(노출 주의)
GenieGo 연동 허브 [등록]에 client_id·client_secret을 입력해 저장하면 자격증명은 AES-256-GCM으로 암호화 저장되고 조회 시 마스킹됩니다.

4. 자주 발생하는 문제

문제원인해결
인증 실패Sandbox 앱의 자격증명을 사용함[Live] 탭에서 생성한 앱의 Client ID·Secret을 사용합니다.
앱을 만들 수 없음일반(개인) 계정으로 로그인함PayPal 비즈니스 계정으로 로그인합니다.

최종 체크리스트

  • PayPal 비즈니스 계정으로 developer.paypal.com 에 로그인했다.
  • [Apps & Credentials]에서 [Live] 탭으로 이동했다.
  • [Create App]으로 앱을 생성하고 Client ID·Secret을 확인했다.
  • GenieGo [등록]에 client_id·client_secret을 입력해 저장했다.

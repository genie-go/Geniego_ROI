Walmart Marketplace
API Key 발급 매뉴얼

Walmart 마켓플레이스 셀러센터의 Developer Portal에서 client_id와 Client Secret을 생성해 GenieGo 연동 허브에 등록하는 과정을 초보자도 그대로 따라 할 수 있도록 정리한 매뉴얼입니다.

소요 시간약 5~15분
필요 계정Walmart 마켓플레이스 셀러 계정
발급 키Client ID · Client Secret
결과물API 자격증명

1. 시작하기 전에

Walmart API 자격증명은 마켓플레이스 셀러센터의 Developer Portal에서 발급합니다.

필요

Walmart 마켓플레이스 셀러 계정

발급을 진행하려면 Walmart 마켓플레이스 셀러 계정이 있어야 합니다. 이 계정으로 셀러센터의 Developer Portal에 접속합니다.

발급 키

Client ID · Client Secret

Developer Portal에서 API Key(client_id)와 Client Secret을 생성합니다.

2. 발급 단계

아래 순서대로 발급하고 GenieGo에 등록합니다.

1

Developer Portal 접속

Walmart 셀러센터 > Developer Portal에 접속합니다.

https://developer.walmart.com
2

자격증명 생성

API Key(client_id)와 Client Secret을 생성합니다.

3

값 확인

발급된 client_id와 client_secret을 확인합니다.

client_id와 Client Secret이 생성되면 정상 발급된 것입니다.

3. GenieGo에 등록할 정보

필드설명
client_id발급된 API Key(Client ID)
client_secret발급된 Client Secret(노출 주의)
GenieGo 연동 허브 [등록]에 client_id와 client_secret을 입력해 저장하면 자격증명은 AES-256-GCM으로 암호화 저장되고 조회 시 마스킹됩니다.

4. 자주 발생하는 문제

문제원인해결
Developer Portal에 접속할 수 없음Walmart 마켓플레이스 셀러 계정이 아님Walmart 마켓플레이스 셀러 계정으로 셀러센터 > Developer Portal에 접속합니다.
인증 실패client_id 또는 client_secret 오기재Developer Portal에서 발급한 client_id와 client_secret을 다시 확인합니다.

최종 체크리스트

  • Walmart 마켓플레이스 셀러 계정으로 셀러센터 > Developer Portal에 접속했다.
  • API Key(client_id)와 Client Secret을 생성했다.
  • client_id와 client_secret 값을 확인했다.
  • GenieGo [등록]에 client_id와 client_secret을 입력해 저장했다.

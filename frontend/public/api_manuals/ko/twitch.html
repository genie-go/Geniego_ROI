1. Twitch 계정 구조 이해

Twitch는 Instagram처럼 비즈니스 계정/크리에이터 계정을 별도로 선택하는 구조가 아닙니다.

기업 공식 채널

브랜드 운영용

회사명 또는 브랜드명으로 일반 Twitch 계정을 만든 뒤 공식 채널처럼 운영합니다. 제품 소개, 라이브커머스, 이벤트 방송에 적합합니다.

스트리머 성장 단계

Affiliate / Partner

방송 실적, 팔로워, 시청 시간 조건을 충족하면 Affiliate 또는 Partner 프로그램에 참여할 수 있습니다.

핵심: 기업도 먼저 일반 Twitch 계정을 만들고, 채널 브랜딩과 2단계 인증, OBS 연결, Developer Console 앱 등록을 진행하면 됩니다.

2. 준비물

  • 기업용 이메일 주소
  • 브랜드명 또는 회사명
  • Twitch 사용자 이름 후보
  • 회사 로고와 채널 배너 이미지
  • 회사 소개 문구와 웹사이트 URL
  • 방송 송출용 PC 또는 노트북
  • OBS Studio 또는 스트리밍 소프트웨어
  • API 연동용 OAuth Redirect URL

3. Twitch 계정 생성

1

Twitch 접속

브라우저에서 Twitch 공식 사이트에 접속합니다.

https://www.twitch.tv
2

회원가입 클릭

우측 상단의 Sign Up 또는 회원가입 버튼을 클릭합니다.

Sign Up
3

기업 정보로 가입

회사 이메일, 사용자 이름, 비밀번호, 생년월일을 입력합니다.

Email / Username / Password
4

이메일 인증

Twitch에서 보낸 인증 메일을 열고 Verify Email을 완료합니다.

Verify Email

4. 브랜드 계정명 정하기

Twitch 사용자명은 채널 주소가 되므로 짧고 기억하기 쉬운 이름이 좋습니다.

목적추천 이름 예시설명
기업 공식 채널Geniego / OCLKorea회사명과 일치해 신뢰도가 높습니다.
글로벌 방송GeniegoGlobal / OCLGlobal해외 시청자를 고려할 때 적합합니다.

기술/플랫폼GeniegoTech / OCLTechIT, SaaS, 플랫폼 소개 방송에 적합합니다.

5. 채널 프로필 설정

1

설정 메뉴 이동

우측 상단 프로필 아이콘을 클릭한 뒤 Settings로 이동합니다.

Profile → Settings
2

프로필 이미지 등록

회사 로고 또는 브랜드 아이콘을 프로필 이미지로 등록합니다.

Profile Picture
3

채널 배너 등록

브랜드 컬러와 서비스 메시지가 담긴 배너 이미지를 등록합니다.

Profile Banner
4

소개 문구 입력

회사 소개, 서비스 설명, 문의 이메일, 웹사이트를 입력합니다.

Bio / Links

6. 2단계 인증 설정

Twitch에서 방송 송출과 보안 설정을 안정적으로 사용하려면 2FA 설정이 매우 중요합니다.

1

보안 메뉴 이동

설정에서 Security and Privacy 메뉴를 엽니다.

Settings → Security and Privacy
2

2FA 활성화

Two-Factor Authentication을 활성화합니다.

Set Up Two-Factor Authentication
3

휴대폰 또는 인증 앱 연결

문자 인증 또는 Google Authenticator, Authy 같은 인증 앱을 연결합니다.

Authenticator App
주의: 기업 계정은 여러 사람이 사용할 수 있으므로 관리자 권한과 복구 이메일을 반드시 관리하세요.

7. 스트림 키 확인 및 OBS 연결

1

크리에이터 대시보드 열기

프로필 메뉴에서 Creator Dashboard로 이동합니다.

Creator Dashboard
2

스트림 메뉴 이동

설정 메뉴에서 Stream 항목을 선택합니다.

Settings → Stream
3

Primary Stream Key 확인

기본 스트림 키를 복사합니다. 이 키는 비밀번호처럼 보관해야 합니다.

Primary Stream Key
4

OBS에 연결

OBS Studio에서 서비스는 Twitch, 스트림 키는 복사한 값을 입력합니다.

OBS → Settings → Stream → Twitch
스트림 키가 노출되면 다른 사람이 내 채널로 방송할 수 있습니다. 절대 공개하지 마세요.

11. Client ID / Client Secret 확인

1

등록한 앱 선택

Developer Console의 Applications 목록에서 방금 만든 앱을 클릭합니다.

Applications → 앱 선택
2

Client ID 복사

화면에 표시된 Client ID를 복사해 서버 설정에 저장합니다.

Client ID
3

Client Secret 생성

New Secret 버튼을 눌러 Client Secret을 생성하고 안전하게 저장합니다.

New Secret
Client Secret은 한 번만 확인 가능한 경우가 많습니다. 생성 즉시 안전한 곳에 저장하세요.

12. Twitch API 인증 구조

Twitch Channel │ ▼ Developer Console │ ▼ Application │ ▼ Client ID / Client Secret │ ▼ OAuth Token │ ▼ Twitch Helix API │ ▼ 채널 정보, 방송 상태, 채팅봇, 시청자 분석

13. OAuth 토큰 발급 흐름

1) Twitch OAuth 인증 URL 생성
2) 사용자가 Twitch 계정으로 로그인
3) Redirect URL로 authorization code 수신
4) authorization code + Client Secret으로 Access Token 요청
5) Access Token으로 Twitch API 호출
6) 필요 시 Refresh Token으로 갱신
단순 공개 데이터 조회는 App Access Token으로 가능하고, 사용자 권한이 필요한 작업은 User Access Token이 필요합니다.

15. 최종 체크리스트

  • 기업용 이메일로 Twitch 계정을 생성했다.
  • 브랜드명과 일치하는 사용자 이름을 설정했다.
  • 이메일 인증을 완료했다.
  • 프로필 이미지, 배너, 소개 문구, 링크를 설정했다.
  • 2단계 인증을 활성화했다.
  • Creator Dashboard에서 Stream Key를 확인했다.
  • OBS Studio와 Twitch 채널을 연결했다.
  • Twitch Developer Console에서 앱을 등록했다.
  • Client ID와 Client Secret을 발급받았다.
  • OAuth Redirect URL과 토큰 발급 흐름을 준비했다.

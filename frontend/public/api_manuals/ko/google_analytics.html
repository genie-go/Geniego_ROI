Google Analytics 4
API Key 발급 매뉴얼

Google Analytics 4 데이터 스트림에서 측정 ID(G-)와 Measurement Protocol API Secret을 발급해 GenieGo 연동 허브에 등록하는 과정을 초보자도 그대로 따라 할 수 있도록 정리한 매뉴얼입니다.

소요 시간약 5~10분
필요 계정GA4 속성 관리 권한
발급 키측정 ID(G-) · API Secret
결과물Measurement Protocol 자격증명

1. 시작하기 전에

GA4 측정 ID와 API Secret은 속성 관리 권한이 있는 계정에서만 발급할 수 있습니다.

필요

GA4 속성 관리 권한

Google Analytics 4 속성을 관리할 수 있는 권한이 있어야 합니다. 보기 권한만으로는 API Secret을 생성할 수 없습니다.

발급 키

측정 ID(G-) · API Secret

웹 스트림의 측정 ID(G-로 시작)와 Measurement Protocol API Secret을 함께 확인합니다.

2. 발급 단계

아래 순서대로 발급하고 GenieGo에 등록합니다.

1

데이터 스트림 이동

Google Analytics에서 [관리] > [데이터 스트림]으로 이동합니다.

analytics.google.com → 관리 → 데이터 스트림
2

측정 ID 확인

웹 스트림을 선택해 측정 ID(G-로 시작)를 확인합니다.

3

API Secret 생성

같은 화면에서 [Measurement Protocol API secret]을 생성합니다.

측정 ID(G-)와 API Secret이 모두 확인되면 정상 발급된 것입니다.

3. GenieGo에 등록할 정보

필드설명
measurement_id측정 ID(G-로 시작)
api_secret생성된 Measurement Protocol API Secret(노출 주의)
GenieGo 연동 허브 [등록]에 입력해 저장하면 자격증명은 AES-256-GCM으로 암호화 저장되고 조회 시 마스킹됩니다.

4. 자주 발생하는 문제

문제원인해결
API Secret을 생성할 수 없음속성 관리 권한 부족GA4 속성 관리 권한이 있는 계정으로 로그인합니다.
데이터가 수집되지 않음측정 ID 또는 API Secret 오기재웹 스트림의 측정 ID(G-)와 API Secret을 다시 확인합니다.

최종 체크리스트

  • GA4 속성 관리 권한이 있는 계정으로 로그인했다.
  • [관리 > 데이터 스트림]에서 웹 스트림의 측정 ID(G-)를 확인했다.
  • Measurement Protocol API Secret을 생성했다.
  • GenieGo [등록]에 measurement_id·api_secret을 입력해 저장했다.

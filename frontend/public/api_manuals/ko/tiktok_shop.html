TikTok Shop
API Key 발급 매뉴얼

TikTok Shop Partner Center에서 앱(App Key·App Secret·Service ID)을 생성하고 샵 인가로 access_token을 발급해 GenieGo 연동 허브에 등록하는 과정을 초보자도 그대로 따라 할 수 있도록 정리한 매뉴얼입니다.

소요 시간약 10~15분
필요 계정TikTok Shop 판매자 계정
발급 키App Key · App Secret · Access Token
결과물API 자격증명

1. 시작하기 전에

Partner Center에서 앱을 만들고 샵을 인가해야 토큰이 발급됩니다.

필요

TikTok Shop 판매자 계정

TikTok Shop 판매자 계정이 있어야 하며, partner.tiktokshop.com(Partner Center)에 개발자로 등록해야 합니다.

발급 키

App Key · App Secret · Access Token

앱 생성으로 App Key·App Secret·Service ID를 확인하고, 샵 인가를 거쳐 access_token을 발급받습니다.

참고: 운영 전환 시 앱 심사가 필요합니다.

2. 발급 단계

아래 순서대로 발급하고 GenieGo에 등록합니다.

1

Partner Center 개발자 등록

partner.tiktokshop.com(Partner Center)에 개발자로 등록합니다.

https://partner.tiktokshop.com
2

앱 생성

[App & Service] > [Create App]으로 앱을 만들고 App Key, App Secret, Service ID를 확인합니다.

App & Service → Create App
3

API 권한 활성화

앱 설정에서 필요한 API 권한(Shop Authorized Information, Product, Order)을 활성화합니다.

4

샵 인가 → access_token 발급

[Get shop authorization]에서 지역(Region)을 선택해 판매자 샵을 인가하면 access_token이 발급됩니다.

Get shop authorization
5

키 확인

App Key·App Secret과 발급된 access_token을 확인합니다. 필요 시 Shop Cipher도 함께 확인합니다.

샵 인가 후 access_token이 발급되면 정상 처리된 것입니다.

3. GenieGo에 등록할 정보

필드설명
app_key발급된 App Key
app_secret발급된 App Secret(노출 주의)
access_token샵 인가로 발급된 액세스 토큰
shop_cipherShop Cipher(선택)
GenieGo 연동 허브 [등록]에 app_key·app_secret·access_token을 입력해 저장하면 자격증명은 AES-256-GCM으로 암호화 저장되고 조회 시 마스킹됩니다.

4. 자주 발생하는 문제

문제원인해결
access_token이 발급되지 않음샵 인가(Get shop authorization) 미완료 또는 지역(Region) 미선택[Get shop authorization]에서 지역을 선택해 판매자 샵을 인가합니다.
권한 오류필요한 API 권한 미활성화앱 설정에서 Shop Authorized Information·Product·Order 권한을 활성화합니다.
운영 호출 거부운영 전환 심사 미통과운영 전환에 필요한 앱 심사를 완료합니다.

최종 체크리스트

  • partner.tiktokshop.com에 개발자로 등록했다.
  • App & Service > Create App으로 App Key·App Secret·Service ID를 확인했다.
  • 필요한 API 권한(Shop/Product/Order)을 활성화했다.
  • 샵 인가(Get shop authorization)로 access_token을 발급했다.
  • GenieGo [등록]에 app_key·app_secret·access_token을 입력해 저장했다.

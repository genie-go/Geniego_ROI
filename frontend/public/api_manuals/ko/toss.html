토스페이먼츠
API Key 발급 매뉴얼

토스페이먼츠 상점관리자의 개발자센터에서 클라이언트 키와 시크릿 키를 확인하고 GenieGo 연동 허브에 등록하는 과정을 정리한 매뉴얼입니다.

소요 시간약 5~10분
필요 계정토스페이먼츠 가맹점 계약·상점관리자 계정
발급 키클라이언트 키 · 시크릿 키
결과물결제 연동 자격증명

1. 시작하기 전에

토스페이먼츠는 가맹점 계약 후 상점관리자에서 키를 확인할 수 있습니다.

필요

가맹점 계약·상점관리자 계정

토스페이먼츠 가맹점 계약이 완료되어 있어야 하며, 상점관리자에 로그인할 수 있어야 합니다.

발급 키

클라이언트 키 · 시크릿 키

개발자센터에서 클라이언트 키와 시크릿 키를 세트로 확인해 함께 사용합니다.

참고: 개발자센터에서 테스트와 라이브를 구분하므로, 실제 연동에는 라이브 키를 사용하는지 확인합니다.

2. 발급 단계

아래 순서대로 확인하고 GenieGo에 등록합니다.

1

상점관리자 로그인

토스페이먼츠 상점관리자에 로그인합니다.

https://dashboard.tosspayments.com
2

개발자센터 이동

좌측 하단 [개발자센터]로 이동합니다.

개발자센터
3

키 확인

테스트/라이브를 선택해 클라이언트 키와 시크릿 키를 확인합니다. 두 키는 세트로 사용합니다.

클라이언트 키와 시크릿 키가 모두 확인되면 등록 준비가 완료된 것입니다.

3. GenieGo에 등록할 정보

필드설명
client_key개발자센터에서 확인한 클라이언트 키
secret_key개발자센터에서 확인한 시크릿 키(노출 주의)
GenieGo 연동 허브 [등록]에 입력해 저장하면 자격증명은 AES-256-GCM으로 암호화 저장되고 조회 시 마스킹됩니다.

4. 자주 발생하는 문제

문제원인해결
인증 오류 발생테스트 키와 라이브 키를 혼동개발자센터에서 테스트/라이브 구분을 확인하고 환경에 맞는 키를 사용합니다.
개발자센터 메뉴가 없음가맹점 계약 미완료가맹점 계약을 완료한 상점관리자 계정으로 로그인합니다.

최종 체크리스트

  • 토스페이먼츠 가맹점 계약을 완료하고 상점관리자에 로그인했다.
  • 좌측 하단 [개발자센터]로 이동했다.
  • 테스트/라이브를 구분해 클라이언트 키·시크릿 키를 확인했다.
  • GenieGo [등록]에 client_key·secret_key를 입력해 저장했다.

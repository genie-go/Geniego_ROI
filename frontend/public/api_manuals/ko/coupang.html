API Key 발급 매뉴얼

쿠팡 WING 판매자 콘솔에서 Open API 인증키(Access Key·Secret Key)를 발급하고 GenieGo 연동 허브에 등록하는 과정을 초보자도 그대로 따라 할 수 있도록 정리한 매뉴얼입니다.

소요 시간약 5~10분
필요 계정쿠팡 WING 판매자 계정
발급 키Access Key · Secret Key
결과물Open API 자격증명

1. 시작하기 전에

쿠팡 Open API는 판매자 계정에서만 발급할 수 있습니다.

필요

WING 판매자 계정

사업자 인증이 완료된 쿠팡 WING 판매자 계정이 있어야 합니다. 일반 구매 회원 계정으로는 발급할 수 없습니다.

발급 키

Access Key · Secret Key

발급 완료 화면에서 Access Key와 Secret Key를 확인하고, 판매자(Vendor) ID도 함께 확인합니다.

참고: 발급 시 회사명·홈페이지/서비스 URL(없으면 wing.coupang.com)·접근 IP(최대 10개)를 입력해야 합니다.

2. Open API Key 발급 단계

아래 순서대로 발급하고 GenieGo에 등록합니다.

1

WING 로그인

판매자 계정으로 로그인합니다.

wing.coupang.com
2

Open API 메뉴 이동

상단/설정 영역의 [Open API] 또는 [API 인증키 관리] 메뉴로 이동합니다.

Open API → API 인증키 관리
3

Open API Key 발급

[Open API Key 발급]을 누르고 회사명·홈페이지 URL·접근 IP(최대 10개)를 입력합니다.

Open API Key 발급
4

키 확인

발급 완료 화면에서 Access Key·Secret Key와 판매자(Vendor) ID를 확인합니다.

발급 완료 화면에 Access Key·Secret Key가 보이면 정상 발급된 것입니다.

3. GenieGo에 등록할 정보

필드설명
access_key발급된 Access Key
secret_key발급된 Secret Key(노출 주의)
vendor_id판매자(Vendor) ID
GenieGo 연동 허브 [등록]에 입력해 저장하면 자격증명은 AES-256-GCM으로 암호화 저장되고 조회 시 마스킹됩니다. 등록 즉시 주문 동기화가 시작됩니다.

4. 자주 발생하는 문제

문제원인해결
발급 메뉴가 없음일반 회원 계정 또는 사업자 인증 미완료사업자 인증을 완료한 WING 판매자 계정으로 로그인합니다.
호출 시 401 오류접근 IP 미등록 또는 Secret Key 오기재발급 시 입력한 접근 IP와 Secret Key를 다시 확인합니다.

최종 체크리스트

  • 사업자 인증이 완료된 WING 판매자 계정으로 로그인했다.
  • Open API Key를 발급하고 접근 IP를 등록했다.
  • Access Key·Secret Key·Vendor ID를 확인했다.
  • GenieGo [등록]에 access_key·secret_key·vendor_id를 입력해 저장했다.

YouTube API Key
발급 매뉴얼

Google Cloud Console에서 YouTube Data API v3를 활성화하고 API Key를 만드는 과정을 초보자도 그대로 따라 할 수 있도록 정리한 실전 매뉴얼입니다.

소요 시간약 5~10분
필요 계정Google 계정
사용 APIYouTube Data API v3
결과물AIzaSy... 형식의 API Key

1. API Key와 OAuth의 차이

처음 시작할 때 가장 많이 헷갈리는 부분입니다.

API Key

공개 데이터 조회용

영상 검색, 채널 정보 조회, 재생목록 조회, 댓글 조회 등 공개 YouTube 데이터를 가져올 때 사용합니다.

OAuth 2.0

내 계정 권한이 필요한 작업용

영상 업로드, 내 채널 관리, 비공개 데이터 접근 등 사용자의 승인이 필요한 작업에 사용합니다.

중요: 영상 자동 업로드를 하려면 API Key만으로는 부족하며 OAuth 2.0 인증이 필요합니다.

2. YouTube Data API v3 활성화

API Key를 만들기 전에 먼저 사용할 API를 프로젝트에 켜야 합니다.

1

Google Cloud Console 접속

Google Cloud Console에 로그인합니다.

console.cloud.google.com
2

프로젝트 선택 또는 생성

상단 프로젝트 선택 영역에서 기존 프로젝트를 선택하거나 새 프로젝트를 만듭니다.

예: Geniego ROI
3

API 라이브러리 이동

왼쪽 메뉴에서 API 및 서비스 → 라이브러리로 이동합니다.

API 및 서비스 → 라이브러리
4

YouTube Data API v3 검색 후 사용

검색창에 YouTube Data API v3를 입력하고 해당 API를 선택한 뒤 사용 버튼을 클릭합니다.

YouTube Data API v3 → 사용
화면에 상태: 사용 설정됨 이 보이면 정상적으로 활성화된 것입니다.

3. API Key 생성하기

YouTube Data API v3가 활성화되면 사용자 인증 정보 화면에서 API Key를 만들 수 있습니다.

1

사용자 인증 정보 메뉴 이동

왼쪽 메뉴에서 API 및 서비스 → 사용자 인증 정보로 이동합니다.

API 및 서비스 → 사용자 인증 정보
2

사용자 인증 정보 만들기 클릭

상단의 + 사용자 인증 정보 만들기 버튼을 누릅니다.

+ 사용자 인증 정보 만들기
3

API 키 선택

나오는 메뉴에서 API 키를 선택합니다.

API 키
4

접근 가능한 API 선택

API 제한사항에서 YouTube Data API v3가 선택되어 있는지 확인합니다.

선택한 API: YouTube Data API v3
5

만들기 버튼 클릭

하단의 만들기 버튼을 누르면 API Key가 생성됩니다.

만들기
생성된 키는 보통 AIzaSy... 로 시작합니다. 이 문자열이 YouTube API Key입니다.

4. 생성 화면에서 어떤 옵션을 선택해야 하나요?

항목 초보자 권장 설정 설명
이름 YouTube API Key 나중에 알아보기 쉬운 이름으로 입력합니다.
API 제한사항 YouTube Data API v3 이 키가 YouTube API에만 사용되도록 제한합니다.
서비스 계정 인증 체크하지 않음 YouTube Data API Key 생성에는 일반적으로 필요하지 않습니다.
애플리케이션 제한사항 처음 테스트는 없음 정상 작동 확인 후 웹사이트 또는 IP 주소 제한을 설정하는 것을 권장합니다.
실제 서비스에 배포할 때는 반드시 키 제한을 설정하세요. 제한 없이 공개되면 다른 사람이 키를 사용할 수 있습니다.

5. API Key 사용 예시

생성한 키를 YOUR_API_KEY 부분에 넣어 테스트할 수 있습니다.

영상 검색 예시

https://www.googleapis.com/youtube/v3/search?part=snippet&q=korea&type=video&key=YOUR_API_KEY

채널 정보 조회 예시

https://www.googleapis.com/youtube/v3/channels?part=snippet&id=CHANNEL_ID&key=YOUR_API_KEY
브라우저 주소창에 위 주소를 입력했을 때 JSON 형태의 결과가 나오면 API Key가 정상 작동하는 것입니다.

6. 자주 발생하는 문제 해결

문제 원인 해결 방법
선택된 API가 없습니다 YouTube Data API v3가 아직 활성화되지 않았거나 API 제한에서 선택하지 않음 API 라이브러리에서 YouTube Data API v3를 사용 설정 후 다시 선택합니다.
표시할 API 키가 없습니다 아직 API Key를 만들지 않음 사용자 인증 정보 만들기 → API 키를 클릭합니다.
403 오류 API 제한, 할당량 초과, 권한 문제 API 제한사항, 할당량, 활성화 상태를 확인합니다.
업로드가 안 됨 API Key만 사용함 영상 업로드는 OAuth 2.0 인증을 사용해야 합니다.

7. 최종 체크리스트

  • Google Cloud 프로젝트를 선택했다.
  • YouTube Data API v3를 사용 설정했다.
  • 사용자 인증 정보에서 API Key를 생성했다.
  • 생성된 API Key가 AIzaSy로 시작하는지 확인했다.
  • API 제한사항을 YouTube Data API v3로 설정했다.
  • 실제 서비스 배포 전 웹사이트 또는 IP 제한을 설정한다.

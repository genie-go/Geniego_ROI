eBay Seller Hub
API Key 발급 매뉴얼

eBay 개발자 프로그램에서 운영(Production) keyset(DevID·AppID·CertID)을 발급하고 OAuth 사용자 토큰을 생성해 GenieGo 연동 허브에 등록하는 과정을 초보자도 그대로 따라 할 수 있도록 정리한 매뉴얼입니다.

소요 시간약 10~20분
필요 계정eBay 계정
발급 키DevID · AppID · CertID + OAuth 토큰
결과물OAuth 액세스 토큰

1. 시작하기 전에

eBay keyset 발급에는 개발자 프로그램 가입이 필요합니다.

필요

eBay 계정

발급을 진행하려면 eBay 계정이 있어야 합니다. 이 계정으로 developer.ebay.com 개발자 프로그램에 가입합니다.

발급 키

DevID · AppID · CertID

운영(Production) keyset을 생성하면 DevID, AppID, CertID가 발급됩니다. 이후 OAuth 사용자 토큰까지 생성해야 GenieGo에 등록할 수 있습니다.

참고: 운영 keyset은 "Your Keyset is currently disabled" 안내가 보일 수 있으며, 이때 마켓플레이스 계정 삭제 알림 구독/거부 절차를 완료해야 활성화됩니다.

2. 발급 단계

아래 순서대로 발급하고 GenieGo에 등록합니다.

1

개발자 프로그램 가입

developer.ebay.com에 접속해 eBay 계정으로 개발자 프로그램에 가입합니다.

developer.ebay.com
2

운영 keyset 생성

[Application Keys] 페이지에서 Production 항목의 [Create a keyset]을 누릅니다.

https://developer.ebay.com/my/keys
3

keyset 활성화 절차

"Your Keyset is currently disabled" 안내가 보이면 링크를 눌러 마켓플레이스 계정 삭제 알림 구독/거부 절차를 완료합니다.

Application Keys → Production
4

키 확인 및 OAuth 토큰 생성

발급된 DevID, AppID, CertID를 확인하고 OAuth 사용자 토큰을 생성합니다.

DevID·AppID·CertID가 발급되고 OAuth 사용자 토큰까지 생성되면 정상 발급된 것입니다.

3. GenieGo에 등록할 정보

필드설명
OAuth 액세스 토큰생성한 OAuth 사용자 토큰(노출 주의)
GenieGo 연동 허브 [등록]에 OAuth 액세스 토큰을 입력해 저장하면 자격증명은 AES-256-GCM으로 암호화 저장되고 조회 시 마스킹됩니다.

4. 자주 발생하는 문제

문제원인해결
keyset이 비활성 상태마켓플레이스 계정 삭제 알림 구독/거부 절차 미완료"Your Keyset is currently disabled" 안내 링크를 눌러 해당 절차를 완료합니다.
OAuth 토큰을 발급할 수 없음운영 keyset(DevID·AppID·CertID) 미발급[Application Keys]에서 Production keyset을 먼저 생성한 뒤 OAuth 토큰을 생성합니다.

최종 체크리스트

  • eBay 계정으로 developer.ebay.com 개발자 프로그램에 가입했다.
  • Production [Create a keyset]으로 DevID·AppID·CertID를 발급했다.
  • keyset 비활성 시 마켓플레이스 계정 삭제 알림 구독/거부 절차를 완료했다.
  • OAuth 사용자 토큰을 생성하고 GenieGo [등록]에 OAuth 액세스 토큰을 입력해 저장했다.

API Key 발급 매뉴얼

지니고 당일배송 서비스를 계약하고 발급받은 당일배송 API 키·머천트 ID를 GenieGo 연동 허브에 등록하는 과정을 정리한 매뉴얼입니다. 이 채널은 자가 콘솔 발급이 아닌 운영팀과의 계약을 통해 자격증명을 수령합니다.

소요 시간계약 진행에 따라 상이
필요 계정지니고 당일배송 서비스 계약
발급 키당일배송 API 키 · 머천트 ID
결과물당일배송 API 자격증명

1. 시작하기 전에

지니고 당일배송은 운영팀과의 서비스 계약 후 자격증명이 발급되는 계약형 채널입니다.

필요

당일배송 서비스 계약

지니고 당일배송을 이용하려면 운영팀과 서비스 이용 계약이 선행되어야 합니다. 계약 없이는 자격증명을 발급받을 수 없습니다.

발급 키

API 키 · 머천트 ID · 서비스 권역

계약 완료 후 운영팀으로부터 머천트 ID·API 키와 함께 서비스 권역을 발급받습니다.

참고: 이 채널은 셀프 콘솔에서 직접 발급하는 방식이 아니라, 운영팀 계약을 통해 자격증명을 수령하는 방식입니다.

2. 발급 신청 절차

아래 순서대로 신청·수령한 뒤 GenieGo에 등록합니다.

1

당일배송 이용 신청

운영팀에 지니고 당일배송 이용을 신청하고 계약을 진행합니다.

2

계약 체결

운영팀과 당일배송 서비스 이용 계약을 체결합니다.

3

자격증명 수령

계약 완료 후 머천트 ID·API 키와 서비스 권역을 발급받습니다.

4

GenieGo 등록

발급받은 api_key, merchant_id, region 을 GenieGo [등록]에 입력해 저장합니다.

운영팀으로부터 머천트 ID·API 키·서비스 권역을 모두 수령했다면 정상적으로 발급된 것입니다.

3. GenieGo에 등록할 정보

필드설명
api_key발급받은 지니고 당일배송 API 키
merchant_id발급받은 머천트 ID
region서비스 권역(예: 수도권)
GenieGo 연동 허브 [등록]에 입력해 저장하면 자격증명은 AES-256-GCM으로 암호화 저장되고 조회 시 마스킹됩니다.

4. 자주 발생하는 문제

문제원인해결
자격증명을 받지 못함당일배송 서비스 계약 미완료운영팀에 당일배송 이용을 신청·계약한 뒤 자격증명을 발급받습니다.
등록 후 권역 처리 오류서비스 권역(region) 미입력 또는 오기재발급받은 서비스 권역을 다시 확인해 정확히 입력합니다.

최종 체크리스트

  • 운영팀에 지니고 당일배송 이용을 신청·계약했다.
  • 계약 완료 후 머천트 ID·API 키·서비스 권역을 발급받았다.
  • api_key·merchant_id·region 을 확인했다.
  • GenieGo [등록]에 api_key·merchant_id·region 을 입력해 저장했다.

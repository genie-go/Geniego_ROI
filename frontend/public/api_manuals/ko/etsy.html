Etsy
API Key 발급 매뉴얼

Etsy 개발자 포털에서 앱을 등록해 API Key(keystring)를 발급하고 OAuth 인가로 access token까지 받아 GenieGo 연동 허브에 등록하는 과정을 초보자도 그대로 따라 할 수 있도록 정리한 매뉴얼입니다.

소요 시간약 5~15분
필요 계정Etsy 셀러 계정
발급 키API Key(keystring) · Shop ID
결과물API 자격증명

1. 시작하기 전에

Etsy API Key는 셀러 계정에서 앱을 등록해 발급합니다.

필요

Etsy 셀러 계정

발급을 진행하려면 Etsy 셀러 계정이 있어야 합니다. 이 계정으로 Etsy 개발자 포털에 접속합니다.

발급 키

API Key(keystring)

앱 정보를 입력하면 API Key(keystring)가 발급됩니다. 이후 OAuth 인가를 거쳐 access token까지 받습니다.

참고: GenieGo 등록에는 발급한 api_key와 함께 상점 식별자인 shop_id가 필요합니다.

2. 발급 단계

아래 순서대로 발급하고 GenieGo에 등록합니다.

1

새 앱 만들기

etsy.com/developers에 접속해 [Create a New App]을 누릅니다.

etsy.com/developers
2

앱 정보 입력 및 키 발급

앱 정보를 입력하면 API Key(keystring)가 발급됩니다.

https://www.etsy.com/developers/your-apps
3

OAuth 인가

OAuth 인가를 거쳐 access token을 발급받습니다.

4

등록 정보 확인

발급된 api_key와 상점의 shop_id를 확인합니다.

API Key(keystring)가 발급되고 OAuth 인가로 access token까지 받으면 정상 발급된 것입니다.

3. GenieGo에 등록할 정보

필드설명
api_key발급된 API Key(keystring)
shop_id연동할 Etsy 상점의 Shop ID
GenieGo 연동 허브 [등록]에 api_key와 shop_id를 입력해 저장하면 자격증명은 AES-256-GCM으로 암호화 저장되고 조회 시 마스킹됩니다.

4. 자주 발생하는 문제

문제원인해결
앱을 만들 수 없음Etsy 셀러 계정이 아니거나 개발자 포털 미접속Etsy 셀러 계정으로 etsy.com/developers에 접속해 [Create a New App]을 진행합니다.
인증 실패OAuth 인가 미완료 또는 access token 미발급API Key(keystring) 발급 후 OAuth 인가를 거쳐 access token을 발급받습니다.

최종 체크리스트

  • Etsy 셀러 계정으로 etsy.com/developers에 접속했다.
  • [Create a New App]으로 API Key(keystring)를 발급했다.
  • OAuth 인가를 거쳐 access token을 발급받았다.
  • GenieGo [등록]에 api_key와 shop_id를 입력해 저장했다.

Auction (옥션)
API Key 발급 매뉴얼

ESM Trading API(etapi.gmarket.com)에서 옥션·G마켓 공통 API 키를 발급받아 GenieGo 연동 허브에 등록하는 과정을 정리한 매뉴얼입니다. 옥션과 G마켓은 ESM Plus로 통합 운영되므로 하나의 API 키로 두 채널을 함께 연동합니다.

소요 시간승인 대기 포함 수일 소요(승인제)
필요 계정옥션·G마켓 판매자 계정 + ESM Plus 마스터 ID
발급 키API 키 · 셀러 ID
결과물ESM Trading API 자격증명

1. 시작하기 전에

옥션·G마켓 모두에 판매자로 가입되어 있고 ESM Plus 마스터 ID가 있어야 합니다.

필요

판매자 계정 + ESM 마스터 ID

옥션과 G마켓 각각의 판매자 계정과 ESM Plus(esmplus.com)에서 생성한 마스터 ID가 모두 필요합니다.

발급 키

API 키 · 셀러 ID

ESM Trading API 승인 후 발급되는 API 키와 셀러 ID를 함께 확인합니다.

참고: ESM Trading API는 승인제로, 신청이 거절될 수 있습니다. 신청 시 옥션 판매자ID·G마켓 판매자ID·ESM 마스터ID 3가지를 제출해야 합니다.

2. 발급 단계

아래 순서대로 신청·발급하고 GenieGo에 등록합니다.

1

판매자 가입 및 ESM 마스터 ID 생성

옥션과 G마켓에 각각 판매자로 가입한 뒤, ESM Plus에 로그인해 마스터 ID를 생성합니다.

esmplus.com
2

ESM Trading API 사용 신청

ESM Trading API에 접속해 API 사용을 신청합니다. 승인제이므로 거절될 수 있습니다.

https://etapi.gmarket.com
3

식별 정보 제출

신청 시 옥션 판매자ID, G마켓 판매자ID, ESM 마스터 ID 3가지를 제출합니다.

4

키 발급

승인되면 API 키가 발급됩니다. 발급된 API 키와 셀러 ID를 확인합니다.

승인 완료 후 API 키가 발급되면 정상 처리된 것입니다. 승인 관련 문의는 etapihelp@gmail.com 으로 연락합니다.

3. GenieGo에 등록할 정보

필드설명
api_key발급된 API 키
seller_id셀러 ID
GenieGo 연동 허브 [등록]에 입력해 저장하면 자격증명은 AES-256-GCM으로 암호화 저장되고 조회 시 마스킹됩니다.

4. 자주 발생하는 문제

문제원인해결
API 사용 신청 거절ESM Trading API는 승인제제출 정보를 확인하고 etapihelp@gmail.com 으로 문의합니다.
신청 정보 불일치옥션·G마켓 판매자ID 또는 ESM 마스터ID 오기재옥션 판매자ID·G마켓 판매자ID·ESM 마스터ID 3가지를 다시 확인합니다.

최종 체크리스트

  • 옥션·G마켓 각각 판매자로 가입했다.
  • ESM Plus에서 마스터 ID를 생성했다.
  • etapi.gmarket.com에서 ESM Trading API 사용을 신청하고 승인받았다.
  • 발급된 api_key와 seller_id를 확인했다.
  • GenieGo [등록]에 api_key·seller_id를 입력해 저장했다.

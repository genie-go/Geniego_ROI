API Key 발급 매뉴얼

네이버 검색광고 관리도구에서 API 인증 정보(액세스라이선스·비밀키)와 CUSTOMER_ID를 발급해 GenieGo 연동 허브에 등록하는 과정을 초보자도 그대로 따라 할 수 있도록 정리한 매뉴얼입니다.

소요 시간약 5~10분
필요 계정네이버 검색광고 광고주 계정
발급 키액세스라이선스 · 비밀키 · CUSTOMER_ID
결과물검색광고 API 자격증명

1. 시작하기 전에

네이버 검색광고 API는 광고주 계정에서만 신청할 수 있습니다.

필요

검색광고 광고주 계정

네이버 검색광고 관리도구를 사용하는 광고주 계정이 있어야 합니다. 일반 네이버 회원 계정으로는 API 사용을 신청할 수 없습니다.

발급 키

액세스라이선스 · 비밀키 · CUSTOMER_ID

API 사용 신청 후 발급되는 CUSTOMER_ID, 액세스라이선스, 비밀키를 함께 확인합니다.

참고: 비밀키는 발급 시점에만 확인할 수 있으므로 발급 직후 안전하게 보관하시기 바랍니다.

2. API 키 발급 단계

아래 순서대로 발급하고 GenieGo에 등록합니다.

1

검색광고 로그인

네이버 검색광고 관리도구에 광고주 계정으로 로그인합니다.

searchad.naver.com
2

API 사용 관리 이동

상단 [도구] 메뉴에서 [API 사용 관리]로 이동합니다.

도구 → API 사용 관리
3

API 서비스 신청

[네이버 검색광고 API 서비스 신청] 버튼을 눌러 신청합니다.

네이버 검색광고 API 서비스 신청
4

발급 정보 확인

발급된 CUSTOMER_ID, 액세스라이선스, 비밀키를 확인합니다.

CUSTOMER_ID·액세스라이선스·비밀키가 모두 확인되면 정상 발급된 것입니다.

3. GenieGo에 등록할 정보

필드설명
api_key발급된 액세스라이선스
api_secret발급된 비밀키(노출 주의)
customer_id발급된 CUSTOMER_ID
GenieGo 연동 허브 [등록]에 입력해 저장하면 자격증명은 AES-256-GCM으로 암호화 저장되고 조회 시 마스킹됩니다.

4. 자주 발생하는 문제

문제원인해결
[API 사용 관리] 메뉴가 없음광고주 계정이 아니거나 권한 부족검색광고 관리도구를 사용하는 광고주 계정으로 로그인합니다.
인증 오류 발생액세스라이선스·비밀키 오기재발급 화면의 액세스라이선스·비밀키와 CUSTOMER_ID를 다시 확인합니다.

최종 체크리스트

  • 네이버 검색광고 광고주 계정으로 로그인했다.
  • [도구 > API 사용 관리]에서 API 서비스 신청을 완료했다.
  • CUSTOMER_ID·액세스라이선스·비밀키를 확인했다.
  • GenieGo [등록]에 api_key·api_secret·customer_id를 입력해 저장했다.

Instagram Live / Facebook Live
API 발급 매뉴얼

Meta for Developers에서 앱을 만들고 Instagram·Facebook 연동에 필요한 App ID, App Secret, Access Token을 준비하는 과정을 초보자도 따라 할 수 있도록 정리했습니다.

핵심 개념API Key가 아니라 Access Token
개발 사이트developers.facebook.com

Instagram전문가 계정/권한 필요

1. 먼저 알아야 할 핵심 차이

YouTube는 API Key를 발급받아 쓰는 방식이 많지만, Meta는 대부분 App ID, App Secret, Access Token 방식으로 동작합니다.

Facebook Live

Live Video API 사용

Facebook 페이지 등에 라이브 방송을 생성하거나 스트리밍 정보를 가져올 때 사용합니다.

Instagram Live

제한이 더 많음

Instagram API는 비즈니스/크리에이터 계정의 미디어, 댓글, 인사이트, 메시징 중심입니다. Live 송출 자동화는 공식 지원 범위를 확인해야 합니다.

중요: Meta는 “API Key 하나”를 복사해서 끝나는 구조가 아닙니다. 보통 앱 생성 → 제품 추가 → 권한 설정 → Access Token 발급 → 앱 검수 순서가 필요합니다.

2. 준비물

  • Meta/Facebook 계정
  • Meta for Developers 접속 권한
  • Facebook 페이지 또는 비즈니스 계정
  • Instagram 비즈니스 또는 크리에이터 계정
  • 실제 서비스용 개인정보처리방침 URL, 서비스 약관 URL

3. Meta Developer 앱 만들기

모든 Meta API 연동은 먼저 앱을 만드는 것에서 시작합니다.

1

Meta for Developers 접속

브라우저에서 Meta for Developers에 로그인합니다.

developers.facebook.com
2

내 앱으로 이동

상단 메뉴에서 My Apps 또는 내 앱 메뉴를 클릭합니다.

My Apps → Create App
3

앱 유형 선택

라이브커머스나 플랫폼 연동은 보통 비즈니스 목적에 가깝습니다.

Business 또는 Other 선택
4

앱 이름과 이메일 입력

앱 이름, 연락 이메일, 비즈니스 계정 연결 여부를 입력하고 앱을 생성합니다.

App Name / Contact Email
앱 생성 후 대시보드에서 App ID와 App Secret을 확인할 수 있습니다.

4. App ID / App Secret 확인 위치

1

앱 대시보드 열기

생성한 앱을 클릭하여 앱 대시보드로 들어갑니다.

My Apps → 생성한 앱 선택
2

기본 설정 메뉴 이동

왼쪽 메뉴에서 앱 설정 → 기본 설정으로 이동합니다.

App Settings → Basic
3

App ID와 App Secret 확인

App ID는 공개 식별자이고, App Secret은 비밀번호처럼 안전하게 보관해야 합니다.

App ID / App Secret
App Secret은 절대 프론트엔드 코드, HTML, GitHub 공개 저장소에 노출하면 안 됩니다.

5. Facebook Live API 설정 흐름

1

앱에 제품 추가

앱 대시보드에서 Facebook Login, Graph API 관련 설정을 준비합니다.

Add Product → Facebook Login
2

Facebook 페이지 연결

라이브 방송을 페이지에 송출하려면 페이지 관리자 권한과 Page Access Token이 필요합니다.

Page Access Token
3

필요 권한 확인

콘텐츠 게시, 페이지 관리, 라이브 비디오 생성 목적에 맞는 권한을 확인합니다.

Permissions / App Review
4

Graph API Explorer 테스트

토큰을 발급하고 권한을 추가한 뒤 API 호출을 테스트합니다.

Tools → Graph API Explorer

Facebook Live 생성 호출 예시

POST /{page-id}/live_videos
access_token={PAGE_ACCESS_TOKEN}
정상적으로 생성되면 방송 ID, 스트림 URL, 스트림 키 등 라이브 송출에 필요한 정보가 반환됩니다.

6. Instagram API 설정 흐름

1

Instagram 계정 유형 확인

Instagram 계정이 비즈니스 또는 크리에이터 계정인지 확인합니다.

Instagram 설정 → 계정 유형
2

Facebook 페이지와 Instagram 연결

Meta Business Suite에서 Facebook 페이지와 Instagram 계정을 연결합니다.

Meta Business Suite → 계정 연결
3

Instagram 제품 추가

앱 대시보드에서 Instagram 관련 제품을 추가하고 필요한 권한을 설정합니다.

Add Product → Instagram
4

Access Token 발급

Graph API Explorer 또는 OAuth 로그인 흐름을 통해 권한이 포함된 토큰을 발급합니다.

Access Token + Permissions
Instagram Live 주의: Instagram Live 송출 자동화는 일반적인 Instagram Graph API 기본 기능처럼 간단히 열려 있지 않습니다. 공식 문서의 현재 지원 범위와 앱 검수 가능 여부를 먼저 확인해야 합니다.

7. Access Token 종류

토큰 종류용도주의사항
User Access Token사용자 로그인 기반 API 호출권한 승인 범위에 따라 접근 가능 데이터가 달라집니다.
Page Access TokenFacebook 페이지 게시, 페이지 라이브 등페이지 관리자 권한과 관련 권한이 필요합니다.
Long-Lived Token짧은 만료 시간을 늘린 장기 토큰운영 서비스에서는 토큰 만료와 갱신 로직이 필요합니다.
App Access Token앱 단위 설정/검증 작업사용자나 페이지 데이터 접근용과는 다릅니다.

8. 실제 서비스 전 체크리스트

  • Meta Developer 앱을 생성했다.
  • App ID와 App Secret을 확인했다.
  • Facebook Login 또는 필요한 제품을 추가했다.
  • Facebook 페이지와 Instagram 전문가 계정을 연결했다.
  • Graph API Explorer에서 Access Token을 발급해 테스트했다.
  • 필요 권한을 정리하고 App Review 제출 자료를 준비했다.
  • App Secret과 Access Token을 서버 환경변수에 안전하게 저장했다.

9. 자주 발생하는 문제

문제원인해결 방법
API Key 메뉴가 보이지 않음Meta는 Google처럼 단일 API Key 중심이 아님App ID, App Secret, Access Token 구조로 진행합니다.
권한 오류Access Token에 필요한 권한이 없음Graph API Explorer에서 권한을 추가하고 다시 토큰을 발급합니다.
페이지 라이브 생성 실패Page Access Token 또는 관리자 권한 부족페이지 관리자 권한과 토큰 권한을 확인합니다.
Instagram Live 자동화 불가공식 API 지원 범위 제한현재 Meta 공식 문서와 앱 검수 가능 범위를 확인합니다.

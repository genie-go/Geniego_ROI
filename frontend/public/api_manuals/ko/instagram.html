Instagram Live
API 발급 매뉴얼

Meta for Developers에서 앱을 만들고 Instagram 비즈니스/크리에이터 계정 연동에 필요한 App ID, App Secret, Access Token과 IG 비즈니스 계정 ID를 준비하는 과정을 초보자도 따라 할 수 있도록 정리했습니다.

핵심 개념API Key가 아니라 Access Token
개발 사이트developers.facebook.com
사용 APIInstagram Graph API
필요 계정비즈니스/크리에이터 계정

1. 먼저 알아야 할 핵심

Instagram API는 비즈니스/크리에이터 계정의 미디어·댓글·인사이트·메시징 중심이며, Facebook 페이지와의 연결이 전제입니다.

Instagram Live 주의: Instagram Live 송출 자동화는 일반 Graph API 기본 기능처럼 단순히 열려 있지 않습니다. 공식 문서의 현재 지원 범위와 앱 검수(App Review) 가능 여부를 먼저 확인하세요. (인사이트·미디어·댓글 조회는 표준 지원)

2. 준비물

  • Meta/Facebook 계정 + Meta for Developers 접속 권한
  • Instagram 비즈니스 또는 크리에이터 계정
  • Instagram과 연결된 Facebook 페이지
  • 실제 서비스용 개인정보처리방침 URL · 서비스 약관 URL

3. Instagram을 비즈니스/크리에이터 계정으로 전환·연결

1

계정 유형 확인·전환

Instagram 앱 설정에서 계정을 비즈니스 또는 크리에이터로 전환합니다.

Instagram 설정 → 계정 유형
2

Facebook 페이지와 연결

Meta Business Suite에서 Facebook 페이지와 Instagram 계정을 연결합니다(Graph API 호출의 전제).

Meta Business Suite → 계정 연결

4. Meta Developer 앱 만들기

모든 Meta API 연동은 앱을 만드는 것에서 시작합니다.

1

Meta for Developers 접속

브라우저에서 Meta for Developers에 로그인합니다.

developers.facebook.com
2

내 앱 → 앱 만들기

상단 My Apps → Create App을 클릭합니다(Business 유형).

My Apps → Create App
3

App ID / App Secret 확인

앱 설정 → 기본 설정에서 App ID·App Secret을 확인합니다. (App Secret은 공개 노출 금지)

App Settings → Basic

5. Instagram 제품 추가 + Access Token 발급

1

Instagram 제품 추가

앱 대시보드에서 Instagram(그래프 API) 제품을 추가합니다.

Add Product → Instagram
2

권한 신청(App Review)

instagram_basic, instagram_manage_insights 등 목적에 맞는 권한을 신청합니다.

Permissions → App Review
3

Access Token 발급

Graph API Explorer 또는 OAuth 로그인 흐름으로 권한이 포함된 토큰을 발급합니다.

Tools → Graph API Explorer
4

IG 비즈니스 계정 ID 확인

연결된 페이지에서 instagram_business_account 필드로 IG 비즈니스 계정 ID(ig_user_id)를 조회합니다.

ig_user_id

IG 비즈니스 계정 ID 조회 예시

GET /{page-id}?fields=instagram_business_account
access_token={ACCESS_TOKEN}

6. GenieGo 연동허브에 등록

1

액세스 토큰 복사

발급한 Access Token을 복사합니다(장기 토큰 권장).

access_token
2

IG 비즈니스 계정 ID 확인

위에서 조회한 ig_user_id를 준비합니다.

ig_user_id
3

연동허브에 등록

GenieGo 연동허브 → Instagram Live → [등록]에 아래 값을 입력해 저장합니다.

access_token · ig_user_id
등록한 키는 AES-256-GCM으로 암호화 저장되며 조회 시 마스킹됩니다.

7. Access Token 종류

토큰 종류용도주의사항
User Access Token사용자 로그인 기반 API 호출권한 승인 범위에 따라 접근 데이터가 달라집니다.
Long-Lived Token만료 시간을 늘린 장기 토큰운영 시 만료·갱신 로직이 필요합니다.
App Access Token앱 단위 설정/검증 작업사용자/미디어 데이터 접근용과는 다릅니다.

8. 체크리스트

  • Instagram을 비즈니스/크리에이터 계정으로 전환했다.
  • Facebook 페이지와 Instagram 계정을 연결했다.
  • Meta Developer 앱을 생성하고 App ID·App Secret을 확인했다.
  • Instagram 제품을 추가하고 필요 권한(App Review)을 신청했다.
  • Access Token과 IG 비즈니스 계정 ID(ig_user_id)를 확인했다.
  • GenieGo [등록]에 access_token·ig_user_id를 입력해 저장했다.

9. 자주 발생하는 문제

문제원인해결 방법
API Key 메뉴가 없음Meta는 단일 API Key 구조가 아님App ID·App Secret·Access Token 구조로 진행합니다.
IG 계정 ID 조회 실패페이지-Instagram 연결 누락 또는 계정 유형 미전환비즈니스/크리에이터 전환 + 페이지 연결을 확인합니다.
권한 오류토큰에 instagram_basic 등 권한 없음Graph API Explorer에서 권한 추가 후 토큰 재발급.
Instagram Live 자동화 불가공식 API 지원 범위 제한Meta 공식 문서의 현재 지원 범위·앱 검수 가능 여부를 확인합니다.

Shopify
API Key 발급 매뉴얼

Shopify 관리자(Admin)에서 커스텀 앱을 만들어 Admin API 액세스 토큰을 발급하고, Shop 도메인과 함께 GenieGo 연동 허브에 등록해 Shopify 스토어를 연동하는 과정을 정리한 매뉴얼입니다.

소요 시간약 5~15분
필요 계정Shopify 스토어 관리자 권한
발급 키Shop 도메인 · Admin API 액세스 토큰
결과물Admin API 자격증명

1. 시작하기 전에

Shopify 커스텀 앱은 스토어 관리자 권한이 있어야 만들 수 있습니다.

필요

스토어 관리자 권한

Shopify 스토어의 관리자(Admin) 권한이 있어야 커스텀 앱을 만들고 토큰을 발급할 수 있습니다.

발급 키

Shop 도메인 · Admin API 토큰

xxx.myshopify.com 형식의 Shop 도메인과, 앱 설치 후 표시되는 Admin API 액세스 토큰을 확인합니다.

참고: Admin API 액세스 토큰은 설치 직후 한 번만 표시되므로, 표시되는 즉시 복사해 두어야 합니다.

2. 발급 단계

아래 순서대로 발급하고 GenieGo에 등록합니다.

1

앱 및 판매 채널로 이동

Shopify 관리자(Admin) > [설정] > [앱 및 판매 채널]로 이동합니다.

https://www.shopify.com/admin/settings/apps
2

커스텀 앱 만들기

[앱 개발] > [앱 만들기]를 눌러 커스텀 앱을 만듭니다.

앱 개발 → 앱 만들기
3

범위 선택 · 앱 설치

Admin API 범위(scope)를 선택하고 앱을 설치합니다.

4

액세스 토큰 복사

설치 후 표시되는 Admin API 액세스 토큰을 즉시 복사합니다(한 번만 표시됨).

앱 설치 후 Admin API 액세스 토큰이 표시되면 정상 발급된 것입니다. 한 번만 표시되므로 즉시 복사하세요.

3. GenieGo에 등록할 정보

필드설명
shop_domainxxx.myshopify.com 형식의 Shop 도메인
access_token발급된 Admin API 액세스 토큰(노출 주의)
GenieGo [등록]에 shop_domain(xxx.myshopify.com)과 access_token을 입력해 저장하면 자격증명은 AES-256-GCM으로 암호화 저장되고 조회 시 마스킹됩니다.

4. 자주 발생하는 문제

문제원인해결
토큰을 다시 못 봄Admin API 액세스 토큰은 1회만 표시토큰을 분실했다면 커스텀 앱에서 토큰을 다시 발급해 즉시 복사합니다.
권한 부족 오류Admin API 범위(scope) 미선택필요한 Admin API 범위(scope)를 선택한 뒤 앱을 다시 설치합니다.
도메인 오류Shop 도메인 형식 오기재xxx.myshopify.com 형식의 Shop 도메인을 정확히 입력합니다.

최종 체크리스트

  • Shopify 스토어 관리자 권한으로 로그인했다.
  • 설정 > 앱 및 판매 채널 > 앱 개발에서 커스텀 앱을 만들었다.
  • Admin API 범위(scope)를 선택하고 앱을 설치했다.
  • 설치 후 표시된 Admin API 액세스 토큰을 즉시 복사했다.
  • GenieGo [등록]에 shop_domain·access_token을 입력해 저장했다.

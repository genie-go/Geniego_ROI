Panduan Penerbitan API Key

Logen Logistics tidak memerlukan penerbitan terpisah dan terhubung melalui kunci pelacakan terpadu Smart Parcel (SweetTracker). Terbitkan t_key di tracking.sweettracker.co.kr dan daftarkan pada saluran [Smart Parcel (Pelacakan Terpadu)] untuk menghubungkan pelacakan pengiriman Logen Logistics.

Perkiraan waktuSekitar 5–15 menit
Akun yang dibutuhkanAkun Smart Parcel (SweetTracker)
Kunci yang diterbitkanKunci API pelacakan Smart Parcel (t_key)
HasilKredensial pelacakan pengiriman terpadu

1. Sebelum Anda Mulai

Logen Logistics terhubung melalui kunci pelacakan terpadu Smart Parcel, tanpa penerbitan khusus.

Metode

Pelacakan terpadu Smart Parcel (SweetTracker)

Logen Logistics tidak memerlukan penerbitan terpisah dan terhubung melalui kunci pelacakan terpadu Smart Parcel (SweetTracker). Satu t_key menangani pelacakan pengiriman untuk semua kurir.

Kunci yang diterbitkan

t_key (Kunci API pelacakan Smart Parcel)

Ini adalah kunci pelacakan terpadu yang diterbitkan setelah mendaftar di tracking.sweettracker.co.kr. Di GenieGo, masukkan pada kolom api_key.

Penting: Jangan pilih saluran Logen Logistics secara langsung; daftarkan t_key Anda pada saluran [Smart Parcel (Pelacakan Terpadu)].

2. Langkah Penerbitan

Terbitkan sesuai urutan di bawah dan daftarkan di GenieGo.

1

Daftar SweetTracker

Daftar Smart Parcel (SweetTracker).

tracking.sweettracker.co.kr
2

Terbitkan t_key

Terbitkan t_key di tracking.sweettracker.co.kr.

3

Pilih saluran Smart Parcel

Di Integration Hub, pilih saluran [Smart Parcel (Pelacakan Terpadu)] alih-alih saluran Logen Logistics.

4

Daftarkan di GenieGo

Masukkan t_key (api_key) di GenieGo [Daftar], dan pelacakan pengiriman Logen Logistics akan terhubung.

Setelah Anda menerbitkan t_key dan mendaftarkannya pada saluran [Smart Parcel (Pelacakan Terpadu)], pelacakan pengiriman Logen Logistics akan terhubung.

3. Informasi yang Perlu Didaftarkan di GenieGo

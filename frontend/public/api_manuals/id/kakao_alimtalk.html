Panduan Penerbitan API

Panduan ini merangkum langkah-langkah praktis mulai dari verifikasi bisnis saluran Kakao Talk, pendaftaran profil pengirim, persetujuan templat Alimtalk, hingga konfirmasi kredensial API.

Struktur intiVerifikasi saluran + persetujuan templat
Akun yang dibutuhkanKakao Business
Hasil utamaSender Profile / API Key
PenggunaanPesan informasional

1. Konsep Penerbitan Kakao Alimtalk

Alimtalk adalah layanan pesan bisnis yang mengirim pesan informasional seperti pesanan, reservasi, pengiriman, dan pembayaran melalui Kakao Talk. Ini bukan struktur yang dapat langsung mengirim hanya dengan kunci aplikasi developer Kakao biasa, melainkan memerlukan verifikasi saluran bisnis serta pengaturan agen pengirim/platform.

Penting: Alimtalk berfokus pada pesan informasional, bukan pesan promosi, dan templat memerlukan peninjauan terlebih dahulu.

2. Yang Perlu Disiapkan

  • Akun Kakao Business
  • Saluran Kakao Talk
  • Dokumen verifikasi bisnis seperti sertifikat pendaftaran usaha
  • Teks templat Alimtalk yang akan dikirim
  • Akun agen pengirim atau layanan pesan cloud
  • Kebijakan pengumpulan nomor telepon penerima dan pengelolaan persetujuan

3. Menyiapkan Saluran Kakao Talk

1

Akses Kakao Business

Masuk ke pusat admin Kakao Business.

business.kakao.com
2

Buat saluran Kakao Talk

Buat saluran dengan nama merek atau layanan Anda.

Pusat Admin Saluran Kakao Talk
3

Lakukan verifikasi bisnis

Kirimkan informasi bisnis dan dokumen pendukung untuk memverifikasi saluran bisnis.

Verifikasi Bisnis
4

Periksa ID pencarian saluran

Periksa ID saluran atau ID pencarian yang diperlukan untuk pengaturan pengiriman Alimtalk.

Profil Saluran → ID Pencarian
Verifikasi bisnis harus selesai sebelum Anda dapat mendaftarkan profil pengirim Alimtalk.

4. Mendaftarkan Profil Pengirim / Sender Profile

1

Pilih platform pengiriman

Pilih penyedia Alimtalk seperti NHN Cloud, Naver Cloud SENS, Infobank, atau Bizgo.

Messaging Provider
2

Buat proyek

Buat proyek Alimtalk pada platform yang Anda pilih.

Project / Service
3

Hubungkan saluran Kakao Talk

Daftarkan informasi saluran Kakao Talk yang telah terverifikasi bisnis.

Sender Profile
4

Peninjauan profil pengirim

Lakukan konfirmasi kepemilikan saluran dan peninjauan profil pengirim.

Profile Review
5

Konfirmasi kredensial API

Periksa App Key, Secret Key, API Key, dan lainnya yang disediakan platform.

API Key / Secret

5. Mendaftarkan dan Menyetujui Templat

1

Tulis templat

Tulis teks informasional seperti konfirmasi pesanan dan pemberitahuan pengiriman.

Template Message
2

Atur variabel

Atur nilai yang berubah secara dinamis seperti nama, nomor pesanan, dan nomor pengiriman sebagai variabel.

#{variabel}
3

Minta peninjauan

Simpan templat dan ajukan permintaan peninjauan Kakao.

Request Approval
4

Konfirmasi persetujuan

Konfirmasi kode templat atau ID yang telah disetujui.

Template Code
Jika teks sebelum peninjauan berbeda dengan teks yang benar-benar dikirim, pengiriman dapat gagal atau sanksi dapat terjadi.

6. Contoh Alur Pengiriman API

1) Panggil Alimtalk API Endpoint dari server Anda
2) Autentikasi dengan API Key atau Secret
3) Kirim nomor telepon penerima, kode templat, dan nilai variabel
4) Terima hasil pengiriman
5) Tangani apakah akan mengirim SMS cadangan saat gagal

7. Daftar Periksa Akhir

  • Telah membuat saluran Kakao Talk.
  • Telah menyelesaikan verifikasi bisnis.
  • Telah memilih platform pengiriman Alimtalk.
  • Telah mendaftarkan Sender Profile.
  • Telah mengonfirmasi API Key atau Secret.
  • Telah mendaftarkan templat dan mendapatkan persetujuan.
  • Telah menyiapkan kebijakan kegagalan pengiriman dan SMS cadangan.

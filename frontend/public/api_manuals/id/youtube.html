1. Perbedaan API Key dan OAuth

Ini adalah bagian yang paling membingungkan saat memulai.

API Key

Untuk mengambil data publik

Digunakan untuk mengambil data YouTube publik seperti mencari video, melihat informasi channel, mengambil playlist, dan membaca komentar.

OAuth 2.0

Untuk tugas yang membutuhkan izin akun Anda

Digunakan untuk tugas yang memerlukan persetujuan pengguna, seperti mengunggah video, mengelola channel Anda sendiri, dan mengakses data pribadi.

Penting: Untuk mengunggah video secara otomatis, API Key saja tidak cukup; diperlukan autentikasi OAuth 2.0.

2. Aktifkan YouTube Data API v3

Sebelum membuat API Key, Anda harus terlebih dahulu mengaktifkan API yang ingin digunakan di proyek Anda.

1

Akses Google Cloud Console

Masuk ke Google Cloud Console.

console.cloud.google.com
2

Pilih atau buat proyek

Di pemilih proyek di bagian atas, pilih proyek yang ada atau buat proyek baru.

mis. Geniego ROI
3

Buka API Library

Di menu kiri, buka API & Layanan → Library.

API & Layanan → Library
4

Cari YouTube Data API v3 lalu aktifkan

Ketik YouTube Data API v3 di kotak pencarian, pilih API tersebut, lalu klik tombol Aktifkan.

YouTube Data API v3 → Aktifkan
Jika Anda melihat Status: Diaktifkan di layar, berarti sudah diaktifkan dengan benar.

3. Buat API Key

Setelah YouTube Data API v3 diaktifkan, Anda dapat membuat API Key di layar Kredensial.

1

Buka menu Kredensial

Di menu kiri, buka API & Layanan → Kredensial.

API & Layanan → Kredensial
2

Klik Buat Kredensial

Klik tombol + Buat Kredensial di bagian atas.

+ Buat Kredensial
3

Pilih Kunci API

Pada menu yang muncul, pilih Kunci API.

Kunci API
4

Pilih API yang dapat diakses

Di bagian Pembatasan API, pastikan YouTube Data API v3 telah dipilih.

API yang dipilih: YouTube Data API v3
5

Klik tombol Buat

Klik tombol Buat di bagian bawah untuk menghasilkan API Key.

Buat
Kunci yang dihasilkan biasanya diawali dengan AIzaSy.... String ini adalah YouTube API Key Anda.

5. Contoh Penggunaan API Key

Anda dapat menguji dengan menempatkan kunci yang dihasilkan di bagian YOUR_API_KEY.

Contoh pencarian videohttps://www.googleapis.com/youtube/v3/search?part=snippet&q=korea&type=video&key=YOUR_API_KEY
Contoh pencarian info channelhttps://www.googleapis.com/youtube/v3/channels?part=snippet&id=CHANNEL_ID&key=YOUR_API_KEY
Jika hasil berformat JSON muncul saat Anda memasukkan URL di atas ke bilah alamat browser, berarti API Key berfungsi dengan benar.

7. Daftar Periksa Akhir

  • Memilih proyek Google Cloud.
  • Mengaktifkan YouTube Data API v3.
  • Membuat API Key di Kredensial.
  • Memastikan API Key yang dihasilkan diawali dengan AIzaSy.
  • Menetapkan Pembatasan API ke YouTube Data API v3.
  • Menetapkan pembatasan situs web atau IP sebelum menerapkan ke layanan nyata.

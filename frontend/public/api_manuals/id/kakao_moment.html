💛 Beginner Guide

Kakao Moment
Panduan Penerbitan API Key

Panduan untuk mengajukan izin Kakao Moment Open API, memperoleh access token dan ad account ID Anda, serta mendaftarkannya di GenieGo Integration Hub. Izin Open API hanya diberikan kepada agensi periklanan resmi Kakao.

Perkiraan waktuBervariasi tergantung periode peninjauan kelayakan
Akun yang dibutuhkanAkun iklan Kakao Moment + Kakao Business Channel
Kunci yang diterbitkanAccess token · Ad account ID
HasilKredensial Open API

1. Sebelum Anda Memulai

Kakao Moment Open API tidak diterbitkan sendiri; ini melalui proses pengajuan dan peninjauan izin.

Penting: Izin Kakao Moment Open API hanya diberikan kepada agensi periklanan resmi Kakao. Pengiklan biasa tidak dapat menerbitkannya sendiri.
Diperlukan

Akun iklan + Business Channel

Anda memerlukan akun iklan Kakao Moment dan Kakao Business Channel.

Kunci yang diterbitkan

Access token · Ad account ID

Setelah izin disetujui, Anda memperoleh access token melalui otorisasi OAuth dan mengonfirmasi ad account ID Anda.

2. Proses Pengajuan Penerbitan

Lakukan secara berurutan: Pengajuan izin Open API → peninjauan/persetujuan → penerbitan token → pendaftaran.

1

Buat aplikasi dan beralih ke Business app

Buat aplikasi di developers.kakao.com dan ubah menjadi [Business App].

developers.kakao.com
2

Ajukan izin Open API

Kirim Kakao Moment Open API [Additional Feature Application]. Izin diberikan setelah kelayakan agensi resmi Anda ditinjau.

Additional Feature Application
3

Terbitkan token

Setelah izin disetujui, peroleh access token melalui otorisasi OAuth dan konfirmasikan ad account ID Anda. Anda dapat menggunakan OAuth satu klik di GenieGo Integration Hub.

Setelah izin disetujui dan access token serta ad account ID Anda dikonfirmasi, Anda dapat melanjutkan pendaftaran.

3. Informasi yang Didaftarkan di GenieGo

BidangDeskripsi
access_tokenAccess token yang diterbitkan melalui otorisasi OAuth (tangani dengan hati-hati)
ad_account_idAd account ID Kakao Moment
Masukkan dan simpan di GenieGo Integration Hub [Register], dan kredensial dienkripsi saat disimpan dengan AES-256-GCM serta disamarkan saat dibaca.

4. Masalah Umum

MasalahPenyebabSolusi
Tidak dapat memperoleh izin Open APIBukan agensi periklanan resmiIzin Open API hanya diberikan kepada agensi periklanan resmi Kakao. Ajukan melalui agensi.
Tidak dapat melanjutkan [Additional Feature Application]Aplikasi belum diubah menjadi Business appUbah aplikasi menjadi [Business App] terlebih dahulu, lalu ajukan fitur tambahan.

Daftar Periksa Akhir

  • Menyiapkan akun iklan Kakao Moment dan Kakao Business Channel.
  • Mengubah aplikasi menjadi [Business App] dan mengirim Open API [Additional Feature Application].
  • Setelah persetujuan izin, mengonfirmasi access token dan ad account ID melalui otorisasi OAuth.
  • Memasukkan access_token·ad_account_id di GenieGo [Register] dan menyimpannya.

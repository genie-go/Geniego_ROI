PayPal
Manual Penerbitan API Key

Panduan langkah demi langkah yang ramah bagi pemula untuk membuat aplikasi (App) di PayPal Developer Portal guna menerbitkan Client ID dan Secret, lalu mendaftarkannya di GenieGo Integration Hub.

Perkiraan waktuSekitar 5–10 menit
Akun yang diperlukanAkun bisnis PayPal
Kunci yang diterbitkanClient ID · Secret
HasilKredensial API

1. Sebelum Memulai

Kredensial API PayPal diterbitkan dengan membuat aplikasi di Developer Portal menggunakan akun bisnis.

Wajib

Akun bisnis PayPal

Anda memerlukan akun bisnis PayPal yang dapat masuk ke developer.paypal.com.

Kunci yang diterbitkan

Client ID · Secret

Membuat aplikasi Live akan menerbitkan Client ID dan Secret.

Catatan: Untuk integrasi sebenarnya, buat aplikasi di tab [Live], bukan Sandbox.

2. Langkah Penerbitan

Terbitkan dalam urutan di bawah ini dan daftarkan di GenieGo.

1

Masuk ke Developer Portal

Masuk dengan akun bisnis PayPal Anda, lalu buka [Apps & Credentials].

developer.paypal.com → Apps & Credentials
2

Buat aplikasi Live

Di tab [Live], klik [Create App] untuk membuat aplikasi.

Live → Create App
3

Periksa kredensial Anda

Periksa Client ID dan Secret yang diterbitkan.

Setelah Client ID dan Secret ditampilkan, berarti keduanya telah diterbitkan dengan benar.

3. Informasi untuk Didaftarkan di GenieGo

BidangDeskripsi
client_idClient ID yang diterbitkan
client_secretSecret yang diterbitkan (tangani dengan hati-hati)
Di GenieGo Integration Hub [Daftar], masukkan client_id·client_secret Anda dan simpan. Kredensial dienkripsi saat disimpan dengan AES-256-GCM dan disamarkan saat dibaca.

4. Masalah Umum

MasalahPenyebabSolusi
Autentikasi gagalMenggunakan kredensial dari aplikasi SandboxGunakan Client ID·Secret dari aplikasi yang dibuat di tab [Live].
Tidak dapat membuat aplikasiMasuk dengan akun biasa (pribadi)Masuk dengan akun bisnis PayPal.

Daftar Periksa Akhir

  • Masuk ke developer.paypal.com dengan akun bisnis PayPal.
  • Berpindah ke tab [Live] di [Apps & Credentials].
  • Membuat aplikasi melalui [Create App] dan memeriksa Client ID·Secret.
  • Memasukkan client_id·client_secret di GenieGo [Daftar] dan menyimpannya.

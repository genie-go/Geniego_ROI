Shopify
Manual Penerbitan API Key

Panduan langkah demi langkah untuk membuat aplikasi khusus di Admin Shopify Anda guna menerbitkan Admin API access token dan mendaftarkannya bersama domain toko Anda di GenieGo Integration Hub untuk menghubungkan toko Shopify Anda.

Perkiraan waktuSekitar 5–15 mnt
Akun yang dibutuhkanIzin admin toko Shopify
Kunci yang diterbitkanDomain toko · Admin API access token
HasilKredensial Admin API

1. Sebelum Anda Mulai

Aplikasi khusus Shopify hanya dapat dibuat dengan izin admin toko.

Diperlukan

Izin admin toko

Anda memerlukan izin admin untuk toko Shopify guna membuat aplikasi khusus dan menerbitkan token.

Kunci yang diterbitkan

Domain toko · Admin API token

Anda akan memperoleh domain toko dalam format xxx.myshopify.com dan Admin API access token yang ditampilkan setelah menginstal aplikasi.

Catatan: Admin API access token hanya ditampilkan satu kali tepat setelah instalasi, jadi Anda harus segera menyalinnya saat muncul.

2. Langkah Penerbitan

Terbitkan sesuai urutan di bawah ini dan daftarkan di GenieGo.

1

Buka Aplikasi dan saluran penjualan

Di Admin Shopify Anda, buka [Pengaturan] > [Aplikasi dan saluran penjualan].

https://www.shopify.com/admin/settings/apps
2

Buat aplikasi khusus

Klik [Kembangkan aplikasi] > [Buat aplikasi] untuk membuat aplikasi khusus.

Kembangkan aplikasi → Buat aplikasi
3

Pilih scope · Instal aplikasi

Pilih Admin API scopes dan instal aplikasi.

4

Salin access token

Segera salin Admin API access token yang ditampilkan setelah instalasi (hanya ditampilkan satu kali).

Setelah Admin API access token muncul usai menginstal aplikasi, berarti penerbitan berhasil. Karena hanya ditampilkan satu kali, segera salin.

3. Informasi untuk Didaftarkan di GenieGo

BidangDeskripsi
shop_domainDomain toko dalam format xxx.myshopify.com
access_tokenAdmin API access token yang diterbitkan (tangani dengan hati-hati)
Di GenieGo [Daftar], masukkan shop_domain (xxx.myshopify.com) dan access_token Anda lalu simpan. Kredensial dienkripsi saat disimpan dengan AES-256-GCM dan disamarkan saat dibaca.

4. Masalah Umum

MasalahPenyebabSolusi
Tidak bisa melihat token lagiAdmin API access token hanya ditampilkan satu kaliJika Anda kehilangan token, terbitkan ulang di aplikasi khusus dan segera salin.
Kesalahan izin tidak cukupAdmin API scopes belum dipilihPilih Admin API scopes yang diperlukan, lalu instal ulang aplikasi.
Kesalahan domainFormat domain toko dimasukkan dengan salahMasukkan domain toko dengan akurat dalam format xxx.myshopify.com.

Daftar Periksa Akhir

  • Masuk dengan izin admin toko Shopify.
  • Membuat aplikasi khusus di Pengaturan > Aplikasi dan saluran penjualan > Kembangkan aplikasi.
  • Memilih Admin API scopes dan menginstal aplikasi.
  • Segera menyalin Admin API access token yang ditampilkan setelah instalasi.
  • Memasukkan shop_domain·access_token di GenieGo [Daftar] dan menyimpannya.

Panduan Penerbitan API Key

Panduan untuk menerbitkan API key bersama Gmarket·Auction dari ESM Trading API (etapi.gmarket.com) dan mendaftarkannya di GenieGo Integration Hub. Gmarket dan Auction dioperasikan bersama melalui ESM Plus, sehingga satu API key menghubungkan kedua kanal sekaligus.

Perkiraan waktuBeberapa hari termasuk masa tunggu persetujuan (berbasis persetujuan)
Akun yang dibutuhkanAkun penjual Gmarket·Auction + ESM Plus master ID
Kunci yang diterbitkanAPI key · Seller ID
HasilKredensial ESM Trading API

1. Sebelum Anda Mulai

Anda harus terdaftar sebagai penjual di Gmarket maupun Auction dan memiliki ESM Plus master ID.

Diperlukan

Akun penjual + ESM master ID

Anda memerlukan akun penjual untuk Auction maupun Gmarket, ditambah master ID yang dibuat di ESM Plus (esmplus.com).

Kunci yang diterbitkan

API key · Seller ID

Setelah ESM Trading API disetujui, periksa API key dan seller ID yang diterbitkan secara bersamaan.

Catatan: ESM Trading API berbasis persetujuan, dan permohonan Anda dapat ditolak. Saat mengajukan permohonan, Anda harus mengirimkan tiga item: Auction seller ID, Gmarket seller ID, dan ESM master ID Anda.

2. Langkah Penerbitan

Ajukan dan terbitkan sesuai urutan di bawah, lalu daftarkan di GenieGo.

1

Daftar sebagai penjual dan buat ESM master ID

Daftar sebagai penjual di Auction maupun Gmarket, lalu masuk ke ESM Plus dan buat master ID.

esmplus.com
2

Ajukan akses ESM Trading API

Buka ESM Trading API dan ajukan akses API. Karena berbasis persetujuan, permohonan dapat ditolak.

https://etapi.gmarket.com
3

Kirim detail identifikasi

Saat mengajukan permohonan, kirimkan tiga item: Auction seller ID, Gmarket seller ID, dan ESM master ID Anda.

4

Terima kunci

Setelah disetujui, sebuah API key akan diterbitkan. Periksa API key dan seller ID yang diterbitkan.

Setelah permohonan Anda disetujui dan API key diterbitkan, berarti telah diproses dengan benar. Untuk pertanyaan terkait persetujuan, hubungi etapihelp@gmail.com.

3. Informasi untuk Didaftarkan di GenieGo

api_keyAPI key yang diterbitkan
seller_idSeller ID
Masukkan dan simpan di GenieGo Integration Hub [Daftar]. Kredensial dienkripsi saat disimpan dengan AES-256-GCM dan disamarkan saat dibaca.

Daftar Periksa Akhir

  • Terdaftar sebagai penjual di Auction maupun Gmarket.
  • Membuat master ID di ESM Plus.
  • Mengajukan dan menerima persetujuan akses ESM Trading API di etapi.gmarket.com.
  • Memeriksa api_key dan seller_id yang diterbitkan.
  • Memasukkan api_key·seller_id di GenieGo [Daftar] dan menyimpannya.

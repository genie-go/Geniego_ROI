1. Memahami Struktur Akun Twitch

Berbeda dengan Instagram, Twitch tidak memiliki struktur pemilihan akun bisnis/kreator secara terpisah.

Kanal resmi perusahaan

Untuk pengoperasian merek

Buat akun Twitch biasa dengan nama perusahaan atau merek, lalu operasikan seperti kanal resmi. Cocok untuk perkenalan produk, live-commerce, dan siaran acara.

Tingkat pertumbuhan streamer

Affiliate / Partner

Setelah memenuhi syarat performa siaran, pengikut, dan waktu tonton, Anda dapat bergabung dengan program Affiliate atau Partner.

Poin utama: Perusahaan pun sebaiknya membuat akun Twitch biasa terlebih dahulu, lalu melakukan branding kanal, autentikasi dua faktor, koneksi OBS, dan pendaftaran aplikasi di Developer Console.

1

Masuk ke Creator Dashboard dari menu profil.

Creator Dashboard
2

Buka menu Stream

Pilih item Stream di menu pengaturan.

Settings → Stream
3

Periksa Primary Stream Key

Salin kunci stream utama. Kunci ini harus disimpan seperti kata sandi.

4

Hubungkan di OBS

Di OBS Studio, atur layanan ke Twitch dan masukkan nilai yang disalin sebagai stream key.

OBS → Settings → Stream → Twitch
Jika stream key Anda terekspos, orang lain dapat menyiarkan di kanal Anda. Jangan pernah dipublikasikan.

12. Struktur Autentikasi Twitch API

Twitch Channel │ ▼ Developer Console │ ▼ Application │ ▼ Client ID / Client Secret │ ▼ OAuth Token │ ▼ Twitch Helix API │ ▼ Info kanal, status siaran, chat bot, analitik pemirsa

13. Alur Penerbitan Token OAuth

1) Buat URL otorisasi Twitch OAuth 2) Pengguna masuk dengan akun Twitch 3) Terima authorization code di Redirect URL 4) Minta Access Token dengan authorization code + Client Secret 5) Panggil Twitch API dengan Access Token 6) Segarkan dengan Refresh Token bila perlu
Kueri data publik sederhana dapat dilakukan dengan App Access Token, sedangkan operasi yang memerlukan izin pengguna membutuhkan User Access Token.

15. Daftar Periksa Akhir

  • Membuat akun Twitch dengan email perusahaan.
  • Mengatur nama pengguna yang sesuai dengan nama merek.
  • Menyelesaikan verifikasi email.
  • Mengatur gambar profil, banner, teks bio, dan tautan.
  • Mengaktifkan autentikasi dua faktor.
  • Memeriksa Stream Key di Creator Dashboard.
  • Menghubungkan OBS Studio dengan kanal Twitch.
  • Mendaftarkan aplikasi di Twitch Developer Console.
  • Memperoleh Client ID dan Client Secret.
  • Menyiapkan OAuth Redirect URL dan alur penerbitan token.

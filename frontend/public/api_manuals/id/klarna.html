Panduan Penerbitan API Key

Panduan langkah demi langkah yang ramah pemula untuk membuat kredensial API (username dan password) di Klarna Merchant Portal dan mendaftarkannya bersama wilayah Anda di GenieGo Integration Hub.

Perkiraan waktuSekitar 5–10 menit
Akun yang dibutuhkanAkun merchant Klarna (BNPL, perlu kontrak)
Kunci yang diterbitkanAPI username · password · wilayah
HasilKredensial API

1. Sebelum Anda Mulai

Kredensial Klarna hanya dapat diterbitkan setelah kontrak merchant Anda selesai.

Diperlukan

Akun merchant Klarna

Pertama-tama Anda memerlukan kontrak merchant Klarna (BNPL). Hanya setelah kontrak selesai, Anda dapat menerbitkan kredensial API di Merchant Portal.

Kunci yang diterbitkan

API username · password · wilayah

Di Merchant Portal pada Settings > Klarna API credentials, buat API username·password dan konfirmasikan wilayah Anda (eu/na/oc).

Catatan: Wilayah adalah salah satu dari eu / na / oc, dan harus dimasukkan bersamaan saat mendaftar.

2. Langkah Penerbitan

Setelah kontrak merchant, terbitkan sesuai urutan di bawah dan daftarkan di GenieGo.

1

Kontrak merchant

Pertama, selesaikan kontrak merchant Klarna (BNPL) Anda. Anda baru dapat melanjutkan ke langkah berikutnya setelah kontrak selesai.

2

Masuk ke Merchant Portal

Masuk ke Klarna Merchant Portal.

portal.klarna.com
3

Buka API credentials

Buka menu [Settings] > [Klarna API credentials].

Settings → Klarna API credentials
4

Buat dan konfirmasi kredensial

Buat API username·password dan konfirmasikan wilayah Anda (eu/na/oc).

Jika API username·password dibuat di Settings > Klarna API credentials dan Anda telah mengonfirmasi wilayah, berarti semuanya beres.

3. Informasi yang Harus Didaftarkan di GenieGo

BidangDeskripsi
usernameAPI Username (MID/PID)
passwordAPI Password (tangani dengan hati-hati)
regionWilayah (eu / na / oc)
Masukkan username·password·region di [Daftar] GenieGo Integration Hub lalu simpan. Kredensial dienkripsi saat disimpan dengan AES-256-GCM dan disamarkan saat dibaca.

4. Masalah Umum

MasalahPenyebabSolusi
Tidak ada opsi penerbitan di menu API credentialsKontrak merchant belum selesaiSelesaikan dulu kontrak merchant Klarna Anda.
Autentikasi gagalWilayah (eu/na/oc) tidak cocok atau password salah dimasukkanPeriksa kembali wilayah serta username·password yang Anda konfirmasi di Merchant Portal.

Daftar Periksa Akhir

  • Menyelesaikan kontrak merchant Klarna (BNPL).
  • Membuat kredensial di Settings > Klarna API credentials dalam Merchant Portal.
  • Mengonfirmasi API username·password dan wilayah (eu/na/oc).
  • Memasukkan username·password·region di GenieGo [Daftar] dan menyimpannya.

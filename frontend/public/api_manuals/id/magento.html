Magento (Adobe Commerce)
Panduan Penerbitan API Key

Panduan langkah demi langkah yang ramah pemula untuk membuat Integration di panel admin Magento (Adobe Commerce) guna menerbitkan token akses dan mendaftarkannya di GenieGo Integration Hub.

Perkiraan waktuSekitar 5–15 menit
Akun yang dibutuhkanHak admin Magento (Adobe Commerce)
Kunci yang diterbitkanToken akses Integration
HasilKredensial REST API

1. Sebelum Anda Mulai

Integration hanya dapat dibuat dengan hak admin.

Diperlukan

Hak admin Magento

Anda memerlukan hak admin Magento (Adobe Commerce). Integration tidak dapat dibuat dengan hak pengguna biasa.

Kunci yang diterbitkan

Token akses Integration

Saat Anda membuat Integration dan mengklik Activate, token akses akan diterbitkan.

2. Langkah Penerbitan

Terbitkan sesuai urutan di bawah dan daftarkan di GenieGo.

1

Buka menu Integration

Di panel admin Magento, buka [Sistem] > [Integrations].

Admin → Sistem → Integrations
2

Buat Integration

Buat Integration dengan [Add New Integration] dan konfigurasikan izin sumber daya.

Add New Integration
3

Terbitkan token melalui Activate

Klik [Activate] untuk memperoleh token akses.

Activate
4

Verifikasi informasi

Konfirmasikan token akses yang diterbitkan dan URL toko (base_url) Anda.

Setelah token akses ditampilkan setelah Activate, berarti token telah diterbitkan dengan benar.

3. Informasi yang Perlu Didaftarkan di GenieGo

BidangDeskripsi
base_urlURL toko
access_tokenToken akses Integration yang diterbitkan (tangani dengan hati-hati)
Masukkan base_url·access_token di [Daftar] pada GenieGo Integration Hub lalu simpan. Kredensial dienkripsi saat disimpan dengan AES-256-GCM dan disamarkan saat dibaca.

4. Masalah Umum

MasalahPenyebabSolusi
Menu Integrations tidak adaHak admin tidak memadaiMasuk dengan akun admin yang memiliki izin untuk membuat Integration.
Kesalahan izin saat pemanggilanIzin sumber daya Integration belum diatur, atau Activate belum selesaiAtur izin sumber daya Integration dan selesaikan Activate untuk menerbitkan token.

Daftar Periksa Akhir

  • Masuk dengan hak admin Magento (Adobe Commerce).
  • Membuat Integration melalui Add New Integration di Sistem > Integrations.
  • Mengatur izin sumber daya dan menerbitkan token akses melalui Activate.
  • Memasukkan base_url·access_token di GenieGo [Daftar] dan menyimpannya.

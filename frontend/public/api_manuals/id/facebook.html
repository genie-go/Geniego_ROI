Facebook Live
Panduan Penerbitan API

Panduan langkah demi langkah yang bisa diikuti siapa saja untuk menerbitkan kredensial Facebook Live (API key/token) dan mendaftarkannya di GenieGo Integration Hub.

Perkiraan waktuSekitar 5–15 menit
Akun diperlukanAkun Facebook Live
Jumlah langkah5 langkah
HasilKredensial API

Langkah Penerbitan

Ikuti langkah-langkah di bawah untuk menerbitkan dan mendaftarkan kredensial Anda.

1

Persyaratan: akun developer Meta (Facebook) + Halaman Facebook yang Anda kelola.

2

Di developers.facebook.com > [My Apps] > [Create App], buat aplikasi bertipe Business.

3

Tambahkan produk [Pages]/[Facebook Login] dan ajukan izin seperti pages_read_engagement melalui App Review.

4

Terbitkan Page access token melalui Graph API Explorer atau system user dan temukan page_id Anda.

5

Di GenieGo [Simpan], masukkan access_token (Page token) dan page_id, lalu simpan.

Daftar Periksa Akhir

  • Persyaratan: akun developer Meta (Facebook) + Halaman Facebook yang Anda kelola.
  • Di developers.facebook.com > [My Apps] > [Create App], buat aplikasi bertipe Business.
  • Tambahkan produk [Pages]/[Facebook Login] dan ajukan izin seperti pages_read_engagement melalui App Review.
  • Terbitkan Page access token melalui Graph API Explorer atau system user dan temukan page_id Anda.
  • Di GenieGo [Simpan], masukkan access_token (Page token) dan page_id, lalu simpan.
Kunci yang dimasukkan di GenieGo Integration Hub [Daftar] akan dienkripsi saat disimpan dengan AES-256-GCM dan disamarkan saat dibaca.
Jaga keamanan kunci/secret Anda dan jangan pernah memublikasikannya. Jika dicurigai bocor, segera terbitkan ulang (rotasi).

Facebook Live
Panduan Penerbitan API

Panduan yang ramah pemula tentang cara membuat aplikasi di Meta for Developers dan menyiapkan App ID, App Secret, serta token akses halaman (Page Access Token) yang diperlukan untuk menghubungkan siaran langsung halaman Facebook.

Konsep intiBukan API Key, melainkan Access Token
Situs pengembangdevelopers.facebook.com
API yang digunakanLive Video API (Graph)
Izin yang diperlukanAdmin halaman + Page Token

1. Hal Inti yang Harus Diketahui Lebih Dulu

YouTube cukup dengan satu API Key, tetapi Facebook (Meta) bekerja dengan struktur App ID · App Secret · Access Token.

Penting: Dengan Meta, "menyalin satu API Key" saja belum cukup. Biasanya Anda perlu mengikuti urutan: buat aplikasi → tambahkan produk → atur izin → terbitkan Page Access Token → tinjauan aplikasi.

2. Yang Perlu Disiapkan

  • Akun Meta/Facebook
  • Akses ke Meta for Developers
  • Halaman Facebook tempat Anda memiliki hak admin
  • URL kebijakan privasi · URL ketentuan layanan untuk layanan nyata

3. Membuat Aplikasi Meta Developer

Setiap integrasi Meta API dimulai dengan membuat aplikasi.

1

Akses Meta for Developers

Masuk ke Meta for Developers di browser Anda.

developers.facebook.com
2

Aplikasi Saya → Buat Aplikasi

Klik My Apps → Create App di bagian atas.

My Apps → Create App
3

Pilih jenis aplikasi

Integrasi platform/live commerce lebih dekat ke tujuan bisnis.

Pilih jenis Business
4

Masukkan nama aplikasi·email → buat

Masukkan nama aplikasi dan email kontak, lalu buat aplikasi.

App Name / Contact Email
Setelah membuat aplikasi, periksa App ID dan App Secret di Dasbor → Pengaturan Aplikasi → Pengaturan Dasar. (Jangan pernah mengekspos App Secret di frontend/repositori publik.)

4. Alur Penyiapan Facebook Live API

1

Tambahkan produk

Pada dasbor aplikasi, siapkan pengaturan terkait Facebook Login / Graph API.

Add Product → Facebook Login
2

Hubungkan halaman + token halaman

Untuk menyiarkan siaran langsung ke halaman, Anda memerlukan hak admin halaman dan Page Access Token.

Page Access Token
3

Ajukan izin yang diperlukan (App Review)

Ajukan izin untuk tujuan memublikasikan konten·mengelola halaman·membuat video langsung (mis. pages_read_engagement, pages_manage_posts).

Permissions → App Review
4

Terbitkan·uji token dengan Graph API Explorer

Terbitkan token yang menyertakan izin dan uji pemanggilan API.

Tools → Graph API Explorer
Contoh pemanggilan pembuatan Facebook Live
POST /{page-id}/live_videos
access_token={PAGE_ACCESS_TOKEN}
Jika berhasil dibuat, informasi siaran langsung seperti ID siaran·URL streaming·kunci streaming akan dikembalikan. Lihat ID halaman (page_id) di informasi halaman.

5. Daftarkan di GenieGo Integration Hub

1

Salin Page Access Token

Salin Page Access Token yang Anda terbitkan (token jangka panjang disarankan).

access_token (Page token)
2

Konfirmasi ID halaman

Konfirmasi ID (page_id) halaman Facebook target.

page_id
3

Daftarkan di Integration Hub

Di GenieGo Integration Hub → Facebook Live → [Daftar], masukkan nilai di bawah ini lalu simpan.

access_token · page_id
Kunci yang terdaftar dienkripsi saat disimpan dengan AES-256-GCM dan disamarkan saat dibaca.

6. Jenis Access Token

Jenis tokenKegunaanCatatan
User Access TokenPemanggilan API berbasis login penggunaData yang dapat diakses bervariasi tergantung cakupan izin yang disetujui.
Page Access TokenPublikasi halaman · pembuatan siaran langsung halamanMemerlukan hak admin halaman dan izin terkait.
Long-Lived TokenToken jangka panjang dengan masa berlaku diperpanjangSaat dioperasikan memerlukan logika kedaluwarsa·penyegaran.

7. Daftar Periksa

  • Membuat aplikasi Meta Developer.
  • Mengonfirmasi App ID dan App Secret.
  • Menghubungkan halaman Facebook dan menerbitkan Page Access Token.
  • Menyusun·mengajukan izin yang diperlukan (App Review).
  • Memasukkan access_token·page_id di GenieGo [Daftar] dan menyimpan.
  • Menyimpan token dengan aman di variabel lingkungan server.

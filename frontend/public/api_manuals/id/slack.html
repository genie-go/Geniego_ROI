💬 Panduan Pemula

Slack Webhook
Panduan Penerbitan API Key

Panduan untuk membuat aplikasi di Slack, menerbitkan Incoming Webhook URL, dan mendaftarkannya di GenieGo Integration Hub agar notifikasi dikirim ke channel Slack yang Anda tentukan.

Perkiraan waktuSekitar 5 menit
Akun yang dibutuhkanIzin admin workspace Slack
Kunci yang diterbitkanWebhook URL
HasilKredensial Incoming Webhook

1. Sebelum Anda Mulai

Slack Webhook hanya dapat dibuat dengan izin admin workspace.

Diperlukan

Izin admin workspace

Anda memerlukan izin admin workspace Slack untuk membuat aplikasi dan mengaktifkan Webhook.

Kunci yang diterbitkan

Webhook URL

Setelah mengaktifkan Incoming Webhooks dan memilih channel, periksa Webhook URL yang dihasilkan.

Catatan: Webhook URL terkait dengan channel tempat notifikasi dikirim. Tentukan channel yang akan menerima notifikasi pada langkah pemilihan channel.

2. Langkah Penerbitan

Terbitkan sesuai urutan di bawah dan daftarkan di GenieGo.

1

Buat aplikasi baru

Klik [Create New App].

https://api.slack.com/apps
2

Aktifkan Incoming Webhooks

Aktifkan [Incoming Webhooks] dan klik [Add New Webhook to Workspace].

3

Pilih channel · Hasilkan URL

Memilih channel akan menghasilkan Webhook URL.

Setelah Anda memilih channel dan Webhook URL dihasilkan, berarti telah diterbitkan dengan benar.

3. Informasi yang Didaftarkan di GenieGo

BidangDeskripsi
webhook_urlIncoming Webhook URL yang dihasilkan (tangani dengan hati-hati)
Masukkan webhook_url di GenieGo [Daftar] lalu simpan. Kredensial dienkripsi saat disimpan dengan AES-256-GCM dan disamarkan saat dibaca.

4. Masalah Umum

MasalahPenyebabSolusi
Tidak dapat membuat aplikasiIzin admin workspace tidak memadaiLanjutkan dengan akun yang memiliki izin admin workspace Slack.
Notifikasi tidak masukWebhook URL salah atau channel tidak dipilihAktifkan Incoming Webhooks, pilih channel, dan periksa kembali Webhook URL yang dihasilkan.

Daftar Periksa Akhir

  • Telah dilanjutkan dengan izin admin workspace Slack.
  • Telah membuat aplikasi di api.slack.com/apps melalui [Create New App].
  • Telah mengaktifkan Incoming Webhooks dan mengklik [Add New Webhook].
  • Telah memilih channel serta menghasilkan dan menyalin Webhook URL.
  • Telah memasukkan webhook_url di GenieGo [Daftar] dan menyimpannya.

🔶 Panduan Pemula

11Street
Panduan Penerbitan API Key

Panduan langkah demi langkah yang ramah pemula untuk menerbitkan kunci Open API dari 11Street Open API Center (openapi.11st.co.kr) dan mendaftarkannya di GenieGo Integration Hub.

Perkiraan waktuSekitar 5–15 menit
Akun yang dibutuhkanAkun penjual 11Street
Kunci yang diterbitkanKunci Open API · Seller ID
HasilKredensial Open API

1. Sebelum Anda Mulai

Kunci Open API 11Street hanya dapat diterbitkan dari akun penjual.

Diperlukan

Akun penjual 11Street

Anda memerlukan akun penjual 11Street. Masuk ke Open API Center dengan Seller ID Anda.

Kunci yang diterbitkan

Kunci Open API · Seller ID

Setelah menyelesaikan pendaftaran layanan dan pendaftaran IP akses, periksa kunci Open API yang diterbitkan bersama dengan Seller ID Anda.

Catatan: Anda harus mendaftarkan IP server yang diizinkan mengakses API sebelum dapat menggunakan kunci.

2. Langkah Penerbitan

Terbitkan sesuai urutan di bawah dan daftarkan di GenieGo.

1

Masuk ke Open API Center

Masuk ke 11Street Open API Center dengan Seller ID Anda.

https://openapi.11st.co.kr/openapi/OpenApiFrontMain.tmall
2

Daftarkan layanan

Dari menu, klik [Pendaftaran·Verifikasi Layanan], masukkan detail penjual, setujui ketentuan, dan daftarkan.

Pendaftaran·Verifikasi Layanan
3

Daftarkan IP akses

Di [Info Seller API] > [Edit], daftarkan IP server yang diizinkan mengakses API.

Info Seller API → Edit
4

Salin API Key

Salin API Key yang diterbitkan.

Setelah pendaftaran layanan dan pendaftaran IP akses selesai, API Key diterbitkan.

3. Informasi yang Harus Didaftarkan di GenieGo

BidangDeskripsi
api_keyKunci Open API yang diterbitkan
seller_idSeller ID
Masukkan dan simpan di GenieGo Integration Hub [Daftar]. Kredensial dienkripsi saat disimpan dengan AES-256-GCM dan disamarkan saat dibaca.

4. Masalah Umum

MasalahPenyebabSolusi
API Key tidak diterbitkanPendaftaran layanan atau pendaftaran IP akses belum selesaiSelesaikan [Pendaftaran·Verifikasi Layanan] dan pendaftaran IP akses di [Info Seller API].
Kesalahan autentikasi saat memanggilIP server yang diizinkan belum didaftarkanPeriksa kembali IP server yang diizinkan di [Info Seller API] > [Edit].

Daftar Periksa Akhir

  • Masuk ke Open API Center dengan akun penjual 11Street.
  • Memasukkan detail penjual dan menyetujui ketentuan di [Pendaftaran·Verifikasi Layanan].
  • Mendaftarkan IP server yang diizinkan di [Info Seller API].
  • Mengonfirmasi kunci Open API yang diterbitkan dan Seller ID.
  • Memasukkan api_key·seller_id di GenieGo [Daftar] dan menyimpan.

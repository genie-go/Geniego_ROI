Panduan Penerbitan API Key

Panduan tentang cara mendaftarkan kunci API Pengiriman Hari Sama dan ID merchant—yang diterbitkan setelah membuat kontrak layanan GenieGo Pengiriman Hari Sama—di GenieGo Integration Hub. Untuk kanal ini, kredensial diterima melalui kontrak dengan tim operasional, bukan diterbitkan sendiri dari konsol.

Akun yang dibutuhkanKontrak layanan GenieGo Pengiriman Hari Sama
Kunci yang diterbitkanKunci API Pengiriman Hari Sama · ID merchant
HasilKredensial API Pengiriman Hari Sama

1. Sebelum Anda Mulai

GenieGo Pengiriman Hari Sama adalah kanal berbasis kontrak, di mana kredensial diterbitkan setelah kontrak layanan dengan tim operasional.

Diperlukan

Kontrak layanan Pengiriman Hari Sama

Untuk menggunakan GenieGo Pengiriman Hari Sama, kontrak penggunaan layanan dengan tim operasional harus didahulukan. Tanpa kontrak, kredensial tidak dapat diterbitkan.

Kunci yang diterbitkan

Kunci API · ID merchant · Wilayah layanan

Setelah kontrak selesai, tim operasional menerbitkan ID merchant dan kunci API beserta wilayah layanan untuk Anda.

Catatan: Kanal ini tidak diterbitkan sendiri dari konsol; sebaliknya, kredensial diterima melalui kontrak dengan tim operasional.

2. Prosedur Permintaan Penerbitan

Ajukan dan terima sesuai urutan di bawah, lalu daftarkan di GenieGo.

1

Ajukan penggunaan Pengiriman Hari Sama

Ajukan kepada tim operasional untuk menggunakan GenieGo Pengiriman Hari Sama dan lanjutkan dengan kontrak.

2

Tanda tangani kontrak

Tanda tangani kontrak penggunaan layanan Pengiriman Hari Sama dengan tim operasional.

3

Terima kredensial

Setelah kontrak selesai, Anda diterbitkan ID merchant, kunci API, dan wilayah layanan.

4

Daftarkan di GenieGo

Masukkan api_key, merchant_id, dan region yang diterbitkan ke GenieGo [Daftar] lalu simpan.

Jika Anda telah menerima ID merchant, kunci API, dan wilayah layanan semuanya dari tim operasional, berarti telah diterbitkan dengan benar.

3. Informasi yang Perlu Didaftarkan di GenieGo

BidangDeskripsi

Setelah dimasukkan dan disimpan di GenieGo Integration Hub [Daftar], kredensial dienkripsi saat disimpan dengan AES-256-GCM dan disamarkan saat dibaca.

Daftar Periksa Akhir

  • Mengajukan dan membuat kontrak penggunaan GenieGo Pengiriman Hari Sama dengan tim operasional.
  • Menerima ID merchant, kunci API, dan wilayah layanan setelah menyelesaikan kontrak.
  • Mengonfirmasi api_key, merchant_id, dan region.
  • Memasukkan api_key, merchant_id, dan region di GenieGo [Daftar] lalu menyimpannya.

KG Inicis
Panduan Penerbitan API Key

Panduan yang merangkum proses menerbitkan INIAPI Key dan kunci tanda tangan standar web (SignKey) di admin merchant KG Inicis dan mendaftarkannya bersama ID toko (MID) di GenieGo Integration Hub.

Perkiraan waktuSekitar 5–15 menit
Akun yang dibutuhkanKontrak merchant KG Inicis · Akun admin merchant
Kunci yang diterbitkanMID · Kunci API · Kunci tanda tangan (SignKey)
HasilKredensial integrasi pembayaran

1. Sebelum Anda Mulai

KG Inicis hanya dapat menerbitkan kunci autentikasi setelah kontrak merchant selesai.

Diperlukan

Kontrak merchant · Akun admin merchant

Anda harus memiliki kontrak merchant KG Inicis yang sudah ditandatangani dan dapat masuk dengan akun admin merchant.

Kunci yang diterbitkan

INIAPI Key · Kunci tanda tangan standar web

Buat INIAPI Key di admin merchant, dan terbitkan kunci tanda tangan standar web sekaligus melalui pembuatan signkey pembayaran web.

Catatan: Pendaftaran memerlukan tiga item: ID toko (MID), kunci tanda tangan (SignKey), dan kunci API.

2. Langkah Penerbitan

Terbitkan sesuai urutan di bawah dan daftarkan di GenieGo.

1

Masuk ke admin merchant

Masuk ke admin merchant Inicis.

https://iniweb.inicis.com
2

Buka menu Informasi Tambahan

Buka [Info Toko] > [Info Kontrak] > [Informasi Tambahan].

Info Toko → Info Kontrak → Informasi Tambahan
3

Buat INIAPI Key

Buat INIAPI Key.

4

Buat kunci tanda tangan standar web

Klik [Pembuatan signkey pembayaran web] untuk menerbitkan kunci tanda tangan standar web (SignKey).

Pembuatan signkey pembayaran web
Setelah INIAPI Key dan kunci tanda tangan standar web keduanya dibuat, berarti telah diterbitkan dengan benar.

3. Informasi yang Akan Didaftarkan di GenieGo

BidangDeskripsi
MIDID toko (MID)
sign_keyKunci tanda tangan standar web yang diterbitkan (SignKey, tangani dengan hati-hati)
api_keyINIAPI Key yang diterbitkan
Masukkan dan simpan di GenieGo Integration Hub [Daftar], maka kredensial dienkripsi saat disimpan dengan AES-256-GCM dan disamarkan saat dibaca.

4. Masalah Umum

MasalahPenyebabSolusi
Menu Informasi Tambahan tidak terlihatKontrak merchant belum selesai, atau akun tanpa izinMasuk dengan akun admin merchant yang memiliki kontrak merchant yang sudah selesai.
Autentikasi pembayaran gagalKunci tanda tangan atau kunci API salah dimasukkanPeriksa kembali kunci tanda tangan·INIAPI Key yang diterbitkan di Info Toko > Info Kontrak > Informasi Tambahan.

Daftar Periksa Akhir

  • Menyelesaikan kontrak merchant KG Inicis dan masuk ke admin merchant.
  • Membuat INIAPI Key di Info Toko > Info Kontrak > Informasi Tambahan.
  • Menerbitkan kunci tanda tangan standar web melalui [Pembuatan signkey pembayaran web].
  • Memasukkan MID·sign_key·api_key di GenieGo [Daftar] dan menyimpan.

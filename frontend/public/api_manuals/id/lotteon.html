롯데온 (LOTTE ON)
Panduan Penerbitan API

Panduan langkah demi langkah yang bisa diikuti siapa saja untuk menerbitkan kredensial 롯데온 (LOTTE ON) (API key/token) dan mendaftarkannya di GenieGo Integration Hub.

Perkiraan waktuSekitar 5–15 menit
Akun diperlukanAkun 롯데온 (LOTTE ON)
Jumlah langkah5 langkah
HasilKredensial API

Langkah Penerbitan

Ikuti langkah-langkah di bawah untuk menerbitkan dan mendaftarkan kredensial Anda.

1

준비물: 롯데온 입점(사업자 인증 완료) 셀러 계정이 필요합니다.

2

롯데온 셀러 스토어센터(store.lotteon.com)에 로그인합니다.

3

[판매자정보] > [OpenAPI관리] > [정보설정]에서 호출 서버 IP를 등록합니다.

4

[키발급] 버튼을 눌러 OpenAPI 인증키를 발급하고 판매자(셀러) ID를 확인합니다.

5

GenieGo [등록]에 api_key(OpenAPI 인증키)와 seller_id를 입력해 저장합니다.

Daftar Periksa Akhir

  • 준비물: 롯데온 입점(사업자 인증 완료) 셀러 계정이 필요합니다.
  • 롯데온 셀러 스토어센터(store.lotteon.com)에 로그인합니다.
  • [판매자정보] > [OpenAPI관리] > [정보설정]에서 호출 서버 IP를 등록합니다.
  • [키발급] 버튼을 눌러 OpenAPI 인증키를 발급하고 판매자(셀러) ID를 확인합니다.
  • GenieGo [등록]에 api_key(OpenAPI 인증키)와 seller_id를 입력해 저장합니다.
Kunci yang dimasukkan di GenieGo Integration Hub [Daftar] akan dienkripsi saat disimpan dengan AES-256-GCM dan disamarkan saat dibaca.
Jaga keamanan kunci/secret Anda dan jangan pernah memublikasikannya. Jika dicurigai bocor, segera terbitkan ulang (rotasi).

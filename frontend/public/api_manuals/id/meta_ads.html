Meta Ads
Panduan Penerbitan API Key

Panduan langkah demi langkah yang ramah pemula untuk menerbitkan token akses pengguna sistem dari aplikasi bisnis Meta Anda dan mendaftarkannya bersama ID akun iklan (act_) di GenieGo Integration Hub untuk menghubungkan data iklan Meta Ads Anda.

Perkiraan waktuSekitar 10–20 menit
Akun yang dibutuhkanAkun pribadi Facebook · Akun iklan Meta
Kunci yang diterbitkanToken akses · ID akun iklan (act_)
HasilKredensial Marketing API

1. Sebelum Anda Mulai

Integrasi Meta Ads memerlukan akun bisnis dan akun iklan yang sedang aktif menjalankan iklan.

Diperlukan

Akun pribadi Facebook · Akun iklan

Anda memerlukan akun pribadi Facebook dan akun iklan Meta yang sedang aktif menjalankan iklan.

Kunci yang diterbitkan

Token akses · ID akun iklan (act_)

Anda akan memperoleh token pengguna sistem (token akses) dan ID akun iklan numerik yang diawali dengan act_.

Catatan: Mengakses data di luar bisnis Anda sendiri memerlukan Tinjauan Aplikasi (App Review). Jika Anda hanya menghubungkan akun iklan dalam bisnis Anda sendiri, Anda dapat melanjutkan tanpa tinjauan.

2. Langkah Penerbitan

Terbitkan sesuai urutan di bawah dan daftarkan di GenieGo.

1

Buat dan verifikasi akun bisnis

Buka situs di browser Anda, masuk, lalu buat akun bisnis melalui [Buat Bisnis] dan selesaikan verifikasi email.

business.facebook.com
2

Buat aplikasi

Klik [Aplikasi Saya] > [Buat Aplikasi] di kanan atas, pilih jenis "Bisnis", dan buat aplikasi.

developers.facebook.com → Aplikasi Saya → Buat Aplikasi
3

Tambahkan Marketing API

Pada dasbor aplikasi, temukan kartu "Marketing API" dan klik [Siapkan] untuk menambahkannya sebagai produk.

4

Buat pengguna sistem dan tetapkan izin

Buka Pengaturan Bisnis > [Pengguna] > [Pengguna Sistem] > [Tambah] untuk membuat pengguna sistem, lalu berikan izin akun iklan di bagian [Tetapkan Aset].

business.facebook.com/settings → Pengguna → Pengguna Sistem
5

Hasilkan token akses

Pada layar yang sama, klik [Hasilkan Token], pilih aplikasi Anda, centang izin (ads_read, ads_management), hasilkan token, dan salin nilainya.

6

Temukan ID akun iklan Anda

Di Pengelola Iklan, temukan ID akun iklan numerik yang diawali dengan act_.

adsmanager.facebook.com
Setelah token dihasilkan dengan izin ads_read dan ads_management dicentang, berarti token telah diterbitkan dengan benar.

3. Informasi untuk Didaftarkan di GenieGo

BidangDeskripsi
access_tokenToken akses pengguna sistem yang diterbitkan (tangani dengan hati-hati)
ad_account_idID akun iklan yang diawali dengan act_
Di GenieGo Integration Hub > [Daftar] kanal terkait, tempel token akses dan ID akun iklan (act_) Anda lalu simpan. Kredensial dienkripsi saat disimpan dengan AES-256-GCM dan disamarkan saat dibaca.

4. Masalah Umum

MasalahPenyebabSolusi
Data tidak dapat diminta karena kesalahan izinIzin token (ads_read·ads_management) tidak dicentang, atau izin akun iklan tidak ditetapkanTetapkan izin akun iklan ke pengguna sistem, dan centang izin ads_read·ads_management saat menghasilkan token.
Akses ke data bisnis lain ditolakTinjauan Aplikasi (App Review) belum selesaiUntuk mengakses data di luar bisnis Anda sendiri, ajukan dan selesaikan Tinjauan Aplikasi.
Kesalahan format ID akun iklanAwalan act_ hilang, atau hanya angkaDi Pengelola Iklan, temukan ID lengkap yang diawali dengan act_ dan masukkan.

Daftar Periksa Akhir

  • Membuat akun bisnis di business.facebook.com dan menyelesaikan verifikasi email.
  • Membuat aplikasi jenis Bisnis di developers.facebook.com dan menambahkan Marketing API.
  • Membuat pengguna sistem dan menetapkan izin akun iklan.
  • Menghasilkan dan menyalin token akses dengan izin ads_read·ads_management.
  • Menemukan ID akun iklan yang diawali dengan act_.
  • Memasukkan access_token·ad_account_id di GenieGo [Daftar] dan menyimpannya.

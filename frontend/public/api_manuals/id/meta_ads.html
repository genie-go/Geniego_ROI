Meta Ads
Panduan Penerbitan API

Panduan langkah demi langkah yang bisa diikuti siapa saja untuk menerbitkan kredensial Meta Ads (API key/token) dan mendaftarkannya di GenieGo Integration Hub.

Perkiraan waktuSekitar 5–15 menit
Akun diperlukanAkun Meta Ads
Jumlah langkah8 langkah
HasilKredensial API

Langkah Penerbitan

Ikuti langkah-langkah di bawah untuk menerbitkan dan mendaftarkan kredensial Anda.

1

Yang dibutuhkan: akun pribadi Facebook dan akun iklan Meta yang sedang menjalankan iklan.

2

Buka business.facebook.com di browser lalu masuk, klik [Buat Bisnis] untuk membuat akun bisnis, kemudian selesaikan verifikasi email.

3

Buka developers.facebook.com → [Aplikasi Saya] di kanan atas → [Buat Aplikasi], lalu pilih tipe "Bisnis" untuk membuat aplikasi.

4

Di dasbor aplikasi, cari kartu "Marketing API" dan klik [Siapkan] untuk menambahkannya sebagai produk.

5

Pengaturan Bisnis (business.facebook.com/settings) → [Pengguna] > [Pengguna Sistem] → [Tambah] untuk membuat pengguna sistem, lalu beri izin atas akun iklan melalui [Tetapkan Aset].

6

Pada layar yang sama, klik [Buat Token], pilih aplikasi, centang izin (ads_read, ads_management), lalu buat token dan salin nilainya.

7

Di Pengelola Iklan (adsmanager.facebook.com), periksa ID akun iklan (angka yang diawali act_).

8

Klik [Daftar] di Hub Integrasi GenieGo untuk channel terkait, tempel "access token" dan "ID akun iklan (act_)" lalu simpan untuk menyelesaikan integrasi.

Jaga keamanan kunci/secret Anda dan jangan pernah memublikasikannya. Jika dicurigai bocor, segera terbitkan ulang (rotasi).

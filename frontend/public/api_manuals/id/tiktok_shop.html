TikTok Shop
Panduan Penerbitan API Key

Panduan langkah demi langkah yang ramah pemula untuk membuat aplikasi (App Key·App Secret·Service ID) di TikTok Shop Partner Center, menerbitkan access_token melalui otorisasi toko, dan mendaftarkannya di GenieGo Integration Hub.

Perkiraan waktuSekitar 10–15 menit
Akun yang dibutuhkanAkun penjual TikTok Shop
Kunci yang diterbitkanApp Key · App Secret · Access Token
HasilKredensial API

1. Sebelum Anda Mulai

Token hanya diterbitkan setelah Anda membuat aplikasi di Partner Center dan mengotorisasi sebuah toko.

Diperlukan

Akun penjual TikTok Shop

Anda memerlukan akun penjual TikTok Shop dan harus mendaftar sebagai pengembang di partner.tiktokshop.com (Partner Center).

Kunci yang diterbitkan

App Key · App Secret · Access Token

Membuat aplikasi memberi Anda App Key·App Secret·Service ID, dan Anda memperoleh access_token melalui otorisasi toko.

Catatan: Tinjauan aplikasi diperlukan saat beralih ke produksi.

2. Langkah Penerbitan

Terbitkan sesuai urutan di bawah dan daftarkan di GenieGo.

1

Daftar sebagai pengembang di Partner Center

Daftar sebagai pengembang di partner.tiktokshop.com (Partner Center).

https://partner.tiktokshop.com
2

Buat aplikasi

Buat aplikasi melalui [App & Service] > [Create App] dan periksa App Key, App Secret, Service ID.

App & Service → Create App
3

Aktifkan izin API

Di pengaturan aplikasi, aktifkan izin API yang diperlukan (Shop Authorized Information, Product, Order).

4

Otorisasi toko → terbitkan access_token

Di [Get shop authorization], pilih Wilayah (Region) untuk mengotorisasi toko penjual, lalu access_token akan diterbitkan.

Get shop authorization
5

Verifikasi kunci

Verifikasi App Key·App Secret dan access_token yang diterbitkan. Jika perlu, periksa juga Shop Cipher.

Setelah access_token diterbitkan usai otorisasi toko, berarti telah diproses dengan benar.

3. Informasi yang Perlu Didaftarkan di GenieGo

BidangDeskripsi
app_keyApp Key yang diterbitkan
app_secretApp Secret yang diterbitkan (tangani dengan hati-hati)
access_tokenToken akses yang diterbitkan melalui otorisasi toko
shop_cipherShop Cipher (opsional)
Masukkan app_key·app_secret·access_token di [Daftar] GenieGo Integration Hub lalu simpan. Kredensial dienkripsi saat disimpan dengan AES-256-GCM dan disamarkan saat dibaca.

4. Masalah Umum

MasalahPenyebabSolusi
access_token tidak diterbitkanOtorisasi toko (Get shop authorization) belum selesai, atau Wilayah (Region) belum dipilihDi [Get shop authorization], pilih Wilayah dan otorisasi toko penjual.
Kesalahan izinIzin API yang diperlukan belum diaktifkanDi pengaturan aplikasi, aktifkan izin Shop Authorized Information·Product·Order.
Panggilan produksi ditolakBelum lulus tinjauan peralihan produksiSelesaikan tinjauan aplikasi yang diperlukan untuk beralih ke produksi.

Daftar Periksa Akhir

  • Telah mendaftar sebagai pengembang di partner.tiktokshop.com.
  • Telah memeriksa App Key·App Secret·Service ID melalui App & Service > Create App.
  • Telah mengaktifkan izin API yang diperlukan (Shop/Product/Order).
  • Telah menerbitkan access_token melalui otorisasi toko (Get shop authorization).
  • Telah memasukkan app_key·app_secret·access_token di GenieGo [Daftar] dan menyimpannya.

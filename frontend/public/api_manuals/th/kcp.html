3. ข้อมูลที่ต้องลงทะเบียนใน GenieGo

ฟิลด์คำอธิบาย
site_cdรหัสไซต์ที่ออกแล้ว
site_keyคีย์ไซต์ที่ออกแล้ว (จัดการด้วยความระมัดระวัง)
ป้อนค่าใน GenieGo Integration Hub ที่ [ลงทะเบียน] แล้วบันทึก ข้อมูลรับรองจะถูกเข้ารหัสขณะจัดเก็บด้วย AES-256-GCM และปิดบังเมื่ออ่าน

4. ปัญหาที่พบบ่อย

ปัญหาสาเหตุวิธีแก้ไข
ไม่สามารถลงชื่อเข้าใช้ผู้ดูแลได้สัญญาร้านค้ายังไม่เสร็จสมบูรณ์ลงชื่อเข้าใช้ด้วยบัญชีผู้ดูแลที่สัญญาร้านค้าเสร็จสมบูรณ์แล้ว
การยืนยันการชำระเงินล้มเหลวป้อนรหัสไซต์หรือคีย์ไซต์ไม่ถูกต้องตรวจสอบ site_cd·site_key ที่ออกในผู้ดูแลร้านค้าอีกครั้ง

รายการตรวจสอบขั้นสุดท้าย

  • ทำสัญญาร้านค้า NHN KCP เสร็จสมบูรณ์และลงชื่อเข้าใช้ผู้ดูแลร้านค้าแล้ว
  • ออกและตรวจสอบรหัสไซต์ (site_cd) และคีย์ไซต์แล้ว
  • ป้อน site_cd·site_key ใน GenieGo [ลงทะเบียน] และบันทึกแล้ว
  • ยืนยันแล้วว่าข้อมูลรับรองแสดงแบบปิดบังหลังจากบันทึก

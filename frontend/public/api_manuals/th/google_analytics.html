Google Analytics 4
คู่มือการออก API Key

คู่มือทีละขั้นตอนที่เป็นมิตรกับผู้เริ่มต้น สำหรับการออกรหัสการวัด (G-) และ Measurement Protocol API Secret จากสตรีมข้อมูล Google Analytics 4 ของคุณ และลงทะเบียนใน GenieGo Integration Hub

เวลาโดยประมาณประมาณ 5–10 นาที
บัญชีที่ต้องใช้สิทธิ์จัดการพร็อพเพอร์ตี้ GA4
คีย์ที่ออกรหัสการวัด (G-) · API Secret
ผลลัพธ์ข้อมูลรับรอง Measurement Protocol

1. ก่อนเริ่มต้น

รหัสการวัด GA4 และ API Secret สามารถออกได้จากบัญชีที่มีสิทธิ์จัดการพร็อพเพอร์ตี้เท่านั้น

จำเป็น

สิทธิ์จัดการพร็อพเพอร์ตี้ GA4

คุณต้องมีสิทธิ์ในการจัดการพร็อพเพอร์ตี้ Google Analytics 4 เพียงสิทธิ์การดูอย่างเดียวไม่สามารถสร้าง API Secret ได้

คีย์ที่ออก

รหัสการวัด (G-) · API Secret

ตรวจสอบทั้งรหัสการวัดของสตรีมเว็บ (ขึ้นต้นด้วย G-) และ Measurement Protocol API Secret

2. ขั้นตอนการออก

ออกตามลำดับด้านล่างและลงทะเบียนใน GenieGo

1

ไปที่สตรีมข้อมูล

ใน Google Analytics ไปที่ [ผู้ดูแลระบบ] > [สตรีมข้อมูล]

analytics.google.com → ผู้ดูแลระบบ → สตรีมข้อมูล
2

ค้นหารหัสการวัด

เลือกสตรีมเว็บและค้นหารหัสการวัด (ขึ้นต้นด้วย G-)

3

สร้าง API Secret

บนหน้าจอเดียวกัน สร้าง [Measurement Protocol API secret]

เมื่อยืนยันทั้งรหัสการวัด (G-) และ API Secret แล้ว แสดงว่าออกอย่างถูกต้อง

3. ข้อมูลที่ต้องลงทะเบียนใน GenieGo

ฟิลด์คำอธิบาย
measurement_idรหัสการวัด (ขึ้นต้นด้วย G-)
api_secretMeasurement Protocol API Secret ที่สร้างแล้ว (จัดการด้วยความระมัดระวัง)
กรอกใน [ลงทะเบียน] ของ GenieGo Integration Hub แล้วบันทึก ข้อมูลรับรองจะถูกเข้ารหัสขณะจัดเก็บด้วย AES-256-GCM และปิดบังเมื่ออ่าน

4. ปัญหาที่พบบ่อย

ปัญหาสาเหตุวิธีแก้ไข
ไม่สามารถสร้าง API Secret ได้สิทธิ์จัดการพร็อพเพอร์ตี้ไม่เพียงพอลงชื่อเข้าใช้ด้วยบัญชีที่มีสิทธิ์จัดการพร็อพเพอร์ตี้ GA4
ไม่มีการเก็บรวบรวมข้อมูลกรอกรหัสการวัดหรือ API Secret ผิดตรวจสอบรหัสการวัด (G-) และ API Secret ของสตรีมเว็บอีกครั้ง

รายการตรวจสอบขั้นสุดท้าย

  • ลงชื่อเข้าใช้ด้วยบัญชีที่มีสิทธิ์จัดการพร็อพเพอร์ตี้ GA4 แล้ว
  • พบรหัสการวัด (G-) ของสตรีมเว็บใน [ผู้ดูแลระบบ > สตรีมข้อมูล] แล้ว
  • สร้าง Measurement Protocol API Secret แล้ว
  • กรอก measurement_id·api_secret ใน GenieGo [ลงทะเบียน] และบันทึกแล้ว

คู่มือการออก API Key

คู่มือทีละขั้นตอนที่เป็นมิตรกับผู้เริ่มต้น สำหรับการสร้างแอป (App Key·App Secret·Service ID) ใน TikTok Shop Partner Center ออก access_token ผ่านการอนุญาตร้านค้า และลงทะเบียนใน GenieGo Integration Hub

1. ก่อนเริ่มต้น

โทเค็นจะออกให้ก็ต่อเมื่อคุณสร้างแอปใน Partner Center และอนุญาตร้านค้าแล้วเท่านั้น

จำเป็น

บัญชีผู้ขาย TikTok Shop

คุณต้องมีบัญชีผู้ขาย TikTok Shop และต้องลงทะเบียนเป็นนักพัฒนาที่ partner.tiktokshop.com (Partner Center)

คีย์ที่ออก

App Key · App Secret · Access Token

การสร้างแอปจะให้ App Key·App Secret·Service ID และคุณจะได้รับ access_token ผ่านการอนุญาตร้านค้า

หมายเหตุ: เมื่อเปลี่ยนไปใช้งานจริง (production) จำเป็นต้องผ่านการตรวจสอบแอป

2. ขั้นตอนการออก

ออกตามลำดับด้านล่างและลงทะเบียนใน GenieGo

1

ลงทะเบียนเป็นนักพัฒนาใน Partner Center

ลงทะเบียนเป็นนักพัฒนาที่ partner.tiktokshop.com (Partner Center)

https://partner.tiktokshop.com
2

สร้างแอป

สร้างแอปผ่าน [App & Service] > [Create App] และตรวจสอบ App Key, App Secret, Service ID

App & Service → Create App
3

เปิดใช้งานสิทธิ์ API

ในการตั้งค่าแอป เปิดใช้งานสิทธิ์ API ที่จำเป็น (Shop Authorized Information, Product, Order)

4

การอนุญาตร้านค้า → ออก access_token

ใน [Get shop authorization] เลือกภูมิภาค (Region) เพื่ออนุญาตร้านค้าของผู้ขาย แล้วระบบจะออก access_token

Get shop authorization
5

ตรวจสอบคีย์

ตรวจสอบ App Key·App Secret และ access_token ที่ออกแล้ว หากจำเป็นให้ตรวจสอบ Shop Cipher ด้วย

เมื่อ access_token ออกหลังจากการอนุญาตร้านค้าแล้ว แสดงว่าดำเนินการอย่างถูกต้อง

3. ข้อมูลที่ต้องลงทะเบียนใน GenieGo

ฟิลด์คำอธิบาย
app_keyApp Key ที่ออกแล้ว
app_secretApp Secret ที่ออกแล้ว (จัดการด้วยความระมัดระวัง)
access_tokenโทเค็นการเข้าถึงที่ออกผ่านการอนุญาตร้านค้า
shop_cipherShop Cipher (ไม่บังคับ)
ป้อน app_key·app_secret·access_token ใน [ลงทะเบียน] ของ GenieGo Integration Hub แล้วบันทึก ข้อมูลรับรองจะถูกเข้ารหัสขณะจัดเก็บด้วย AES-256-GCM และปิดบังเมื่ออ่าน

4. ปัญหาที่พบบ่อย

ปัญหาสาเหตุวิธีแก้ไข
ไม่ออก access_tokenการอนุญาตร้านค้า (Get shop authorization) ยังไม่เสร็จสมบูรณ์ หรือไม่ได้เลือกภูมิภาค (Region)ใน [Get shop authorization] เลือกภูมิภาคเพื่ออนุญาตร้านค้าของผู้ขาย
ข้อผิดพลาดด้านสิทธิ์ยังไม่ได้เปิดใช้งานสิทธิ์ API ที่จำเป็นในการตั้งค่าแอป เปิดใช้งานสิทธิ์ Shop Authorized Information·Product·Order
การเรียกใช้งานจริงถูกปฏิเสธไม่ผ่านการตรวจสอบการเปลี่ยนไปใช้งานจริงดำเนินการตรวจสอบแอปที่จำเป็นสำหรับการเปลี่ยนไปใช้งานจริงให้เสร็จสมบูรณ์

รายการตรวจสอบขั้นสุดท้าย

  • ลงทะเบียนเป็นนักพัฒนาที่ partner.tiktokshop.com แล้ว
  • ตรวจสอบ App Key·App Secret·Service ID ผ่าน App & Service > Create App แล้ว
  • เปิดใช้งานสิทธิ์ API ที่จำเป็น (Shop/Product/Order) แล้ว
  • ออก access_token ผ่านการอนุญาตร้านค้า (Get shop authorization) แล้ว
  • ป้อน app_key·app_secret·access_token ใน GenieGo [ลงทะเบียน] และบันทึกแล้ว

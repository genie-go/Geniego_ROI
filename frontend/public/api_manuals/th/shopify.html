คู่มือการออก API Key

คู่มือทีละขั้นตอนในการสร้างแอปแบบกำหนดเองในผู้ดูแลระบบ Shopify (Admin) เพื่อออก Admin API access token และลงทะเบียนพร้อมกับโดเมนร้านค้าใน GenieGo Integration Hub เพื่อเชื่อมต่อร้านค้า Shopify ของคุณ

1. ก่อนเริ่มต้น

แอปแบบกำหนดเองของ Shopify สามารถสร้างได้เฉพาะผู้ที่มีสิทธิ์ผู้ดูแลระบบร้านค้าเท่านั้น

จำเป็น

สิทธิ์ผู้ดูแลระบบร้านค้า

คุณต้องมีสิทธิ์ผู้ดูแลระบบ (Admin) ของร้านค้า Shopify จึงจะสามารถสร้างแอปแบบกำหนดเองและออก token ได้

คีย์ที่ออก

โดเมนร้านค้า · Admin API token

คุณจะได้รับโดเมนร้านค้าในรูปแบบ xxx.myshopify.com และ Admin API access token ที่แสดงหลังจากติดตั้งแอป

หมายเหตุ: Admin API access token จะแสดงเพียงครั้งเดียวทันทีหลังการติดตั้ง ดังนั้นคุณต้องคัดลอกทันทีเมื่อปรากฏ

2. ขั้นตอนการออก

ออกตามลำดับด้านล่างและลงทะเบียนใน GenieGo

1

ไปที่แอปและช่องทางการขาย

ในผู้ดูแลระบบ Shopify (Admin) ไปที่ [การตั้งค่า] > [แอปและช่องทางการขาย]

https://www.shopify.com/admin/settings/apps
2

สร้างแอปแบบกำหนดเอง

คลิก [พัฒนาแอป] > [สร้างแอป] เพื่อสร้างแอปแบบกำหนดเอง

พัฒนาแอป → สร้างแอป
3

เลือกขอบเขต · ติดตั้งแอป

เลือก Admin API scopes และติดตั้งแอป

4

คัดลอก access token

คัดลอก Admin API access token ที่แสดงหลังการติดตั้งทันที (แสดงเพียงครั้งเดียว)

เมื่อ Admin API access token ปรากฏหลังจากติดตั้งแอป แสดงว่าออกสำเร็จแล้ว เนื่องจากแสดงเพียงครั้งเดียว โปรดคัดลอกทันที

3. ข้อมูลที่ต้องลงทะเบียนใน GenieGo

ฟิลด์คำอธิบาย
shop_domainโดเมนร้านค้าในรูปแบบ xxx.myshopify.com
access_tokenAdmin API access token ที่ออกแล้ว (โปรดจัดการด้วยความระมัดระวัง)
ใน GenieGo [ลงทะเบียน] ให้ป้อน shop_domain (xxx.myshopify.com) และ access_token ของคุณแล้วบันทึก ข้อมูลรับรองจะถูกเข้ารหัสขณะจัดเก็บด้วย AES-256-GCM และถูกปกปิดเมื่ออ่าน

4. ปัญหาที่พบบ่อย

ปัญหาสาเหตุวิธีแก้ไข
ไม่สามารถดู token อีกครั้งได้Admin API access token แสดงเพียงครั้งเดียวหากคุณทำ token หาย ให้ออก token ใหม่ในแอปแบบกำหนดเองและคัดลอกทันที
ข้อผิดพลาดสิทธิ์ไม่เพียงพอยังไม่ได้เลือก Admin API scopesเลือก Admin API scopes ที่จำเป็น จากนั้นติดตั้งแอปใหม่
ข้อผิดพลาดโดเมนป้อนรูปแบบโดเมนร้านค้าไม่ถูกต้องป้อนโดเมนร้านค้าให้ถูกต้องในรูปแบบ xxx.myshopify.com

รายการตรวจสอบขั้นสุดท้าย

  • เข้าสู่ระบบด้วยสิทธิ์ผู้ดูแลระบบร้านค้า Shopify แล้ว
  • สร้างแอปแบบกำหนดเองใน การตั้งค่า > แอปและช่องทางการขาย > พัฒนาแอป แล้ว
  • เลือก Admin API scopes และติดตั้งแอปแล้ว
  • คัดลอก Admin API access token ที่แสดงหลังการติดตั้งทันทีแล้ว
  • ป้อน shop_domain·access_token ใน GenieGo [ลงทะเบียน] และบันทึกแล้ว

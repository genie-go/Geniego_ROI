Auction (옥션)
คู่มือการออก API

คู่มือทีละขั้นตอนที่ใครก็ทำตามได้ เพื่อออกข้อมูลรับรอง Auction (옥션) (API key/โทเค็น) และลงทะเบียนใน GenieGo Integration Hub

เวลาโดยประมาณประมาณ 5–15 นาที
บัญชีที่ต้องใช้บัญชี Auction (옥션)
จำนวนขั้นตอน6 ขั้นตอน
ผลลัพธ์ข้อมูลรับรอง API

ขั้นตอนการออก

ทำตามขั้นตอนด้านล่างเพื่อออกและลงทะเบียนข้อมูลรับรองของคุณ

1

สิ่งที่ต้องเตรียม: บัญชีผู้ขาย Auction และ Gmarket แยกกัน และ ID มาสเตอร์ ESM Plus

2

หลังสมัครเป็นผู้ขายที่ Auction และ Gmarket แล้ว เข้าสู่ระบบ ESM Plus (esmplus.com) แล้วสร้าง ID มาสเตอร์

3

เข้า etapi.gmarket.com (ESM Trading API) และสมัครใช้งาน API (ใช้ระบบอนุมัติ — อาจถูกปฏิเสธได้)

4

ตอนสมัคร ให้ส่ง 3 อย่าง: ID ผู้ขาย Auction, ID ผู้ขาย Gmarket และ ID มาสเตอร์ ESM

5

เมื่ออนุมัติแล้ว API key จะถูกออกให้ (สอบถาม: etapihelp@gmail.com)

6

กรอก api_key และ seller_id ที่ออกให้ ที่ [ลงทะเบียน] ของ GenieGo แล้วบันทึก

  • เข้า etapi.gmarket.com (ESM Trading API) และสมัครใช้งาน API (ใช้ระบบอนุมัติ — อาจถูกปฏิเสธได้)
  • ตอนสมัคร ให้ส่ง 3 อย่าง: ID ผู้ขาย Auction, ID ผู้ขาย Gmarket และ ID มาสเตอร์ ESM
  • เมื่ออนุมัติแล้ว API key จะถูกออกให้ (สอบถาม: etapihelp@gmail.com)

คีย์ที่ออก

API key · Seller ID

หลังจาก ESM Trading API อนุมัติแล้ว ให้ยืนยัน API key ที่ออกพร้อมกับ Seller ID

หมายเหตุ: ESM Trading API ใช้ระบบอนุมัติ และคำขออาจถูกปฏิเสธได้ เมื่อยื่นคำขอ คุณต้องส่งสามรายการ ได้แก่ Auction seller ID, Gmarket seller ID และ ESM master ID

การออกคีย์

เมื่อมีการออก API key หลังจากการอนุมัติเสร็จสมบูรณ์ แสดงว่าดำเนินการถูกต้องแล้ว สำหรับคำถามเกี่ยวกับการอนุมัติ ติดต่อ etapihelp@gmail.com

  • ยื่นขอใช้ ESM Trading API ที่ etapi.gmarket.com และได้รับอนุมัติแล้ว
  • ยืนยัน api_key และ seller_id ที่ออกแล้ว

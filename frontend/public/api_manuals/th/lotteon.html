1. ก่อนเริ่มต้น

คีย์การยืนยันตัวตน OpenAPI สามารถออกได้จากบัญชีผู้ขายที่ลงทะเบียนและยืนยันตัวตนธุรกิจเสร็จสมบูรณ์แล้วเท่านั้น

จำเป็น

บัญชีผู้ขายที่ลงทะเบียน

คุณต้องมีบัญชีผู้ขายที่ลงทะเบียนกับ LOTTE ON (ยืนยันตัวตนธุรกิจเสร็จสมบูรณ์แล้ว) หากยังยืนยันตัวตนธุรกิจไม่เสร็จสมบูรณ์ จะไม่สามารถออกคีย์ได้

คีย์ที่ออก

คีย์การยืนยันตัวตน OpenAPI · รหัสผู้ขาย

ตรวจสอบทั้งคีย์การยืนยันตัวตน OpenAPI ที่ออกผ่าน [ออกคีย์] และรหัสผู้ขายของคุณ

หมายเหตุ: คุณต้องลงทะเบียน IP ของเซิร์ฟเวอร์ที่เรียกใช้ก่อนการออกคีย์

2. ขั้นตอนการออก

ออกตามลำดับด้านล่างและลงทะเบียนใน GenieGo

1

ลงชื่อเข้าใช้ Seller Store Center

ลงชื่อเข้าใช้ LOTTE ON Seller Store Center

https://store.lotteon.com
2

เข้าสู่การจัดการ OpenAPI และลงทะเบียน IP

ลงทะเบียน IP ของเซิร์ฟเวอร์ที่เรียกใช้ในส่วน [ข้อมูลผู้ขาย] > [การจัดการ OpenAPI] > [การตั้งค่าข้อมูล]

ข้อมูลผู้ขาย → การจัดการ OpenAPI → การตั้งค่าข้อมูล
3

ออกคีย์

คลิกปุ่ม [ออกคีย์] เพื่อออกคีย์การยืนยันตัวตน OpenAPI และยืนยันรหัสผู้ขายของคุณ

ออกคีย์
เมื่อออกคีย์การยืนยันตัวตน OpenAPI หลังจาก [ออกคีย์] แล้ว แสดงว่าดำเนินการอย่างถูกต้อง

3. ข้อมูลที่ต้องลงทะเบียนใน GenieGo

ฟิลด์คำอธิบาย
api_keyคีย์การยืนยันตัวตน OpenAPI ที่ออกแล้ว
seller_idรหัสผู้ขาย
เมื่อคุณป้อนและบันทึกใน GenieGo Integration Hub [ลงทะเบียน] แล้ว ข้อมูลรับรองจะถูกเข้ารหัสขณะจัดเก็บด้วย AES-256-GCM และปิดบังเมื่ออ่าน

4. ปัญหาที่พบบ่อย

ปัญหาสาเหตุวิธีแก้ไข
ไม่เห็นปุ่ม [ออกคีย์]ยังยืนยันตัวตนธุรกิจไม่เสร็จสมบูรณ์ หรือลงทะเบียนร้านค้าไม่เสร็จสมบูรณ์ลงชื่อเข้าใช้ด้วยบัญชีผู้ขายที่ลงทะเบียนและยืนยันตัวตนธุรกิจเสร็จสมบูรณ์แล้ว
ข้อผิดพลาดการยืนยันตัวตนเมื่อเรียกใช้ยังไม่ได้ลงทะเบียน IP ของเซิร์ฟเวอร์ที่เรียกใช้ตรวจสอบ IP ของเซิร์ฟเวอร์ที่เรียกใช้อีกครั้งในส่วน [การจัดการ OpenAPI] > [การตั้งค่าข้อมูล]

รายการตรวจสอบขั้นสุดท้าย

  • ลงชื่อเข้าใช้ด้วยบัญชีผู้ขายที่ลงทะเบียนกับ LOTTE ON และยืนยันตัวตนธุรกิจเสร็จสมบูรณ์แล้ว
  • ลงทะเบียน IP ของเซิร์ฟเวอร์ที่เรียกใช้ในส่วน [การจัดการ OpenAPI] > [การตั้งค่าข้อมูล] แล้ว
  • ออกคีย์การยืนยันตัวตน OpenAPI ผ่าน [ออกคีย์] และยืนยันรหัสผู้ขายแล้ว
  • ป้อน api_key·seller_id ใน GenieGo [ลงทะเบียน] และบันทึกแล้ว

1. ก่อนเริ่มต้น

CJ Logistics International Express เป็นช่องทางแบบสัญญาที่จะออกรหัสลูกค้า (สัญญา) หลังจากทำสัญญาการขาย

จำเป็น

สัญญา International Express

ในการใช้ CJ Logistics International Express ต้องทำสัญญากับผู้ดูแลฝ่ายขายก่อน หากไม่มีสัญญา จะไม่สามารถออกรหัสลูกค้า (สัญญา) ได้

คีย์ที่ออก

API key · รหัสลูกค้า (สัญญา)

หลังจากทำสัญญาเสร็จสมบูรณ์ คุณจะได้รับรหัสลูกค้า (สัญญา) และ API key

หมายเหตุ: ช่องทางนี้ไม่ใช่วิธีการออกโดยตรงจากคอนโซลแบบบริการตนเอง แต่เป็นวิธีการรับข้อมูลรับรองผ่านสัญญาการขาย

3. ข้อมูลที่ต้องลงทะเบียนใน GenieGo

4. ปัญหาที่พบบ่อย

ปัญหาสาเหตุวิธีแก้ไข
ไม่ได้รับรหัสลูกค้า (สัญญา)สัญญา International Express ยังไม่เสร็จสมบูรณ์ดำเนินการทำสัญญากับผู้ดูแลฝ่ายขายของ CJ Logistics International Express จากนั้นจึงรับข้อมูลรับรอง
ข้อผิดพลาดการเชื่อมต่อ International Expressป้อนรหัสลูกค้า (สัญญา) (cust_code) ผิดตรวจสอบรหัสลูกค้า (สัญญา) ที่ได้รับอีกครั้งและป้อนให้ถูกต้อง

  • ได้รับรหัสลูกค้า (สัญญา) และ API key หลังจากทำสัญญาเสร็จสมบูรณ์แล้ว
  • ตรวจสอบ api_key·cust_code แล้ว
  • ป้อน api_key·cust_code ใน GenieGo [ลงทะเบียน] และบันทึกแล้ว

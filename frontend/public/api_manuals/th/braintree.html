คู่มือทีละขั้นตอนที่เป็นมิตรกับผู้เริ่มต้น สำหรับการค้นหาข้อมูลรับรอง Production API (Merchant ID·Public Key·Private Key) ใน Braintree (PayPal) Control Panel และลงทะเบียนใน GenieGo Integration Hub

เวลาโดยประมาณประมาณ 5–10 นาที
บัญชีที่ต้องใช้บัญชีร้านค้า Braintree (PayPal) (ต้องได้รับการอนุมัติ)
คีย์ที่ออกMerchant ID · Public Key · Private Key
ผลลัพธ์ข้อมูลรับรอง Production API

1. ก่อนเริ่มต้น

ข้อมูลรับรอง Braintree (PayPal) จะดูได้ก็ต่อเมื่อบัญชีร้านค้าของคุณได้รับการอนุมัติแล้วเท่านั้น

จำเป็น

บัญชีร้านค้า Braintree (PayPal)

ก่อนอื่นคุณต้องมีบัญชีร้านค้า Braintree (PayPal) ที่ได้รับการอนุมัติ ข้อมูลรับรอง Production จะดูได้ก็ต่อเมื่อบัญชีร้านค้าของคุณได้รับการอนุมัติแล้วเท่านั้น

คีย์ที่ออก

Merchant ID · Public/Private Key

ใน Control Panel ไปที่ Settings > API Keys เพื่อดู Merchant ID·Public Key·Private Key ของสภาพแวดล้อม Production

หมายเหตุ: ต้องใช้ข้อมูลรับรองของสภาพแวดล้อม Production เท่านั้น ระวังอย่าสับสนกับคีย์ Sandbox

2. ขั้นตอนการยื่นขอออก

หลังจากบัญชีร้านค้าได้รับการอนุมัติแล้ว ให้ดูข้อมูลรับรองตามลำดับด้านล่างและลงทะเบียนใน GenieGo

1

การอนุมัติบัญชีร้านค้า

ดำเนินการอนุมัติบัญชีร้านค้า Braintree (PayPal) ให้เสร็จก่อน คุณจะไปยังขั้นตอนถัดไปได้ก็ต่อเมื่อการอนุมัติเสร็จสมบูรณ์แล้ว

2

เข้าสู่ระบบ Control Panel

เข้าสู่ระบบ Braintree Control Panel

braintreegateway.com
3

ไปที่ API Keys

ไปที่เมนู [Settings] > [API Keys]

Settings → API Keys
4

ดูข้อมูลรับรอง

ดู Merchant ID·Public Key·Private Key ของสภาพแวดล้อม Production

หากเห็น Merchant ID·Public Key·Private Key ของ Production ใน Settings > API Keys แสดงว่าปกติ

3. ข้อมูลที่ต้องลงทะเบียนใน GenieGo

ฟิลด์คำอธิบาย
merchant_idProduction Merchant ID
public_keyProduction Public Key
private_keyProduction Private Key (จัดการด้วยความระมัดระวัง)
ป้อน merchant_id·public_key·private_key ใน [ลงทะเบียน] ของ GenieGo Integration Hub แล้วบันทึก ข้อมูลรับรองจะถูกเข้ารหัสขณะจัดเก็บด้วย AES-256-GCM และปิดบังเมื่ออ่าน

4. ปัญหาที่พบบ่อย

ปัญหาสาเหตุวิธีแก้ไข
ไม่มีข้อมูลรับรองในเมนู API Keysการอนุมัติบัญชีร้านค้ายังไม่เสร็จสมบูรณ์ดำเนินการอนุมัติบัญชีร้านค้า Braintree (PayPal) ให้เสร็จก่อน
การยืนยันตัวตนล้มเหลวใช้คีย์ Sandbox หรือป้อน Private Key ผิดตรวจสอบ Merchant ID·Public Key·Private Key ของสภาพแวดล้อม Production อีกครั้ง

รายการตรวจสอบขั้นสุดท้าย

  • บัญชีร้านค้า Braintree (PayPal) ได้รับการอนุมัติแล้ว
  • ดูข้อมูลรับรอง Production ใน Settings > API Keys ของ Control Panel แล้ว
  • ยืนยัน Merchant ID·Public Key·Private Key แล้ว
  • ป้อน merchant_id·public_key·private_key ใน GenieGo [ลงทะเบียน] และบันทึกแล้ว

1. ก่อนเริ่มต้น

Smart Tracker (Unified Tracking) เป็นช่องทางออกคีย์ด้วยตนเองที่ใช้งานได้เพียงสมัครสมาชิกโดยไม่ต้องทำสัญญาแยกต่างหาก

จำเป็น

บัญชี SweetTracker

คุณเพียงต้องมีบัญชีที่ลงทะเบียนไว้ที่ tracking.sweettracker.co.kr สามารถออกคีย์ได้เพียงสมัครสมาชิกโดยไม่ต้องทำสัญญาแยกต่างหาก

คีย์ที่ออก

t_key (คีย์ติดตามแบบรวม)

ออก t_key เพียงหนึ่งคีย์ ซึ่งเป็นคีย์ติดตามแบบรวมของทุกบริษัทขนส่ง แล้วการติดตามการจัดส่งของทุกบริษัทขนส่งก็จะเชื่อมต่อ

หมายเหตุ: ป้อน t_key ในฟิลด์ api_key เมื่อลงทะเบียนใน GenieGo

ลงทะเบียนใน GenieGo

ป้อน t_key (api_key) ใน GenieGo [ลงทะเบียน] แล้วการติดตามการจัดส่งของทุกบริษัทขนส่งจะเชื่อมต่อ

เมื่อออก t_key แล้ว แสดงว่าออกอย่างถูกต้อง และการติดตามการจัดส่งของทุกบริษัทขนส่งจะเชื่อมต่อทันทีเมื่อลงทะเบียน

3. ข้อมูลที่ต้องลงทะเบียนใน GenieGo

ฟิลด์คำอธิบาย
api_keyt_key ที่ออกแล้ว (คีย์ติดตามแบบรวมของทุกบริษัทขนส่ง)
ป้อนใน GenieGo Integration Hub [ลงทะเบียน] แล้วบันทึก ข้อมูลรับรองจะถูกเข้ารหัสขณะจัดเก็บด้วย AES-256-GCM และปิดบังเมื่ออ่าน การติดตามการจัดส่งของทุกบริษัทขนส่งจะเชื่อมต่อทันทีเมื่อลงทะเบียน

4. ปัญหาที่พบบ่อย

ปัญหาสาเหตุวิธีแก้ไข
ไม่พบเมนูการออก t_keyยังไม่สมัครหรือยังไม่เข้าสู่ระบบ SweetTrackerสมัครและเข้าสู่ระบบที่ tracking.sweettracker.co.kr จากนั้นยื่นขอออก t_key
การติดตามการจัดส่งไม่เชื่อมต่อป้อน t_key ผิดหรือไม่ได้ป้อนตรวจสอบอีกครั้งว่าได้ป้อน t_key ที่ออกแล้วในฟิลด์ api_key อย่างถูกต้องหรือไม่

รายการตรวจสอบขั้นสุดท้าย

  • สมัครและเข้าสู่ระบบที่ tracking.sweettracker.co.kr แล้ว
  • ยื่นขอออก t_key (คีย์ติดตามแบบรวมของทุกบริษัทขนส่ง) แล้ว
  • ตรวจสอบ t_key ที่ออกแล้ว
  • ป้อน t_key ในฟิลด์ api_key ใน GenieGo [ลงทะเบียน] และบันทึกแล้ว

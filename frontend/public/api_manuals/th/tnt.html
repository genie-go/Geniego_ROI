1. ก่อนเริ่มต้น

TNT ถูกรวมเข้ากับ FedEx และไม่มีพอร์ทัลนักพัฒนา TNT แยกต่างหาก โดยใช้ข้อมูลรับรองของ FedEx Developer Portal

คำแนะนำ: TNT ถูกเข้าซื้อกิจการโดย FedEx และออกข้อมูลรับรอง (API Key/Secret) ที่ FedEx Developer Portal (developer.fedex.com) ไม่มีพอร์ทัลนักพัฒนา TNT แยกต่างหาก และจำเป็นต้องมีหมายเลขบัญชี FedEx

4. ปัญหาที่พบบ่อย

ปัญหาสาเหตุวิธีแก้ไข
ไม่พบพอร์ทัลนักพัฒนาเฉพาะของ TNTTNT ถูกรวมเข้ากับ FedEx จึงไม่มีพอร์ทัลแยกต่างหากสร้างโปรเจกต์ที่ developer.fedex.com เพื่อออก API Key/Secret
ข้อผิดพลาดการตรวจสอบสิทธิ์เมื่อเรียกใช้พิมพ์ Secret ผิด หรือขาดหมายเลขบัญชี FedExตรวจสอบ api_secret และหมายเลขบัญชี (account_number) อีกครั้ง

รายการตรวจสอบขั้นสุดท้าย

  • ยืนยันแล้วว่า TNT ใช้ข้อมูลรับรองของ FedEx
  • สมัครที่ developer.fedex.com และสร้างโปรเจกต์แล้ว
  • ตรวจสอบ API Key·Secret และหมายเลขบัญชี FedEx แล้ว
  • ป้อน api_key·api_secret·account_number ใน [ลงทะเบียน] ของ GenieGo และบันทึกแล้ว

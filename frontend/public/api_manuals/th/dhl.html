1. ก่อนเริ่มต้น

ข้อมูลรับรอง DHL Express ต้องยื่นคำขอบนพอร์ทัลนักพัฒนา และจำเป็นต้องมีหมายเลขบัญชีลูกค้า

สิ่งที่ต้องเตรียม

หมายเลขบัญชีลูกค้า DHL Express

การมีหมายเลขบัญชีลูกค้า DHL Express ช่วยให้การยื่นขอข้อมูลรับรองราบรื่นขึ้น หมายเลขบัญชีจะถูกป้อนเมื่อลงทะเบียนใน GenieGo ด้วยเช่นกัน

คีย์ที่ออก

API Key · Secret

คุณจะได้รับ API Key และ Secret ผ่านแอปของคุณเอง (ชื่อบริษัท) บนพอร์ทัล

หมายเหตุ: เมื่อคุณลงทะเบียนข้อมูลลูกค้าผ่าน [Get Access] และยื่นขอข้อมูลรับรอง โดยปกติคุณจะได้รับอีเมลอนุมัติ Test/Production ในวันทำการถัดไป

2. ขั้นตอนการออก

ออกตามลำดับด้านล่างและลงทะเบียนใน GenieGo

1

สมัครพอร์ทัลนักพัฒนา

ไปที่พอร์ทัลนักพัฒนา DHL Express สมัครด้วยอีเมลและยืนยัน

https://developer.dhl.com
2

ยื่นขอข้อมูลรับรอง

หลังจากลงชื่อเข้าใช้ ลงทะเบียนข้อมูลลูกค้าของคุณในส่วน [Get Access] และยื่นขอข้อมูลรับรอง

Get Access
3

รับอีเมลอนุมัติ

โดยปกติคุณจะได้รับอีเมลอนุมัติ Test/Production ในวันทำการถัดไป

4

ค้นหา API Key·Secret

บนพอร์ทัล ไปที่แอปของคุณเอง (ชื่อบริษัท) > [Show Key] เพื่อค้นหา API Key และ Secret

แอปของคุณเอง (ชื่อบริษัท) → Show Key
เมื่อคุณได้รับอีเมลอนุมัติและเห็น API Key·Secret บนพอร์ทัล แสดงว่าออกอย่างถูกต้องแล้ว

3. ข้อมูลที่ต้องลงทะเบียนใน GenieGo

ฟิลด์คำอธิบาย
api_keyDHL API Key ที่พบบนพอร์ทัล
account_numberหมายเลขบัญชีลูกค้า DHL Express
เมื่อคุณป้อนข้อมูลใน GenieGo Integration Hub [ลงทะเบียน] และบันทึก ข้อมูลรับรองจะถูกเข้ารหัสขณะจัดเก็บด้วย AES-256-GCM และปิดบังเมื่ออ่าน

4. ปัญหาที่พบบ่อย

ปัญหาสาเหตุวิธีแก้ไข
ไม่ได้รับอีเมลอนุมัติกำลังรอการประมวลผลอนุมัติในวันทำการถัดไป หรือยังไม่ได้ลงทะเบียนข้อมูลลูกค้าตรวจสอบใน [Get Access] ว่าการลงทะเบียนข้อมูลลูกค้าและการยื่นขอข้อมูลรับรองเสร็จสมบูรณ์แล้ว และรอตามวันทำการ
ข้อผิดพลาดการยืนยันตัวตนเมื่อเรียกใช้สับสนระหว่างสภาพแวดล้อม Test/Production หรือขาดหมายเลขบัญชีตรวจสอบ API Key·Secret ของสภาพแวดล้อมที่อนุมัติแล้วและหมายเลขบัญชี (account_number) อีกครั้ง

รายการตรวจสอบขั้นสุดท้าย

  • ยืนยันหมายเลขบัญชีลูกค้า DHL Express แล้ว
  • สมัครที่ developer.dhl.com และยื่นขอข้อมูลรับรองผ่าน [Get Access] แล้ว
  • ได้รับอีเมลอนุมัติ Test/Production ในวันทำการถัดไปแล้ว
  • พบ API Key·Secret บนพอร์ทัลแล้ว
  • ป้อน api_key·account_number ใน GenieGo [ลงทะเบียน] และบันทึกแล้ว

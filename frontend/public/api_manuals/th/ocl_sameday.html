1. ก่อนเริ่มต้น

GenieGo Same-Day Delivery เป็นช่องทางแบบทำสัญญา ซึ่งข้อมูลรับรองจะออกให้หลังจากทำสัญญาบริการกับทีมปฏิบัติการ

จำเป็น

สัญญาบริการการจัดส่งภายในวัน

หากต้องการใช้ GenieGo Same-Day Delivery จำเป็นต้องมีสัญญาการใช้บริการกับทีมปฏิบัติการก่อน หากไม่มีสัญญาจะไม่สามารถออกข้อมูลรับรองได้

คีย์ที่ออก

คีย์ API · รหัสร้านค้า · เขตพื้นที่บริการ

หลังจากทำสัญญาเสร็จสมบูรณ์ ทีมปฏิบัติการจะออกรหัสร้านค้าและคีย์ API พร้อมกับเขตพื้นที่บริการให้คุณ

หมายเหตุ: ช่องทางนี้ไม่ใช่การออกด้วยตนเองจากคอนโซล แต่เป็นการรับข้อมูลรับรองผ่านสัญญากับทีมปฏิบัติการ

2. ขั้นตอนการขอออก

ยื่นขอและรับตามลำดับด้านล่าง จากนั้นลงทะเบียนใน GenieGo

1

ยื่นขอใช้การจัดส่งภายในวัน

ยื่นขอใช้ GenieGo Same-Day Delivery กับทีมปฏิบัติการและดำเนินการทำสัญญา

2

ลงนามในสัญญา

ลงนามในสัญญาการใช้บริการการจัดส่งภายในวันกับทีมปฏิบัติการ

3

รับข้อมูลรับรอง

หลังจากทำสัญญาเสร็จสมบูรณ์ คุณจะได้รับรหัสร้านค้า คีย์ API และเขตพื้นที่บริการ

4

ลงทะเบียนใน GenieGo

ป้อน api_key, merchant_id และ region ที่ออกให้ลงใน GenieGo [ลงทะเบียน] แล้วบันทึก

หากคุณได้รับรหัสร้านค้า คีย์ API และเขตพื้นที่บริการครบทั้งหมดจากทีมปฏิบัติการแล้ว แสดงว่าออกอย่างถูกต้อง

3. ข้อมูลที่ต้องลงทะเบียนใน GenieGo

ฟิลด์คำอธิบาย
api_keyคีย์ API การจัดส่งภายในวันของ GenieGo ที่ออกให้
merchant_idรหัสร้านค้าที่ออกให้
regionเขตพื้นที่บริการ (เช่น เขตเมืองหลวง)
เมื่อป้อนและบันทึกใน GenieGo Integration Hub [ลงทะเบียน] แล้ว ข้อมูลรับรองจะถูกเข้ารหัสขณะจัดเก็บด้วย AES-256-GCM และปิดบังเมื่ออ่าน

4. ปัญหาที่พบบ่อย

ปัญหาสาเหตุวิธีแก้ไข
ไม่ได้รับข้อมูลรับรองสัญญาบริการการจัดส่งภายในวันยังไม่เสร็จสมบูรณ์ยื่นขอและทำสัญญาการใช้การจัดส่งภายในวันกับทีมปฏิบัติการ จากนั้นรับข้อมูลรับรอง
ข้อผิดพลาดในการประมวลผลเขตพื้นที่หลังลงทะเบียนไม่ได้ป้อนหรือป้อนเขตพื้นที่บริการ (region) ผิดตรวจสอบเขตพื้นที่บริการที่ออกให้อีกครั้งและป้อนให้ถูกต้อง

รายการตรวจสอบขั้นสุดท้าย

  • ยื่นขอและทำสัญญาการใช้ GenieGo Same-Day Delivery กับทีมปฏิบัติการแล้ว
  • ได้รับรหัสร้านค้า คีย์ API และเขตพื้นที่บริการหลังจากทำสัญญาเสร็จสมบูรณ์แล้ว
  • ยืนยัน api_key, merchant_id และ region แล้ว
  • ป้อน api_key, merchant_id และ region ใน GenieGo [ลงทะเบียน] และบันทึกแล้ว

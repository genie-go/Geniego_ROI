Naver Search Ads
คู่มือการออก API Key

คู่มือทีละขั้นตอนที่เป็นมิตรกับผู้เริ่มต้น สำหรับการออกข้อมูลรับรอง API (access license, secret key) และ CUSTOMER_ID จากเครื่องมือจัดการ Naver Search Ad และลงทะเบียนใน GenieGo Integration Hub

เวลาโดยประมาณประมาณ 5–10 นาที
บัญชีที่ต้องใช้บัญชีผู้ลงโฆษณา Naver Search Ad
คีย์ที่ออกAccess license · Secret key · CUSTOMER_ID
ผลลัพธ์ข้อมูลรับรอง Search Ad API

1. ก่อนเริ่มต้น

Naver Search Ad API สามารถขอได้จากบัญชีผู้ลงโฆษณาเท่านั้น

จำเป็น

บัญชีผู้ลงโฆษณา Search Ad

คุณต้องมีบัญชีผู้ลงโฆษณาที่ใช้เครื่องมือจัดการ Naver Search Ad คุณไม่สามารถขอสิทธิ์เข้าถึง API ด้วยบัญชีสมาชิก Naver ทั่วไปได้

คีย์ที่ออก

Access license · Secret key · CUSTOMER_ID

ยืนยัน CUSTOMER_ID, access license และ secret key ที่ออกมาพร้อมกันหลังจากคุณขอสิทธิ์เข้าถึง API

หมายเหตุ: Secret key สามารถดูได้เฉพาะในขณะที่ออกเท่านั้น ดังนั้นโปรดเก็บรักษาให้ปลอดภัยทันทีหลังจากออก

2. ขั้นตอนการออก API Key

ออกตามลำดับด้านล่างและลงทะเบียนใน GenieGo

1

ลงชื่อเข้าใช้ Search Ads

ลงชื่อเข้าใช้เครื่องมือจัดการ Naver Search Ad ด้วยบัญชีผู้ลงโฆษณาของคุณ

searchad.naver.com
2

ไปที่การจัดการการใช้ API

จากเมนู [เครื่องมือ] ด้านบน ไปที่ [การจัดการการใช้ API]

เครื่องมือ → การจัดการการใช้ API
3

ขอใช้บริการ API

คลิกปุ่ม [ขอใช้บริการ Naver Search Ad API] เพื่อยื่นคำขอ

ขอใช้บริการ Naver Search Ad API
4

ตรวจสอบข้อมูลที่ออก

ยืนยัน CUSTOMER_ID, access license และ secret key ที่ออกมา

เมื่อยืนยัน CUSTOMER_ID, access license และ secret key ครบทั้งหมดแล้ว แสดงว่าออกได้อย่างถูกต้อง

3. ข้อมูลที่ต้องลงทะเบียนใน GenieGo

ฟิลด์คำอธิบาย
api_keyAccess license ที่ออก
api_secretSecret key ที่ออก (จัดการด้วยความระมัดระวัง)
customer_idCUSTOMER_ID ที่ออก
ป้อนและบันทึกใน [ลงทะเบียน] ของ GenieGo Integration Hub แล้วข้อมูลรับรองจะถูกเข้ารหัสขณะจัดเก็บด้วย AES-256-GCM และถูกปิดบังเมื่ออ่าน

4. ปัญหาที่พบบ่อย

ปัญหาสาเหตุวิธีแก้ไข
ไม่มีเมนู [การจัดการการใช้ API]ไม่ใช่บัญชีผู้ลงโฆษณา หรือสิทธิ์ไม่เพียงพอลงชื่อเข้าใช้ด้วยบัญชีผู้ลงโฆษณาที่ใช้เครื่องมือจัดการ Search Ad
เกิดข้อผิดพลาดในการยืนยันตัวตนป้อน access license หรือ secret key ไม่ถูกต้องตรวจสอบ access license, secret key และ CUSTOMER_ID จากหน้าจอการออกอีกครั้ง

รายการตรวจสอบขั้นสุดท้าย

  • ลงชื่อเข้าใช้ด้วยบัญชีผู้ลงโฆษณา Naver Search Ad แล้ว
  • ทำการยื่นคำขอใช้บริการ API ที่ [เครื่องมือ > การจัดการการใช้ API] เสร็จสมบูรณ์แล้ว
  • ยืนยัน CUSTOMER_ID, access license และ secret key แล้ว
  • ป้อน api_key·api_secret·customer_id ใน GenieGo [ลงทะเบียน] และบันทึกแล้ว

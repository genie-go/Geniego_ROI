Coupang Wing
คู่มือการออก API Key

คู่มือทีละขั้นตอนที่เหมาะสำหรับมือใหม่ ในการออกข้อมูลรับรอง Open API (Access Key·Secret Key) ในคอนโซลผู้ขาย Coupang WING และลงทะเบียนในศูนย์เชื่อมต่อ GenieGo

เวลาโดยประมาณประมาณ 5–10 นาที
บัญชีที่ต้องใช้บัญชีผู้ขาย Coupang WING
คีย์ที่ออกAccess Key · Secret Key
ผลลัพธ์ข้อมูลรับรอง Open API

1. ก่อนเริ่มต้น

Coupang Open API สามารถออกได้จากบัญชีผู้ขายเท่านั้น

จำเป็น

บัญชีผู้ขาย WING

คุณต้องมีบัญชีผู้ขาย Coupang WING ที่ผ่านการยืนยันธุรกิจแล้ว ไม่สามารถออกด้วยบัญชีสมาชิกผู้ซื้อทั่วไปได้

คีย์ที่ออก

Access Key · Secret Key

ในหน้าจอแสดงผลการออกที่เสร็จสมบูรณ์ ให้ค้นหา Access Key และ Secret Key และตรวจสอบ vendor ID ด้วย

หมายเหตุ: เมื่อทำการออก คุณต้องกรอกชื่อบริษัท, URL หน้าเว็บ/บริการ (หากไม่มีให้ใช้ wing.coupang.com) และ IP ที่เข้าถึง (สูงสุด 10 รายการ)

2. ขั้นตอนการออก Open API Key

ออกตามลำดับด้านล่างและลงทะเบียนใน GenieGo

1

เข้าสู่ระบบ WING

เข้าสู่ระบบด้วยบัญชีผู้ขายของคุณ

wing.coupang.com
2

ไปที่เมนู Open API

ไปที่เมนู [Open API] หรือ [การจัดการ API Key] ในส่วนด้านบน/การตั้งค่า

Open API → การจัดการ API Key
3

ออก Open API Key

คลิก [ออก Open API Key] และกรอกชื่อบริษัท, URL หน้าเว็บ และ IP ที่เข้าถึง (สูงสุด 10 รายการ)

ออก Open API Key
4

ตรวจสอบคีย์

ในหน้าจอแสดงผลการออกที่เสร็จสมบูรณ์ ให้ค้นหา Access Key·Secret Key และ vendor ID

เมื่อ Access Key·Secret Key ปรากฏบนหน้าจอแสดงผลการออกที่เสร็จสมบูรณ์ แสดงว่าออกได้อย่างถูกต้องแล้ว

3. ข้อมูลที่ต้องลงทะเบียนใน GenieGo

ฟิลด์คำอธิบาย
access_keyAccess Key ที่ออกแล้ว
secret_keySecret Key ที่ออกแล้ว (จัดการด้วยความระมัดระวัง)
vendor_idVendor ID
กรอกข้อมูลในศูนย์เชื่อมต่อ GenieGo [ลงทะเบียน] แล้วบันทึก ข้อมูลรับรองจะถูกเข้ารหัสขณะจัดเก็บด้วย AES-256-GCM และถูกปิดบังเมื่ออ่าน การซิงค์คำสั่งซื้อจะเริ่มทันทีหลังลงทะเบียน

4. ปัญหาที่พบบ่อย

ปัญหาสาเหตุวิธีแก้
ไม่มีเมนูการออกบัญชีสมาชิกทั่วไป หรือยังไม่ผ่านการยืนยันธุรกิจเข้าสู่ระบบด้วยบัญชีผู้ขาย WING ที่ผ่านการยืนยันธุรกิจแล้ว
ข้อผิดพลาด 401 เมื่อเรียกใช้ยังไม่ได้ลงทะเบียน IP ที่เข้าถึง หรือกรอก Secret Key ผิดตรวจสอบ IP ที่เข้าถึงที่กรอกตอนออก และ Secret Key อีกครั้ง

รายการตรวจสอบสุดท้าย

  • เข้าสู่ระบบด้วยบัญชีผู้ขาย WING ที่ผ่านการยืนยันธุรกิจแล้ว
  • ออก Open API Key และลงทะเบียน IP ที่เข้าถึงแล้ว
  • ตรวจสอบ Access Key·Secret Key·Vendor ID แล้ว
  • กรอก access_key·secret_key·vendor_id ใน GenieGo [ลงทะเบียน] และบันทึกแล้ว

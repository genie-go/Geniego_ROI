1. ก่อนเริ่มต้น

การเชื่อมต่อ Meta Ads ต้องใช้บัญชีธุรกิจและบัญชีโฆษณาที่กำลังลงโฆษณาอยู่

จำเป็น

บัญชีส่วนตัว Facebook · บัญชีโฆษณา

คุณต้องมีบัญชีส่วนตัว Facebook และบัญชีโฆษณา Meta ที่กำลังลงโฆษณาอยู่

คีย์ที่ออก

โทเค็นการเข้าถึง · ID บัญชีโฆษณา (act_)

คุณจะได้รับโทเค็นผู้ใช้ระบบ (โทเค็นการเข้าถึง) และ ID บัญชีโฆษณาที่เป็นตัวเลขซึ่งขึ้นต้นด้วย act_

หมายเหตุ: การเข้าถึงข้อมูลนอกธุรกิจของคุณเองจำเป็นต้องผ่านการตรวจสอบแอป (App Review) หากคุณเชื่อมต่อเฉพาะบัญชีโฆษณาภายในธุรกิจของคุณเอง คุณสามารถดำเนินการได้โดยไม่ต้องตรวจสอบ

2. ขั้นตอนการออก

ออกตามลำดับด้านล่างและลงทะเบียนใน GenieGo

1

สร้างและยืนยันบัญชีธุรกิจ

เปิดเว็บไซต์ในเบราว์เซอร์ของคุณ ลงชื่อเข้าใช้ จากนั้นสร้างบัญชีธุรกิจผ่าน [สร้างธุรกิจ] และทำการยืนยันอีเมลให้เสร็จสมบูรณ์

business.facebook.com
2

สร้างแอป

คลิก [แอปของฉัน] > [สร้างแอป] ที่มุมขวาบน เลือกประเภท "ธุรกิจ" และสร้างแอป

developers.facebook.com → แอปของฉัน → สร้างแอป
3

เพิ่ม Marketing API

บนแดชบอร์ดของแอป ค้นหาการ์ด "Marketing API" และคลิก [ตั้งค่า] เพื่อเพิ่มเป็นผลิตภัณฑ์

4

สร้างผู้ใช้ระบบและกำหนดสิทธิ์

ไปที่การตั้งค่าธุรกิจ > [ผู้ใช้] > [ผู้ใช้ระบบ] > [เพิ่ม] เพื่อสร้างผู้ใช้ระบบ จากนั้นให้สิทธิ์บัญชีโฆษณาภายใต้ [กำหนดสินทรัพย์]

business.facebook.com/settings → ผู้ใช้ → ผู้ใช้ระบบ
5

สร้างโทเค็นการเข้าถึง

บนหน้าจอเดียวกัน คลิก [สร้างโทเค็น] เลือกแอปของคุณ เลือกสิทธิ์ (ads_read, ads_management) สร้างโทเค็น และคัดลอกค่าของมัน

6

ค้นหา ID บัญชีโฆษณาของคุณ

ใน Ads Manager ค้นหา ID บัญชีโฆษณาที่เป็นตัวเลขซึ่งขึ้นต้นด้วย act_

adsmanager.facebook.com
เมื่อสร้างโทเค็นโดยเลือกสิทธิ์ ads_read และ ads_management แล้ว แสดงว่าออกอย่างถูกต้อง

3. ข้อมูลที่ต้องลงทะเบียนใน GenieGo

ฟิลด์คำอธิบาย
access_tokenโทเค็นการเข้าถึงของผู้ใช้ระบบที่ออกแล้ว (จัดการด้วยความระมัดระวัง)
ad_account_idID บัญชีโฆษณาที่ขึ้นต้นด้วย act_
ใน GenieGo Integration Hub > [ลงทะเบียน] ของช่องทางนั้น วางโทเค็นการเข้าถึงและ ID บัญชีโฆษณา (act_) แล้วบันทึก ข้อมูลรับรองจะถูกเข้ารหัสขณะจัดเก็บด้วย AES-256-GCM และปิดบังเมื่ออ่าน

4. ปัญหาที่พบบ่อย

ปัญหาสาเหตุวิธีแก้ไข
ไม่สามารถดึงข้อมูลได้เนื่องจากข้อผิดพลาดด้านสิทธิ์ไม่ได้เลือกสิทธิ์โทเค็น (ads_read·ads_management) หรือไม่ได้กำหนดสิทธิ์บัญชีโฆษณากำหนดสิทธิ์บัญชีโฆษณาให้กับผู้ใช้ระบบ และเลือกสิทธิ์ ads_read·ads_management เมื่อสร้างโทเค็น
การเข้าถึงข้อมูลของธุรกิจอื่นถูกปฏิเสธการตรวจสอบแอป (App Review) ยังไม่เสร็จสมบูรณ์หากต้องการเข้าถึงข้อมูลนอกธุรกิจของคุณเอง โปรดยื่นขอและดำเนินการตรวจสอบแอปให้เสร็จสมบูรณ์
ข้อผิดพลาดรูปแบบ ID บัญชีโฆษณาขาดคำนำหน้า act_ หรือใส่เฉพาะตัวเลขใน Ads Manager ค้นหา ID เต็มที่ขึ้นต้นด้วย act_ แล้วป้อนเข้าไป

รายการตรวจสอบขั้นสุดท้าย

  • สร้างบัญชีธุรกิจที่ business.facebook.com และทำการยืนยันอีเมลเสร็จสมบูรณ์แล้ว
  • สร้างแอปประเภทธุรกิจที่ developers.facebook.com และเพิ่ม Marketing API แล้ว
  • สร้างผู้ใช้ระบบและกำหนดสิทธิ์บัญชีโฆษณาแล้ว
  • สร้างและคัดลอกโทเค็นการเข้าถึงด้วยสิทธิ์ ads_read·ads_management แล้ว
  • พบ ID บัญชีโฆษณาที่ขึ้นต้นด้วย act_ แล้ว
  • ป้อน access_token·ad_account_id ใน GenieGo [ลงทะเบียน] และบันทึกแล้ว

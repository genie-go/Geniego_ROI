Meta Ads
คู่มือการออก API

คู่มือทีละขั้นตอนที่ใครก็ทำตามได้ เพื่อออกข้อมูลรับรอง Meta Ads (API key/โทเค็น) และลงทะเบียนใน GenieGo Integration Hub

เวลาโดยประมาณประมาณ 5–15 นาที
บัญชีที่ต้องใช้บัญชี Meta Ads
จำนวนขั้นตอน8 ขั้นตอน
ผลลัพธ์ข้อมูลรับรอง API

ขั้นตอนการออก

ทำตามขั้นตอนด้านล่างเพื่อออกและลงทะเบียนข้อมูลรับรองของคุณ

1

สิ่งที่ต้องเตรียม: บัญชี Facebook ส่วนตัว และบัญชีโฆษณา Meta ที่กำลังลงโฆษณาอยู่

2

เปิด business.facebook.com ในเบราว์เซอร์แล้วเข้าสู่ระบบ กด [สร้างธุรกิจ] เพื่อสร้างบัญชีธุรกิจ จากนั้นยืนยันอีเมลให้เสร็จ

3

เข้า developers.facebook.com → [แอปของฉัน] ที่มุมขวาบน → [สร้างแอป] แล้วเลือกประเภท "ธุรกิจ" เพื่อสร้างแอป

4

ในแดชบอร์ดของแอป ค้นหาการ์ด "Marketing API" แล้วกด [ตั้งค่า] เพื่อเพิ่มเป็นผลิตภัณฑ์

5

การตั้งค่าธุรกิจ (business.facebook.com/settings) → [ผู้ใช้] > [ผู้ใช้ระบบ] → [เพิ่ม] เพื่อสร้างผู้ใช้ระบบ จากนั้นให้สิทธิ์บัญชีโฆษณาผ่าน [กำหนดสินทรัพย์]

6

ที่หน้าจอเดียวกัน กด [สร้างโทเคน] เลือกแอป ติ๊กสิทธิ์ (ads_read, ads_management) แล้วสร้างโทเคนและคัดลอกค่า

7

ที่ตัวจัดการโฆษณา (adsmanager.facebook.com) ตรวจสอบ ID บัญชีโฆษณา (ตัวเลขที่ขึ้นต้นด้วย act_)

8

กด [ลงทะเบียน] ที่ Integration Hub ของ GenieGo สำหรับช่องทางที่เกี่ยวข้อง วาง "access token" และ "ID บัญชีโฆษณา (act_)" แล้วบันทึกเพื่อเชื่อมต่อให้เสร็จ

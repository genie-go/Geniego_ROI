1. ก่อนเริ่มต้น

Kakao Moment Open API ไม่ใช่การออกด้วยตนเอง แต่ต้องผ่านกระบวนการขอสิทธิ์และการตรวจสอบ

สำคัญ: สิทธิ์ Kakao Moment Open API จะมอบให้เฉพาะตัวแทนโฆษณาอย่างเป็นทางการของ Kakao เท่านั้น ผู้โฆษณาทั่วไปไม่สามารถออกด้วยตนเองได้

บัญชีโฆษณา + Business Channel

คุณต้องมีบัญชีโฆษณา Kakao Moment และ Kakao Business Channel

คีย์ที่ออกให้

Access token · Ad account ID

Additional Feature Application
3

ออก token

หลังจากได้รับอนุมัติสิทธิ์ ให้รับ access token ผ่านการอนุญาต OAuth และยืนยัน ad account ID ของคุณ คุณสามารถใช้ OAuth แบบคลิกเดียวในศูนย์เชื่อมต่อ GenieGo ได้

เมื่อสิทธิ์ได้รับการอนุมัติและ access token รวมถึง ad account ID ของคุณได้รับการยืนยันแล้ว คุณสามารถดำเนินการลงทะเบียนต่อได้

3. ข้อมูลที่ต้องลงทะเบียนใน GenieGo

ฟิลด์คำอธิบาย
access_tokenAccess token ที่ออกผ่านการอนุญาต OAuth (โปรดจัดการด้วยความระมัดระวัง)
ad_account_idAd account ID ของ Kakao Moment
กรอกและบันทึกในศูนย์เชื่อมต่อ GenieGo [Register] ข้อมูลรับรองจะถูกเข้ารหัสขณะจัดเก็บด้วย AES-256-GCM และปกปิดเมื่ออ่าน

4. ปัญหาที่พบบ่อย

ปัญหาสาเหตุวิธีแก้ไข
ไม่สามารถรับสิทธิ์ Open API ได้ไม่ใช่ตัวแทนโฆษณาอย่างเป็นทางการสิทธิ์ Open API จะมอบให้เฉพาะตัวแทนโฆษณาอย่างเป็นทางการของ Kakao เท่านั้น โปรดยื่นขอผ่านตัวแทน
ไม่สามารถดำเนินการ [Additional Feature Application] ได้แอปยังไม่ได้เปลี่ยนเป็น Business appเปลี่ยนแอปเป็น [Business App] ก่อน จากนั้นจึงขอฟีเจอร์เพิ่มเติม

รายการตรวจสอบขั้นสุดท้าย

  • เตรียมบัญชีโฆษณา Kakao Moment และ Kakao Business Channel แล้ว
  • เปลี่ยนแอปเป็น [Business App] และยื่น Open API [Additional Feature Application] แล้ว
  • หลังจากได้รับอนุมัติสิทธิ์ ยืนยัน access token และ ad account ID ผ่านการอนุญาต OAuth แล้ว
  • กรอก access_token·ad_account_id ใน GenieGo [Register] และบันทึกแล้ว

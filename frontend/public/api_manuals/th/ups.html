3. ข้อมูลที่ต้องลงทะเบียนใน GenieGo

ฟิลด์คำอธิบาย
client_idClient ID ที่ออกแล้ว
client_secretClient Secret ที่ออกแล้ว (จัดการด้วยความระมัดระวัง)
account_numberหมายเลขบัญชี UPS
ป้อนใน GenieGo Integration Hub [ลงทะเบียน] แล้วบันทึก ข้อมูลรับรองจะถูกเข้ารหัสขณะจัดเก็บด้วย AES-256-GCM และปิดบังเมื่ออ่าน

4. ปัญหาที่พบบ่อย

ปัญหาสาเหตุวิธีแก้ไข
ไม่เห็นข้อมูลรับรองในแอปการสร้างแอปยังไม่เสร็จสมบูรณ์สร้างแอปที่ developer.ups.com แล้วจะมีการออก client_id·client_secret
ข้อผิดพลาดการตรวจสอบสิทธิ์เมื่อเรียกใช้client_secret ผิด หรือขาดหมายเลขบัญชีตรวจสอบ client_secret และหมายเลขบัญชี (account_number) อีกครั้ง

รายการตรวจสอบขั้นสุดท้าย

  • ตรวจสอบหมายเลขบัญชี UPS แล้ว
  • ลงทะเบียน developer.ups.com และสร้างแอปแล้ว
  • ออก client_id·client_secret แล้ว
  • ป้อน client_id·client_secret·account_number ใน GenieGo [ลงทะเบียน] และบันทึกแล้ว

Mollie
API Key 申请手册

本手册手把手指导新手在 Mollie 仪表盘中查找 Live API 密钥(以 live_ 开头),并将其注册到 GenieGo 集成中心。

预计时间约 5~10 分钟
所需账户Mollie 账户(以欧洲为主)
签发密钥Live API 密钥(live_)
成果Live API 凭证

1. 开始之前

Mollie API 密钥可在 Mollie 账户的仪表盘中查看。

必需

Mollie 账户

您需要一个 Mollie 账户(以欧洲为主)。您必须能够使用该账户登录仪表盘。

签发密钥

Live API 密钥

在仪表盘的 Developers > API keys 中查找以 live_ 开头的 Live API 密钥。

注意: 务必查找以 live_ 开头的 Live API 密钥。注意不要与 Test 密钥混淆。

2. 申请步骤

按以下顺序签发并在 GenieGo 中注册。

1

访问仪表盘

登录 Mollie 仪表盘。

my.mollie.com
2

前往 API keys

前往 [Developers] > [API keys]。

https://my.mollie.com/dashboard/developers/api-keys
3

查找 Live API 密钥

查找以 live_ 开头的 Live API 密钥。

在 Developers > API keys 中能看到以 live_ 开头的密钥即表示正常。

3. 需在 GenieGo 中注册的信息

字段说明
api_keyLive API 密钥(以 live_ 开头,请妥善保管)
在 GenieGo 集成中心 [注册] 中输入 api_key 并保存后,凭证将以 AES-256-GCM 加密存储,并在查询时进行掩码处理。

4. 常见问题

问题原因解决方法
认证失败使用了 Test 密钥再次确认是否为以 live_ 开头的 Live API 密钥。
API keys 菜单中没有密钥需要确认仪表盘登录账户确认是否已使用 Mollie 账户登录 my.mollie.com 仪表盘。

最终检查清单

  • 已使用 Mollie 账户登录 my.mollie.com 仪表盘。
  • 已在 Developers > API keys 中查看 Live API 密钥。
  • 已查找到以 live_ 开头的 Live API 密钥。
  • 已在 GenieGo [注册] 中输入 api_key 并保存。

Shopee
API Key 申请手册

本手册手把手指导新手在 Shopee Open Platform 完成开发者注册与审核后创建应用以签发 Partner ID 和 Partner Key，经店铺授权后注册到 GenieGo 集成中心的整个流程。

预计时间约 10~20 分钟(审核等待另计)
所需账户Shopee 卖家账户
签发密钥Partner ID · Partner Key · Shop ID
成果API 凭证

1. 开始之前

Shopee Open Platform 需要开发者注册与审核，以及店铺授权步骤。

必需

Shopee 卖家账户

要进行签发，您需要一个 Shopee 卖家账户。使用该账户在 Shopee Open Platform 上注册为开发者。

签发密钥

Partner ID · Partner Key

创建应用后会签发 Partner ID 和 Partner Key。随后经店铺授权并获取 access_token。

注意: 在 Shopee Open Platform 完成开发者注册后需要等待审核，且签发的 access_token 每 4 小时刷新一次。

2. 申请步骤

按以下顺序签发并在 GenieGo 中注册。

1

开发者注册与审核

在 Shopee Open Platform(open.shopee.com)上注册为开发者并等待审核。

https://open.shopee.com
2

创建应用

在 [App Management] > [App List] 中创建应用，即可签发 Partner ID 和 Partner Key。

App Management → App List
3

店铺授权

生成店铺授权(authorization)链接并授权卖家店铺，随后会签发 code。

4

签发 access_token

使用 code 获取 access_token。access_token 每 4 小时刷新一次。

当签发了 Partner ID 和 Partner Key，并通过店铺授权获取到 access_token 时，即表示已正确签发。

3. 需在 GenieGo 中注册的信息

字段说明
partner_id签发的 Partner ID
partner_key签发的 Partner Key(请妥善保管)
shop_id已授权卖家店铺的 Shop ID
在 GenieGo 集成中心 [注册] 中输入 partner_id、partner_key、shop_id 并保存后，凭证将以 AES-256-GCM 加密存储，并在查询时进行掩码处理。

4. 常见问题

问题原因解决方法
无法创建应用开发者注册尚未通过审核在 Shopee Open Platform 注册为开发者，等待审核通过后再创建应用。
认证/调用失败店铺授权未完成，或 access_token 已过期通过店铺授权链接授权店铺，并每 4 小时刷新一次 access_token。

最终检查清单

  • 已使用 Shopee 卖家账户完成 Open Platform 开发者注册与审核。
  • 已在 App Management > App List 中创建应用并签发 Partner ID 和 Partner Key。
  • 已通过店铺授权链接授权店铺并获取 access_token。
  • 已在 GenieGo [注册] 中输入 partner_id、partner_key、shop_id 并保存。

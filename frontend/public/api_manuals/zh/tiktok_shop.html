TikTok Shop
API Key 申请手册

本手册手把手指导新手在 TikTok Shop Partner Center 中创建应用(App Key·App Secret·Service ID),通过店铺授权签发 access_token,并注册到 GenieGo 集成中心。

预计时间约 10~15 分钟
所需账户TikTok Shop 卖家账户
签发密钥App Key · App Secret · Access Token
成果API 凭证

1. 开始之前

只有在 Partner Center 中创建应用并授权店铺后才会签发令牌。

必需

TikTok Shop 卖家账户

您需要一个 TikTok Shop 卖家账户,并在 partner.tiktokshop.com(Partner Center)注册为开发者。

签发密钥

App Key · App Secret · Access Token

创建应用后可获得 App Key·App Secret·Service ID,并通过店铺授权获取 access_token。

注意: 切换到生产环境时需要进行应用审核。

2. 申请步骤

按以下顺序签发并在 GenieGo 中注册。

1

在 Partner Center 注册为开发者

在 partner.tiktokshop.com(Partner Center)注册为开发者。

https://partner.tiktokshop.com
2

创建应用

通过 [App & Service] > [Create App] 创建应用,并查看 App Key、App Secret、Service ID。

App & Service → Create App
3

启用 API 权限

在应用设置中启用所需的 API 权限(Shop Authorized Information、Product、Order)。

4

店铺授权 → 签发 access_token

在 [Get shop authorization] 中选择地区(Region)以授权卖家店铺,即可签发 access_token。

Get shop authorization
5

确认密钥

确认 App Key·App Secret 和签发的 access_token。如有需要,一并确认 Shop Cipher。

店铺授权后签发 access_token,即表示已正常处理。

3. 需在 GenieGo 中注册的信息

字段说明
app_key签发的 App Key
app_secret签发的 App Secret(请妥善保管)
access_token通过店铺授权签发的访问令牌
shop_cipherShop Cipher(可选)
在 GenieGo 集成中心的 [注册] 中输入 app_key·app_secret·access_token 并保存后,凭证将以 AES-256-GCM 加密存储,并在查询时进行掩码处理。

4. 常见问题

问题原因解决方法
未签发 access_token店铺授权(Get shop authorization)未完成,或未选择地区(Region)在 [Get shop authorization] 中选择地区以授权卖家店铺。
权限错误未启用所需的 API 权限在应用设置中启用 Shop Authorized Information·Product·Order 权限。
生产调用被拒绝未通过生产切换审核完成切换到生产环境所需的应用审核。

最终检查清单

  • 已在 partner.tiktokshop.com 注册为开发者。
  • 已通过 App & Service > Create App 查看 App Key·App Secret·Service ID。
  • 已启用所需的 API 权限(Shop/Product/Order)。
  • 已通过店铺授权(Get shop authorization)签发 access_token。
  • 已在 GenieGo 的 [注册] 中输入 app_key·app_secret·access_token 并保存。

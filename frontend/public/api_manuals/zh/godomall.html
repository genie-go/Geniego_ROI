godomall
API Key 申请手册

本手册手把手指导新手在 NHN Commerce 开发者中心申请 godomall 开放 API 密钥，获取合作伙伴密钥和 API 密钥，并注册到 GenieGo 集成中心。

预计时间约 10~20 分钟(含审核等待)
所需账户godomall(NHN Commerce)商城运营者账户
签发密钥合作伙伴密钥 · API 密钥
成果开放 API 凭证

1. 开始之前

godomall 开放 API 需在 NHN Commerce 开发者中心完成开发者注册和审核后申请。

必需

商城运营者账户

您需要一个运营 godomall(NHN Commerce)商城的运营者账户。使用该账户在开发者中心完成会员和开发者注册。

签发密钥

合作伙伴密钥 · API 密钥

密钥签发申请通过审核后，将签发开放 API 密钥(合作伙伴密钥 / API 密钥)。

注意: 开发者注册和密钥签发申请需经过审核流程，因此审核可能需要一些时间。

2. 开放 API 密钥签发步骤

按以下顺序申请、签发，然后在 GenieGo 中注册。

1

注册并通过开发者审核

在 NHN Commerce 开发者中心注册为会员和开发者并获得审核通过。

devcenter.nhn-commerce.com
2

申请密钥签发

在[开放 API] > [申请密钥签发]中填写申请表并提交。

开放 API → 申请密钥签发
3

查看密钥

查看已签发的开放 API 密钥(合作伙伴密钥 / API 密钥)。

当签发出合作伙伴密钥和 API 密钥时，即表示已正确申请并通过审核。

3. 需在 GenieGo 中注册的信息

字段说明
partner_key已签发的合作伙伴密钥
api_key已签发的 API 密钥(请妥善保管)
在 GenieGo 集成中心 [注册] 中输入并保存后，凭证将以 AES-256-GCM 加密存储，并在查询时进行掩码处理。

4. 常见问题

问题原因解决方法
无法申请密钥签发开发者注册 / 审核未完成请先在开发者中心完成会员和开发者注册及审核。
申请后未签发密钥正在等待审核请等待审核流程完成后再次查看。

最终检查清单

  • 已在 NHN Commerce 开发者中心注册为会员和开发者并通过审核。
  • 已在[开放 API > 申请密钥签发]中填写并提交申请表。
  • 已查看签发的合作伙伴密钥和 API 密钥。
  • 已在 GenieGo [注册] 中输入 partner_key·api_key 并保存。

Magento (Adobe Commerce)
API Key 申请手册

本手册手把手指导新手在 Magento(Adobe Commerce)管理后台中创建集成(Integration)以签发访问令牌，并将其注册到 GenieGo 集成中心。

预计时间约 5~15 分钟
所需账户Magento(Adobe Commerce)管理员权限
签发密钥Integration 访问令牌
成果REST API 凭证

1. 开始之前

集成(Integration)只能在管理员权限下创建。

必需

Magento 管理员权限

您需要 Magento(Adobe Commerce)管理员权限。使用普通用户权限无法创建集成。

签发密钥

Integration 访问令牌

创建集成(Integration)并 Activate 后，即会签发访问令牌。

2. 申请步骤

按以下顺序签发并在 GenieGo 中注册。

1

进入集成(Integration)菜单

在 Magento 管理后台中，进入 [系统] > [集成(Integration)]。

管理后台 → 系统 → 集成(Integration)
2

创建集成

通过 [Add New Integration] 创建集成并设置资源权限。

Add New Integration
3

通过 Activate 签发令牌

点击 [Activate] 以获取访问令牌。

Activate
4

确认信息

确认签发的访问令牌和店铺 URL(base_url)。

Activate 后显示访问令牌时，即表示已正确签发。

3. 需在 GenieGo 中注册的信息

字段说明
base_url店铺 URL
access_token签发的 Integration 访问令牌(请妥善保管)
在 GenieGo 集成中心的 [注册] 中输入 base_url·access_token 并保存后，凭证将以 AES-256-GCM 加密存储，并在查询时进行掩码处理。

4. 常见问题

问题原因解决方法
没有集成菜单管理员权限不足使用具有创建集成权限的管理员账户登录。
调用时权限错误未设置集成的资源权限，或未完成 Activate设置集成的资源权限并完成 Activate 以签发令牌。

最终检查清单

  • 已使用 Magento(Adobe Commerce)管理员权限登录。
  • 已在 系统 > 集成(Integration) 中通过 Add New Integration 创建集成。
  • 已设置资源权限并通过 Activate 签发访问令牌。
  • 已在 GenieGo 的 [注册] 中输入 base_url·access_token 并保存。

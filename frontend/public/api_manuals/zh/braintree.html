Braintree
API Key 申请手册

本手册手把手指导新手在 Braintree(PayPal) Control Panel 中查看 Production API 凭证(Merchant ID·Public Key·Private Key),并将其注册到 GenieGo 集成中心。

预计时间约 5~10 分钟
所需账户Braintree(PayPal) 商户账户(需审核通过)
签发密钥Merchant ID · Public Key · Private Key
成果Production API 凭证

1. 开始之前

Braintree(PayPal) 凭证只有在商户账户审核通过后才能查看。

必需

Braintree(PayPal) 商户账户

首先需要通过 Braintree(PayPal) 商户账户审核。只有商户账户审核完成后，才能查看 Production 凭证。

签发密钥

Merchant ID · Public/Private Key

在 Control Panel 的 Settings > API Keys 中查看 Production 环境的 Merchant ID·Public Key·Private Key。

注意: 请务必查看 Production 环境的凭证。注意不要与 Sandbox 密钥混淆。

2. 申请流程

商户账户审核通过后，按以下顺序查看凭证并在 GenieGo 中注册。

1

商户账户审核

首先完成 Braintree(PayPal) 商户账户审核。只有审核完成后才能进入下一步。

2

登录 Control Panel

登录 Braintree Control Panel。

braintreegateway.com
3

前往 API Keys

进入 [Settings] > [API Keys] 菜单。

Settings → API Keys
4

查看凭证

查看 Production 环境的 Merchant ID·Public Key·Private Key。

如果在 Settings > API Keys 中能看到 Production 的 Merchant ID·Public Key·Private Key，即表示正常。

3. 需在 GenieGo 中注册的信息

字段说明
merchant_idProduction Merchant ID
public_keyProduction Public Key
private_keyProduction Private Key(请妥善保管)
在 GenieGo 集成中心的 [注册] 中输入 merchant_id·public_key·private_key 并保存后，凭证将以 AES-256-GCM 加密存储，并在查询时进行掩码处理。

4. 常见问题

问题原因解决方法
API Keys 菜单中没有凭证商户账户审核未完成请先完成 Braintree(PayPal) 商户账户审核。
认证失败使用了 Sandbox 密钥，或 Private Key 填写错误重新查看 Production 环境的 Merchant ID·Public Key·Private Key。

最终检查清单

  • Braintree(PayPal) 商户账户审核已通过。
  • 已在 Control Panel 的 Settings > API Keys 中查看 Production 凭证。
  • 已确认 Merchant ID·Public Key·Private Key。
  • 已在 GenieGo 的 [注册] 中输入 merchant_id·public_key·private_key 并保存。

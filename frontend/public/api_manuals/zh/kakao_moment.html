Kakao Moment
API 密钥申请手册

本手册指导您申请 Kakao Moment 开放API权限，获取访问令牌和广告账户ID，并在 GenieGo 集成中心进行注册。开放API权限仅授予 Kakao 官方广告代理商。

成果开放API 凭证

1. 开始之前

Kakao Moment 开放API 并非自助申请，而需经过权限申请与审核流程。

重要: Kakao Moment 开放API权限仅授予 Kakao 官方广告代理商。普通广告主无法自助申请。
必需

广告账户 + 商业频道

您需要一个 Kakao Moment 广告账户和一个 Kakao 商业频道。

申请的密钥

访问令牌 · 广告账户ID

权限获批后，通过 OAuth 授权获取访问令牌并确认您的广告账户ID。

2. 申请流程

按以下顺序进行：开放API权限申请 → 审核/批准 → 令牌申请 → 注册。

1

创建应用并切换为商业应用

在 developers.kakao.com 创建应用，并将其切换为[商业应用]。

developers.kakao.com
2

申请开放API权限

提交 Kakao Moment 开放API[附加功能申请]。在审核您的官方代理商资格后授予权限。

附加功能申请
3

申请令牌

权限获批后，通过 OAuth 授权获取访问令牌并确认您的广告账户ID。您可以使用 GenieGo 集成中心的一键 OAuth。

权限获批且访问令牌和广告账户ID确认后，您即可进行注册。

3. 需在 GenieGo 注册的信息

字段说明
access_token通过 OAuth 授权申请的访问令牌(请谨慎处理)
ad_account_idKakao Moment 广告账户ID
在 GenieGo 集成中心[注册]中输入并保存，凭证将以 AES-256-GCM 静态加密存储，并在读取时进行掩码处理。

4. 常见问题

问题原因解决方法
无法获取开放API权限不是官方广告代理商开放API权限仅授予 Kakao 官方广告代理商。请通过代理商申请。
无法进行[附加功能申请]应用尚未切换为商业应用请先将应用切换为[商业应用]，然后再申请附加功能。

最终检查清单

  • 已准备好 Kakao Moment 广告账户和 Kakao 商业频道。
  • 已将应用切换为[商业应用]并提交了开放API[附加功能申请]。
  • 权限获批后，已通过 OAuth 授权确认访问令牌和广告账户ID。
  • 已在 GenieGo[注册]中输入 access_token·ad_account_id 并保存。

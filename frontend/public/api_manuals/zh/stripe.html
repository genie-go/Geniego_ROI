Stripe
API 密钥签发手册

一份面向初学者的分步指南，帮助您从 Stripe 仪表板的 Developers 菜单签发 API 密钥（Publishable Key·Secret Key），并将其注册到 GenieGo 集成中心。

预计时间约 5–10 分钟
所需账户Stripe 账户
签发密钥Publishable Key · Secret Key
结果API 凭据

1. 开始之前

Stripe API 密钥从您的 Stripe 账户仪表板签发。

必需

Stripe 账户

您必须能够使用 Stripe 账户登录 dashboard.stripe.com。

签发密钥

Publishable Key · Secret Key

在 Developers > API keys 界面查找您的 Publishable key 和 Secret key。

注意：对于真实的支付集成，请使用实时密钥，而非测试密钥。

2. 签发步骤

按以下顺序签发并在 GenieGo 中注册。

1

登录 Stripe

使用您准备好的 Stripe 账户登录仪表板。

dashboard.stripe.com
2

进入 API keys 菜单

进入 [Developers] > [API keys] 菜单。

Developers → API keys
3

查看密钥

查找您的 Publishable key 和 Secret key。使用实时密钥。

https://dashboard.stripe.com/apikeys
当 Publishable key 和 Secret key 都已确认后，即表示签发成功。

3. 需在 GenieGo 中注册的信息

字段说明
publishable_key已签发的 Publishable Key
secret_key已签发的 Secret Key（请妥善保管）
在 GenieGo 集成中心的[注册]中输入 publishable_key·secret_key 并保存。凭据会以 AES-256-GCM 静态加密存储，并在读取时进行掩码处理。

4. 常见问题

问题原因解决方法
认证失败输入了测试密钥检查您是否正在使用实时密钥（Publishable·Secret）。
看不到 Secret key该密钥仅显示一次或处于隐藏状态在 Developers > API keys 界面再次查看，或重新签发一个。

最终检查清单

  • 已使用 Stripe 账户登录 dashboard.stripe.com。
  • 已进入 Developers > API keys。
  • 已确认实时的 Publishable Key·Secret Key。
  • 已在 GenieGo [注册] 中输入 publishable_key·secret_key 并保存。

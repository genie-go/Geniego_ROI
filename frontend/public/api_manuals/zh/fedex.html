FedEx
API Key 申请手册

本手册整理了在 FedEx 开发者门户(developer.fedex.com)创建项目以签发 API Key 和 Secret Key，并连同 FedEx 账户编号一起注册到 GenieGo 集成中心的流程。

预计时间约 5~15 分钟
所需账户FedEx 账户(账户编号)
签发密钥API Key · Secret Key
成果API 凭证 · 账户编号

1. 开始之前

FedEx 凭证通过在开发者门户创建项目来签发，并且需要账户编号。

准备物

FedEx 账户编号

您需要一个 FedEx 账户编号。账户编号需在注册 GenieGo 时一并输入。

签发密钥

API Key · Secret Key

在开发者门户创建项目后，将签发 API Key 和 Secret Key。

2. 申请步骤

按以下顺序签发并在 GenieGo 中注册。

1

注册并登录开发者门户

注册 FedEx 开发者门户并登录。

https://developer.fedex.com
2

创建项目

创建项目后，将签发 API Key 和 Secret Key。

创建项目
3

确认密钥与账户编号

确认已签发的 API Key·Secret Key 以及 FedEx 账户编号。

创建项目后签发 API Key·Secret Key 即表示正常。

3. 需在 GenieGo 中注册的信息

字段说明
api_key签发的 API Key
api_secret签发的 Secret Key(请妥善保管)
account_numberFedEx 账户编号
在 GenieGo 集成中心 [注册] 中输入并保存后，凭证将以 AES-256-GCM 加密存储，并在查询时进行掩码处理。

4. 常见问题

问题原因解决方法
项目中看不到密钥项目创建未完成在 developer.fedex.com 创建项目后，将签发 API Key·Secret Key。
调用时出现认证错误Secret Key 填写错误或缺少账户编号重新确认 api_secret 和账户编号(account_number)。

最终检查清单

  • 已确认 FedEx 账户编号。
  • 已在 developer.fedex.com 注册并创建项目。
  • 已签发 API Key·Secret Key。
  • 已在 GenieGo [注册] 中输入 api_key·api_secret·account_number 并保存。

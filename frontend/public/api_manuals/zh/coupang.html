Coupang Wing
API Key 发放手册

本手册详细介绍如何在 Coupang WING 卖家控制台发放 Open API 认证密钥（Access Key·Secret Key）并将其注册到 GenieGo 集成中心，初学者也能照着操作完成。

预计耗时约5〜10分钟
所需账户Coupang WING 卖家账户
发放的密钥Access Key · Secret Key
成果Open API 认证凭据

1. 开始之前

Coupang Open API 只能从卖家账户发放。

必需

WING 卖家账户

您需要一个已完成企业认证的 Coupang WING 卖家账户。普通买家会员账户无法发放。

发放的密钥

Access Key · Secret Key

在发放完成页面查看 Access Key 和 Secret Key，并一并确认卖家（vendor）ID。

提示：发放时，您必须填写公司名称、主页／服务URL（如无则填 wing.coupang.com）以及访问IP（最多10个）。

2. Open API Key 发放步骤

按以下顺序发放并注册到 GenieGo。

1

登录 WING

使用卖家账户登录。

wing.coupang.com
2

进入 Open API 菜单

进入顶部／设置区域的 [Open API] 或 [API 认证密钥管理] 菜单。

Open API → API 认证密钥管理
3

发放 Open API Key

点击 [发放 Open API Key]，并填写公司名称、主页URL以及访问IP（最多10个）。

发放 Open API Key
4

确认密钥

在发放完成页面查看 Access Key·Secret Key 和卖家（vendor）ID。

当发放完成页面显示 Access Key·Secret Key 时，即表示已正常发放。

3. 需要注册到 GenieGo 的信息

字段说明
access_key已发放的 Access Key
secret_key已发放的 Secret Key（请妥善保管）
vendor_id卖家（vendor）ID
在 GenieGo 集成中心的 [注册] 中填写并保存。凭据将以 AES-256-GCM 加密存储，读取时进行掩码处理。注册后立即开始订单同步。

4. 常见问题

问题原因解决
没有发放菜单普通会员账户，或企业认证未完成请使用已完成企业认证的 WING 卖家账户登录。
调用时出现401错误访问IP未注册，或 Secret Key 填写错误请重新确认发放时填写的访问IP和 Secret Key。

最终检查清单

  • 已使用完成企业认证的 WING 卖家账户登录。
  • 已发放 Open API Key 并注册了访问IP。
  • 已确认 Access Key·Secret Key·Vendor ID。
  • 已在 GenieGo [注册] 中填写 access_key·secret_key·vendor_id 并保存。

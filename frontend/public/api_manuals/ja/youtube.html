YouTube API Key
発行マニュアル

初心者でもそのまま実践できるよう、Google Cloud Console で YouTube Data API v3 を有効化し API Key を作成する手順を整理したハンズオンガイドです。

所要時間約5〜10分
必要なアカウントGoogle アカウント
使用 APIYouTube Data API v3
成果物AIzaSy... 形式の API Key

1. API Key と OAuth の違い

始めたばかりの頃に最も混乱しやすい部分です。

API Key

公開データの取得用

動画検索、チャンネル情報の照会、再生リストの取得、コメントの取得など、公開された YouTube データを取得する際に使用します。

OAuth 2.0

アカウントの権限が必要な作業用

動画のアップロード、自分のチャンネル管理、非公開データへのアクセスなど、ユーザーの承認が必要な作業に使用します。

重要: 動画を自動でアップロードするには API Key だけでは不十分で、OAuth 2.0 認証が必要です。

2. YouTube Data API v3 の有効化

API Key を作成する前に、まず使用する API をプロジェクトで有効にする必要があります。

1

Google Cloud Console にアクセス

Google Cloud Console にログインします。

console.cloud.google.com
2

プロジェクトの選択または作成

上部のプロジェクト選択欄で既存のプロジェクトを選択するか、新しいプロジェクトを作成します。

例: Geniego ROI
3

API ライブラリへ移動

左メニューから API とサービス → ライブラリへ移動します。

API とサービス → ライブラリ
4

YouTube Data API v3 を検索して有効化

検索ボックスに YouTube Data API v3 を入力し、その API を選択して有効にするボタンをクリックします。

YouTube Data API v3 → 有効にする
画面に ステータス: 有効 と表示されれば、正常に有効化されています。

3. API Key の作成

YouTube Data API v3 が有効になると、認証情報画面で API Key を作成できます。

1

認証情報メニューへ移動

左メニューから API とサービス → 認証情報へ移動します。

API とサービス → 認証情報
2

認証情報を作成をクリック

上部の + 認証情報を作成ボタンをクリックします。

+ 認証情報を作成
3

API キーを選択

表示されるメニューから API キーを選択します。

API キー
4

アクセス可能な API を選択

API の制限で YouTube Data API v3 が選択されていることを確認します。

選択した API: YouTube Data API v3
5

作成ボタンをクリック

下部の作成ボタンをクリックすると API Key が生成されます。

作成
生成されたキーは通常 AIzaSy... で始まります。この文字列が YouTube API Key です。

4. 作成画面ではどのオプションを選べばよいですか？

項目初心者向け推奨設定説明
名前YouTube API Key後で分かりやすい名前を入力します。
API の制限YouTube Data API v3このキーが YouTube API にのみ使用されるよう制限します。
サービスアカウント認証チェックしないYouTube Data API Key の作成には通常必要ありません。
アプリケーションの制限最初のテストはなし正常に動作することを確認した後、ウェブサイトまたは IP アドレスの制限を設定することをおすすめします。
実際のサービスにデプロイする際は必ずキーの制限を設定してください。制限なしで公開されると、他人がキーを使用できてしまいます。

5. API Key の使用例

生成したキーを YOUR_API_KEY の部分に入れてテストできます。

動画検索の例https://www.googleapis.com/youtube/v3/search?part=snippet&q=korea&type=video&key=YOUR_API_KEY
チャンネル情報照会の例https://www.googleapis.com/youtube/v3/channels?part=snippet&id=CHANNEL_ID&key=YOUR_API_KEY
ブラウザのアドレスバーに上記の URL を入力したときに JSON 形式の結果が表示されれば、API Key は正常に動作しています。

6. よくある問題の解決

問題原因解決方法
選択された API がありませんYouTube Data API v3 がまだ有効化されていない、または API の制限で選択されていないAPI ライブラリで YouTube Data API v3 を有効にしてから、もう一度選択します。
表示する API キーがありませんまだ API Key を作成していない認証情報を作成 → API キーをクリックします。
403 エラーAPI の制限、割り当て超過、権限の問題API の制限、割り当て、有効化状態を確認します。
アップロードできないAPI Key のみを使用している動画のアップロードには OAuth 2.0 認証を使用する必要があります。

7. 最終チェックリスト

  • Google Cloud プロジェクトを選択した。
  • YouTube Data API v3 を有効にした。
  • 認証情報で API Key を作成した。
  • 生成された API Key が AIzaSy で始まることを確認した。
  • API の制限を YouTube Data API v3 に設定した。
  • 実際のサービスにデプロイする前にウェブサイトまたは IP の制限を設定する。

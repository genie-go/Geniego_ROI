Smart Tracker (Unified Tracking)
API Key 発行マニュアル

スウィートトラッカー(tracking.sweettracker.co.kr)に登録し、全配送会社の統合追跡キー(t_key)を発行して GenieGo 連携ハブに登録する手順を、初心者でもそのまま実行できるようにまとめたマニュアルです。t_key 一つですべての配送会社の配送追跡が連携されます。

所要時間約 5〜15 分
必要なアカウントスウィートトラッカー登録アカウント
発行するキーt_key (全配送会社の統合追跡キー)
成果物統合配送追跡 API 認証情報

1. はじめる前に

スマート配送(統合追跡)は、別途契約なしで登録のみで利用できる自己発行チャネルです。

必須

スウィートトラッカー登録アカウント

tracking.sweettracker.co.kr に登録したアカウントがあれば大丈夫です。別途契約なしで登録のみで発行できます。

発行するキー

t_key (統合追跡キー)

全配送会社の統合追跡キーである t_key を一つ発行すれば、すべての配送会社の配送追跡が連携されます。

注意: t_key は GenieGo 登録時に api_key フィールドに入力します。

2. t_key 発行手順

以下の順序で発行し、GenieGo に登録します。

1

スウィートトラッカー登録・ログイン

tracking.sweettracker.co.kr にアクセスして登録し、ログインします。

https://tracking.sweettracker.co.kr
2

t_key 発行申請

t_key(全配送会社の統合追跡キー)の発行を申請します。

3

t_key を確認

発行された t_key を確認します。

4

GenieGo 登録

GenieGo の [登録] に t_key(api_key)を入力すると、すべての配送会社の配送追跡が連携されます。

t_key が発行されれば正常に発行されたもので、登録後すぐに全配送会社の配送追跡が連携されます。

3. GenieGo に登録する情報

フィールド説明
api_key発行された t_key (全配送会社の統合追跡キー)
GenieGo 連携ハブの [登録] に入力して保存すると、認証情報は AES-256-GCM で暗号化して保存され、閲覧時にはマスキングされます。登録後すぐにすべての配送会社の配送追跡が連携されます。

4. よくある問題

問題原因解決方法
t_key 発行メニューが見つからないスウィートトラッカー未登録またはログインしていないtracking.sweettracker.co.kr に登録・ログインしてから t_key の発行を申請します。
配送追跡が連携されないt_key の誤記入または未入力発行された t_key を api_key フィールドに正確に入力したか再度確認します。

最終チェックリスト

  • tracking.sweettracker.co.kr に登録・ログインした。
  • t_key(全配送会社の統合追跡キー)の発行を申請した。
  • 発行された t_key を確認した。
  • GenieGo の [登録] に t_key を api_key フィールドとして入力して保存した。

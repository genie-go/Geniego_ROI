Stripe
APIキー発行マニュアル

StripeダッシュボードのDevelopersメニューからAPIキー（Publishable Key・Secret Key）を発行し、GenieGo連携ハブに登録する手順を、初心者でもそのまま実行できるようにまとめたマニュアルです。

所要時間約5〜10分
必要なアカウントStripeアカウント
発行するキーPublishable Key · Secret Key
成果物API認証情報

1. 始める前に

Stripe APIキーはStripeアカウントのダッシュボードから発行します。

必須

Stripeアカウント

Stripeアカウントで dashboard.stripe.com にログインできる必要があります。

発行するキー

Publishable Key · Secret Key

Developers > API keys 画面でPublishable keyとSecret keyを確認します。

注意: 実際の決済連携には、テストキーではなくライブキーを使用します。

2. 発行手順

以下の順序で発行し、GenieGoに登録します。

1

Stripeにログイン

準備したStripeアカウントでダッシュボードにログインします。

dashboard.stripe.com
2

API keysメニューへ移動

[Developers] > [API keys] メニューへ移動します。

Developers → API keys
3

キーの確認

Publishable keyとSecret keyを確認します。ライブキーを使用します。

https://dashboard.stripe.com/apikeys
Publishable keyとSecret keyの両方が確認できれば、正常に発行されています。

3. GenieGoに登録する情報

フィールド説明
publishable_key発行されたPublishable Key
secret_key発行されたSecret Key（取り扱い注意）
GenieGo連携ハブの[登録]でpublishable_key・secret_keyを入力して保存すると、認証情報はAES-256-GCMで暗号化して保存され、照会時にはマスキングされます。

4. よくある問題

問題原因解決方法
認証失敗テストキーを入力したライブキー（Publishable・Secret）を使用しているか確認します。
Secret keyが表示されないキーは一度しか表示されないか、隠された状態Developers > API keys 画面で再確認するか、新たに発行します。

最終チェックリスト

  • Stripeアカウントで dashboard.stripe.com にログインした。
  • Developers > API keys へ移動した。
  • ライブのPublishable Key・Secret Keyを確認した。
  • GenieGoの[登録]にpublishable_key・secret_keyを入力して保存した。

11Street
API Key 発行マニュアル

11Street オープンAPIセンター(openapi.11st.co.kr)でオープンAPI キーを発行し、GenieGo 連携ハブに登録する手順を、初心者でもそのまま実行できるようにまとめたマニュアルです。

所要時間約 5〜15 分
必要なアカウント11Street セラー(販売者)アカウント
発行するキーオープンAPI キー · セラー ID
成果物オープンAPI 認証情報

1. はじめる前に

11Street オープンAPI は、セラー(販売者)アカウントからのみ発行できます。

必須

11Street セラーアカウント

11Street セラー(販売者)アカウントが必要です。オープンAPIセンターにセラー ID でログインします。

発行するキー

オープンAPI キー · セラー ID

サービス登録とアクセス IP 登録を完了すると発行されるオープンAPI キーとセラー ID を併せて確認します。

注意: API にアクセスできるサーバー IP を登録しないとキーを使用できません。

2. 発行手順

以下の順序で発行し、GenieGo に登録します。

1

オープンAPIセンターにログイン

セラー ID で 11Street オープンAPIセンターにログインします。

https://openapi.11st.co.kr/openapi/OpenApiFrontMain.tmall
2

サービス登録

メニューで [サービス登録・確認] をクリックし、セラー情報を入力して規約に同意して登録します。

サービス登録・確認
3

アクセス IP 登録

[Seller API 情報] > [修正] で API にアクセスできるサーバー IP を登録します。

Seller API 情報 → 修正
4

API Key をコピー

発行された API Key をコピーします。

サービス登録とアクセス IP 登録を完了すると、API Key が発行されます。

3. GenieGo に登録する情報

フィールド説明
api_key発行されたオープンAPI キー
seller_idセラー ID
GenieGo 連携ハブの [登録] に入力して保存すると、認証情報は AES-256-GCM で暗号化して保存され、閲覧時にはマスキングされます。

4. よくある問題

問題原因解決方法
API Key が発行されないサービス登録またはアクセス IP 登録が未完了[サービス登録・確認] と [Seller API 情報] のアクセス IP 登録を完了します。
呼び出し時に認証エラーアクセス可能なサーバー IP が未登録[Seller API 情報] > [修正] でアクセス可能なサーバー IP を再確認します。

5. 商品登録の前に（重要）

APIキーが発行されても、以下の3点が揃っていないと11番街は商品登録を拒否します。

  • 表示カテゴリ（dispCtgrNo）— GenieGo [カタログ同期] > [カテゴリマッピング]、または「カテゴリ・ブランドの確定が必要」パネルで指定
  • ブランド — GenieGo [カタログ同期] > [ブランド管理]で登録し、商品に指定（ブランドコードがなくてもブランド名だけで送信されます）
  • 販売方式 — GenieGoが固定価格販売（01）として自動送信するため、入力は不要です
エラーメッセージ本当の意味対処法
-997 登録されたAPI情報が存在しません認証エラーではありません。呼び出したAPIのアドレス（パス＋メソッド）が11番街に存在しないという意味です。キーを再発行しないでください。連携担当者に呼び出しアドレスの確認を依頼してください。
ブランドコードがありません商品にブランドが指定されていません。[カタログ同期] > [ブランド管理]でブランドを登録し、商品に指定してください。
APIで利用できない販売方式です販売方式コードがAPIの許容値ではありません。GenieGoが固定価格販売（01）を自動送信します。対応は不要です。
キーを再発行する前に確認してください。 -997 はキー・IP・API利用申請の問題ではありません。3つすべて正常でも、呼び出しアドレスが誤っていれば -997 になります。

最終チェックリスト

  • 11Street セラーアカウントでオープンAPIセンターにログインした。
  • [サービス登録・確認] でセラー情報を入力し、規約に同意した。
  • [Seller API 情報] でアクセス可能なサーバー IP を登録した。
  • 発行されたオープンAPI キーとセラー ID を確認した。
  • GenieGo の [登録] に api_key·seller_id を入力して保存した。

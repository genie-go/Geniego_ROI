Naver Smart Store
API 発行マニュアル

Naver Smart Store の認証情報（API キー/トークン）を発行し、GenieGo 連携ハブに登録する手順を、初心者でもそのまま実行できるように段階的にまとめました。

所要時間約 5〜15 分
必要なアカウントNaver Smart Store アカウント
ステップ数6 ステップ
成果物API 認証情報

発行手順

以下の手順どおりに進めると認証情報を発行・登録できます。

1

ご用意いただくもの: スマートストア販売者アカウントの「統合マネージャー」権限が必要です（統合マネージャーのみ発行可能）。

2

apicenter.commerce.naver.com（NAVER コマース API センター）にアクセスします。

3

右上の [アカウント作成] をクリックし、開発業者のアカウント名・障害対応連絡先を入力し、規約に同意して登録します。

4

[マイストアアプリケーション] でアプリケーションを登録すると、アプリケーション ID と Secret が発行されます。

5

API を呼び出すサーバー IP を登録し、案内に従って定期的にアプリケーション認証を更新します。

6

GenieGo の [登録] に client_id、client_secret を入力して保存します。

GenieGo 連携ハブ [登録] に入力したキーは AES-256-GCM で暗号化保存され、閲覧時はマスキングされます。
発行したキー/シークレットは外部に漏れないよう安全に保管してください。漏洩が疑われる場合は直ちに再発行（ローテーション）してください。

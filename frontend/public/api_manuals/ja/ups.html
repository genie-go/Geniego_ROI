UPS
API Key 発行マニュアル

UPS 開発者ポータル(developer.ups.com)でアプリを作成して client_id と client_secret を発行し、UPS アカウント番号とともに GenieGo 連携ハブに登録する手順をまとめたマニュアルです。

所要時間約 5〜15 分
必要なアカウントUPS アカウント(アカウント番号)
発行するキーClient ID · Client Secret
成果物OAuth 認証情報 · アカウント番号

1. はじめる前に

UPS の認証情報は開発者ポータルでアプリを作成して発行し、アカウント番号が必要です。

準備物

UPS アカウント番号

UPS アカウント番号が必要です。アカウント番号は GenieGo 登録時に一緒に入力します。

発行するキー

Client ID · Client Secret

開発者ポータルでアプリを作成すると client_id と client_secret が発行されます。

2. 発行手順

以下の順序で発行し、GenieGo に登録します。

1

開発者ポータルに登録・アプリ作成

UPS 開発者ポータルに登録してアプリを作成します。

https://developer.ups.com
2

client_id·client_secret を発行

アプリを作成すると client_id と client_secret が発行されます。

アプリ作成
3

アカウント番号を確認

発行された client_id·client_secret と UPS アカウント番号を確認します。

アプリ作成後に client_id·client_secret が発行されれば正常です。

3. GenieGo に登録する情報

フィールド説明
client_id発行された Client ID
client_secret発行された Client Secret(取り扱い注意)
account_numberUPS アカウント番号
GenieGo 連携ハブ [登録] に入力して保存すると、認証情報は AES-256-GCM で暗号化して保存され、閲覧時にはマスキングされます。

4. よくある問題

問題原因解決方法
アプリで認証情報が表示されないアプリ作成が未完了developer.ups.com でアプリを作成すると client_id·client_secret が発行されます。
呼び出し時に認証エラーclient_secret の誤記、またはアカウント番号の欠落client_secret とアカウント番号(account_number)を再確認します。

最終チェックリスト

  • UPS アカウント番号を確認した。
  • developer.ups.com に登録してアプリを作成した。
  • client_id·client_secret を発行した。
  • GenieGo の [登録] に client_id·client_secret·account_number を入力して保存した。

Toss Payments
API Key 発行マニュアル

トスペイメンツの加盟店管理ツールの開発者センターでクライアントキーとシークレットキーを確認し、GenieGo 連携ハブに登録する手順をまとめたマニュアルです。

所要時間約 5〜10 分
必要なアカウントトスペイメンツ加盟店契約 · 加盟店管理アカウント
発行するキークライアントキー · シークレットキー
成果物決済連携の認証情報

1. はじめる前に

トスペイメンツは加盟店契約後、加盟店管理ツールでキーを確認できます。

必須

加盟店契約 · 加盟店管理アカウント

トスペイメンツの加盟店契約が完了しており、加盟店管理ツールにログインできる必要があります。

発行するキー

クライアントキー · シークレットキー

開発者センターでクライアントキーとシークレットキーをセットで確認し、一緒に使用します。

注意: 開発者センターではテストとライブが区別されるため、実際の連携にはライブキーを使用しているか確認します。

2. 発行手順

以下の順序で確認し、GenieGo に登録します。

1

加盟店管理ツールにログイン

トスペイメンツの加盟店管理ツールにログインします。

https://dashboard.tosspayments.com
2

開発者センターへ移動

左下の [開発者センター] へ移動します。

開発者センター
3

キーの確認

テスト/ライブを選択してクライアントキーとシークレットキーを確認します。2 つのキーはセットで使用します。

クライアントキーとシークレットキーの両方が確認できれば、登録準備が完了です。

3. GenieGo に登録する情報

フィールド説明
client_key開発者センターで確認したクライアントキー
secret_key開発者センターで確認したシークレットキー(取り扱い注意)
GenieGo 連携ハブ [登録] に入力して保存すると、認証情報は AES-256-GCM で暗号化して保存され、閲覧時にはマスキングされます。

4. よくある問題

問題原因解決方法
認証エラーが発生するテストキーとライブキーを混同している開発者センターでテスト/ライブの区別を確認し、環境に合ったキーを使用します。
開発者センターのメニューがない加盟店契約が未完了加盟店契約が完了した加盟店管理アカウントでログインします。

最終チェックリスト

  • トスペイメンツの加盟店契約を完了し、加盟店管理ツールにログインした。
  • 左下の [開発者センター] へ移動した。
  • テスト/ライブを区別してクライアントキー · シークレットキーを確認した。
  • GenieGo [登録] に client_key · secret_key を入力して保存した。

🛍 Beginner Guide

Shopify
API Key 発行マニュアル

Shopify 管理画面（Admin）でカスタムアプリを作成して Admin API アクセストークンを発行し、Shop ドメインとともに GenieGo 連携ハブに登録して Shopify ストアを連携する手順をまとめたマニュアルです。

所要時間約5～15分
必要なアカウントShopify ストアの管理者権限
発行キーShop ドメイン · Admin API アクセストークン
成果物Admin API 認証情報

1. 始める前に

Shopify のカスタムアプリは、ストアの管理者権限がないと作成できません。

必要

ストア管理者権限

カスタムアプリを作成してトークンを発行するには、Shopify ストアの管理者（Admin）権限が必要です。

発行キー

Shop ドメイン · Admin API トークン

xxx.myshopify.com 形式の Shop ドメインと、アプリのインストール後に表示される Admin API アクセストークンを確認します。

注意: Admin API アクセストークンはインストール直後に一度だけ表示されるため、表示されたらすぐにコピーしておく必要があります。

2. 発行手順

以下の順序で発行し、GenieGo に登録します。

1

アプリと販売チャネルへ移動

Shopify 管理画面（Admin）> [設定] > [アプリと販売チャネル] へ移動します。

https://www.shopify.com/admin/settings/apps
2

カスタムアプリを作成

[アプリ開発] > [アプリを作成] をクリックしてカスタムアプリを作成します。

アプリ開発 → アプリを作成
3

スコープを選択 · アプリをインストール

Admin API のスコープ（scope）を選択してアプリをインストールします。

4

アクセストークンをコピー

インストール後に表示される Admin API アクセストークンをすぐにコピーします（一度だけ表示されます）。

アプリのインストール後に Admin API アクセストークンが表示されれば、正常に発行されています。一度しか表示されないため、すぐにコピーしてください。

3. GenieGo に登録する情報

フィールド説明
shop_domainxxx.myshopify.com 形式の Shop ドメイン
access_token発行された Admin API アクセストークン（取り扱い注意）
GenieGo の [登録] に shop_domain（xxx.myshopify.com）と access_token を入力して保存すると、認証情報は AES-256-GCM で暗号化保存され、参照時にはマスキングされます。

4. よくある問題

問題原因解決
トークンを再表示できないAdmin API アクセストークンは1回のみ表示トークンを紛失した場合は、カスタムアプリでトークンを再発行してすぐにコピーします。
権限不足エラーAdmin API スコープ（scope）未選択必要な Admin API スコープ（scope）を選択してから、アプリを再インストールします。
ドメインエラーShop ドメイン形式の誤記xxx.myshopify.com 形式の Shop ドメインを正確に入力します。

最終チェックリスト

  • Shopify ストアの管理者権限でログインした。
  • 設定 > アプリと販売チャネル > アプリ開発でカスタムアプリを作成した。
  • Admin API スコープ（scope）を選択してアプリをインストールした。
  • インストール後に表示された Admin API アクセストークンをすぐにコピーした。
  • GenieGo の [登録] に shop_domain·access_token を入力して保存した。

API Key 発行マニュアル

ロジェン宅配は別途発行不要で、スマート宅配(スイートトラッカー)の統合追跡キーで連携します。tracking.sweettracker.co.kr で t_key を発行し、[スマート宅配(統合追跡)] チャネルに登録すると、ロジェン宅配の配送追跡が連携されます。

所要時間約 5〜15 分
必要なアカウントスマート宅配(スイートトラッカー)アカウント
発行するキースマート宅配 追跡 API キー(t_key)
成果物統合配送追跡の認証情報

1. はじめる前に

ロジェン宅配は専用発行なしで、スマート宅配の統合追跡キーで連携します。

方式

スマート宅配(スイートトラッカー)統合追跡

ロジェン宅配は別途発行不要で、スマート宅配(スイートトラッカー)の統合追跡キーで連携します。t_key 一つで全宅配会社の配送追跡を処理します。

発行するキー

t_key (スマート宅配 追跡 API キー)

tracking.sweettracker.co.kr に登録して発行する統合追跡キーです。GenieGo には api_key フィールドに入力します。

重要: ロジェン宅配チャネルを直接選択せず、[スマート宅配(統合追跡)] チャネルに t_key を登録してください。

2. 発行手順

以下の順序で発行し、GenieGo に登録します。

1

スイートトラッカーに登録

スマート宅配(スイートトラッカー)に登録します。

tracking.sweettracker.co.kr
2

t_key を発行

tracking.sweettracker.co.kr で t_key を発行します。

3

スマート宅配チャネルを選択

連携ハブでロジェン宅配チャネルの代わりに [スマート宅配(統合追跡)] チャネルを選択します。

4

GenieGo に登録

GenieGo [登録] に t_key(api_key)を入力すると、ロジェン宅配の配送追跡が連携されます。

t_key を発行して [スマート宅配(統合追跡)] チャネルに登録すると、ロジェン宅配の配送追跡が連携されます。

3. GenieGo に登録する情報

フィールド説明
api_keyスイートトラッカーで発行した t_key(スマート宅配 追跡 API キー)
GenieGo 連携ハブ [登録] に入力して保存すると、認証情報は AES-256-GCM で暗号化して保存され、閲覧時にはマスキングされます。

4. よくある問題

問題原因解決方法
ロジェン宅配チャネルにキーを登録できないロジェン宅配は専用発行なしで統合追跡キーで連携される[スマート宅配(統合追跡)] チャネルを選択して t_key を登録します。
配送追跡が連携されないt_key の誤記入または未発行tracking.sweettracker.co.kr で発行した t_key を再確認します。

最終チェックリスト

  • tracking.sweettracker.co.kr に登録した。
  • t_key(スマート宅配 追跡 API キー)を発行した。
  • 連携ハブで [スマート宅配(統合追跡)] チャネルを選択した。
  • GenieGo [登録] に t_key(api_key)を入力して保存した。

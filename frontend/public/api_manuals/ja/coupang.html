Coupang Wing
API 発行マニュアル

Coupang Wing の認証情報（API キー/トークン）を発行し、GenieGo 連携ハブに登録する手順を、初心者でもそのまま実行できるように段階的にまとめました。

所要時間約 5〜15 分
必要なアカウントCoupang Wing アカウント
ステップ数6 ステップ
成果物API 認証情報

発行手順

以下の手順どおりに進めると認証情報を発行・登録できます。

1

ご用意いただくもの: Coupang WING の販売者アカウント（事業者認証完了済み）が必要です。一般会員では発行できません。

2

wing.coupang.com に販売者アカウントでログインします。

3

上部/設定エリアの [Open API] または [API 認証キー管理] メニューに移動します。

4

[Open API Key 発行] をクリックし、会社名とホームページ/サービス URL（ない場合は wing.coupang.com）、アクセス IP（最大 10 個）を入力します。

5

発行完了画面で Access Key と Secret Key を確認し、販売者（Vendor）ID もあわせて確認します。

6

GenieGo の [登録] に access_key、secret_key、vendor_id を入力して保存します。

GenieGo 連携ハブ [登録] に入力したキーは AES-256-GCM で暗号化保存され、閲覧時はマスキングされます。
発行したキー/シークレットは外部に漏れないよう安全に保管してください。漏洩が疑われる場合は直ちに再発行してください。

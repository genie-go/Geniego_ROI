Coupang Wing
API Key 発行マニュアル

Coupang WING 販売者コンソールで Open API 認証キー（Access Key・Secret Key）を発行し、GenieGo 連携ハブに登録する手順を、初心者でもそのまま進められるようにまとめたマニュアルです。

所要時間約5〜10分
必要なアカウントCoupang WING 販売者アカウント
発行するキーAccess Key · Secret Key
成果物Open API 認証情報

1. 始める前に

Coupang Open API は販売者アカウントからのみ発行できます。

必要

WING 販売者アカウント

事業者認証が完了した Coupang WING 販売者アカウントが必要です。一般の購入会員アカウントでは発行できません。

発行するキー

Access Key · Secret Key

発行完了画面で Access Key と Secret Key を確認し、販売者（vendor）ID も併せて確認します。

注記: 発行時には会社名・ホームページ／サービスURL（無い場合は wing.coupang.com）・アクセスIP（最大10件）を入力する必要があります。

2. Open API Key の発行手順

以下の順序で発行し、GenieGo に登録します。

1

WING にログイン

販売者アカウントでログインします。

wing.coupang.com
2

Open API メニューへ移動

上部／設定エリアの [Open API] または [API 認証キー管理] メニューへ移動します。

Open API → API 認証キー管理
3

Open API Key を発行

[Open API Key 発行] をクリックし、会社名・ホームページURL・アクセスIP（最大10件）を入力します。

Open API Key 発行
4

キーを確認

発行完了画面で Access Key・Secret Key と販売者（vendor）ID を確認します。

発行完了画面に Access Key・Secret Key が表示されれば、正常に発行されています。

3. GenieGo に登録する情報

フィールド説明
access_key発行された Access Key
secret_key発行された Secret Key（取り扱い注意）
vendor_id販売者（vendor）ID
GenieGo 連携ハブの [登録] に入力して保存します。認証情報は AES-256-GCM で暗号化保存され、参照時にはマスキングされます。登録後すぐに注文同期が開始されます。

4. よくある問題

問題原因解決
発行メニューが表示されない一般会員アカウント、または事業者認証が未完了事業者認証が完了した WING 販売者アカウントでログインします。
呼び出し時に401エラーアクセスIPが未登録、または Secret Key の入力ミス発行時に入力したアクセスIPと Secret Key を再確認します。

最終チェックリスト

  • 事業者認証が完了した WING 販売者アカウントでログインした。
  • Open API Key を発行し、アクセスIPを登録した。
  • Access Key・Secret Key・Vendor ID を確認した。
  • GenieGo の [登録] に access_key・secret_key・vendor_id を入力して保存した。

Manual de emisión de API Key

Una guía paso a paso y apta para principiantes para emitir un conjunto de claves de producción (Production) (DevID·AppID·CertID) en el Programa para Desarrolladores de eBay, generar un token de usuario OAuth y registrarlo en el GenieGo Integration Hub.

Tiempo estimadoUnos 10–20 min
Cuenta necesariaCuenta de eBay
Claves emitidasDevID · AppID · CertID + token OAuth
ResultadoToken de acceso OAuth

1. Antes de empezar

Emitir un conjunto de claves de eBay requiere la inscripción en el Programa para Desarrolladores.

Requerido

Cuenta de eBay

Para continuar con la emisión necesitas una cuenta de eBay. Usa esta cuenta para inscribirte en el Programa para Desarrolladores en developer.ebay.com.

Claves emitidas

DevID · AppID · CertID

Al crear un conjunto de claves de producción (Production) se emiten un DevID, un AppID y un CertID. Después también debes generar un token de usuario OAuth para registrarlo en GenieGo.

Nota: Un conjunto de claves de producción puede mostrar el aviso "Your Keyset is currently disabled". En ese caso, se activa solo después de completar el procedimiento de suscripción/rechazo de la notificación de eliminación de cuentas del marketplace.

2. Pasos de emisión

Emite en el orden siguiente y registra en GenieGo.

1

Inscríbete en el Programa para Desarrolladores

Ve a developer.ebay.com e inscríbete en el Programa para Desarrolladores con tu cuenta de eBay.

developer.ebay.com
2

Crea un conjunto de claves de producción

En la página [Application Keys], haz clic en [Create a keyset] en la sección Production.

https://developer.ebay.com/my/keys
3

Procedimiento de activación del conjunto de claves

Si ves el aviso "Your Keyset is currently disabled", haz clic en el enlace para completar el procedimiento de suscripción/rechazo de la notificación de eliminación de cuentas del marketplace.

Application Keys → Production
4

Verifica las claves y genera un token OAuth

Verifica el DevID, el AppID y el CertID emitidos y genera un token de usuario OAuth.

Una vez que se emiten el DevID·AppID·CertID y se genera un token de usuario OAuth, la emisión está completa.

3. Información para registrar en GenieGo

CampoDescripción
Token de acceso OAuthEl token de usuario OAuth que generaste (manéjalo con cuidado)
En el GenieGo Integration Hub [Registrar], introduce tu token de acceso OAuth y guarda. Las credenciales se cifran en reposo con AES-256-GCM y se enmascaran al leerlas.

4. Problemas frecuentes

ProblemaCausaSolución
El conjunto de claves está deshabilitadoNo se ha completado el procedimiento de suscripción/rechazo de la notificación de eliminación de cuentas del marketplaceHaz clic en el enlace del aviso "Your Keyset is currently disabled" y completa ese procedimiento.
No se puede emitir el token OAuthEl conjunto de claves de producción (DevID·AppID·CertID) aún no se ha emitidoEn [Application Keys], crea primero el conjunto de claves de producción y luego genera el token OAuth.

Lista de verificación final

  • Te inscribiste en el Programa para Desarrolladores en developer.ebay.com con tu cuenta de eBay.
  • Emitiste el DevID·AppID·CertID mediante Production [Create a keyset].
  • Cuando el conjunto de claves estaba deshabilitado, completaste el procedimiento de suscripción/rechazo de la notificación de eliminación de cuentas del marketplace.
  • Generaste un token de usuario OAuth e introdujiste el token de acceso OAuth en GenieGo [Registrar] y guardaste.

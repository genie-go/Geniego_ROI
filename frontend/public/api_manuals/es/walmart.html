🏪 Guía para principiantes

Walmart Marketplace
Manual de emisión de API Key

Una guía paso a paso y apta para principiantes para crear un client_id y un Client Secret en el Developer Portal del Seller Center de Walmart Marketplace y registrarlos en el GenieGo Integration Hub.

Tiempo estimadoUnos 5–15 min
Cuenta necesariaCuenta de vendedor de Walmart Marketplace
Claves emitidasClient ID · Client Secret
ResultadoCredenciales de API

1. Antes de empezar

Las credenciales de la API de Walmart se emiten desde el Developer Portal del Seller Center del Marketplace.

Requerido

Cuenta de vendedor de Walmart Marketplace

Para proceder con la emisión, necesita una cuenta de vendedor de Walmart Marketplace. Use esta cuenta para acceder al Developer Portal en el Seller Center.

Claves emitidas

Client ID · Client Secret

Cree una API Key (client_id) y un Client Secret en el Developer Portal.

2. Pasos de emisión

Emita en el orden siguiente y registre en GenieGo.

1

Acceder al Developer Portal

Vaya a Walmart Seller Center > Developer Portal.

https://developer.walmart.com
2

Crear credenciales

Cree una API Key (client_id) y un Client Secret.

3

Verificar los valores

Verifique el client_id y el client_secret emitidos.

Una vez creados el client_id y el Client Secret, se han emitido correctamente.

3. Información para registrar en GenieGo

CampoDescripción
client_idLa API Key emitida (Client ID)
client_secretEl Client Secret emitido (manéjelo con cuidado)
Introduzca su client_id y client_secret en el GenieGo Integration Hub [Registrar] y guarde. Las credenciales se cifran en reposo con AES-256-GCM y se enmascaran al leerse.

4. Problemas frecuentes

ProblemaCausaSolución
No se puede acceder al Developer PortalNo es una cuenta de vendedor de Walmart MarketplaceAcceda al Seller Center > Developer Portal con una cuenta de vendedor de Walmart Marketplace.
Fallo de autenticaciónclient_id o client_secret introducidos incorrectamenteVuelva a comprobar el client_id y el client_secret emitidos en el Developer Portal.

Lista de comprobación final

  • Accedí al Seller Center > Developer Portal con una cuenta de vendedor de Walmart Marketplace.
  • Creé una API Key (client_id) y un Client Secret.
  • Verifiqué los valores de client_id y client_secret.
  • Introduje el client_id y el client_secret en GenieGo [Registrar] y guardé.

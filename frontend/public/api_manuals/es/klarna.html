Klarna
Manual de emisión de API

Una guía paso a paso que cualquiera puede seguir para emitir credenciales de Klarna (clave/token de API) y registrarlas en el GenieGo Integration Hub.

Tiempo estimadoUnos 5–15 min
Cuenta necesariaCuenta de Klarna
Pasos4 pasos
ResultadoCredenciales de API

Pasos de emisión

Siga los pasos siguientes para emitir y registrar sus credenciales.

1

Requisitos: un contrato de comercio de Klarna aprobado (BNPL).

2

Inicie sesión en el Merchant Portal en portal.klarna.com.

3

En [Settings] > [Klarna API credentials], cree su API username/password y anote su region (eu/na/oc).

4

En GenieGo [Register], introduzca el username, password y region, y guarde.

Lista de verificación final

  • Requisitos: un contrato de comercio de Klarna aprobado (BNPL).
  • Inicie sesión en el Merchant Portal en portal.klarna.com.
  • En [Settings] > [Klarna API credentials], cree su API username/password y anote su region (eu/na/oc).
  • En GenieGo [Register], introduzca el username, password y region, y guarde.
Las claves introducidas en GenieGo Integration Hub [Register] se almacenan cifradas con AES-256-GCM y se enmascaran al consultarlas.
Guarde sus claves/secretos de forma segura y nunca los exponga públicamente. Si sospecha una filtración, vuelva a emitirlas (rotar) de inmediato.

TikTok Shop
Manual de emisión de API Key

Una guía paso a paso fácil para principiantes para crear una app (App Key·App Secret·Service ID) en TikTok Shop Partner Center, emitir un access_token mediante la autorización de la tienda y registrarlo en el GenieGo Integration Hub.

Tiempo estimadoUnos 10–15 min
Cuenta necesariaCuenta de vendedor de TikTok Shop
Claves emitidasApp Key · App Secret · Access Token
ResultadoCredenciales de API

1. Antes de empezar

Un token solo se emite después de crear una app en Partner Center y autorizar una tienda.

Requerido

Cuenta de vendedor de TikTok Shop

Necesita una cuenta de vendedor de TikTok Shop y debe registrarse como desarrollador en partner.tiktokshop.com (Partner Center).

Claves emitidas

App Key · App Secret · Access Token

Al crear la app obtiene App Key·App Secret·Service ID, y obtiene un access_token mediante la autorización de la tienda.

Nota: Se requiere una revisión de la app al pasar a producción.

2. Pasos de emisión

Emita en el orden siguiente y regístrelo en GenieGo.

1

Registrarse como desarrollador en Partner Center

Regístrese como desarrollador en partner.tiktokshop.com (Partner Center).

https://partner.tiktokshop.com
2

Crear una app

Cree una app mediante [App & Service] > [Create App] y verifique el App Key, App Secret y Service ID.

App & Service → Create App
3

Habilitar permisos de API

En la configuración de la app, habilite los permisos de API necesarios (Shop Authorized Information, Product, Order).

4

Autorización de la tienda → emitir access_token

En [Get shop authorization], seleccione la Región (Region) para autorizar la tienda del vendedor, y se emitirá un access_token.

Get shop authorization
5

Verificar las claves

Verifique el App Key·App Secret y el access_token emitido. Si es necesario, verifique también el Shop Cipher.

Una vez emitido el access_token tras la autorización de la tienda, se ha procesado correctamente.

3. Información para registrar en GenieGo

CampoDescripción
app_keyEl App Key emitido
app_secretEl App Secret emitido (manéjelo con cuidado)
access_tokenEl token de acceso emitido mediante la autorización de la tienda
shop_cipherShop Cipher (opcional)
Introduzca app_key·app_secret·access_token en [Registrar] del GenieGo Integration Hub y guarde. Las credenciales se cifran en reposo con AES-256-GCM y se enmascaran al leerlas.

4. Problemas frecuentes

ProblemaCausaSolución
No se emite el access_tokenAutorización de la tienda (Get shop authorization) no completada, o Región (Region) no seleccionadaEn [Get shop authorization], seleccione la Región y autorice la tienda del vendedor.
Error de permisosPermisos de API necesarios no habilitadosEn la configuración de la app, habilite los permisos Shop Authorized Information·Product·Order.
Llamadas de producción denegadasNo se superó la revisión del cambio a producciónComplete la revisión de la app necesaria para pasar a producción.

Lista de verificación final

  • Me registré como desarrollador en partner.tiktokshop.com.
  • Verifiqué el App Key·App Secret·Service ID mediante App & Service > Create App.
  • Habilité los permisos de API necesarios (Shop/Product/Order).
  • Emití un access_token mediante la autorización de la tienda (Get shop authorization).
  • Introduje app_key·app_secret·access_token en GenieGo [Registrar] y guardé.

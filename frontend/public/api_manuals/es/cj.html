CJ Logistics
Manual de emisión de API Key

La API dedicada de CJ Logistics se emite en función de un código de contrato (cliente). Este manual le guía para obtener el código de contrato (cliente) y la clave de API de su representante de ventas y registrarlos en el GenieGo Integration Hub.

Tiempo estimadoDe varios días a semanas, según el avance del contrato
Cuenta necesariaContrato de CJ Logistics (cliente corporativo)
Claves emitidasClave de API · Código de cliente (contrato)
ResultadoCredenciales de API

1. Antes de empezar

La API dedicada de CJ Logistics es un canal basado en contrato y no se emite al instante desde una consola de autoservicio.

Obligatorio

Código de contrato (cliente) de CJ Logistics

Debe contar con un código de contrato (cliente) emitido mediante un contrato con CJ Logistics. No se emite directamente desde una consola aparte.

Recomendado

Smart Parcel (seguimiento unificado)

Si solo necesita seguimiento de envíos, puede conectar todas las empresas de mensajería con una sola t_key a través del canal [Smart Parcel (seguimiento unificado)].

Nota: Si su objetivo es el seguimiento de envíos, recomendamos conectarse a través de la clave unificada de Smart Parcel (SweetTracker) sin un contrato de API dedicado.

2. Procedimiento de solicitud de emisión

Solicite y registre en el orden que se indica a continuación.

1

Prepare el código de contrato (cliente)

Prepare su código de contrato (cliente) de CJ Logistics. La API dedicada se proporciona en régimen de contrato.

2

Decida si necesita seguimiento unificado

Si solo necesita seguimiento de envíos, puede conectar todas las empresas de mensajería con una sola t_key a través del canal [Smart Parcel (seguimiento unificado)] en lugar del canal dedicado de CJ Logistics.

3

Solicite la API dedicada

Si necesita la API dedicada de CJ Logistics, obtenga el código de contrato (cliente) y la clave de API de su representante de ventas.

4

Registre en GenieGo

Introduzca api_key y cust_code en [Registrar] del GenieGo Integration Hub y guarde.

Una vez que reciba la clave de API y el código de contrato (cliente) de su representante de ventas, estará listo para registrarse.

3. Información para registrar en GenieGo

CampoDescripción
api_keyLa clave de API dedicada de CJ Logistics emitida
cust_codeCódigo de contrato (cliente) de CJ Logistics
Cuando lo introduzca y guarde en [Registrar] del GenieGo Integration Hub, las credenciales se cifran en reposo con AES-256-GCM y se enmascaran al leerlas.

Lista de verificación final

  • Conseguí el código de contrato (cliente) de CJ Logistics.
  • Revisé la integración de Smart Parcel (seguimiento unificado) si solo se necesita seguimiento de envíos.
  • Obtuve la clave de API y el código de contrato (cliente) del representante de ventas.
  • Introduje api_key·cust_code en [Registrar] de GenieGo y guardé.

Rakuten
Manual de emisión de API Key

Una guía paso a paso para principiantes sobre cómo emitir las credenciales del RMS Web Service (Service Secret y License Key) desde la consola de vendedor de Rakuten RMS y registrarlas en el GenieGo Integration Hub.

Tiempo estimadoUnos 5–15 min
Cuenta necesariaCuenta de vendedor de Rakuten (Ichiba)
Claves emitidasService Secret · License Key
ResultadoCredenciales del RMS Web Service

1. Antes de empezar

Las claves de vendedor (Ichiba) se emiten desde el RMS Web Service.

Requerido

Cuenta de vendedor de Rakuten

Necesitas una cuenta de vendedor de Rakuten (Ichiba). Necesitas una cuenta que pueda iniciar sesión mediante RMS R-Login (glogin.rms.rakuten.co.jp).

Claves emitidas

Service Secret · License Key

Obtendrás un serviceSecret y un licenseKey del RMS Web Service. La License Key caduca a los 90 días y se puede volver a emitir.

Atención: El RMS Web Service (glogin.rms.rakuten.co.jp) que emite las claves de vendedor (Ichiba) es distinto de webservice.rakuten.co.jp, que es para afiliados/búsqueda de productos. Asegúrate de emitir tus claves en el lado de RMS.

2. Pasos de emisión

Emite en el orden indicado a continuación y regístralas en GenieGo.

1

Inicia sesión en el portal de desarrolladores de RMS

Inicia sesión en el portal de desarrolladores de Rakuten (RMS/Webservice) con tu cuenta de vendedor.

https://glogin.rms.rakuten.co.jp
2

Ve al RMS Web Service

Tras iniciar sesión mediante RMS R-Login, navega al área del RMS Web Service.

RMS R-Login → RMS Web Service
3

Emite el Service Secret · License Key

Desde el RMS Web Service, emite el Service Secret y la License Key. La License Key caduca a los 90 días, así que vuelve a emitirla cuando caduque.

4

Verifica las claves

Verifica el Service Secret y la License Key emitidos, junto con la Shop URL de tu tienda.

Una vez que el Service Secret y la License Key aparezcan en la pantalla de emisión, se han emitido correctamente.

3. Información para registrar en GenieGo

CampoDescripción
service_secretEl Service Secret emitido
license_keyLa License Key emitida (caduca a los 90 días·se puede volver a emitir, manéjala con cuidado)
shop_urlLa Shop URL de tu tienda
En el GenieGo Integration Hub [Registrar], introduce service_secret·license_key·shop_url y guarda. Las credenciales se cifran en reposo con AES-256-GCM y se enmascaran al leerlas.

4. Problemas frecuentes

ProblemaCausaSolución
Se emitieron claves incorrectasEmitidas desde webservice.rakuten.co.jp (para afiliados/búsqueda de productos)Vuelve a emitirlas desde el RMS Web Service en RMS R-Login (glogin.rms.rakuten.co.jp).
La autenticación falla al cabo de un tiempoLa License Key caducó a los 90 díasVuelve a emitir la License Key desde el RMS Web Service y reintrodúcela en [Registrar].

Lista de verificación final

  • Inicié sesión en RMS R-Login con una cuenta de vendedor de Rakuten (Ichiba).
  • Emití el Service Secret·License Key desde el RMS Web Service.
  • Verifiqué la Shop URL de mi tienda.
  • Introduje service_secret·license_key·shop_url en GenieGo [Registrar] y guardé.

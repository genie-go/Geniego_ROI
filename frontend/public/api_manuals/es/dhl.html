DHL Express
Manual de emisión de API Key

Una guía paso a paso, apta para principiantes, para emitir una API Key y un Secret en el portal de desarrolladores de DHL Express (developer.dhl.com) y registrarlos junto con su número de cuenta de cliente de DHL Express en el GenieGo Integration Hub.

Tiempo estimadoUnos 10–15 min (espera de aprobación aparte)
Cuenta necesariaCuenta de cliente de DHL Express (número de cuenta)
Claves emitidasAPI Key · Secret
ResultadoCredenciales de API · número de cuenta

1. Antes de empezar

Las credenciales de DHL Express se solicitan en el portal de desarrolladores y se requiere un número de cuenta de cliente.

Preparación

Número de cuenta de cliente de DHL Express

Tener un número de cuenta de cliente de DHL Express facilita la solicitud de credenciales. El número de cuenta también se introduce al registrarse en GenieGo.

Claves emitidas

API Key · Secret

Obtendrá la API Key y el Secret a través de su propia app (nombre de la empresa) en el portal.

Nota: Cuando registra la información de su cliente mediante [Get Access] y solicita credenciales, normalmente recibe el correo de aprobación Test/Production al día hábil siguiente.

2. Pasos de emisión

Emita en el orden siguiente y registre en GenieGo.

1

Registrarse en el portal de desarrolladores

Vaya al portal de desarrolladores de DHL Express, regístrese con su correo electrónico y verifíquelo.

https://developer.dhl.com
2

Solicitar credenciales

Tras iniciar sesión, registre la información de su cliente en la sección [Get Access] y solicite credenciales.

Get Access
3

Recibir el correo de aprobación

Normalmente recibe el correo de aprobación Test/Production al día hábil siguiente.

4

Encontrar la API Key·Secret

En el portal, vaya a su propia app (nombre de la empresa) > [Show Key] para encontrar la API Key y el Secret.

Su app (nombre de la empresa) → Show Key
Una vez que reciba el correo de aprobación y la API Key·Secret sean visibles en el portal, se habrán emitido correctamente.

3. Información para registrar en GenieGo

CampoDescripción
api_keyLa DHL API Key encontrada en el portal
account_numberNúmero de cuenta de cliente de DHL Express
Una vez que las introduce en el GenieGo Integration Hub [Registrar] y guarda, las credenciales se cifran en reposo con AES-256-GCM y se enmascaran al leerse.

4. Problemas frecuentes

ProblemaCausaSolución
El correo de aprobación no llegaA la espera del procesamiento de aprobación del día hábil siguiente, o información de cliente no registradaCompruebe en [Get Access] si el registro del cliente y la solicitud de credenciales están completos, y espere según los días hábiles.
Error de autenticación al llamarConfusión entre los entornos Test/Production, o número de cuenta faltanteVuelva a comprobar la API Key·Secret del entorno aprobado y el número de cuenta (account_number).

Lista de verificación final

  • Confirmó el número de cuenta de cliente de DHL Express.
  • Se registró en developer.dhl.com y solicitó credenciales mediante [Get Access].
  • Recibió el correo de aprobación Test/Production al día hábil siguiente.
  • Encontró la API Key·Secret en el portal.
  • Introdujo api_key·account_number en GenieGo [Registrar] y guardó.

Stripe
Manual de Emisión de Claves API

Una guía paso a paso, apta para principiantes, para emitir claves API (Publishable Key·Secret Key) desde el menú Developers del panel de Stripe y registrarlas en el GenieGo Integration Hub.

Tiempo estimadoUnos 5–10 min
Cuenta necesariaCuenta de Stripe
Claves emitidasPublishable Key · Secret Key
ResultadoCredenciales de API

1. Antes de Empezar

Las claves API de Stripe se emiten desde el panel de tu cuenta de Stripe.

Requerido

Cuenta de Stripe

Debes poder iniciar sesión en dashboard.stripe.com con tu cuenta de Stripe.

Claves emitidas

Publishable Key · Secret Key

Encuentra tu Publishable key y tu Secret key en la pantalla Developers > API keys.

Nota: Para la integración de pagos real, usa claves live en lugar de claves de prueba.

2. Pasos de Emisión

Emite en el orden siguiente y regístralas en GenieGo.

1

Iniciar sesión en Stripe

Inicia sesión en el panel con la cuenta de Stripe que preparaste.

dashboard.stripe.com
2

Ir al menú API keys

Ve al menú [Developers] > [API keys].

Developers → API keys
3

Comprobar las claves

Encuentra tu Publishable key y Secret key. Usa las claves live.

https://dashboard.stripe.com/apikeys
Una vez confirmadas tanto la Publishable key como la Secret key, se han emitido correctamente.

3. Información para Registrar en GenieGo

CampoDescripción
publishable_keyLa Publishable Key emitida
secret_keyLa Secret Key emitida (manéjala con cuidado)
En el GenieGo Integration Hub [Registrar], introduce publishable_key·secret_key y guarda. Las credenciales se cifran en reposo con AES-256-GCM y se enmascaran al leerlas.

4. Problemas Frecuentes

ProblemaCausaSolución
Fallo de autenticaciónSe introdujo una clave de pruebaComprueba si estás usando las claves live (Publishable·Secret).
La Secret key no se veLa clave se muestra solo una vez o está ocultaVuelve a comprobarla en la pantalla Developers > API keys o emite una nueva.

Lista de Verificación Final

  • Inicié sesión en dashboard.stripe.com con mi cuenta de Stripe.
  • Fui a Developers > API keys.
  • Confirmé la Publishable Key·Secret Key live.
  • Introduje publishable_key·secret_key en GenieGo [Registrar] y guardé.

Beginner Guide

YouTube API Key
Manual de emisión

Una guía práctica paso a paso que incluso los principiantes pueden seguir para habilitar la YouTube Data API v3 en la Google Cloud Console y crear una API Key.

Tiempo estimadoUnos 5–10 min
Cuenta necesariaCuenta de Google
API utilizadaYouTube Data API v3
ResultadoUna API Key con el formato AIzaSy...

1. API Key frente a OAuth

Esta es la parte que más confunde a la gente al empezar.

API Key

Para consultar datos públicos

Se usa para obtener datos públicos de YouTube, como buscar vídeos, consultar la información de un canal, recuperar listas de reproducción y leer comentarios.

OAuth 2.0

Para tareas que requieren los permisos de tu cuenta

Se usa para tareas que requieren la aprobación del usuario, como subir vídeos, gestionar tu propio canal y acceder a datos privados.

Importante: Para subir vídeos automáticamente, una API Key por sí sola no basta; se requiere autenticación OAuth 2.0.

2. Habilitar la YouTube Data API v3

Antes de crear una API Key, primero debes activar en tu proyecto la API que quieres usar.

1

Acceder a la Google Cloud Console

Inicia sesión en la Google Cloud Console.

console.cloud.google.com
2

Seleccionar o crear un proyecto

En el selector de proyectos de la parte superior, elige un proyecto existente o crea uno nuevo.

p. ej., Geniego ROI
3

Ir a la Biblioteca de API

En el menú de la izquierda, ve a APIs y servicios → Biblioteca.

APIs y servicios → Biblioteca
4

Buscar la YouTube Data API v3 y habilitarla

Escribe YouTube Data API v3 en el cuadro de búsqueda, selecciona esa API y haz clic en el botón Habilitar.

YouTube Data API v3 → Habilitar
Si ves Estado: Habilitada en la pantalla, se ha habilitado correctamente.

3. Crear una API Key

Una vez habilitada la YouTube Data API v3, puedes crear una API Key en la pantalla de Credenciales.

1

Ir al menú Credenciales

En el menú de la izquierda, ve a APIs y servicios → Credenciales.

APIs y servicios → Credenciales
2

Hacer clic en Crear credenciales

Haz clic en el botón + Crear credenciales de la parte superior.

+ Crear credenciales
3

Seleccionar Clave de API

En el menú que aparece, selecciona Clave de API.

Clave de API
4

Seleccionar la API accesible

En Restricciones de API, confirma que la YouTube Data API v3 está seleccionada.

API seleccionada: YouTube Data API v3
5

Hacer clic en el botón Crear

Haz clic en el botón Crear de la parte inferior para generar la API Key.

Crear
La clave generada suele empezar por AIzaSy.... Esta cadena es tu YouTube API Key.

4. ¿Qué opciones debes elegir en la pantalla de creación?

ElementoConfiguración recomendada para principiantesDescripción
NombreYouTube API KeyIntroduce un nombre que sea fácil de reconocer más adelante.
Restricciones de APIYouTube Data API v3Restringe esta clave para que solo se use con la YouTube API.
Autenticación de cuenta de servicioDejar sin marcarPor lo general no es necesaria para crear una YouTube Data API Key.
Restricciones de aplicaciónNinguna para la prueba inicialTras confirmar que funciona correctamente, recomendamos establecer restricciones por sitio web o dirección IP.
Al implementar en un servicio real, asegúrate de establecer restricciones de clave. Si se expone sin restricciones, otras personas pueden usar tu clave.

5. Ejemplos de uso de la API Key

Puedes probarla colocando tu clave generada en la parte YOUR_API_KEY.

Ejemplo de búsqueda de vídeoshttps://www.googleapis.com/youtube/v3/search?part=snippet&q=korea&type=video&key=YOUR_API_KEY
Ejemplo de consulta de información de canalhttps://www.googleapis.com/youtube/v3/channels?part=snippet&id=CHANNEL_ID&key=YOUR_API_KEY
Si aparece un resultado en formato JSON al introducir la URL anterior en la barra de direcciones de tu navegador, la API Key funciona correctamente.

6. Solución de problemas comunes

ProblemaCausaSolución
No hay ninguna API seleccionadaLa YouTube Data API v3 aún no se ha habilitado, o no se seleccionó en Restricciones de APIHabilita la YouTube Data API v3 en la Biblioteca de API y vuelve a seleccionarla.
No hay claves de API para mostrarAún no has creado una API KeyHaz clic en Crear credenciales → Clave de API.
Error 403Restricciones de API, cuota superada o problemas de permisosComprueba las restricciones de API, la cuota y el estado de habilitación.
Las subidas no funcionanSe usa solo una API KeyLas subidas de vídeos deben usar autenticación OAuth 2.0.

7. Lista de verificación final

  • Seleccionaste un proyecto de Google Cloud.
  • Habilitaste la YouTube Data API v3.
  • Creaste una API Key en Credenciales.
  • Confirmaste que la API Key generada empieza por AIzaSy.
  • Estableciste las restricciones de API en la YouTube Data API v3.
  • Establece restricciones por sitio web o IP antes de implementar en un servicio real.

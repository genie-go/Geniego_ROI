Manual de apertura de canal empresarial y emisión de API

Una guía práctica y apta para principiantes dirigida a empresas, marcas, live-commerce y operadores de plataformas de TI/logística para crear un canal oficial de Twitch y completar la configuración de la transmisión, la conexión con OBS y la emisión de la Twitch Developer API.

Nivel de monetizaciónAffiliate / Partner
Herramienta de transmisiónOBS Studio
Resultado de la APIClient ID / Secret

1. Comprender la estructura de la cuenta de Twitch

A diferencia de Instagram, Twitch no tiene una estructura para elegir por separado una cuenta de empresa/creador.

Canal oficial de la empresa

Para la operación de la marca

Crea una cuenta normal de Twitch con el nombre de tu empresa o marca y opérala como un canal oficial. Es ideal para presentaciones de productos, live-commerce y transmisiones de eventos.

Nivel de crecimiento del streamer

Affiliate / Partner

Una vez que cumplas los requisitos de rendimiento de transmisión, seguidores y tiempo de visualización, puedes unirte al programa Affiliate o Partner.

Punto clave: Las empresas también deben crear primero una cuenta normal de Twitch y luego ocuparse de la identidad de marca del canal, la autenticación de dos factores, la conexión con OBS y el registro de la app en la Developer Console.

2. Lo que necesitas

  • Dirección de correo electrónico empresarial
  • Nombre de marca o de empresa
  • Nombre de usuario de Twitch candidato
  • Logotipo de la empresa e imagen de banner del canal
  • Texto de presentación de la empresa y URL del sitio web
  • PC o portátil para transmitir
  • OBS Studio o software de streaming
  • OAuth Redirect URL para la integración de la API

3. Crear una cuenta de Twitch

1

Abrir Twitch

Abre el sitio oficial de Twitch en tu navegador.

https://www.twitch.tv
2

Hacer clic en Registrarse

Haz clic en el botón Sign Up o Registrarse en la parte superior derecha.

Sign Up
3

Registrarse con datos de la empresa

Introduce el correo de la empresa, el nombre de usuario, la contraseña y la fecha de nacimiento.

Email / Username / Password
4

Verificar el correo electrónico

Abre el correo de verificación enviado por Twitch y completa Verify Email.

Verify Email

4. Elegir el nombre de la cuenta de marca

El nombre de usuario de Twitch se convierte en la dirección del canal, por lo que es mejor un nombre corto y fácil de recordar.

PropósitoEjemplos de nombres recomendadosDescripción

Transmisión globalGeniegoGlobal / OCLGlobalAdecuado cuando se consideran espectadores del extranjero.
Live-commerceGeniegoLive / GeniegoCommerceTransmite con claridad un propósito de venta o de compra.
Tecnología/plataformaGeniegoTech / OCLTechAdecuado para transmisiones de presentación de TI, SaaS y plataformas.

Ir al menú de ajustes

Haz clic en el icono de perfil en la parte superior derecha y luego ve a Settings.

Profile → Settings
2

Subir una imagen de perfil

Establece el logotipo de la empresa o el icono de la marca como imagen de perfil.

Profile Picture
3

Subir un banner del canal

Sube una imagen de banner con los colores de tu marca y el mensaje del servicio.

Profile Banner
4

Introducir el texto de la bio

Introduce la presentación de la empresa, la descripción del servicio, el correo de contacto y el sitio web.

Bio / Links

6. Configurar la autenticación de dos factores

La configuración de 2FA es muy importante para usar de forma fiable los ajustes de transmisión y seguridad en Twitch.

1

Ir al menú de seguridad

Abre el menú Security and Privacy en los ajustes.

Settings → Security and Privacy
2

Activar 2FA

Activa Two-Factor Authentication.

Set Up Two-Factor Authentication
3

Conectar un teléfono o una app de autenticación

Conecta la verificación por SMS o una app de autenticación como Google Authenticator o Authy.

Authenticator App
Precaución: Las cuentas de empresa pueden ser usadas por varias personas, así que asegúrate de gestionar los permisos de administrador y el correo de recuperación.

7. Comprobar la Stream Key y conectar OBS

1

Abrir el Creator Dashboard

Ve al Creator Dashboard desde el menú de perfil.

Creator Dashboard
2

Ir al menú Stream

Selecciona el elemento Stream en el menú de ajustes.

Settings → Stream
3

Comprobar la Primary Stream Key

Copia la clave de transmisión principal. Esta clave debe guardarse como una contraseña.

Primary Stream Key
4

Conectar en OBS

En OBS Studio, configura el servicio como Twitch e introduce el valor copiado como clave de transmisión.

OBS → Settings → Stream → Twitch
Si tu clave de transmisión queda expuesta, otra persona puede transmitir en tu canal. Nunca la hagas pública.

9. Registrarse en la Twitch Developer Console

Para crear chat bots, analítica de espectadores, comprobación del estado de la transmisión y funciones de automatización, debes registrar una app de Twitch Developer.

1

Abrir Twitch Developer

Abre el sitio para desarrolladores de Twitch.

https://dev.twitch.tv
2

Ir a la Console

Haz clic en Console o Developer Console en el menú superior.

Developer Console
3

Seleccionar el menú Applications

Selecciona Applications en el menú izquierdo o superior.

Applications
4

Hacer clic en Register Your Application

Haz clic en el botón para registrar una nueva aplicación.

Register Your Application

10. Introducir los datos de registro de la app de Twitch API

Campo de entradaEjemplo de entradaDescripción
NameGeniego Twitch AppEl nombre de la app. Introdúcelo según el nombre de tu servicio.
OAuth Redirect URLshttps://yourdomain.com/auth/twitch/callbackLa dirección del servidor a la que se vuelve tras el inicio de sesión con OAuth.
CategoryWebsite Integration / Chat BotSelecciona la categoría que se ajuste al propósito de tu app.

Una vez completado el registro, se muestra el Client ID y puedes generar un Client Secret con el botón New Secret.

11. Comprobar el Client ID / Client Secret

1

Seleccionar la app registrada

Haz clic en la app que acabas de crear en la lista Applications de la Developer Console.

Applications → seleccionar app
2

Copiar el Client ID

Copia el Client ID que aparece en pantalla y guárdalo en los ajustes de tu servidor.

Client ID
3

Generar el Client Secret

Haz clic en el botón New Secret para generar un Client Secret y guárdalo de forma segura.

New Secret
El Client Secret a menudo solo se puede ver una vez. Guárdalo en un lugar seguro inmediatamente después de generarlo.

12. Estructura de autenticación de la Twitch API

Twitch Channel │ ▼ Developer Console │ ▼ Application │ ▼ Client ID / Client Secret │ ▼ OAuth Token │ ▼ Twitch Helix API │ ▼ Información del canal, estado de la transmisión, chat bot, analítica de espectadores

13. Flujo de emisión del token OAuth

1) Generar la URL de autorización de Twitch OAuth 2) El usuario inicia sesión con su cuenta de Twitch 3) Recibir el authorization code en la Redirect URL 4) Solicitar un Access Token con el authorization code + Client Secret 5) Llamar a la Twitch API con el Access Token 6) Renovar con el Refresh Token cuando sea necesario
Las consultas simples de datos públicos son posibles con un App Access Token, mientras que las operaciones que requieren permisos de usuario necesitan un User Access Token.

15. Lista de verificación final

  • Creé una cuenta de Twitch con un correo empresarial.
  • Establecí un nombre de usuario que coincide con el nombre de la marca.
  • Completé la verificación del correo electrónico.
  • Configuré la imagen de perfil, el banner, el texto de la bio y los enlaces.
  • Activé la autenticación de dos factores.
  • Comprobé la Stream Key en el Creator Dashboard.
  • Conecté OBS Studio con el canal de Twitch.
  • Registré la app en la Twitch Developer Console.
  • Obtuve el Client ID y el Client Secret.
  • Preparé la OAuth Redirect URL y el flujo de emisión del token.

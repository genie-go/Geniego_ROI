Slack Webhook
Manual de emisión de la API Key

Una guía para crear una app en Slack, emitir una URL de Incoming Webhook y registrarla en el GenieGo Integration Hub para que las notificaciones se envíen al canal de Slack que especifiques.

Tiempo estimadoUnos 5 min
Cuenta necesariaPermiso de administrador del workspace de Slack
Claves emitidasWebhook URL
ResultadoCredenciales de Incoming Webhook

1. Antes de empezar

Un Webhook de Slack solo se puede crear con permiso de administrador del workspace.

Obligatorio

Permiso de administrador del workspace

Necesitas permiso de administrador del workspace de Slack para crear una app y activar el Webhook.

Claves emitidas

Webhook URL

Después de activar Incoming Webhooks y seleccionar un canal, comprueba la Webhook URL generada.

Nota: La Webhook URL está vinculada al canal al que se envían las notificaciones. Especifica el canal que debe recibir las notificaciones en el paso de selección de canal.

2. Pasos de emisión

Emite en el orden siguiente y regístrala en GenieGo.

1

Crear una nueva app

Haz clic en [Create New App].

https://api.slack.com/apps
2

Activar Incoming Webhooks

Activa [Incoming Webhooks] y haz clic en [Add New Webhook to Workspace].

3

Seleccionar un canal · Generar URL

Al seleccionar un canal se genera la Webhook URL.

Una vez que seleccionas un canal y se genera la Webhook URL, se ha emitido correctamente.

3. Información para registrar en GenieGo

CampoDescripción
webhook_urlLa Incoming Webhook URL generada (manéjala con cuidado)
Introduce la webhook_url en GenieGo [Registrar] y guarda. Las credenciales se cifran en reposo con AES-256-GCM y se enmascaran al leerlas.

4. Problemas frecuentes

ProblemaCausaSolución
No se puede crear la appPermiso de administrador del workspace insuficienteContinúa con una cuenta que tenga permiso de administrador del workspace de Slack.
No llegan las notificacionesWebhook URL incorrecta o canal no seleccionadoActiva Incoming Webhooks, selecciona un canal y vuelve a comprobar la Webhook URL generada.

Lista de verificación final

  • Continué con permiso de administrador del workspace de Slack.
  • Creé una app en api.slack.com/apps mediante [Create New App].
  • Activé Incoming Webhooks y hice clic en [Add New Webhook].
  • Seleccioné un canal y generé y copié la Webhook URL.
  • Introduje la webhook_url en GenieGo [Registrar] y la guardé.

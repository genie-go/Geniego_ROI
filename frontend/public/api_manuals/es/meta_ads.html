Meta Ads
Manual de emisión de la API Key

Una guía paso a paso para principiantes sobre cómo emitir un token de acceso de usuario del sistema desde tu app empresarial de Meta y registrarlo junto con tu ID de cuenta publicitaria (act_) en el GenieGo Integration Hub para conectar tus datos publicitarios de Meta Ads.

Tiempo estimadoUnos 10–20 min
Cuenta necesariaCuenta personal de Facebook · Cuenta publicitaria de Meta
Claves emitidasToken de acceso · ID de cuenta publicitaria (act_)
ResultadoCredenciales de Marketing API

1. Antes de empezar

La integración de Meta Ads requiere una cuenta empresarial y una cuenta publicitaria que esté publicando anuncios activamente.

Requerido

Cuenta personal de Facebook · Cuenta publicitaria

Necesitas una cuenta personal de Facebook y una cuenta publicitaria de Meta que esté publicando anuncios activamente.

Claves emitidas

Token de acceso · ID de cuenta publicitaria (act_)

Obtendrás un token de usuario del sistema (token de acceso) y un ID numérico de cuenta publicitaria que comienza con act_.

Nota: Acceder a datos fuera de tu propio negocio requiere una revisión de la app (App Review). Si solo conectas cuentas publicitarias dentro de tu propio negocio, puedes continuar sin revisión.

2. Pasos de emisión

Emite en el orden indicado a continuación y regístralo en GenieGo.

1

Crear y verificar una cuenta empresarial

Abre el sitio en tu navegador, inicia sesión y luego crea una cuenta empresarial mediante [Crear empresa] y completa la verificación por correo electrónico.

business.facebook.com
2

Crear una app

Haz clic en [Mis apps] > [Crear app] en la parte superior derecha, elige el tipo "Empresa" y crea la app.

developers.facebook.com → Mis apps → Crear app
3

Agregar la Marketing API

En el panel de la app, busca la tarjeta "Marketing API" y haz clic en [Configurar] para agregarla como producto.

4

Crear un usuario del sistema y asignar permisos

Ve a Configuración de la empresa > [Usuarios] > [Usuarios del sistema] > [Agregar] para crear un usuario del sistema y, luego, otorga permisos de cuenta publicitaria en [Asignar activos].

business.facebook.com/settings → Usuarios → Usuarios del sistema
5

Generar un token de acceso

En la misma pantalla, haz clic en [Generar token], selecciona tu app, marca los permisos (ads_read, ads_management), genera el token y copia su valor.

6

Encontrar tu ID de cuenta publicitaria

En el Administrador de anuncios, busca el ID numérico de cuenta publicitaria que comienza con act_.

adsmanager.facebook.com
Una vez que el token se genera con los permisos ads_read y ads_management marcados, se ha emitido correctamente.

3. Información para registrar en GenieGo

CampoDescripción
access_tokenEl token de acceso de usuario del sistema emitido (manéjalo con cuidado)
ad_account_idID de cuenta publicitaria que comienza con act_
En el GenieGo Integration Hub > el [Registrar] del canal, pega tu token de acceso y tu ID de cuenta publicitaria (act_) y guarda. Las credenciales se cifran en reposo con AES-256-GCM y se enmascaran al consultarlas.

4. Problemas comunes

ProblemaCausaSolución
No se pueden consultar los datos por errores de permisosPermisos del token (ads_read·ads_management) no marcados, o permiso de cuenta publicitaria no asignadoAsigna el permiso de cuenta publicitaria al usuario del sistema y marca los permisos ads_read·ads_management al generar el token.
Se deniega el acceso a los datos de otro negocioRevisión de la app (App Review) no completadaPara acceder a datos fuera de tu propio negocio, solicita y completa la revisión de la app.
Error de formato del ID de cuenta publicitariaFalta el prefijo act_, o solo se ingresaron númerosEn el Administrador de anuncios, busca el ID completo que comienza con act_ e ingrésalo.

Lista de verificación final

  • Creé una cuenta empresarial en business.facebook.com y completé la verificación por correo electrónico.
  • Creé una app de tipo Empresa en developers.facebook.com y agregué la Marketing API.
  • Creé un usuario del sistema y asigné el permiso de cuenta publicitaria.
  • Generé y copié un token de acceso con los permisos ads_read·ads_management.
  • Encontré el ID de cuenta publicitaria que comienza con act_.
  • Ingresé access_token·ad_account_id en GenieGo [Registrar] y guardé.

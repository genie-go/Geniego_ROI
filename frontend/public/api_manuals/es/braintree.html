Braintree
Manual de emisión de API Key

Una guía paso a paso y apta para principiantes para encontrar tus credenciales de Production API (Merchant ID, Public Key, Private Key) en el Braintree (PayPal) Control Panel y registrarlas en el GenieGo Integration Hub.

Tiempo estimadoUnos 5–10 min
Cuenta necesariaCuenta de comercio de Braintree (PayPal) (requiere aprobación)
Claves emitidasMerchant ID · Public Key · Private Key
ResultadoCredenciales de Production API

1. Antes de empezar

Las credenciales de Braintree (PayPal) solo están disponibles después de que se apruebe tu cuenta de comercio.

Requerido

Cuenta de comercio de Braintree (PayPal)

Primero necesitas una cuenta de comercio de Braintree (PayPal) aprobada. Las credenciales de Production solo están disponibles una vez que tu cuenta de comercio ha sido aprobada.

Claves emitidas

Merchant ID · Public/Private Key

En el Control Panel, ve a Settings > API Keys para encontrar el Merchant ID, la Public Key y la Private Key del entorno Production.

Nota: Asegúrate de usar las credenciales del entorno Production. Ten cuidado de no confundirlas con las claves de Sandbox.

2. Procedimiento de emisión

Tras la aprobación de tu cuenta de comercio, encuentra las credenciales en el orden siguiente y regístralas en GenieGo.

1

Aprobación de la cuenta de comercio

Completa primero la aprobación de la cuenta de comercio de Braintree (PayPal). Solo puedes pasar al siguiente paso una vez que la aprobación se haya completado.

2

Iniciar sesión en el Control Panel

Inicia sesión en el Braintree Control Panel.

braintreegateway.com
3

Ir a API Keys

Navega al menú [Settings] > [API Keys].

Settings → API Keys
4

Encontrar las credenciales

Encuentra el Merchant ID, la Public Key y la Private Key del entorno Production.

Si el Merchant ID, la Public Key y la Private Key de Production aparecen en Settings > API Keys, todo está correcto.

3. Información para registrar en GenieGo

CampoDescripción
merchant_idProduction Merchant ID
public_keyProduction Public Key
private_keyProduction Private Key (manéjala con cuidado)
Introduce merchant_id, public_key y private_key en [Registrar] del GenieGo Integration Hub y guarda. Las credenciales se cifran en reposo con AES-256-GCM y se enmascaran al leerlas.

4. Problemas frecuentes

ProblemaCausaSolución
No hay credenciales en el menú API KeysAprobación de la cuenta de comercio no completadaCompleta primero la aprobación de la cuenta de comercio de Braintree (PayPal).
Fallo de autenticaciónSe usaron claves de Sandbox, o la Private Key se introdujo malVuelve a comprobar el Merchant ID, la Public Key y la Private Key del entorno Production.

Lista de verificación final

  • La cuenta de comercio de Braintree (PayPal) ha sido aprobada.
  • Encontradas las credenciales de Production en Settings > API Keys del Control Panel.
  • Confirmados el Merchant ID, la Public Key y la Private Key.
  • Introducidos merchant_id, public_key y private_key en GenieGo [Registrar] y guardados.

Coupang Wing
Manual de emisión de API Key

Una guía paso a paso apta para principiantes sobre cómo emitir credenciales de Open API (Access Key·Secret Key) en la consola de vendedor de Coupang WING y registrarlas en el GenieGo Integration Hub.

Tiempo estimadoUnos 5–10 min
Cuenta necesariaCuenta de vendedor de Coupang WING
Claves emitidasAccess Key · Secret Key
ResultadoCredenciales de Open API

1. Antes de empezar

La Open API de Coupang solo se puede emitir desde una cuenta de vendedor.

Requerido

Cuenta de vendedor de WING

Necesitas una cuenta de vendedor de Coupang WING con la verificación de empresa completada. No se puede emitir con una cuenta de miembro comprador normal.

Claves emitidas

Access Key · Secret Key

En la pantalla de emisión completada, busca la Access Key y la Secret Key, y comprueba también el vendor ID.

Nota: Al emitir, debes introducir el nombre de tu empresa, la URL de la página de inicio/servicio (usa wing.coupang.com si no tienes ninguna) y las IP de acceso (hasta 10).

2. Pasos para emitir la Open API Key

Emite en el orden indicado a continuación y registra en GenieGo.

1

Inicia sesión en WING

Inicia sesión con tu cuenta de vendedor.

wing.coupang.com
2

Ve al menú Open API

Ve al menú [Open API] o [Gestión de API Key] en la zona superior/de ajustes.

Open API → Gestión de API Key
3

Emite una Open API Key

Haz clic en [Emitir Open API Key] e introduce el nombre de tu empresa, la URL de la página de inicio y las IP de acceso (hasta 10).

Emitir Open API Key
4

Comprueba las claves

En la pantalla de emisión completada, busca la Access Key·Secret Key y el vendor ID.

Una vez que la Access Key·Secret Key aparecen en la pantalla de emisión completada, se han emitido correctamente.

3. Información para registrar en GenieGo

CampoDescripción
access_keyLa Access Key emitida
secret_keyLa Secret Key emitida (manéjala con cuidado)
vendor_idVendor ID
Introdúcelas en el GenieGo Integration Hub [Registrar] y guarda. Las credenciales se cifran en reposo con AES-256-GCM y se enmascaran al leerlas. La sincronización de pedidos comienza inmediatamente tras el registro.

4. Problemas frecuentes

ProblemaCausaSolución
Falta el menú de emisiónCuenta de miembro normal, o verificación de empresa no completadaInicia sesión con una cuenta de vendedor de WING que tenga la verificación de empresa completada.
Error 401 al llamarIP de acceso no registrada, o Secret Key introducida incorrectamenteVuelve a comprobar las IP de acceso que introdujiste durante la emisión y la Secret Key.

Lista de comprobación final

  • Inicié sesión con una cuenta de vendedor de WING con la verificación de empresa completada.
  • Emití una Open API Key y registré las IP de acceso.
  • Comprobé la Access Key·Secret Key·Vendor ID.
  • Introduje access_key·secret_key·vendor_id en GenieGo [Registrar] y guardé.

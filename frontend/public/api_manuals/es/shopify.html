Shopify
Manual de emisión de API Key

Una guía paso a paso para crear una app personalizada en tu Admin de Shopify a fin de emitir un Admin API access token y registrarlo junto con el dominio de tu tienda en el GenieGo Integration Hub para conectar tu tienda de Shopify.

Tiempo estimadoAprox. 5–15 min
Cuenta necesariaPermiso de administrador de la tienda Shopify
Claves emitidasDominio de la tienda · Admin API access token
ResultadoCredenciales de Admin API

1. Antes de comenzar

Una app personalizada de Shopify solo se puede crear con permiso de administrador de la tienda.

Requerido

Permiso de administrador de la tienda

Necesitas permiso de administrador de la tienda Shopify para crear una app personalizada y emitir un token.

Claves emitidas

Dominio de la tienda · Admin API token

Obtendrás un dominio de tienda con el formato xxx.myshopify.com y el Admin API access token que se muestra después de instalar la app.

Nota: El Admin API access token se muestra solo una vez justo después de la instalación, por lo que debes copiarlo de inmediato cuando aparezca.

2. Pasos de emisión

Emite en el orden indicado a continuación y regístralo en GenieGo.

1

Ir a Aplicaciones y canales de venta

En tu Admin de Shopify, ve a [Configuración] > [Aplicaciones y canales de venta].

https://www.shopify.com/admin/settings/apps
2

Crear una app personalizada

Haz clic en [Desarrollar aplicaciones] > [Crear una aplicación] para crear una app personalizada.

Desarrollar aplicaciones → Crear una aplicación
3

Seleccionar scopes · Instalar la app

Selecciona los Admin API scopes e instala la app.

4

Copiar el access token

Copia de inmediato el Admin API access token que se muestra tras la instalación (se muestra solo una vez).

Una vez que el Admin API access token aparezca después de instalar la app, significa que se ha emitido correctamente. Se muestra solo una vez, así que cópialo de inmediato.

3. Información para registrar en GenieGo

CampoDescripción
shop_domainDominio de la tienda con el formato xxx.myshopify.com
access_tokenEl Admin API access token emitido (manéjalo con cuidado)
En GenieGo [Registrar], ingresa tu shop_domain (xxx.myshopify.com) y access_token y guarda. Las credenciales se cifran en reposo con AES-256-GCM y se enmascaran al leerlas.

4. Problemas frecuentes

ProblemaCausaSolución
No se puede volver a ver el tokenEl Admin API access token se muestra solo una vezSi perdiste el token, vuelve a emitirlo en la app personalizada y cópialo de inmediato.
Error de permiso insuficienteAdmin API scopes no seleccionadosSelecciona los Admin API scopes necesarios y luego reinstala la app.
Error de dominioFormato del dominio de la tienda ingresado incorrectamenteIngresa el dominio de la tienda con precisión en el formato xxx.myshopify.com.

Lista de verificación final

  • Inicié sesión con permiso de administrador de la tienda Shopify.
  • Creé una app personalizada en Configuración > Aplicaciones y canales de venta > Desarrollar aplicaciones.
  • Seleccioné los Admin API scopes e instalé la app.
  • Copié de inmediato el Admin API access token mostrado tras la instalación.
  • Ingresé shop_domain·access_token en GenieGo [Registrar] y guardé.

دليل إصدار مفتاح API

دليل تفصيلي ملائم للمبتدئين لإنشاء مفاتيح API في وضع Live (Key ID·Key Secret) والتحقق منها في Razorpay Dashboard وتسجيلها في GenieGo Integration Hub.

الوقت المقدّرحوالي 5–10 دقائق
الحساب المطلوبحساب Razorpay (يركّز على الهند)
المفاتيح المُصدَرةKey ID (rzp_live_) · Key Secret
النتيجةبيانات اعتماد Live API

1. قبل أن تبدأ

يتم إنشاء مفاتيح API الخاصة بـ Razorpay في وضع Live على Dashboard.

مطلوب

حساب Razorpay

أنت بحاجة إلى حساب Razorpay (يركّز على الهند). يجب أن تتمكن من تسجيل الدخول إلى Dashboard باستخدامه.

المفاتيح المُصدَرة

Key ID · Key Secret

أنشئ وتحقق من Key ID وKey Secret لوضع Live في Dashboard ضمن Settings > API Keys.

ملاحظة: احرص على إنشاء المفاتيح في وضع Live. يبدأ Key ID بـ rzp_live_.

2. خطوات الإصدار

أصدر بالترتيب أدناه وسجّل في GenieGo.

1

افتح Dashboard

سجّل الدخول إلى Razorpay Dashboard.

dashboard.razorpay.com
2

انتقل إلى API Keys

انتقل إلى [Settings] > [API Keys].

Settings → API Keys
3

أنشئ المفاتيح وتحقق منها

في وضع Live، أنشئ Key ID وKey Secret وتحقق منهما.

إذا كنت ترى Key ID لوضع Live (rzp_live_) وKey Secret ضمن Settings > API Keys، فهذا يعني أنه تم إنشاؤهما بشكل صحيح.

3. المعلومات المطلوب تسجيلها في GenieGo

الحقلالوصف
key_idKey ID لوضع Live (يبدأ بـ rzp_live_)
key_secretKey Secret لوضع Live (تعامل معه بحذر)
في GenieGo Integration Hub [التسجيل]، أدخل key_id·key_secret ثم احفظ. تُشفَّر بيانات الاعتماد أثناء التخزين باستخدام AES-256-GCM ويتم إخفاؤها عند القراءة.

4. المشكلات الشائعة

المشكلةالسببالحل
فشل المصادقةاستخدام مفتاح وضع Testتحقق من أنك تستخدم Key ID (rzp_live_)·Key Secret تم إنشاؤه في وضع Live.
تعذّر عرض Key Secret مرة أخرىيُعرض Key Secret مرة واحدة فقط عند الإنشاءاحفظ Key Secret بأمان عند الإنشاء؛ وإذا فُقد، أنشئ مفتاحًا جديدًا.

قائمة التحقق النهائية

  • سجّلت الدخول إلى dashboard.razorpay.com بحساب Razorpay.
  • أنشأت مفاتيح وضع Live ضمن Settings > API Keys.
  • تحققت من Key ID (rzp_live_)·Key Secret.
  • أدخلت key_id·key_secret في GenieGo [التسجيل] وحفظت.

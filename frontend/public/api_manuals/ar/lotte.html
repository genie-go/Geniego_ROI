1. قبل أن تبدأ

واجهة برمجة التطبيقات المخصصة لشركة Lotte Global Logistics هي قناة قائمة على العقد، وليست قناة تُصدر فوريًا من وحدة تحكم الخدمة الذاتية.

مطلوب

رمز العقد (العميل) لشركة Lotte Global Logistics

تحتاج إلى رمز عقد (عميل) يتم إصداره من خلال عقد مع Lotte Global Logistics. ولا يتم إصداره مباشرة من وحدة تحكم منفصلة.

موصى به

Smart Parcel (التتبع الموحد)

إذا كنت بحاجة إلى تتبع الشحنات فقط، يمكنك ربط جميع شركات الشحن بمفتاح t_key واحد على قناة [Smart Parcel (التتبع الموحد)].

ملاحظة: إذا كان هدفك هو تتبع الشحنات، نوصي بالربط عبر مفتاح Smart Parcel الموحد دون الحاجة إلى عقد API مخصص.

2. إجراءات طلب الإصدار

قدّم الطلب وسجّل بالترتيب أدناه.

1

تجهيز رمز العقد (العميل)

جهّز رمز العقد (العميل) الخاص بشركة Lotte Global Logistics. يتم توفير واجهة برمجة التطبيقات المخصصة بموجب عقد.

2

تحديد ما إذا كنت ستستخدم التتبع الموحد

إذا كنت بحاجة إلى تتبع الشحنات فقط، فبدلاً من القناة المخصصة لشركة Lotte Global Logistics يمكنك ربط جميع شركات الشحن بمفتاح t_key واحد على قناة [Smart Parcel (التتبع الموحد)].

3

تقديم طلب إصدار واجهة برمجة التطبيقات المخصصة

إذا كنت بحاجة إلى واجهة برمجة التطبيقات المخصصة، فاحصل على رمز العقد (العميل) ومفتاح API من مندوب المبيعات.

4

التسجيل في GenieGo

أدخل api_key و cust_code في [التسجيل] داخل GenieGo Integration Hub واحفظ.

بمجرد استلامك لمفتاح API ورمز العقد (العميل) من مندوب المبيعات، تكون جاهزًا للتسجيل.

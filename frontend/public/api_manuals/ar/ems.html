1. قبل أن تبدأ

‏EMS ليست إصدارًا ذاتيًا عبر وحدة تحكم، بل هي خدمة قائمة على عقد تعتمد على رمز العقد (العميل).

ملاحظة: ‏Korea Post International Express ‏(EMS) تعتمد على رمز العقد (العميل). ليست طريقة تُصدر فيها بيانات الاعتماد بنفسك، بل هي إجراء توقّع فيه عقد EMS ثم تتلقى رمز العميل (العقد) ومفتاح API الخاص بـ Korea Post.

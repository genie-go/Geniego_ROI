TNT Express
دليل إصدار مفتاح API

تم الاستحواذ على TNT Express من قبل FedEx وتستخدم مفتاح وسر API (API Key·Secret) الصادر في بوابة مطوري FedEx (developer.fedex.com) إلى جانب رقم حساب FedEx الخاص بك. يرشدك هذا الدليل إلى تسجيل بيانات الاعتماد هذه في GenieGo Integration Hub.

الوقت المقدرحوالي 5–15 دقيقة
الحساب المطلوبحساب FedEx (رقم الحساب)
المفاتيح الصادرةAPI Key · API Secret
النتيجةبيانات اعتماد API · رقم الحساب

1. قبل أن تبدأ

تم دمج TNT في FedEx ولا توجد بوابة مطورين منفصلة لـ TNT؛ فهي تستخدم بيانات اعتماد FedEx Developer Portal.

ملاحظة: تم الاستحواذ على TNT من قبل FedEx وتُصدر بيانات الاعتماد (API Key/Secret) في FedEx Developer Portal (developer.fedex.com). لا توجد بوابة مطورين منفصلة لـ TNT، ورقم حساب FedEx مطلوب.
المتطلبات

رقم حساب FedEx

تحتاج إلى رقم حساب FedEx. يتم إدخال رقم الحساب معًا عند التسجيل في GenieGo.

المفاتيح الصادرة

API Key · API Secret

عند إنشاء مشروع في بوابة مطوري FedEx، يتم إصدار API Key وSecret.

2. خطوات الإصدار

أصدر بيانات الاعتماد الخاصة بك في FedEx Developer Portal، ثم سجّلها في GenieGo.

https://developer.fedex.com
2

إصدار API Key·Secret

إنشاء مشروع
3

تحقق من رقم الحساب

تحقق من رقم حساب FedEx الخاص بك.

بمجرد إصدار API Key·Secret في FedEx Developer Portal، يكون كل شيء على ما يرام.

3. المعلومات التي يجب تسجيلها في GenieGo

1. قبل أن تبدأ

تُطلب بيانات اعتماد DHL Express على بوابة المطورين، ويلزم وجود رقم حساب عميل.

التحضير

رقم حساب عميل DHL Express

وجود رقم حساب عميل DHL Express يجعل طلب بيانات الاعتماد أكثر سلاسة. يُدخل رقم الحساب أيضًا عند التسجيل في GenieGo.

المفاتيح الصادرة

API Key · Secret

ستحصل على API Key وSecret من خلال تطبيقك الخاص (اسم الشركة) على البوابة.

ملاحظة: عند تسجيل معلومات العميل عبر [Get Access] وطلب بيانات الاعتماد، تتلقى عادةً بريد موافقة Test/Production في يوم العمل التالي.

2. خطوات الإصدار

أصدر بالترتيب أدناه وسجّل في GenieGo.

1

التسجيل في بوابة المطورين

انتقل إلى بوابة مطوري DHL Express، وسجّل ببريدك الإلكتروني وقم بالتحقق منه.

https://developer.dhl.com
2

طلب بيانات الاعتماد

بعد تسجيل الدخول، سجّل معلومات العميل الخاصة بك في قسم [Get Access] واطلب بيانات الاعتماد.

Get Access
3

استلام بريد الموافقة

تتلقى عادةً بريد موافقة Test/Production في يوم العمل التالي.

4

العثور على API Key·Secret

في البوابة، انتقل إلى تطبيقك الخاص (اسم الشركة) > [Show Key] للعثور على API Key وSecret.

تطبيقك (اسم الشركة) → Show Key
بمجرد تلقي بريد الموافقة وظهور API Key·Secret على البوابة، يكون قد تم إصدارهما بشكل صحيح.

3. المعلومات المطلوب تسجيلها في GenieGo

الحقلالوصف
api_keyDHL API Key الذي تم العثور عليه على البوابة
account_numberرقم حساب عميل DHL Express
بمجرد إدخالها في GenieGo Integration Hub [تسجيل] والحفظ، يتم تشفير بيانات الاعتماد أثناء التخزين باستخدام AES-256-GCM وإخفاؤها عند القراءة.

4. المشكلات الشائعة

المشكلةالسببالحل
بريد الموافقة لا يصلفي انتظار معالجة الموافقة في يوم العمل التالي، أو لم يتم تسجيل معلومات العميلتحقق في [Get Access] مما إذا كان تسجيل العميل وطلب بيانات الاعتماد قد اكتملا، وانتظر على أساس أيام العمل.
خطأ في المصادقة عند الاستدعاءالخلط بين بيئتي Test/Production، أو رقم الحساب مفقودأعد التحقق من API Key·Secret الخاصين بالبيئة المعتمدة ورقم الحساب (account_number).

قائمة التحقق النهائية

  • تم تأكيد رقم حساب عميل DHL Express.
  • تم التسجيل في developer.dhl.com وطلب بيانات الاعتماد عبر [Get Access].
  • تم استلام بريد موافقة Test/Production في يوم العمل التالي.
  • تم العثور على API Key·Secret على البوابة.
  • تم إدخال api_key·account_number في GenieGo [تسجيل] والحفظ.

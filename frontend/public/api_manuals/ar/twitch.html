1. فهم بنية حساب Twitch

على عكس Instagram، لا تتضمن Twitch بنية لاختيار حساب أعمال/منشئ محتوى منفصل.

القناة الرسمية للشركة

لتشغيل العلامة التجارية

أنشئ حساب Twitch عاديًا باسم شركتك أو علامتك التجارية، ثم شغّله كقناة رسمية. وهو مناسب لعروض المنتجات والتجارة المباشرة وبث الفعاليات.

مرحلة نمو المذيع

Affiliate / Partner

بمجرد استيفائك لشروط أداء البث والمتابعين ووقت المشاهدة، يمكنك الانضمام إلى برنامج Affiliate أو Partner.

النقطة الأساسية: ينبغي للشركات أيضًا إنشاء حساب Twitch عادي أولاً، ثم القيام بتمييز القناة والمصادقة الثنائية وربط OBS وتسجيل التطبيق في Developer Console.

2. ما تحتاج إليه

  • عنوان بريد إلكتروني للأعمال
  • اسم العلامة التجارية أو اسم الشركة
  • اسم مستخدم Twitch مقترح
  • شعار الشركة وصورة بانر القناة
  • نص تعريف الشركة وعنوان URL للموقع
  • حاسوب أو حاسوب محمول للبث
  • OBS Studio أو برنامج بث
  • OAuth Redirect URL لتكامل API

3. إنشاء حساب Twitch

1

الدخول إلى Twitch

افتح موقع Twitch الرسمي في متصفحك.

https://www.twitch.tv
2

النقر على التسجيل

انقر على زر Sign Up أو التسجيل في أعلى اليمين.

Sign Up
3

التسجيل بمعلومات الشركة

أدخل بريد الشركة الإلكتروني واسم المستخدم وكلمة المرور وتاريخ الميلاد.

Email / Username / Password
4

التحقق من البريد الإلكتروني

افتح بريد التحقق الذي أرسلته Twitch وأكمل Verify Email.

Verify Email

4. تحديد اسم حساب العلامة التجارية

يصبح اسم مستخدم Twitch عنوان القناة، لذا فإن الاسم القصير وسهل التذكّر هو الأفضل.

الغرضأمثلة على الأسماء الموصى بهاالوصف
القناة الرسمية للشركةGeniego / OCLKoreaيطابق اسم الشركة فيمنح مصداقية عالية.
البث العالميGeniegoGlobal / OCLGlobalمناسب عند مراعاة المشاهدين في الخارج.
التجارة المباشرةGeniegoLive / GeniegoCommerceيوضّح بجلاء غرض بث البيع أو التسوّق.
التقنية/المنصةGeniegoTech / OCLTechمناسب لبث عروض تقنية المعلومات وSaaS والمنصات.

5. إعداد ملف القناة الشخصي

1

الانتقال إلى قائمة الإعدادات

انقر على أيقونة الملف الشخصي في أعلى اليمين ثم انتقل إلى Settings.

Profile → Settings
2

رفع صورة الملف الشخصي

اضبط شعار الشركة أو أيقونة العلامة التجارية كصورة للملف الشخصي.

Profile Picture
3

رفع بانر القناة

ارفع صورة بانر تحمل ألوان علامتك التجارية ورسالة الخدمة.

Profile Banner
4

إدخال نص التعريف

أدخل تعريف الشركة ووصف الخدمة وبريد التواصل والموقع الإلكتروني.

Bio / Links

6. إعداد المصادقة الثنائية

إعداد المصادقة الثنائية (2FA) مهم جدًا لاستخدام إعدادات البث والأمان بشكل موثوق على Twitch.

1

الانتقال إلى قائمة الأمان

افتح قائمة Security and Privacy في الإعدادات.

Settings → Security and Privacy
2

تفعيل 2FA

فعّل Two-Factor Authentication.

Set Up Two-Factor Authentication
3

ربط هاتف أو تطبيق مصادقة

اربط التحقق عبر الرسائل النصية أو تطبيق مصادقة مثل Google Authenticator أو Authy.

Authenticator App
تنبيه: يمكن أن يستخدم حساب الشركة عدة أشخاص، لذا احرص على إدارة صلاحيات المسؤول وبريد الاسترداد.

7. التحقق من Stream Key وربط OBS

1

فتح Creator Dashboard

انتقل إلى Creator Dashboard من قائمة الملف الشخصي.

Creator Dashboard
2

الانتقال إلى قائمة البث

اختر العنصر Stream في قائمة الإعدادات.

Settings → Stream
3

التحقق من Primary Stream Key

انسخ مفتاح البث الأساسي. يجب الاحتفاظ بهذا المفتاح كأنه كلمة مرور.

Primary Stream Key
4

الربط في OBS

في OBS Studio، اضبط الخدمة على Twitch وأدخل القيمة المنسوخة كمفتاح بث.

OBS → Settings → Stream → Twitch
إذا تسرّب مفتاح البث، يمكن لشخص آخر البث على قناتك. لا تكشفه أبدًا.

11. التحقق من Client ID / Client Secret

1

اختيار التطبيق المسجَّل

انقر على التطبيق الذي أنشأته للتو في قائمة Applications بـ Developer Console.

Applications → اختر التطبيق
2

نسخ Client ID

انسخ Client ID المعروض على الشاشة واحفظه في إعدادات الخادم.

Client ID
3

إنشاء Client Secret

انقر على زر New Secret لإنشاء Client Secret واحفظه بأمان.

New Secret
غالبًا ما يمكن عرض Client Secret مرة واحدة فقط. احفظه في مكان آمن فور إنشائه.

12. بنية مصادقة Twitch API

Twitch Channel │ ▼ Developer Console │ ▼ Application │ ▼ Client ID / Client Secret │ ▼ OAuth Token │ ▼ Twitch Helix API │ ▼ معلومات القناة وحالة البث وروبوت الدردشة وتحليلات المشاهدين

13. تدفق إصدار رمز OAuth

1) إنشاء عنوان URL لتفويض Twitch OAuth 2) يسجّل المستخدم الدخول بحساب Twitch 3) استقبال authorization code عند Redirect URL 4) طلب Access Token باستخدام authorization code + Client Secret 5) استدعاء Twitch API باستخدام Access Token 6) التحديث باستخدام Refresh Token عند الحاجة
يمكن إجراء الاستعلامات البسيطة عن البيانات العامة باستخدام App Access Token، بينما تتطلّب العمليات التي تحتاج إلى صلاحيات المستخدم استخدام User Access Token.

15. قائمة التحقق النهائية

  • أنشأتُ حساب Twitch ببريد إلكتروني للأعمال.
  • ضبطتُ اسم مستخدم يطابق اسم العلامة التجارية.
  • أكملتُ التحقق من البريد الإلكتروني.
  • ضبطتُ صورة الملف الشخصي والبانر ونص التعريف والروابط.
  • فعّلتُ المصادقة الثنائية.
  • تحققتُ من Stream Key في Creator Dashboard.
  • ربطتُ OBS Studio بقناة Twitch.
  • سجّلتُ التطبيق في Twitch Developer Console.
  • حصلتُ على Client ID وClient Secret.
  • جهّزتُ OAuth Redirect URL وتدفق إصدار الرمز.

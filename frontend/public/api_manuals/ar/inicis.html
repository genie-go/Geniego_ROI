دليل إصدار API

دليل تفصيلي يمكن لأي شخص اتباعه لإصدار بيانات اعتماد KG이니시스 (مفتاح/رمز API) وتسجيلها في GenieGo Integration Hub.

الوقت التقريبيحوالي 5–15 دقيقة
الحساب المطلوبحساب KG이니시스
عدد الخطوات5 خطوات
النتيجةبيانات اعتماد API

خطوات الإصدار

اتبع الخطوات أدناه لإصدار بيانات الاعتماد وتسجيلها.

1

المتطلبات: تحتاج إلى عقد متجر KG Inicis وحساب مدير المتجر.

2

سجّل الدخول إلى مدير متجر Inicis.

3

انتقل إلى [معلومات المتجر] > [معلومات العقد] > [معلومات إضافية].

4

أنشئ INIAPI Key، وأصدِر أيضًا مفتاح توقيع الويب القياسي عبر [إنشاء signkey للدفع الإلكتروني].

5

في GenieGo [تسجيل]، أدخل MID (معرّف المتجر) و sign_key و api_key ثم احفظ.

قائمة التحقق النهائية

  • المتطلبات: تحتاج إلى عقد متجر KG Inicis وحساب مدير المتجر.
  • سجّل الدخول إلى مدير متجر Inicis.
  • انتقل إلى [معلومات المتجر] > [معلومات العقد] > [معلومات إضافية].
  • أنشئ INIAPI Key، وأصدِر أيضًا مفتاح توقيع الويب القياسي عبر [إنشاء signkey للدفع الإلكتروني].
  • في GenieGo [تسجيل]، أدخل MID (معرّف المتجر) و sign_key و api_key ثم احفظ.
المفاتيح المُدخلة في GenieGo Integration Hub [تسجيل] يتم تخزينها مشفّرة بـ AES-256-GCM وإخفاؤها عند العرض.
احتفظ بمفاتيحك/أسرارك بأمان ولا تكشفها علنًا أبدًا. عند الاشتباه بتسرّب، أعد الإصدار (التدوير) فورًا.

1. قبل أن تبدأ

يتم تقديم طلب Open API الخاص بـ godomall في NHN Commerce Developer Center بعد تسجيل المطوّر والموافقة عليه.

مطلوب

حساب مشغّل المتجر

تحتاج إلى حساب مشغّل يدير متجر godomall (NHN Commerce). استخدم هذا الحساب لإكمال تسجيل العضو والمطوّر في Developer Center.

المفاتيح الصادرة

partner key · API key

بمجرد الموافقة على طلب إصدار المفتاح، يتم إصدار مفاتيح Open API (partner key / API key).

ملاحظة: يمر تسجيل المطوّر وطلب إصدار المفتاح بعملية موافقة، لذا قد تستغرق الموافقة بعض الوقت.

2. خطوات إصدار مفتاح Open API

قدّم الطلب وأصدر بالترتيب أدناه، ثم سجّل في GenieGo.

1

التسجيل والحصول على الموافقة كمطوّر

سجّل كعضو ومطوّر في NHN Commerce Developer Center واحصل على الموافقة.

devcenter.nhn-commerce.com
2

تقديم طلب إصدار المفتاح

في [Open API] > [تقديم طلب إصدار المفتاح]، املأ نموذج الطلب وأرسله.

Open API → تقديم طلب إصدار المفتاح
3

التحقق من المفاتيح

تحقق من مفاتيح Open API الصادرة (partner key / API key).

بمجرد إصدار partner key و API key، يكون الطلب قد قُدّم وتمت الموافقة عليه بشكل صحيح.

3. المعلومات المراد تسجيلها في GenieGo

الحقلالوصف
partner_keypartner key الصادر
api_keyAPI key الصادر (تعامل معه بحذر)
أدخل القيم واحفظها في GenieGo Integration Hub [تسجيل]. يتم تشفير بيانات الاعتماد أثناء التخزين باستخدام AES-256-GCM وإخفاؤها عند القراءة.

قائمة التحقق النهائية

  • سجّلت كعضو ومطوّر في NHN Commerce Developer Center وحصلت على الموافقة.
  • ملأت نموذج الطلب وأرسلته في [Open API > تقديم طلب إصدار المفتاح].
  • تحققت من partner key و API key الصادرين.

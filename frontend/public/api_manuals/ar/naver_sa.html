دليل إصدار مفتاح API

دليل تفصيلي خطوة بخطوة مناسب للمبتدئين لإصدار بيانات اعتماد API (ترخيص الوصول، المفتاح السري) ومعرّف CUSTOMER_ID الخاص بك من أداة إدارة Naver Search Ad وتسجيلها في GenieGo Integration Hub.

1. قبل أن تبدأ

لا يمكن طلب Naver Search Ad API إلا من حساب معلِن.

مطلوب

حساب معلِن Search Ad

تحتاج إلى حساب معلِن يستخدم أداة إدارة Naver Search Ad. لا يمكنك التقدّم بطلب للوصول إلى API باستخدام حساب عضوية Naver عادي.

المفاتيح الصادرة

ترخيص الوصول · المفتاح السري · CUSTOMER_ID

أكّد على CUSTOMER_ID وترخيص الوصول والمفتاح السري الصادرة معًا بعد التقدّم بطلب للوصول إلى API.

ملاحظة: لا يمكن عرض المفتاح السري إلا لحظة الإصدار، لذا احفظه بأمان مباشرة بعد إصداره.

2. خطوات إصدار مفتاح API

أصدِرها بالترتيب أدناه وسجّلها في GenieGo.

1

تسجيل الدخول إلى Search Ads

سجّل الدخول إلى أداة إدارة Naver Search Ad باستخدام حساب المعلِن الخاص بك.

searchad.naver.com
2

الانتقال إلى إدارة استخدام API

من قائمة [الأدوات] العلوية، انتقل إلى [إدارة استخدام API].

الأدوات → إدارة استخدام API
3

التقدّم بطلب لخدمة API

انقر على زر [التقدّم بطلب لخدمة Naver Search Ad API] لتقديم الطلب.

التقدّم بطلب لخدمة Naver Search Ad API
4

التحقق من المعلومات الصادرة

أكّد على CUSTOMER_ID وترخيص الوصول والمفتاح السري الصادرة.

بمجرد تأكيد CUSTOMER_ID وترخيص الوصول والمفتاح السري جميعها، يكون قد تم إصدارها بشكل صحيح.

3. المعلومات المطلوب تسجيلها في GenieGo

الحقلالوصف
api_keyترخيص الوصول الصادر
api_secretالمفتاح السري الصادر (تعامل معه بحذر)
customer_idمعرّف CUSTOMER_ID الصادر
أدخِلها واحفظها في [تسجيل] ضمن GenieGo Integration Hub، وتُشفَّر بيانات الاعتماد عند التخزين باستخدام AES-256-GCM وتُخفى عند القراءة.

4. المشكلات الشائعة

المشكلةالسببالحل
لا توجد قائمة [إدارة استخدام API]ليس حساب معلِن، أو الأذونات غير كافيةسجّل الدخول بحساب معلِن يستخدم أداة إدارة Search Ad.
حدوث خطأ في المصادقةإدخال ترخيص الوصول أو المفتاح السري بشكل غير صحيحأعد التحقق من ترخيص الوصول والمفتاح السري ومعرّف CUSTOMER_ID من شاشة الإصدار.

قائمة التحقق النهائية

  • تم تسجيل الدخول بحساب معلِن Naver Search Ad.
  • تم إكمال طلب خدمة API ضمن [الأدوات > إدارة استخدام API].
  • تم تأكيد CUSTOMER_ID وترخيص الوصول والمفتاح السري.
  • تم إدخال api_key·api_secret·customer_id في GenieGo [تسجيل] وحفظها.

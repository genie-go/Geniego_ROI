دليل إصدار مفتاح API

دليل تفصيلي خطوة بخطوة سهل للمبتدئين لتسجيل تطبيق في بوابة مطوري Etsy لإصدار API Key (keystring)، والحصول على access token عبر تفويض OAuth، وتسجيله في GenieGo Integration Hub.

النتيجةبيانات اعتماد API

1. قبل أن تبدأ

يتم إصدار مفتاح API الخاص بـ Etsy عن طريق تسجيل تطبيق ضمن حساب البائع الخاص بك.

مطلوب

حساب بائع Etsy

للمتابعة في الإصدار، تحتاج إلى حساب بائع Etsy. ستستخدم هذا الحساب للوصول إلى بوابة مطوري Etsy.

المفاتيح المُصدَرة

API Key (keystring)

بمجرد إدخال معلومات تطبيقك، يتم إصدار API Key (keystring). ثم تحصل على access token عبر تفويض OAuth.

ملاحظة: يتطلب التسجيل في GenieGo مفتاح api_key المُصدَر إلى جانب shop_id الذي يحدد متجرك.

2. خطوات الإصدار

أصدر بالترتيب أدناه وسجّل في GenieGo.

1

إنشاء تطبيق جديد

انتقل إلى etsy.com/developers وانقر على [Create a New App].

etsy.com/developers
2

إدخال معلومات التطبيق وإصدار المفتاح

بمجرد إدخال معلومات تطبيقك، يتم إصدار API Key (keystring).

https://www.etsy.com/developers/your-apps
3

تفويض OAuth

احصل على access token عبر تفويض OAuth.

4

تأكيد معلومات التسجيل

تحقق من api_key المُصدَر و shop_id الخاص بمتجرك.

بمجرد إصدار API Key (keystring) وحصولك على access token عبر تفويض OAuth، يكون قد تم إصداره بشكل صحيح.

3. المعلومات المطلوب تسجيلها في GenieGo

الحقلالوصف
api_keyمفتاح API Key (keystring) المُصدَر
shop_idمعرّف المتجر Shop ID لمتجر Etsy الذي تريد ربطه
في GenieGo Integration Hub [تسجيل]، أدخل api_key و shop_id ثم احفظ. يتم تشفير بيانات الاعتماد أثناء التخزين باستخدام AES-256-GCM وإخفاؤها عند القراءة.

4. المشكلات الشائعة

المشكلةالسببالحل
تعذر إنشاء تطبيقليس حساب بائع Etsy، أو لم يتم الوصول إلى بوابة المطورينادخل إلى etsy.com/developers بحساب بائع Etsy الخاص بك وتابع مع [Create a New App].
فشل المصادقةلم يكتمل تفويض OAuth، أو لم يتم إصدار access tokenبعد إصدار API Key (keystring)، احصل على access token عبر تفويض OAuth.

قائمة التحقق النهائية

  • دخلت إلى etsy.com/developers بحساب بائع Etsy الخاص بي.
  • أصدرت API Key (keystring) عبر [Create a New App].
  • حصلت على access token عبر تفويض OAuth.
  • أدخلت api_key و shop_id في GenieGo [تسجيل] وحفظت.

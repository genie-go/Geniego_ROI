دليل إصدار API Key

دليل تفصيلي خطوة بخطوة لإنشاء تطبيق مخصص في لوحة إدارة Shopify (Admin) لإصدار Admin API access token وتسجيله مع نطاق المتجر في GenieGo Integration Hub لربط متجر Shopify الخاص بك.

1. قبل أن تبدأ

لا يمكن إنشاء تطبيق Shopify مخصص إلا بصلاحية مدير المتجر.

مطلوب

صلاحية مدير المتجر

تحتاج إلى صلاحية المدير (Admin) لمتجر Shopify لإنشاء تطبيق مخصص وإصدار token.

المفاتيح المُصدرة

نطاق المتجر · Admin API token

ستحصل على نطاق متجر بصيغة xxx.myshopify.com وعلى Admin API access token الذي يظهر بعد تثبيت التطبيق.

ملاحظة: يظهر Admin API access token مرة واحدة فقط مباشرة بعد التثبيت، لذا يجب نسخه فورًا عند ظهوره.

2. خطوات الإصدار

أصدر بالترتيب أدناه وسجّل في GenieGo.

1

الانتقال إلى التطبيقات وقنوات البيع

في لوحة إدارة Shopify (Admin)، انتقل إلى [الإعدادات] > [التطبيقات وقنوات البيع].

https://www.shopify.com/admin/settings/apps
2

إنشاء تطبيق مخصص

انقر على [تطوير التطبيقات] > [إنشاء تطبيق] لإنشاء تطبيق مخصص.

تطوير التطبيقات → إنشاء تطبيق
3

تحديد النطاقات · تثبيت التطبيق

حدد Admin API scopes وثبّت التطبيق.

4

نسخ access token

انسخ فورًا Admin API access token الذي يظهر بعد التثبيت (يظهر مرة واحدة فقط).

بمجرد ظهور Admin API access token بعد تثبيت التطبيق، يكون قد تم إصداره بشكل صحيح. يظهر مرة واحدة فقط، لذا انسخه فورًا.

3. المعلومات المطلوب تسجيلها في GenieGo

الحقلالوصف
shop_domainنطاق المتجر بصيغة xxx.myshopify.com
access_tokenAdmin API access token المُصدر (تعامل معه بحذر)
في GenieGo [تسجيل]، أدخل shop_domain (xxx.myshopify.com) و access_token الخاص بك واحفظ. يتم تشفير بيانات الاعتماد عند التخزين باستخدام AES-256-GCM وإخفاؤها عند القراءة.

قائمة التحقق النهائية

  • سجّلت الدخول بصلاحية مدير متجر Shopify.
  • أنشأت تطبيقًا مخصصًا ضمن الإعدادات > التطبيقات وقنوات البيع > تطوير التطبيقات.
  • حددت Admin API scopes وثبّت التطبيق.
  • نسخت فورًا Admin API access token الذي ظهر بعد التثبيت.
  • أدخلت shop_domain·access_token في GenieGo [تسجيل] وحفظت.

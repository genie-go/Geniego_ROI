1. قبل أن تبدأ

Adyen هي بوابة دفع عالمية على مستوى المؤسسات، لذا يجب توفر عقد تاجر والموافقة على المراجعة أولاً قبل أن يتسنى إصدار بيانات الاعتماد.

مطلوب

عقد تاجر · الموافقة على المراجعة

Adyen هي بوابة دفع عالمية على مستوى المؤسسات تتطلب عقد تاجر والموافقة على المراجعة. لا يمكنك إصدار بيانات الاعتماد إلا بعد الموافقة.

المفاتيح الصادرة

X-API-Key · Merchant Account

بعد الموافقة، أنشئ مفتاح API (X-API-Key) في [Developers] > [API credentials] داخل Customer Area وأكّد اسم Merchant Account.

شرط مسبق: الإصدار الذاتي غير ممكن قبل الموافقة على عقد التاجر والمراجعة. يجب عليك أولاً إتمام عقد التاجر والمراجعة مع Adyen.

2. إجراءات طلب الإصدار

بعد الموافقة على المراجعة، أصدر بيانات الاعتماد وسجّلها بالترتيب أدناه.

1

عقد تاجر · الموافقة على المراجعة

وقّع عقد تاجر مع Adyen واحصل على الموافقة على المراجعة.

مبيعات/عقود Adyen
2

تسجيل الدخول إلى Customer Area

بعد الموافقة، سجّل الدخول إلى Customer Area.

https://ca-live.adyen.com
3

إصدار بيانات اعتماد API

في [Developers] > [API credentials]، أنشئ مفتاح API (X-API-Key) وأكّد اسم Merchant Account.

Developers → API credentials
4

تأكيد رقم دفعة التسوية الابتدائية

لأول عملية جمع للتسوية، ابحث عن أحدث رقم settlement batch في Customer Area.

بمجرد تأكيد مفتاح API (X-API-Key) و Merchant Account، تكون بيانات اعتمادك جاهزة.

3. المعلومات المطلوب تسجيلها في GenieGo

الحقلالوصف
api_keyمفتاح API الصادر (X-API-Key، تعامل معه بحذر)
merchant_accountاسم Merchant Account المؤكَّد
batch_startرقم دفعة التسوية الابتدائية لأول عملية جمع للتسوية (أحدث رقم settlement batch في Customer Area)
أدخل api_key·merchant_account·batch_start في [تسجيل] داخل GenieGo Integration Hub واحفظ. تُشفَّر بيانات الاعتماد أثناء التخزين باستخدام AES-256-GCM وتُخفى عند القراءة.

قائمة التحقق النهائية

  • حصلت على عقد تاجر Adyen · الموافقة على المراجعة.
  • سجّلت الدخول إلى Customer Area (ca-live.adyen.com).
  • أكّدت مفتاح API (X-API-Key) و Merchant Account في Developers > API credentials.
  • أكّدت أحدث رقم settlement batch (batch_start).
  • أدخلت api_key·merchant_account·batch_start في [تسجيل] داخل GenieGo وحفظت.

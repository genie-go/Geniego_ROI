1. مفهوم إصدار Kakao Alimtalk

Alimtalk هي خدمة رسائل تجارية ترسل الرسائل المعلوماتية مثل الطلبات والحجوزات والشحن والمدفوعات عبر Kakao Talk. وهي ليست بنية يمكن الإرسال بها مباشرة باستخدام مفتاح تطبيق مطوّر Kakao العادي وحده، بل تتطلب التحقق من القناة التجارية وإعداد وكالة إرسال/منصة.

مهم: تركّز Alimtalk على الرسائل المعلوماتية وليست الرسائل الإعلانية، وتتطلب القوالب مراجعة مسبقة.

2. المتطلبات

  • حساب Kakao Business
  • قناة Kakao Talk
  • مستندات التحقق التجاري مثل شهادة التسجيل التجاري
  • نص قالب Alimtalk المراد إرساله
  • حساب وكالة إرسال أو خدمة مراسلة سحابية
  • سياسة جمع أرقام هواتف المستلمين وإدارة الموافقة

3. تجهيز قناة Kakao Talk

1

الوصول إلى Kakao Business

سجّل الدخول إلى مركز إدارة Kakao Business.

business.kakao.com
2

إنشاء قناة Kakao Talk

أنشئ قناة باسم علامتك التجارية أو خدمتك.

مركز إدارة قناة Kakao Talk
3

إجراء التحقق التجاري

قدّم معلومات عملك والمستندات الداعمة للتحقق من القناة التجارية.

التحقق التجاري
4

التحقق من معرّف البحث للقناة

تحقق من معرّف القناة أو معرّف البحث اللازم لإعدادات إرسال Alimtalk.

ملف القناة ← معرّف البحث
يجب إكمال التحقق التجاري قبل أن تتمكن من تسجيل ملف مُرسِل Alimtalk.

4. تسجيل ملف المُرسِل / Sender Profile

1

اختيار منصة الإرسال

اختر مزوّد Alimtalk مثل NHN Cloud أو Naver Cloud SENS أو Infobank أو Bizgo.

Messaging Provider
2

إنشاء مشروع

أنشئ مشروع Alimtalk على المنصة التي اخترتها.

Project / Service
3

ربط قناة Kakao Talk

سجّل معلومات قناة Kakao Talk التي تم التحقق منها تجاريًا.

Sender Profile
4

مراجعة ملف المُرسِل

أجرِ تأكيد ملكية القناة ومراجعة ملف المُرسِل.

Profile Review
5

تأكيد بيانات اعتماد API

تحقق من App Key وSecret Key وAPI Key وغيرها التي توفّرها المنصة.

API Key / Secret

5. تسجيل القوالب والموافقة عليها

1

كتابة القالب

اكتب نصوصًا معلوماتية مثل تأكيد الطلب وإشعار الشحن.

Template Message
2

ضبط المتغيرات

اضبط القيم المتغيرة ديناميكيًا مثل الاسم ورقم الطلب ورقم الشحن كمتغيرات.

#{متغير}
3

طلب المراجعة

احفظ القالب وأرسل طلب مراجعة إلى Kakao.

Request Approval
4

تأكيد الموافقة

أكّد رمز القالب أو المعرّف المعتمد.

Template Code
إذا اختلف النص قبل المراجعة عن النص المُرسَل فعليًا، فقد يفشل الإرسال أو تُفرَض عقوبات.

6. مثال على مسار الإرسال عبر API

1) استدعِ Alimtalk API Endpoint من خادمك
2) صادِق باستخدام API Key أو Secret
3) مرّر رقم هاتف المستلم ورمز القالب وقيم المتغيرات
4) استقبل نتيجة الإرسال
5) عالِج ما إذا كان سيتم إرسال SMS بديلة عند الفشل

7. قائمة التحقق النهائية

  • أنشأتُ قناة Kakao Talk.
  • أكملتُ التحقق التجاري.
  • اخترتُ منصة إرسال Alimtalk.
  • سجّلتُ Sender Profile.
  • أكّدتُ API Key أو Secret.
  • سجّلتُ القالب وحصلتُ على الموافقة.
  • جهّزتُ سياسة فشل الإرسال والرسائل النصية البديلة.

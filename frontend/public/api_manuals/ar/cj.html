1. قبل أن تبدأ

واجهة برمجة التطبيقات المخصصة لـ CJ Logistics هي قناة قائمة على العقد ولا تُصدر فوريًا من وحدة تحكم الخدمة الذاتية.

مطلوب

رمز العقد (العميل) لـ CJ Logistics

يجب أن يكون لديك رمز عقد (عميل) يُصدر من خلال عقد مع CJ Logistics. ولا يُصدر مباشرة من وحدة تحكم منفصلة.

موصى به

Smart Parcel (التتبع الموحد)

إذا كنت بحاجة إلى تتبع الشحن فقط، يمكنك ربط جميع شركات الشحن بمفتاح t_key واحد عبر قناة [Smart Parcel (التتبع الموحد)].

ملاحظة: إذا كان هدفك هو تتبع الشحن، نوصي بالربط عبر المفتاح الموحد Smart Parcel (SweetTracker) دون عقد API مخصص.

2. إجراء طلب الإصدار

قدّم الطلب وسجّل حسب الترتيب أدناه.

1

تجهيز رمز العقد (العميل)

جهّز رمز العقد (العميل) الخاص بك لـ CJ Logistics. تُقدَّم واجهة برمجة التطبيقات المخصصة على أساس تعاقدي.

2

تحديد ما إذا كنت بحاجة إلى التتبع الموحد

إذا كنت بحاجة إلى تتبع الشحن فقط، يمكنك ربط جميع شركات الشحن بمفتاح t_key واحد عبر قناة [Smart Parcel (التتبع الموحد)] بدلاً من قناة CJ Logistics المخصصة.

3

تقديم طلب واجهة برمجة التطبيقات المخصصة

إذا كنت بحاجة إلى واجهة برمجة التطبيقات المخصصة لـ CJ Logistics، احصل على رمز العقد (العميل) ومفتاح API من مندوب المبيعات الخاص بك.

4

التسجيل في GenieGo

أدخل api_key و cust_code في [تسجيل] بـ GenieGo Integration Hub واحفظ.

بمجرد استلامك لمفتاح API ورمز العقد (العميل) من مندوب المبيعات، تكون جاهزًا للتسجيل.

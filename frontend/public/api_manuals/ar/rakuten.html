دليل إصدار مفتاح API

دليل تفصيلي خطوة بخطوة مناسب للمبتدئين لإصدار بيانات اعتماد RMS Web Service (Service Secret و License Key) من وحدة تحكم البائع في Rakuten RMS وتسجيلها في GenieGo Integration Hub.

النتيجةبيانات اعتماد RMS Web Service

1. قبل أن تبدأ

تُصدَر مفاتيح البائع (Ichiba) من RMS Web Service.

مطلوب

حساب بائع Rakuten

تحتاج إلى حساب بائع Rakuten (Ichiba). تحتاج إلى حساب يمكنه تسجيل الدخول عبر RMS R-Login (glogin.rms.rakuten.co.jp).

المفاتيح الصادرة

Service Secret · License Key

ستحصل على serviceSecret و licenseKey من RMS Web Service. تنتهي صلاحية License Key بعد 90 يومًا ويمكن إعادة إصداره.

تنبيه: يختلف RMS Web Service (glogin.rms.rakuten.co.jp) الذي يُصدر مفاتيح البائع (Ichiba) عن webservice.rakuten.co.jp المخصص للتسويق بالعمولة/البحث عن المنتجات. تأكد من إصدار مفاتيحك من جهة RMS.

2. خطوات الإصدار

أصدر بالترتيب أدناه وسجّل في GenieGo.

1

تسجيل الدخول إلى بوابة مطوري RMS

سجّل الدخول إلى بوابة مطوري Rakuten (RMS/Webservice) باستخدام حساب البائع الخاص بك.

https://glogin.rms.rakuten.co.jp
2

الانتقال إلى RMS Web Service

بعد تسجيل الدخول عبر RMS R-Login، انتقل إلى منطقة RMS Web Service.

RMS R-Login → RMS Web Service
3

إصدار Service Secret · License Key

من RMS Web Service، أصدر Service Secret و License Key. تنتهي صلاحية License Key بعد 90 يومًا، لذا أعد إصداره عند انتهاء صلاحيته.

4

التحقق من المفاتيح

تحقق من Service Secret و License Key الصادرين، إلى جانب Shop URL الخاص بمتجرك.

بمجرد ظهور Service Secret و License Key على شاشة الإصدار، فهذا يعني أنهما صدرا بشكل صحيح.

3. المعلومات المطلوب تسجيلها في GenieGo

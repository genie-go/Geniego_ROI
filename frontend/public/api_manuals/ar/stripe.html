1. قبل أن تبدأ

تُصدر مفاتيح Stripe API من لوحة تحكم حساب Stripe الخاص بك.

مطلوب

حساب Stripe

يجب أن تكون قادرًا على تسجيل الدخول إلى dashboard.stripe.com باستخدام حساب Stripe الخاص بك.

المفاتيح الصادرة

Publishable Key · Secret Key

ابحث عن Publishable key و Secret key في شاشة Developers > API keys.

ملاحظة: لتكامل الدفع الحقيقي، استخدم المفاتيح الحية (live) بدلاً من مفاتيح الاختبار.

2. خطوات الإصدار

أصدر بالترتيب أدناه وسجّل في GenieGo.

1

تسجيل الدخول إلى Stripe

سجّل الدخول إلى لوحة التحكم باستخدام حساب Stripe الذي أعددته.

dashboard.stripe.com
2

الانتقال إلى قائمة API keys

انتقل إلى قائمة [Developers] > [API keys].

Developers → API keys
3

التحقق من المفاتيح

ابحث عن Publishable key و Secret key. استخدم المفاتيح الحية (live).

https://dashboard.stripe.com/apikeys
بمجرد التأكد من كل من Publishable key و Secret key، تكون قد صدرت بشكل صحيح.

3. المعلومات المطلوب تسجيلها في GenieGo

الحقلالوصف
publishable_keyPublishable Key الصادر
secret_keySecret Key الصادر (تعامل معه بحذر)
في GenieGo Integration Hub [تسجيل]، أدخل publishable_key·secret_key واحفظ. تُشفَّر بيانات الاعتماد أثناء التخزين باستخدام AES-256-GCM وتُخفى عند القراءة.

4. المشكلات الشائعة

المشكلةالسببالحل
فشل المصادقةتم إدخال مفتاح اختبارتحقق مما إذا كنت تستخدم المفاتيح الحية (Publishable·Secret).
Secret key غير ظاهريُعرض المفتاح مرة واحدة فقط أو يكون مخفيًاتحقق مرة أخرى في شاشة Developers > API keys أو أصدر مفتاحًا جديدًا.

قائمة التحقق النهائية

  • سجّلت الدخول إلى dashboard.stripe.com باستخدام حساب Stripe الخاص بك.
  • انتقلت إلى Developers > API keys.
  • أكّدت Publishable Key·Secret Key الحية.
  • أدخلت publishable_key·secret_key في GenieGo [تسجيل] وحفظت.

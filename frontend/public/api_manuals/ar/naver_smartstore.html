Naver Smart Store
دليل إصدار مفتاح API

دليل خطوة بخطوة لتسجيل تطبيق في مركز Naver Commerce API (apicenter.commerce.naver.com) للحصول على Client ID و Client Secret، ثم تسجيلهما في GenieGo Integration Hub.

الوقت المقدرحوالي 10–15 دقيقة
الحساب المطلوبحساب Smart Store يتمتع بصلاحية "المدير المتكامل"
المفاتيح الصادرةClient ID · Client Secret
النتيجةبيانات اعتماد Commerce API

1. قبل أن تبدأ

لا يمكن الإصدار إلا بصلاحية المدير المتكامل على حساب بائع في Smart Store.

مطلوب

صلاحية المدير المتكامل

يجب أن تتمتع بصلاحية "المدير المتكامل" على حساب بائع في Smart Store. المدير المتكامل وحده يمكنه إصدار المفاتيح.

المفاتيح الصادرة

Client ID · Client Secret

عند تسجيل تطبيق، يتم إصدار معرّف تطبيق (Client ID) وسر (Client Secret).

ملاحظة: يلزم تسجيل عنوان IP الخاص بالخادم الذي سيستدعي API وتجديد مصادقة التطبيق بشكل دوري.

2. خطوات الإصدار

أصدِر بالترتيب أدناه وسجّل في GenieGo.

1

الوصول إلى مركز Commerce API

سجّل الدخول إلى مركز Naver Commerce API بحساب المدير المتكامل الخاص بك.

https://apicenter.commerce.naver.com
2

إنشاء حساب

انقر على [إنشاء حساب] في أعلى اليمين، وأدخل اسم حساب شركة المطوّر وجهة اتصال للاستجابة للأعطال، ووافق على الشروط، ثم سجّل.

إنشاء حساب
3

تسجيل تطبيق

ضمن [تطبيقات متجري]، سجّل تطبيقًا؛ بعد ذلك يتم إصدار معرّف تطبيق وسر.

تطبيقات متجري
4

تسجيل عنوان IP وتجديد المصادقة

سجّل عنوان IP الخاص بالخادم الذي سيستدعي API، وجدّد مصادقة التطبيق بشكل دوري حسب الإرشادات.

بمجرد تسجيل التطبيق وإصدار معرّف التطبيق والسر، يكون قد تمت معالجته بشكل صحيح.

3. المعلومات المطلوب تسجيلها في GenieGo

الحقلالوصف
client_idمعرّف التطبيق الصادر (Client ID)
client_secretالسر الصادر (Client Secret، تعامل معه بحذر)
أدخلهما في [تسجيل] ضمن GenieGo Integration Hub واحفظ؛ تُشفَّر بيانات الاعتماد أثناء التخزين باستخدام AES-256-GCM وتُخفى عند القراءة.

قائمة التحقق النهائية

  • تم الوصول إلى مركز Commerce API بحساب المدير المتكامل.
  • تم إدخال معلومات شركة المطوّر وجهة اتصال للاستجابة للأعطال عبر [إنشاء حساب] والموافقة على الشروط.
  • تم تسجيل تطبيق ضمن [تطبيقات متجري] والحصول على المعرّف/السر.
  • تم تسجيل عنوان IP لخادم المستدعي وإعداد مصادقة دورية.
  • تم إدخال client_id·client_secret في GenieGo [تسجيل] وحفظهما.

Руководство по выпуску API

Пошаговое руководство, которому может следовать каждый, чтобы выпустить учётные данные 11Street (11번가) (ключ/токен API) и зарегистрировать их в GenieGo Integration Hub.

Примерное времяОколо 5–15 мин
Нужен аккаунтАккаунт 11Street (11번가)
Шагов6 шагов
РезультатУчётные данные API

Шаги выпуска

Следуйте шагам ниже, чтобы выпустить и зарегистрировать учётные данные.

1

Что понадобится: аккаунт продавца (seller) 11st.

2

Войдите на openapi.11st.co.kr под идентификатором продавца.

3

В меню нажмите [Регистрация и проверка сервиса], введите данные продавца, примите условия и зарегистрируйтесь.

4

В разделе [Информация Seller API] > [Изменить] зарегистрируйте IP сервера, имеющего доступ к API.

5

Скопируйте выданный API Key.

6

В GenieGo [Зарегистрировать] введите api_key и seller_id (ID продавца) и сохраните.

Итоговый чек-лист

  • Что понадобится: аккаунт продавца (seller) 11st.
  • Войдите на openapi.11st.co.kr под идентификатором продавца.
  • В меню нажмите [Регистрация и проверка сервиса], введите данные продавца, примите условия и зарегистрируйтесь.
  • В разделе [Информация Seller API] > [Изменить] зарегистрируйте IP сервера, имеющего доступ к API.
  • Скопируйте выданный API Key.
  • В GenieGo [Зарегистрировать] введите api_key и seller_id (ID продавца) и сохраните.
Ключи, введённые в GenieGo Integration Hub [Register], хранятся в зашифрованном виде с AES-256-GCM и маскируются при просмотре.
Храните ключи/секреты в безопасности и никогда не публикуйте их. При подозрении на утечку немедленно перевыпустите (ротируйте).

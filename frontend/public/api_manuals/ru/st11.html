11Street
Руководство по выпуску API Key

Понятное новичкам пошаговое руководство по выпуску ключа Open API в Центре Open API 11Street (openapi.11st.co.kr) и его регистрации в GenieGo Integration Hub.

Примерное времяОколо 5–15 мин
Требуемый аккаунтАккаунт продавца 11Street
Выпускаемые ключиКлюч Open API · Seller ID
РезультатУчётные данные Open API

1. Прежде чем начать

Ключи Open API 11Street можно выпустить только из аккаунта продавца.

Требуется

Аккаунт продавца 11Street

Вам нужен аккаунт продавца 11Street. Войдите в Центр Open API, используя свой Seller ID.

Выпускаемые ключи

Ключ Open API · Seller ID

После завершения регистрации сервиса и регистрации IP-адреса доступа проверьте выпущенный ключ Open API вместе со своим Seller ID.

Примечание: Прежде чем использовать ключ, необходимо зарегистрировать IP-адрес сервера, которому разрешён доступ к API.

2. Шаги выпуска

Выпустите в указанном ниже порядке и зарегистрируйте в GenieGo.

1

Войдите в Центр Open API

Войдите в Центр Open API 11Street, используя свой Seller ID.

https://openapi.11st.co.kr/openapi/OpenApiFrontMain.tmall
2

Зарегистрируйте сервис

В меню нажмите [Регистрация·Проверка сервиса], введите данные продавца, примите условия и зарегистрируйтесь.

Регистрация·Проверка сервиса
3

Зарегистрируйте IP-адрес доступа

В разделе [Информация Seller API] > [Изменить] зарегистрируйте IP-адрес сервера, которому разрешён доступ к API.

Информация Seller API → Изменить
4

Скопируйте API Key

Скопируйте выпущенный API Key.

После завершения регистрации сервиса и регистрации IP-адреса доступа API Key будет выпущен.

3. Информация для регистрации в GenieGo

ПолеОписание
api_keyВыпущенный ключ Open API
seller_idSeller ID
Введите и сохраните в GenieGo Integration Hub [Регистрация]. Учётные данные шифруются при хранении с помощью AES-256-GCM и маскируются при чтении.

4. Частые проблемы

ПроблемаПричинаРешение
API Key не выпускаетсяНе завершена регистрация сервиса или регистрация IP-адреса доступаЗавершите [Регистрация·Проверка сервиса] и регистрацию IP-адреса доступа в [Информация Seller API].
Ошибка аутентификации при вызовахНе зарегистрирован разрешённый IP-адрес сервераПовторно проверьте разрешённый IP-адрес сервера в [Информация Seller API] > [Изменить].

5. Перед регистрацией товаров (важно)

Даже с действующим ключом API 11Street отклонит регистрацию товара, если нет трёх пунктов ниже.

  • Отображаемая категория (dispCtgrNo) — задайте в GenieGo [Синхронизация каталога] > [Сопоставление категорий] или в панели «Нужно указать категорию и бренд»
  • Бренд — добавьте в GenieGo [Синхронизация каталога] > [Управление брендами] и назначьте товару (достаточно названия, код не нужен)
  • Способ продажи — GenieGo автоматически отправляет фиксированную цену (01); вводить ничего не нужно

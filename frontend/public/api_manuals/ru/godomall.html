Руководство по выпуску API Key

Понятное пошаговое руководство для начинающих по подаче заявки на ключи Open API godomall в NHN Commerce Developer Center, получению partner key и API key и их регистрации в GenieGo Integration Hub.

Необходимая учётная записьУчётная запись оператора магазина godomall (NHN Commerce)
Выпускаемые ключиpartner key · API key
РезультатУчётные данные Open API

1. Перед началом

Заявка на Open API godomall подаётся в NHN Commerce Developer Center после регистрации разработчика и одобрения.

Требуется

Учётная запись оператора магазина

Вам нужна учётная запись оператора, управляющего магазином godomall (NHN Commerce). Используйте эту учётную запись для завершения регистрации участника и разработчика в Developer Center.

Выпускаемые ключи

partner key · API key

После одобрения вашей заявки на выпуск ключей выпускаются ключи Open API (partner key / API key).

Примечание: Регистрация разработчика и заявка на выпуск ключей проходят процесс одобрения, поэтому одобрение может занять некоторое время.

2. Этапы выпуска ключа Open API

Подайте заявку и выпустите в указанном ниже порядке, затем зарегистрируйте в GenieGo.

1

Зарегистрироваться и получить одобрение как разработчик

Зарегистрируйтесь как участник и разработчик в NHN Commerce Developer Center и получите одобрение.

devcenter.nhn-commerce.com
2

Подать заявку на выпуск ключей

В [Open API] > [Подать заявку на выпуск ключей] заполните форму заявки и отправьте её.

Open API → Подать заявку на выпуск ключей
3

Проверить ключи

Проверьте выпущенные ключи Open API (partner key / API key).

Как только partner key и API key выпущены, заявка была корректно подана и одобрена.

3. Информация для регистрации в GenieGo

ПолеОписание
partner_keyВыпущенный partner key
api_keyВыпущенный API key (обращайтесь осторожно)
Введите и сохраните значения в GenieGo Integration Hub [Регистрация]. Учётные данные шифруются при хранении с помощью AES-256-GCM и маскируются при чтении.

Руководство по выпуску API Key

Пошаговое руководство для начинающих по регистрации и получению одобрения в качестве разработчика на Shopee Open Platform, созданию приложения для выпуска Partner ID и Partner Key, авторизации вашего магазина и регистрации всего этого в GenieGo Integration Hub.

Примерное времяОколо 10–20 мин (ожидание одобрения отдельно)
Необходимый аккаунтАккаунт продавца Shopee
Выпускаемые ключиPartner ID · Partner Key · Shop ID
РезультатУчётные данные API

1. Перед началом

Shopee Open Platform требует регистрации и одобрения разработчика, а также шага авторизации магазина.

Требуется

Аккаунт продавца Shopee

Чтобы продолжить выпуск, вам нужен аккаунт продавца Shopee. С этим аккаунтом вы регистрируетесь как разработчик на Shopee Open Platform.

Выпускаемые ключи

Partner ID · Partner Key

Когда вы создаёте приложение, выпускаются Partner ID и Partner Key. После этого вы авторизуете свой магазин и получаете access_token.

Примечание: После регистрации в качестве разработчика на Shopee Open Platform необходимо дождаться одобрения, и выпущенный access_token обновляется каждые 4 часа.

2. Шаги выпуска

Выпускайте в указанном ниже порядке и зарегистрируйте в GenieGo.

1

Регистрация и получение одобрения как разработчик

Зарегистрируйтесь как разработчик на Shopee Open Platform (open.shopee.com) и дождитесь одобрения.

https://open.shopee.com
2

Создание приложения

Создайте приложение в разделе [App Management] > [App List], при этом выпускаются Partner ID и Partner Key.

App Management → App List
3

Авторизация магазина

Сгенерируйте ссылку авторизации (authorization) магазина и авторизуйте магазин продавца; после этого выпускается code.

4

Выпуск access_token

Используйте code для получения access_token. access_token обновляется каждые 4 часа.

После того как Partner ID и Partner Key выпущены и вы получили access_token через авторизацию магазина, он выпущен правильно.

3. Информация для регистрации в GenieGo

ПолеОписание
partner_idВыпущенный Partner ID
partner_keyВыпущенный Partner Key (обращайтесь осторожно)
shop_idShop ID авторизованного магазина продавца
В GenieGo Integration Hub [Зарегистрировать] введите partner_id, partner_key и shop_id и сохраните. Учётные данные шифруются при хранении с помощью AES-256-GCM и маскируются при чтении.

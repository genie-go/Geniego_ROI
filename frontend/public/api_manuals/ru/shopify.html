Руководство по выпуску API Key

Пошаговое руководство по созданию пользовательского приложения в панели администратора Shopify (Admin) для выпуска Admin API access token и его регистрации вместе с доменом магазина в GenieGo Integration Hub для подключения вашего магазина Shopify.

Примерное времяОколо 5–15 мин
Требуемая учётная записьПрава администратора магазина Shopify
Выпущенные ключиДомен магазина · Admin API access token
РезультатУчётные данные Admin API

1. Перед началом

Пользовательское приложение Shopify можно создать только с правами администратора магазина.

Требуется

Права администратора магазина

Чтобы создать пользовательское приложение и выпустить token, вам нужны права администратора (Admin) магазина Shopify.

Выпущенные ключи

Домен магазина · Admin API token

Вы получите домен магазина в формате xxx.myshopify.com и Admin API access token, который отображается после установки приложения.

Примечание: Admin API access token отображается только один раз сразу после установки, поэтому вы должны скопировать его немедленно, как только он появится.

2. Шаги выпуска

Выполните выпуск в указанном ниже порядке и зарегистрируйте в GenieGo.

1

Перейти к Приложениям и каналам продаж

В панели администратора Shopify (Admin) перейдите в [Настройки] > [Приложения и каналы продаж].

https://www.shopify.com/admin/settings/apps
2

Создать пользовательское приложение

Нажмите [Разработка приложений] > [Создать приложение], чтобы создать пользовательское приложение.

Разработка приложений → Создать приложение
3

Выбрать scopes · Установить приложение

Выберите Admin API scopes и установите приложение.

4

Скопировать access token

Немедленно скопируйте Admin API access token, отображаемый после установки (отображается только один раз).

Как только Admin API access token появится после установки приложения, он выпущен корректно. Он отображается только один раз, поэтому скопируйте его немедленно.

3. Информация для регистрации в GenieGo

ПолеОписание
shop_domainДомен магазина в формате xxx.myshopify.com
access_tokenВыпущенный Admin API access token (обращайтесь осторожно)
В GenieGo [Регистрация] введите ваш shop_domain (xxx.myshopify.com) и access_token и сохраните. Учётные данные шифруются при хранении с помощью AES-256-GCM и маскируются при чтении.

4. Частые проблемы

Финальный чек-лист

  • Вошёл с правами администратора магазина Shopify.
  • Создал пользовательское приложение в Настройки > Приложения и каналы продаж > Разработка приложений.
  • Выбрал Admin API scopes и установил приложение.
  • Немедленно скопировал Admin API access token, отображённый после установки.
  • Ввёл shop_domain·access_token в GenieGo [Регистрация] и сохранил.

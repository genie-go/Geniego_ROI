YouTube API Key
Руководство по выпуску

Практическое пошаговое руководство, которому могут следовать даже новички, чтобы включить YouTube Data API v3 в Google Cloud Console и создать API Key.

Примерное времяОколо 5–10 мин
Необходимая учётная записьАккаунт Google
Используемый APIYouTube Data API v3
РезультатAPI Key в формате AIzaSy...

1. Разница между API Key и OAuth

Именно эта часть вызывает больше всего путаницы в начале работы.

API Key

Для запроса общедоступных данных

Используется для получения общедоступных данных YouTube, таких как поиск видео, просмотр сведений о канале, получение плейлистов и чтение комментариев.

OAuth 2.0

Для задач, требующих разрешений вашего аккаунта

Используется для задач, требующих согласия пользователя, таких как загрузка видео, управление собственным каналом и доступ к личным данным.

Важно: Для автоматической загрузки видео одного API Key недостаточно; требуется аутентификация OAuth 2.0.

2. Включение YouTube Data API v3

Прежде чем создавать API Key, сначала необходимо включить нужный API в вашем проекте.

1

Откройте Google Cloud Console

Войдите в Google Cloud Console.

console.cloud.google.com
2

Выберите или создайте проект

В селекторе проектов вверху выберите существующий проект или создайте новый.

напр., Geniego ROI
3

Перейдите в библиотеку API

В левом меню перейдите в API и сервисы → Библиотека.

API и сервисы → Библиотека
4

Найдите YouTube Data API v3 и включите его

Введите YouTube Data API v3 в поле поиска, выберите этот API и нажмите кнопку Включить.

YouTube Data API v3 → Включить
Если на экране отображается Статус: Включён, значит он включён правильно.

3. Создание API Key

После включения YouTube Data API v3 вы можете создать API Key на экране Учётные данные.

1

Перейдите в меню Учётные данные

В левом меню перейдите в API и сервисы → Учётные данные.

API и сервисы → Учётные данные
2

Нажмите Создать учётные данные

Нажмите кнопку + Создать учётные данные вверху.

+ Создать учётные данные
3

Выберите Ключ API

В появившемся меню выберите Ключ API.

Ключ API
4

Выберите доступный API

В разделе Ограничения API убедитесь, что выбран YouTube Data API v3.

Выбранный API: YouTube Data API v3
5

Нажмите кнопку Создать

Нажмите кнопку Создать внизу, чтобы сгенерировать API Key.

Создать
Сгенерированный ключ обычно начинается с AIzaSy.... Эта строка и есть ваш YouTube API Key.

4. Какие параметры выбрать на экране создания?

ПараметрРекомендуемая настройка для начинающихОписание
ИмяYouTube API KeyВведите имя, которое легко будет узнать позже.
Ограничения APIYouTube Data API v3Ограничьте этот ключ, чтобы он использовался только с YouTube API.
Аутентификация сервисного аккаунтаОставьте без отметкиОбычно не требуется для создания YouTube Data API Key.
Ограничения приложенияНет для первого тестаПосле подтверждения корректной работы рекомендуем задать ограничения по веб-сайту или IP-адресу.
При развёртывании в реальном сервисе обязательно задайте ограничения ключа. Если он будет открыт без ограничений, другие смогут использовать ваш ключ.

5. Примеры использования API Key

Вы можете протестировать, подставив сгенерированный ключ в часть YOUR_API_KEY.

Пример поиска видеоhttps://www.googleapis.com/youtube/v3/search?part=snippet&q=korea&type=video&key=YOUR_API_KEY
Пример запроса сведений о каналеhttps://www.googleapis.com/youtube/v3/channels?part=snippet&id=CHANNEL_ID&key=YOUR_API_KEY
Если при вводе указанного выше URL в адресную строку браузера появляется результат в формате JSON, значит API Key работает правильно.

  • Включили YouTube Data API v3.
  • Создали API Key в разделе Учётные данные.
  • Подтвердили, что сгенерированный API Key начинается с AIzaSy.
  • Установили ограничения API на YouTube Data API v3.
  • Задайте ограничения по веб-сайту или IP перед развёртыванием в реальном сервисе.

Руководство по выпуску API Key

Руководство, описывающее, как получить общий ключ API для Auction·Gmarket через ESM Trading API (etapi.gmarket.com) и зарегистрировать его в GenieGo Integration Hub. Поскольку Auction и Gmarket управляются совместно через ESM Plus, один ключ API подключает оба канала.

1. Прежде чем начать

Вы должны быть зарегистрированы как продавец и в Auction, и в Gmarket и иметь мастер-ID ESM Plus.

Требуется

Аккаунт продавца + мастер-ID ESM

Вам нужен аккаунт продавца для каждого из сервисов Auction и Gmarket, а также мастер-ID, созданный в ESM Plus (esmplus.com).

Выпускаемые ключи

Ключ API · ID продавца

После одобрения ESM Trading API подтвердите выпущенный ключ API вместе с ID продавца.

Примечание: ESM Trading API требует одобрения, и заявки могут быть отклонены. При подаче заявки необходимо предоставить три элемента: ваш ID продавца Auction, ID продавца Gmarket и мастер-ID ESM.

2. Шаги выпуска

Подайте заявку и получите ключ в указанном ниже порядке, затем зарегистрируйте его в GenieGo.

1

Зарегистрироваться как продавец и создать мастер-ID ESM

Зарегистрируйтесь как продавец и в Auction, и в Gmarket, затем войдите в ESM Plus и создайте мастер-ID.

esmplus.com
2

Подать заявку на использование ESM Trading API

Перейдите в ESM Trading API и подайте заявку на доступ к API. Так как требуется одобрение, заявка может быть отклонена.

https://etapi.gmarket.com
3

Предоставить идентификационную информацию

При подаче заявки предоставьте три элемента: ваш ID продавца Auction, ID продавца Gmarket и мастер-ID ESM.

4

Выпуск ключа

После одобрения выпускается ключ API. Подтвердите выпущенный ключ API и ID продавца.

Когда ключ API выпущен после завершения одобрения, процесс прошёл успешно. По вопросам, связанным с одобрением, обращайтесь по адресу etapihelp@gmail.com.

3. Информация для регистрации в GenieGo

ПолеОписание
api_keyВыпущенный ключ API
seller_idID продавца
Введите значения в GenieGo Integration Hub [Регистрация] и сохраните. Учётные данные шифруются при хранении с помощью AES-256-GCM и маскируются при чтении.

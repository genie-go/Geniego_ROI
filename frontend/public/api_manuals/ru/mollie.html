Руководство по выпуску API-ключа

Пошаговое руководство для начинающих по поиску вашего ключа Live API (начинается с live_) в панели управления Mollie и его регистрации в GenieGo Integration Hub.

Примерное времяОколо 5–10 мин
Необходимая учётная записьУчётная запись Mollie (ориентирована на Европу)
Выпускаемый ключКлюч Live API (live_)
РезультатУчётные данные Live API

1. Прежде чем начать

Ваш API-ключ Mollie находится в панели управления вашей учётной записи Mollie.

Требуется

Учётная запись Mollie

Вам нужна учётная запись Mollie (ориентирована на Европу). Вы должны иметь возможность войти в панель управления с её помощью.

Выпускаемый ключ

Ключ Live API

В панели управления перейдите в Developers > API keys и найдите ключ Live API, начинающийся с live_.

Примечание: Обязательно найдите ключ Live API, начинающийся с live_. Будьте осторожны, чтобы не перепутать его с ключом Test.

2. Шаги выпуска

Выпустите в указанном ниже порядке и зарегистрируйте в GenieGo.

1

Доступ к панели управления

Войдите в панель управления Mollie.

my.mollie.com
2

Перейти в API keys

Перейдите в [Developers] > [API keys].

https://my.mollie.com/dashboard/developers/api-keys
3

Найти ключ Live API

Найдите ключ Live API, начинающийся с live_.

Если в разделе Developers > API keys вы видите ключ, начинающийся с live_, значит всё правильно.

3. Информация для регистрации в GenieGo

ПолеОписание
api_keyКлюч Live API (начинается с live_; обращайтесь осторожно)
В GenieGo Integration Hub [Регистрация] введите ваш api_key и сохраните. Учётные данные шифруются при хранении с помощью AES-256-GCM и маскируются при чтении.

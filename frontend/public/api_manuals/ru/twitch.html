1. Понимание структуры аккаунта Twitch

В отличие от Instagram, в Twitch нет структуры с отдельным выбором бизнес-аккаунта/аккаунта автора.

Официальный канал компании

Для управления брендом

Создайте обычный аккаунт Twitch под названием вашей компании или бренда и ведите его как официальный канал. Подходит для презентаций товаров, лайв-коммерции и трансляций мероприятий.

Уровень роста стримера

Affiliate / Partner

Как только вы выполните условия по показателям трансляций, подписчикам и времени просмотра, вы сможете присоединиться к программе Affiliate или Partner.

Ключевой момент: Компаниям также следует сначала создать обычный аккаунт Twitch, а затем заняться брендингом канала, двухфакторной аутентификацией, подключением OBS и регистрацией приложения в Developer Console.

2. Что вам понадобится

  • Корпоративный адрес электронной почты
  • Название бренда или компании
  • Кандидат на имя пользователя Twitch
  • Логотип компании и изображение баннера канала
  • Текст представления компании и URL сайта
  • ПК или ноутбук для трансляции
  • OBS Studio или ПО для стриминга
  • OAuth Redirect URL для интеграции API

3. Создание аккаунта Twitch

1

Открыть Twitch

Откройте официальный сайт Twitch в браузере.

https://www.twitch.tv
2

Нажать «Зарегистрироваться»

Нажмите кнопку Sign Up или «Зарегистрироваться» в правом верхнем углу.

Sign Up
3

Регистрация с данными компании

Введите корпоративную почту, имя пользователя, пароль и дату рождения.

Email / Username / Password
4

Подтвердить электронную почту

Откройте письмо для подтверждения, отправленное Twitch, и завершите Verify Email.

Verify Email

4. Выбор имени аккаунта бренда

Имя пользователя Twitch становится адресом канала, поэтому лучше выбрать короткое и легко запоминающееся имя.

5. Настройка профиля канала

1

Перейти в меню настроек

Нажмите значок профиля в правом верхнем углу, затем перейдите в Settings.

Profile → Settings
2

Загрузить изображение профиля

Установите логотип компании или значок бренда в качестве изображения профиля.

Profile Picture
3

Загрузить баннер канала

Загрузите изображение баннера с цветами вашего бренда и сообщением о сервисе.

Profile Banner
4

Ввести текст биографии

Введите представление компании, описание сервиса, контактную почту и сайт.

Bio / Links

6. Настройка двухфакторной аутентификации

Настройка 2FA очень важна для надёжного использования настроек трансляции и безопасности на Twitch.

1

Перейти в меню безопасности

Откройте меню Security and Privacy в настройках.

Settings → Security and Privacy
2

Включить 2FA

Включите Two-Factor Authentication.

Set Up Two-Factor Authentication
3

Подключить телефон или приложение-аутентификатор

Подключите подтверждение по SMS или приложение-аутентификатор, такое как Google Authenticator или Authy.

Authenticator App
Внимание: Корпоративный аккаунт могут использовать несколько человек, поэтому обязательно управляйте правами администратора и адресом для восстановления.

7. Проверка Stream Key и подключение OBS

1

Открыть Creator Dashboard

Перейдите в Creator Dashboard из меню профиля.

Creator Dashboard
2

Перейти в меню Stream

Выберите пункт Stream в меню настроек.

Settings → Stream
3

Проверить Primary Stream Key

Скопируйте основной стрим-ключ. Этот ключ нужно хранить как пароль.

Primary Stream Key
4

Подключить в OBS

В OBS Studio установите сервис Twitch и введите скопированное значение в качестве стрим-ключа.

OBS → Settings → Stream → Twitch
Если ваш стрим-ключ будет раскрыт, кто-то другой сможет вести трансляцию на вашем канале. Никогда не публикуйте его.

9. Регистрация в Twitch Developer Console

Чтобы создавать чат-ботов, аналитику зрителей, проверку статуса трансляции и функции автоматизации, необходимо зарегистрировать приложение Twitch Developer.

1

Открыть Twitch Developer

Откройте сайт для разработчиков Twitch.

https://dev.twitch.tv
2

Перейти в Console

Нажмите Console или Developer Console в верхнем меню.

Developer Console
3

Выбрать меню Applications

Выберите Applications в левом или верхнем меню.

Applications
4

Нажать Register Your Application

Нажмите кнопку регистрации нового приложения.

Register Your Application

11. Проверка Client ID / Client Secret

1

Выбрать зарегистрированное приложение

Нажмите на только что созданное приложение в списке Applications в Developer Console.

Applications → выбрать приложение
2

Скопировать Client ID

Скопируйте отображаемый на экране Client ID и сохраните его в настройках сервера.

Client ID
3

Сгенерировать Client Secret

Нажмите кнопку New Secret, чтобы сгенерировать Client Secret, и надёжно сохраните его.

New Secret
Client Secret часто можно просмотреть только один раз. Сохраните его в надёжном месте сразу после генерации.

12. Структура аутентификации Twitch API

Twitch Channel │ ▼ Developer Console │ ▼ Application │ ▼ Client ID / Client Secret │ ▼ OAuth Token │ ▼ Twitch Helix API │ ▼ Информация о канале, статус трансляции, чат-бот, аналитика зрителей

13. Процесс выпуска токена OAuth

1) Сгенерировать URL авторизации Twitch OAuth 2) Пользователь входит со своим аккаунтом Twitch 3) Получить authorization code на Redirect URL 4) Запросить Access Token с помощью authorization code + Client Secret 5) Вызвать Twitch API с помощью Access Token 6) При необходимости обновить с помощью Refresh Token
Простые запросы публичных данных возможны с App Access Token, тогда как операции, требующие пользовательских разрешений, нуждаются в User Access Token.

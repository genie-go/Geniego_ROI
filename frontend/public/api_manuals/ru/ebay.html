Руководство по выпуску API Key

Понятное новичкам пошаговое руководство по выпуску рабочего (Production) набора ключей (DevID·AppID·CertID) в Программе разработчиков eBay, созданию пользовательского токена OAuth и регистрации его в GenieGo Integration Hub.

Примерное времяОколо 10–20 мин
Необходимая учётная записьУчётная запись eBay
Выпускаемые ключиDevID · AppID · CertID + токен OAuth
РезультатТокен доступа OAuth

1. Прежде чем начать

Для выпуска набора ключей eBay необходима регистрация в Программе разработчиков.

Необходимо

Учётная запись eBay

Чтобы продолжить выпуск, вам нужна учётная запись eBay. Используйте её для регистрации в Программе разработчиков на developer.ebay.com.

Выпускаемые ключи

DevID · AppID · CertID

При создании рабочего (Production) набора ключей выпускаются DevID, AppID и CertID. После этого необходимо также создать пользовательский токен OAuth, чтобы зарегистрировать его в GenieGo.

Примечание: Для рабочего набора ключей может отображаться уведомление "Your Keyset is currently disabled". В этом случае он активируется только после завершения процедуры подписки/отказа от уведомлений об удалении учётных записей маркетплейса.

2. Этапы выпуска

Выпускайте в указанном ниже порядке и регистрируйте в GenieGo.

1

Регистрация в Программе разработчиков

Перейдите на developer.ebay.com и зарегистрируйтесь в Программе разработчиков с помощью своей учётной записи eBay.

developer.ebay.com
2

Создание рабочего набора ключей

На странице [Application Keys] нажмите [Create a keyset] в разделе Production.

https://developer.ebay.com/my/keys
3

Процедура активации набора ключей

Если отображается уведомление "Your Keyset is currently disabled", нажмите ссылку, чтобы завершить процедуру подписки/отказа от уведомлений об удалении учётных записей маркетплейса.

Application Keys → Production
4

Проверка ключей и создание токена OAuth

Проверьте выпущенные DevID, AppID и CertID и создайте пользовательский токен OAuth.

Как только DevID·AppID·CertID выпущены и создан пользовательский токен OAuth, выпуск завершён.

3. Информация для регистрации в GenieGo

ПолеОписание
Токен доступа OAuthСозданный вами пользовательский токен OAuth (обращайтесь осторожно)
В GenieGo Integration Hub [Регистрация] введите токен доступа OAuth и сохраните. Учётные данные шифруются при хранении с помощью AES-256-GCM и маскируются при чтении.

4. Частые проблемы

ПроблемаПричинаРешение
Набор ключей в отключённом состоянииПроцедура подписки/отказа от уведомлений об удалении учётных записей маркетплейса не завершенаНажмите ссылку уведомления "Your Keyset is currently disabled" и завершите эту процедуру.
Невозможно выпустить токен OAuthРабочий набор ключей (DevID·AppID·CertID) ещё не выпущенВ [Application Keys] сначала создайте рабочий набор ключей, затем создайте токен OAuth.

Итоговый контрольный список

  • Зарегистрировались в Программе разработчиков на developer.ebay.com с помощью своей учётной записи eBay.
  • Выпустили DevID·AppID·CertID через Production [Create a keyset].
  • При отключённом наборе ключей завершили процедуру подписки/отказа от уведомлений об удалении учётных записей маркетплейса.
  • Создали пользовательский токен OAuth и ввели токен доступа OAuth в GenieGo [Регистрация] и сохранили.

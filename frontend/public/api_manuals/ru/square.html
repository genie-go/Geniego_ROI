Руководство по выпуску API Key

Понятное новичкам пошаговое руководство по созданию приложения в Square Developer Portal для выпуска Access Token и Location ID и их регистрации в GenieGo Integration Hub.

Примерное времяОколо 5–10 мин
Необходимый аккаунтАккаунт Square
Выпускаемые ключиAccess Token · Location ID
РезультатУчётные данные API

1. Прежде чем начать

Учётные данные Square API выпускаются путём создания приложения в Developer Portal.

Требуется

Аккаунт Square

Вам нужен аккаунт Square, позволяющий создать приложение на developer.squareup.com/apps.

Выпускаемые ключи

Access Token · Location ID

Найдите Access Token на вкладке Production и Location ID в разделе [Locations].

Примечание: Для реальной интеграции используйте токен Production, а не Sandbox.

2. Шаги выпуска

Выпустите в указанном ниже порядке и зарегистрируйте в GenieGo.

1

Создать приложение

Создайте приложение в Square Developer Portal.

developer.squareup.com/apps
2

Найти Access Token

Найдите Access Token на вкладке Production.

Production
3

Найти Location ID

Найдите Location ID в разделе [Locations].

Locations
После того как вы подтвердили Production Access Token и Location ID, они выпущены правильно.

3. Информация для регистрации в GenieGo

ПолеОписание
access_tokenAccess Token, найденный на вкладке Production (обращайтесь осторожно)
location_idLocation ID, найденный в разделе [Locations]
Введите access_token·location_id в [Регистрация] GenieGo Integration Hub и сохраните. Учётные данные шифруются при хранении с помощью AES-256-GCM и маскируются при чтении.

4. Частые проблемы

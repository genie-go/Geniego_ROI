Пошаговое руководство для начинающих по выпуску токена доступа системного пользователя из вашего бизнес-приложения Meta и его регистрации вместе с идентификатором рекламного аккаунта (act_) в GenieGo Integration Hub для подключения рекламных данных Meta Ads.

Выпускаемые ключиТокен доступа · Идентификатор рекламного аккаунта (act_)
РезультатУчётные данные Marketing API

1. Перед началом

Для интеграции Meta Ads требуется бизнес-аккаунт и рекламный аккаунт, который активно показывает рекламу.

Требуется

Личный аккаунт Facebook · Рекламный аккаунт

Вам нужен личный аккаунт Facebook и рекламный аккаунт Meta, который активно показывает рекламу.

Выпускаемые ключи

Токен доступа · Идентификатор рекламного аккаунта (act_)

Вы получите токен системного пользователя (токен доступа) и числовой идентификатор рекламного аккаунта, начинающийся с act_.

Примечание: Для доступа к данным за пределами вашего собственного бизнеса требуется проверка приложения (App Review). Если вы подключаете только рекламные аккаунты в рамках своего бизнеса, вы можете продолжить без проверки.

2. Шаги выпуска

Выпускайте в указанном ниже порядке и зарегистрируйте в GenieGo.

1

Создание и подтверждение бизнес-аккаунта

Откройте сайт в браузере, войдите в систему, затем создайте бизнес-аккаунт через [Создать бизнес] и завершите подтверждение по электронной почте.

business.facebook.com
2

Создание приложения

Нажмите [Мои приложения] > [Создать приложение] в правом верхнем углу, выберите тип "Бизнес" и создайте приложение.

developers.facebook.com → Мои приложения → Создать приложение
3

Добавление Marketing API

На панели управления приложением найдите карточку "Marketing API" и нажмите [Настроить], чтобы добавить её как продукт.

4

Создание системного пользователя и назначение разрешений

Перейдите в Настройки бизнеса > [Пользователи] > [Системные пользователи] > [Добавить], чтобы создать системного пользователя, затем предоставьте разрешения рекламного аккаунта в разделе [Назначить ресурсы].

business.facebook.com/settings → Пользователи → Системные пользователи
5

Создание токена доступа

На том же экране нажмите [Создать токен], выберите своё приложение, отметьте разрешения (ads_read, ads_management), создайте токен и скопируйте его значение.

6

Поиск идентификатора рекламного аккаунта

В Ads Manager найдите числовой идентификатор рекламного аккаунта, начинающийся с act_.

adsmanager.facebook.com
После того как токен создан с отмеченными разрешениями ads_read и ads_management, он выпущен правильно.

3. Информация для регистрации в GenieGo

ПолеОписание
access_tokenВыпущенный токен доступа системного пользователя (обращайтесь осторожно)
ad_account_idИдентификатор рекламного аккаунта, начинающийся с act_
В GenieGo Integration Hub > в разделе [Зарегистрировать] соответствующего канала вставьте токен доступа и идентификатор рекламного аккаунта (act_) и сохраните. Учётные данные шифруются при хранении с помощью AES-256-GCM и маскируются при чтении.

Руководство по выпуску API Key

Руководство по выпуску общего API-ключа Gmarket·Auction через ESM Trading API (etapi.gmarket.com) и его регистрации в GenieGo Integration Hub. Gmarket и Auction работают совместно через ESM Plus, поэтому один API-ключ подключает оба канала одновременно.

1. Прежде чем начать

Вы должны быть зарегистрированы как продавец и в Gmarket, и в Auction, а также иметь мастер-ID ESM Plus.

Требуется

Учётные записи продавца + мастер-ID ESM

Вам нужны учётные записи продавца как в Auction, так и в Gmarket, а также мастер-ID, созданный в ESM Plus (esmplus.com).

Выпускаемые ключи

API-ключ · ID продавца

После одобрения ESM Trading API проверьте вместе выпущенный API-ключ и ID продавца.

Примечание: ESM Trading API работает по одобрению, и ваша заявка может быть отклонена. При подаче заявки необходимо предоставить три элемента: ваш ID продавца Auction, ID продавца Gmarket и мастер-ID ESM.

2. Этапы выпуска

Подайте заявку и выпустите в указанном ниже порядке, затем зарегистрируйте в GenieGo.

1

Зарегистрируйтесь как продавец и создайте мастер-ID ESM

Зарегистрируйтесь как продавец и в Auction, и в Gmarket, затем войдите в ESM Plus и создайте мастер-ID.

esmplus.com
2

Подайте заявку на доступ к ESM Trading API

Перейдите в ESM Trading API и подайте заявку на доступ к API. Доступ предоставляется по одобрению, поэтому заявка может быть отклонена.

https://etapi.gmarket.com
3

Предоставьте идентификационные данные

При подаче заявки предоставьте три элемента: ваш ID продавца Auction, ID продавца Gmarket и мастер-ID ESM.

4

Получите ключ

После одобрения выпускается API-ключ. Проверьте выпущенный API-ключ и ID продавца.

Как только ваша заявка одобрена и API-ключ выпущен, обработка прошла корректно. По вопросам, связанным с одобрением, обращайтесь по адресу etapihelp@gmail.com.

3. Информация для регистрации в GenieGo

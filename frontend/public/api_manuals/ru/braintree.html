Руководство по выпуску API Key

Пошаговое руководство для начинающих по поиску учётных данных Production API (Merchant ID·Public Key·Private Key) в Braintree (PayPal) Control Panel и их регистрации в GenieGo Integration Hub.

Примерное времяОколо 5–10 мин
Требуемый аккаунтАккаунт продавца Braintree (PayPal) (требуется одобрение)
Выпущенные ключиMerchant ID · Public Key · Private Key
РезультатУчётные данные Production API

1. Прежде чем начать

Учётные данные Braintree (PayPal) становятся доступны только после одобрения вашего аккаунта продавца.

Обязательно

Аккаунт продавца Braintree (PayPal)

Сначала вам нужен одобренный аккаунт продавца Braintree (PayPal). Учётные данные Production доступны только после одобрения вашего аккаунта продавца.

Выпущенные ключи

Merchant ID · Public/Private Key

В Control Panel перейдите в Settings > API Keys, чтобы найти Merchant ID, Public Key и Private Key среды Production.

Примечание: Обязательно используйте учётные данные среды Production. Будьте внимательны, чтобы не перепутать их с ключами Sandbox.

2. Процедура выпуска

После одобрения вашего аккаунта продавца найдите учётные данные в указанном ниже порядке и зарегистрируйте их в GenieGo.

1

Одобрение аккаунта продавца

Сначала завершите одобрение аккаунта продавца Braintree (PayPal). Перейти к следующему шагу можно только после завершения одобрения.

2

Войти в Control Panel

Войдите в Braintree Control Panel.

braintreegateway.com
3

Перейти в API Keys

Перейдите в меню [Settings] > [API Keys].

Settings → API Keys
4

Найти учётные данные

Найдите Merchant ID, Public Key и Private Key среды Production.

Если Merchant ID, Public Key и Private Key среды Production отображаются в Settings > API Keys, значит всё в порядке.

3. Информация для регистрации в GenieGo

ПолеОписание
merchant_idProduction Merchant ID
public_keyProduction Public Key
private_keyProduction Private Key (обращайтесь осторожно)
Введите merchant_id·public_key·private_key в разделе [Регистрация] в GenieGo Integration Hub и сохраните. Учётные данные шифруются при хранении с помощью AES-256-GCM и маскируются при чтении.

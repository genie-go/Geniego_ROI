Walmart Marketplace
Руководство по выпуску API Key

Пошаговое руководство для начинающих по созданию client_id и Client Secret в Developer Portal Центра продавцов Walmart Marketplace и их регистрации в GenieGo Integration Hub.

Расчётное времяОколо 5–15 мин
Требуемый аккаунтАккаунт продавца Walmart Marketplace
Выпускаемые ключиClient ID · Client Secret
РезультатУчётные данные API

1. Прежде чем начать

Учётные данные API Walmart выпускаются в Developer Portal Центра продавцов Marketplace.

Обязательно

Аккаунт продавца Walmart Marketplace

Чтобы продолжить выпуск, вам нужен аккаунт продавца Walmart Marketplace. Используйте этот аккаунт для доступа к Developer Portal в Центре продавцов.

Выпускаемые ключи

Client ID · Client Secret

Создайте API Key (client_id) и Client Secret в Developer Portal.

2. Шаги выпуска

Выпустите в указанном ниже порядке и зарегистрируйте в GenieGo.

1

Доступ к Developer Portal

Перейдите в Walmart Seller Center > Developer Portal.

https://developer.walmart.com
2

Создать учётные данные

Создайте API Key (client_id) и Client Secret.

3

Проверить значения

Проверьте выпущенные client_id и client_secret.

После того как client_id и Client Secret созданы, они выпущены корректно.

3. Информация для регистрации в GenieGo

ПолеОписание
client_idВыпущенный API Key (Client ID)
client_secretВыпущенный Client Secret (обращайтесь осторожно)
Введите ваши client_id и client_secret в GenieGo Integration Hub [Регистрация] и сохраните. Учётные данные шифруются при хранении с помощью AES-256-GCM и маскируются при чтении.

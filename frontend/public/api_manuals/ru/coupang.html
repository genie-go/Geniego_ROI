Руководство по выпуску API Key

Понятное для новичков пошаговое руководство по выпуску учётных данных Open API (Access Key·Secret Key) в консоли продавца Coupang WING и их регистрации в GenieGo Integration Hub.

Примерное времяОколо 5–10 мин
Требуемый аккаунтАккаунт продавца Coupang WING
Выпускаемые ключиAccess Key · Secret Key
РезультатУчётные данные Open API

1. Прежде чем начать

Coupang Open API можно выпустить только из аккаунта продавца.

Требуется

Аккаунт продавца WING

Вам нужен аккаунт продавца Coupang WING с завершённой бизнес-верификацией. Выпуск невозможен с обычным аккаунтом участника-покупателя.

Выпускаемые ключи

Access Key · Secret Key

На экране завершения выпуска найдите Access Key и Secret Key, а также проверьте vendor ID.

Примечание: При выпуске необходимо указать название вашей компании, URL главной страницы/сервиса (используйте wing.coupang.com, если его нет) и IP-адреса доступа (до 10).

2. Шаги выпуска Open API Key

Выпустите в указанном ниже порядке и зарегистрируйте в GenieGo.

1

Войдите в WING

Войдите под своим аккаунтом продавца.

wing.coupang.com
2

Перейдите в меню Open API

Перейдите в меню [Open API] или [Управление API Key] в верхней области/настройках.

Open API → Управление API Key
3

Выпустите Open API Key

Нажмите [Выпустить Open API Key] и укажите название компании, URL главной страницы и IP-адреса доступа (до 10).

Выпустить Open API Key
4

Проверьте ключи

На экране завершения выпуска найдите Access Key·Secret Key и vendor ID.

Как только Access Key·Secret Key появятся на экране завершения выпуска, они выпущены правильно.

3. Сведения для регистрации в GenieGo

ПолеОписание
access_keyВыпущенный Access Key
secret_keyВыпущенный Secret Key (обращайтесь осторожно)
vendor_idVendor ID
Введите их в GenieGo Integration Hub [Регистрация] и сохраните. Учётные данные шифруются при хранении с помощью AES-256-GCM и маскируются при чтении. Синхронизация заказов начинается сразу после регистрации.

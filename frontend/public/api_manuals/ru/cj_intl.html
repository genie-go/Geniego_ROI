Руководство, описывающее процесс заключения договора CJ Logistics International Express и регистрации выданного ключа API и кода клиента (договора) в GenieGo Integration Hub. Этот канал получает учётные данные через договор с отделом продаж, а не путём самостоятельного выпуска через консоль.

1. Перед началом

CJ Logistics International Express — это канал на основе договора, в котором код клиента (договора) выдаётся после заключения договора с отделом продаж.

Требуется

Договор International Express

Чтобы использовать CJ Logistics International Express, сначала необходимо заключить договор с представителем отдела продаж. Без договора код клиента (договора) не может быть выдан.

Выданные ключи

Ключ API · Код клиента (договора)

После завершения договора вы получаете код клиента (договора) и ключ API.

Примечание: Этот канал не выпускается напрямую через консоль самообслуживания; вместо этого учётные данные получаются через договор с отделом продаж.

2. Процедура подачи заявки на выпуск

Подайте заявку и получите в указанном ниже порядке, затем зарегистрируйте в GenieGo.

1

Заключение договора с представителем отдела продаж

Заключите договор с представителем отдела продаж CJ Logistics International Express.

2

Заключение договора

Заключите договор на использование International Express.

3

Получение учётных данных

После завершения договора вы получаете код клиента (договора) и ключ API.

4

Регистрация в GenieGo

Введите выданные api_key и cust_code в GenieGo [Регистрация] и сохраните.

Если вы получили от представителя отдела продаж как код клиента (договора), так и ключ API, значит они выданы корректно.

3. Информация для регистрации в GenieGo

ПолеОписание
api_keyВыданный ключ API International Express
cust_codeВыданный код клиента (договора)
При вводе и сохранении в GenieGo Integration Hub [Регистрация] учётные данные шифруются при хранении с помощью AES-256-GCM и маскируются при чтении.

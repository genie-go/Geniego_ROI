CJ Logistics
Руководство по выпуску API-ключа

Выделенный API CJ Logistics выпускается на основе кода контракта (клиента). Это руководство проведёт вас через получение кода контракта (клиента) и ключа API у вашего торгового представителя и их регистрацию в GenieGo Integration Hub.

Требуемый аккаунтКонтракт CJ Logistics (корпоративный клиент)
Выпускаемые ключиКлюч API · Код клиента (контракта)
РезультатУчётные данные API

1. Прежде чем начать

Выделенный API CJ Logistics — это канал на основе контракта, и он не выпускается мгновенно из консоли самообслуживания.

Обязательно

Код контракта (клиента) CJ Logistics

У вас должен быть код контракта (клиента), выпущенный по контракту с CJ Logistics. Он не выпускается напрямую из отдельной консоли.

Рекомендуется

Smart Parcel (единое отслеживание)

Если вам нужно только отслеживание доставки, вы можете подключить всех курьеров с помощью одного t_key через канал [Smart Parcel (единое отслеживание)].

Примечание: Если ваша цель — отслеживание доставки, рекомендуем подключаться через единый ключ Smart Parcel (SweetTracker) без выделенного контракта на API.

2. Процедура подачи заявки на выпуск

Подайте заявку и зарегистрируйте в указанном ниже порядке.

1

Подготовьте код контракта (клиента)

Подготовьте ваш код контракта (клиента) CJ Logistics. Выделенный API предоставляется на контрактной основе.

2

Решите, нужно ли вам единое отслеживание

Если вам нужно только отслеживание доставки, вы можете подключить всех курьеров с помощью одного t_key через канал [Smart Parcel (единое отслеживание)] вместо выделенного канала CJ Logistics.

3

Подайте заявку на выделенный API

Если вам нужен выделенный API CJ Logistics, получите код контракта (клиента) и ключ API у вашего торгового представителя.

4

Зарегистрируйте в GenieGo

Введите api_key и cust_code в [Регистрация] GenieGo Integration Hub и сохраните.

Как только вы получите ключ API и код контракта (клиента) от вашего торгового представителя, вы готовы к регистрации.

3. Информация для регистрации в GenieGo

ПолеОписание
api_keyВыпущенный выделенный ключ API CJ Logistics
cust_codeКод контракта (клиента) CJ Logistics
Когда вы введёте и сохраните его в [Регистрация] GenieGo Integration Hub, учётные данные шифруются при хранении с помощью AES-256-GCM и маскируются при чтении.

Руководство по выпуску API Key

Понятное пошаговое руководство для начинающих по выпуску учётных данных RMS Web Service (Service Secret и License Key) из консоли продавца Rakuten RMS и их регистрации в GenieGo Integration Hub.

Примерное времяОколо 5–15 мин.
Требуемый аккаунтАккаунт продавца Rakuten (Ichiba)
Выпускаемые ключиService Secret · License Key
РезультатУчётные данные RMS Web Service

1. Прежде чем начать

Ключи продавца (Ichiba) выпускаются в RMS Web Service.

Требуется

Аккаунт продавца Rakuten

Вам нужен аккаунт продавца Rakuten (Ichiba). Вам нужен аккаунт, который может войти через RMS R-Login (glogin.rms.rakuten.co.jp).

Выпускаемые ключи

Service Secret · License Key

Вы получите serviceSecret и licenseKey в RMS Web Service. License Key истекает через 90 дней и может быть выпущен повторно.

Внимание: RMS Web Service (glogin.rms.rakuten.co.jp), который выпускает ключи продавца (Ichiba), отличается от webservice.rakuten.co.jp, предназначенного для партнёрской программы/поиска товаров. Обязательно выпускайте свои ключи на стороне RMS.

2. Шаги выпуска

Выпускайте в указанном ниже порядке и регистрируйте в GenieGo.

1

Войдите в портал разработчика RMS

Войдите в портал разработчика Rakuten (RMS/Webservice) с помощью своего аккаунта продавца.

https://glogin.rms.rakuten.co.jp
2

Перейдите в RMS Web Service

После входа через RMS R-Login перейдите в раздел RMS Web Service.

RMS R-Login → RMS Web Service
3

Выпустите Service Secret · License Key

В RMS Web Service выпустите Service Secret и License Key. License Key истекает через 90 дней, поэтому выпускайте его повторно по истечении срока.

4

Проверьте ключи

Проверьте выпущенные Service Secret и License Key, а также Shop URL вашего магазина.

Как только Service Secret и License Key появятся на экране выпуска, они выпущены правильно.

3. Информация для регистрации в GenieGo

ПолеОписание
service_secretВыпущенный Service Secret
license_keyВыпущенный License Key (истекает через 90 дней·можно выпустить повторно, обращайтесь осторожно)
shop_urlShop URL вашего магазина
В GenieGo Integration Hub [Регистрация] введите service_secret·license_key·shop_url и сохраните. Учётные данные шифруются при хранении с помощью AES-256-GCM и маскируются при чтении.

Kakao Moment
Руководство по выпуску API-ключа

Руководство по запросу разрешения Kakao Moment Open API, получению токена доступа и идентификатора рекламного аккаунта, а также их регистрации в центре интеграции GenieGo. Разрешение Open API предоставляется только официальным рекламным агентствам Kakao.

1. Прежде чем начать

Kakao Moment Open API не выпускается самостоятельно; он проходит процесс подачи заявки на разрешение и её рассмотрения.

Важно: Разрешение Kakao Moment Open API предоставляется только официальным рекламным агентствам Kakao. Обычные рекламодатели не могут выпустить его самостоятельно.
Требуется

Рекламный аккаунт + Business Channel

Вам нужны рекламный аккаунт Kakao Moment и Kakao Business Channel.

Выпускаемые ключи

Токен доступа · Идентификатор рекламного аккаунта

После одобрения разрешения вы получаете токен доступа через авторизацию OAuth и подтверждаете идентификатор своего рекламного аккаунта.

2. Процедура подачи заявки на выпуск

Действуйте в следующем порядке: подача заявки на разрешение Open API → рассмотрение/одобрение → выпуск токена → регистрация.

1

Создайте приложение и переключите его на Business-приложение

Создайте приложение на developers.kakao.com и переключите его на [Business App].

developers.kakao.com
2

Подайте заявку на разрешение Open API

Отправьте Kakao Moment Open API [Additional Feature Application]. Разрешение предоставляется после проверки вашего соответствия требованиям официального агентства.

Additional Feature Application
3

Выпустите токен

После одобрения разрешения получите токен доступа через авторизацию OAuth и подтвердите идентификатор своего рекламного аккаунта. Вы можете использовать OAuth в один клик в центре интеграции GenieGo.

Как только разрешение будет одобрено, а ваш токен доступа и идентификатор рекламного аккаунта подтверждены, вы можете перейти к регистрации.

3. Информация для регистрации в GenieGo

ПолеОписание
access_tokenТокен доступа, выпущенный через авторизацию OAuth (обращайтесь с осторожностью)
ad_account_idИдентификатор рекламного аккаунта Kakao Moment
Введите и сохраните их в центре интеграции GenieGo [Register]; учётные данные шифруются при хранении с помощью AES-256-GCM и маскируются при чтении.

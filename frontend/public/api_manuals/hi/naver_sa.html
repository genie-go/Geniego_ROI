1. शुरू करने से पहले

Naver Search Ad API केवल विज्ञापनदाता खाते से ही अनुरोध की जा सकती है।

आवश्यक

Search Ad विज्ञापनदाता खाता

आपको एक विज्ञापनदाता खाता चाहिए जो Naver Search Ad प्रबंधन टूल का उपयोग करता हो। आप सामान्य Naver सदस्य खाते से API एक्सेस के लिए आवेदन नहीं कर सकते।

जारी की गई कुंजियाँ

Access license · Secret key · CUSTOMER_ID

API एक्सेस के लिए आवेदन करने के बाद एक साथ जारी की गई CUSTOMER_ID, access license और secret key की पुष्टि करें।

नोट: Secret key केवल जारी किए जाने के समय ही देखी जा सकती है, इसलिए जारी होने के तुरंत बाद इसे सुरक्षित रूप से संग्रहित करें।

2. API Key जारी करने के चरण

नीचे दिए गए क्रम में जारी करें और GenieGo में पंजीकृत करें।

1

Search Ads में साइन इन करें

अपने विज्ञापनदाता खाते से Naver Search Ad प्रबंधन टूल में साइन इन करें।

searchad.naver.com
2

API उपयोग प्रबंधन पर जाएँ

शीर्ष [टूल] मेनू से [API उपयोग प्रबंधन] पर जाएँ।

टूल → API उपयोग प्रबंधन
3

API सेवा के लिए आवेदन करें

आवेदन करने के लिए [Naver Search Ad API सेवा के लिए आवेदन करें] बटन पर क्लिक करें।

Naver Search Ad API सेवा के लिए आवेदन करें
4

जारी की गई जानकारी जाँचें

जारी की गई CUSTOMER_ID, access license और secret key की पुष्टि करें।

जब CUSTOMER_ID, access license और secret key सभी की पुष्टि हो जाती है, तो वे सही ढंग से जारी कर दिए गए हैं।

3. GenieGo में पंजीकृत करने की जानकारी

फ़ील्डविवरण
api_keyजारी की गई access license
api_secretजारी की गई secret key (सावधानी से संभालें)
customer_idजारी की गई CUSTOMER_ID
इन्हें GenieGo Integration Hub के [पंजीकरण] में दर्ज करके सहेजें, और क्रेडेंशियल AES-256-GCM के साथ संग्रहण के समय एन्क्रिप्ट किए जाते हैं और पढ़ते समय मास्क किए जाते हैं।

4. सामान्य समस्याएँ

समस्याकारणसमाधान
[API उपयोग प्रबंधन] मेनू नहीं हैविज्ञापनदाता खाता नहीं है, या अनुमतियाँ अपर्याप्त हैंSearch Ad प्रबंधन टूल का उपयोग करने वाले विज्ञापनदाता खाते से साइन इन करें।
प्रमाणीकरण त्रुटि होती हैAccess license या secret key गलत दर्ज की गईजारी करने वाली स्क्रीन से access license, secret key और CUSTOMER_ID को फिर से जाँचें।

अंतिम चेकलिस्ट

  • Naver Search Ad विज्ञापनदाता खाते से साइन इन किया।
  • [टूल > API उपयोग प्रबंधन] के अंतर्गत API सेवा आवेदन पूरा किया।
  • CUSTOMER_ID, access license और secret key की पुष्टि की।
  • GenieGo [पंजीकरण] में api_key·api_secret·customer_id दर्ज करके सहेजा।

1. शुरू करने से पहले

CJ Logistics की समर्पित API एक अनुबंध-आधारित चैनल है और इसे स्वयं-सेवा कंसोल से तुरंत जारी नहीं किया जाता।

आवश्यक

CJ Logistics अनुबंध (ग्राहक) कोड

आपके पास CJ Logistics के साथ अनुबंध के माध्यम से जारी किया गया अनुबंध (ग्राहक) कोड होना चाहिए। यह किसी अलग कंसोल से सीधे जारी नहीं किया जाता।

अनुशंसित

Smart Parcel (एकीकृत ट्रैकिंग)

यदि आपको केवल डिलीवरी ट्रैकिंग चाहिए, तो आप [Smart Parcel (एकीकृत ट्रैकिंग)] चैनल के माध्यम से एक ही t_key से सभी कूरियरों को जोड़ सकते हैं।

नोट: यदि आपका उद्देश्य डिलीवरी ट्रैकिंग है, तो हम समर्पित API अनुबंध के बिना Smart Parcel (SweetTracker) एकीकृत कुंजी के माध्यम से जोड़ने की अनुशंसा करते हैं।

2. जारी करने की आवेदन प्रक्रिया

नीचे दिए गए क्रम में आवेदन करें और पंजीकृत करें।

1

अनुबंध (ग्राहक) कोड तैयार करें

अपना CJ Logistics अनुबंध (ग्राहक) कोड तैयार करें। समर्पित API अनुबंध के आधार पर प्रदान की जाती है।

2

तय करें कि क्या आपको एकीकृत ट्रैकिंग चाहिए

यदि आपको केवल डिलीवरी ट्रैकिंग चाहिए, तो आप CJ Logistics के समर्पित चैनल के बजाय [Smart Parcel (एकीकृत ट्रैकिंग)] चैनल के माध्यम से एक ही t_key से सभी कूरियरों को जोड़ सकते हैं।

3

समर्पित API के लिए आवेदन करें

यदि आपको CJ Logistics की समर्पित API चाहिए, तो अपने बिक्री प्रतिनिधि से अनुबंध (ग्राहक) कोड और API कुंजी प्राप्त करें।

4

GenieGo में पंजीकृत करें

GenieGo Integration Hub के [पंजीकृत करें] में api_key और cust_code दर्ज करें और सहेजें।

एक बार जब आप अपने बिक्री प्रतिनिधि से API कुंजी और अनुबंध (ग्राहक) कोड प्राप्त कर लेते हैं, तो आप पंजीकरण के लिए तैयार हैं।

API Key जारी करने की मैनुअल

Braintree (PayPal) Control Panel में अपने Production API क्रेडेंशियल (Merchant ID·Public Key·Private Key) खोजने और उन्हें GenieGo Integration Hub में पंजीकृत करने के लिए शुरुआती-अनुकूल चरण-दर-चरण मार्गदर्शिका।

1. शुरू करने से पहले

Braintree (PayPal) क्रेडेंशियल आपके मर्चेंट खाते के अनुमोदित होने के बाद ही उपलब्ध होते हैं।

आवश्यक

Braintree (PayPal) मर्चेंट खाता

सबसे पहले आपको एक अनुमोदित Braintree (PayPal) मर्चेंट खाता चाहिए। Production क्रेडेंशियल केवल तभी उपलब्ध होते हैं जब आपका मर्चेंट खाता अनुमोदित हो जाता है।

जारी की गई कुंजियाँ

Merchant ID · Public/Private Key

Control Panel में, Production वातावरण का Merchant ID·Public Key·Private Key खोजने के लिए Settings > API Keys पर जाएँ।

नोट: सुनिश्चित करें कि आप Production वातावरण के क्रेडेंशियल का उपयोग करें। उन्हें Sandbox कुंजियों के साथ भ्रमित न करने का ध्यान रखें।

2. जारी करने की प्रक्रिया

आपके मर्चेंट खाते के अनुमोदन के बाद, नीचे दिए गए क्रम में क्रेडेंशियल खोजें और उन्हें GenieGo में पंजीकृत करें।

1

मर्चेंट खाते का अनुमोदन

पहले Braintree (PayPal) मर्चेंट खाते का अनुमोदन पूरा करें। अनुमोदन पूरा होने के बाद ही आप अगले चरण पर आगे बढ़ सकते हैं।

2

Control Panel में साइन इन करें

Braintree Control Panel में साइन इन करें।

braintreegateway.com
3

API Keys पर जाएँ

[Settings] > [API Keys] मेनू पर नेविगेट करें।

Settings → API Keys
4

क्रेडेंशियल खोजें

Production वातावरण का Merchant ID·Public Key·Private Key खोजें।

यदि Settings > API Keys में Production का Merchant ID·Public Key·Private Key दिखाई देता है, तो सब कुछ ठीक है।

3. GenieGo में पंजीकृत करने योग्य जानकारी

फ़ील्डविवरण
merchant_idProduction Merchant ID
public_keyProduction Public Key
private_keyProduction Private Key (सावधानी से संभालें)
GenieGo Integration Hub के [पंजीकरण] में merchant_id·public_key·private_key दर्ज करें और सहेजें। क्रेडेंशियल को संग्रहण के समय AES-256-GCM से एन्क्रिप्ट किया जाता है और पढ़ते समय मास्क किया जाता है।

4. सामान्य समस्याएँ

समस्याकारणसमाधान
API Keys मेनू में कोई क्रेडेंशियल नहींमर्चेंट खाते का अनुमोदन पूरा नहीं हुआपहले Braintree (PayPal) मर्चेंट खाते का अनुमोदन पूरा करें।
प्रमाणीकरण विफलSandbox कुंजियों का उपयोग, या Private Key गलत दर्ज की गईProduction वातावरण का Merchant ID·Public Key·Private Key फिर से जाँचें।

अंतिम चेकलिस्ट

  • Braintree (PayPal) मर्चेंट खाता अनुमोदित हो गया है।
  • Control Panel के Settings > API Keys में Production क्रेडेंशियल खोज लिए।
  • Merchant ID·Public Key·Private Key की पुष्टि कर ली।
  • GenieGo [पंजीकरण] में merchant_id·public_key·private_key दर्ज करके सहेज लिया।

API Key जारी करने की नियमावली

Razorpay Dashboard में Live मोड की API कुंजियाँ (Key ID·Key Secret) बनाने और सत्यापित करने तथा उन्हें GenieGo Integration Hub में पंजीकृत करने की चरण-दर-चरण, शुरुआती-अनुकूल नियमावली।

अनुमानित समयलगभग 5–10 मिनट
आवश्यक खाताRazorpay खाता (भारत-केंद्रित)
जारी कुंजियाँKey ID (rzp_live_) · Key Secret
परिणामLive API क्रेडेंशियल

1. शुरू करने से पहले

Razorpay API कुंजियाँ Dashboard के Live मोड में बनाई जाती हैं।

आवश्यक

Razorpay खाता

आपको एक Razorpay खाता चाहिए (भारत-केंद्रित)। आपको उससे Dashboard में लॉगिन करने में सक्षम होना चाहिए।

जारी कुंजियाँ

Key ID · Key Secret

Dashboard में Settings > API Keys के अंतर्गत Live मोड की Key ID और Key Secret बनाएँ और सत्यापित करें।

नोट: कुंजियाँ अवश्य Live मोड में ही बनाएँ। Key ID rzp_live_ से शुरू होती है।

2. जारी करने के चरण

नीचे दिए गए क्रम में जारी करें और GenieGo में पंजीकृत करें।

1

Dashboard खोलें

Razorpay Dashboard में लॉगिन करें।

dashboard.razorpay.com
2

API Keys पर जाएँ

[Settings] > [API Keys] पर जाएँ।

Settings → API Keys
3

कुंजियाँ बनाएँ और सत्यापित करें

Live मोड में Key ID और Key Secret बनाएँ और सत्यापित करें।

यदि आपको Settings > API Keys के अंतर्गत Live मोड की Key ID (rzp_live_) और Key Secret दिखाई देती हैं, तो वे सही ढंग से बन गई हैं।

3. GenieGo में पंजीकृत की जाने वाली जानकारी

फ़ील्डविवरण
key_idLive मोड की Key ID (rzp_live_ से शुरू)
key_secretLive मोड की Key Secret (सावधानी से संभालें)
GenieGo Integration Hub [पंजीकरण] में अपनी key_id·key_secret दर्ज करें और सहेजें। क्रेडेंशियल विश्राम में AES-256-GCM से एन्क्रिप्ट किए जाते हैं और पढ़ते समय मास्क किए जाते हैं।

4. आम समस्याएँ

समस्याकारणसमाधान
प्रमाणीकरण विफलTest मोड की कुंजी का उपयोगजाँचें कि आप Live मोड में बनाई गई Key ID (rzp_live_)·Key Secret का उपयोग कर रहे हैं।
Key Secret को दोबारा नहीं देखा जा सकताKey Secret केवल बनाते समय एक बार दिखाई जाती हैबनाते समय Key Secret को सुरक्षित रखें; खो जाने पर नई बनाएँ।

अंतिम जाँच-सूची

  • Razorpay खाते से dashboard.razorpay.com में लॉगिन किया।
  • Settings > API Keys में Live मोड की कुंजियाँ बनाईं।
  • Key ID (rzp_live_)·Key Secret सत्यापित की।
  • GenieGo [पंजीकरण] में key_id·key_secret दर्ज करके सहेजा।

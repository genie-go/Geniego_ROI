जारी करने के चरण

अपने क्रेडेंशियल जारी और पंजीकृत करने के लिए नीचे दिए चरणों का पालन करें।

1

आवश्यकताएँ: आपको एक LINE Developers खाता + LINE आधिकारिक खाता (Messaging API चैनल) चाहिए।

2

developers.line.biz कंसोल में एक प्रोवाइडर और Messaging API चैनल बनाएँ।

3

[Basic settings] में Channel Secret, और [Messaging API] में Channel Access Token जारी करें।

4

GenieGo [पंजीकरण] में channel_secret, channel_access_token दर्ज करके सहेजें।

अंतिम चेकलिस्ट

  • आवश्यकताएँ: आपको एक LINE Developers खाता + LINE आधिकारिक खाता (Messaging API चैनल) चाहिए।
  • developers.line.biz कंसोल में एक प्रोवाइडर और Messaging API चैनल बनाएँ।
  • [Basic settings] में Channel Secret, और [Messaging API] में Channel Access Token जारी करें।
  • GenieGo [पंजीकरण] में channel_secret, channel_access_token दर्ज करके सहेजें।
GenieGo Integration Hub [पंजीकरण] में दर्ज कुंजियाँ AES-256-GCM से एन्क्रिप्ट करके संग्रहित की जाती हैं और देखने पर मास्क की जाती हैं।
अपनी कुंजियों/सीक्रेट को सुरक्षित रखें और कभी सार्वजनिक न करें। लीक का संदेह हो तो तुरंत पुनः जारी करें।

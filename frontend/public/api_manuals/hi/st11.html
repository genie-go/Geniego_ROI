API Key जारी करने की नियमावली

11Street Open API Center (openapi.11st.co.kr) से Open API कुंजी जारी करने और उसे GenieGo Integration Hub में पंजीकृत करने के लिए शुरुआती-अनुकूल चरण-दर-चरण मार्गदर्शिका।

1. शुरू करने से पहले

11Street Open API कुंजियाँ केवल विक्रेता खाते से ही जारी की जा सकती हैं।

आवश्यक

11Street विक्रेता खाता

आपको एक 11Street विक्रेता खाते की आवश्यकता है। अपनी Seller ID से Open API Center में साइन इन करें।

जारी कुंजियाँ

Open API कुंजी · Seller ID

सेवा पंजीकरण और एक्सेस IP पंजीकरण पूरा करने के बाद, जारी की गई Open API कुंजी को अपनी Seller ID के साथ जाँचें।

ध्यान दें: कुंजी का उपयोग करने से पहले आपको एक ऐसा सर्वर IP पंजीकृत करना होगा जिसे API तक पहुँचने की अनुमति हो।

2. जारी करने के चरण

नीचे दिए गए क्रम में जारी करें और GenieGo में पंजीकृत करें।

1

Open API Center में साइन इन करें

अपनी Seller ID से 11Street Open API Center में साइन इन करें।

https://openapi.11st.co.kr/openapi/OpenApiFrontMain.tmall
2

सेवा पंजीकृत करें

मेनू से [सेवा पंजीकरण·सत्यापन] पर क्लिक करें, विक्रेता विवरण दर्ज करें, शर्तों से सहमत हों और पंजीकृत करें।

सेवा पंजीकरण·सत्यापन
3

एक्सेस IP पंजीकृत करें

[Seller API जानकारी] > [संपादित करें] में, API तक पहुँचने के लिए अनुमत सर्वर IP पंजीकृत करें।

Seller API जानकारी → संपादित करें
4

API Key कॉपी करें

जारी की गई API Key कॉपी करें।

सेवा पंजीकरण और एक्सेस IP पंजीकरण पूरा होते ही API Key जारी हो जाती है।

3. GenieGo में पंजीकृत करने योग्य जानकारी

फ़ील्डविवरण
api_keyजारी की गई Open API कुंजी
seller_idSeller ID
इसे GenieGo Integration Hub [पंजीकरण] में दर्ज करके सहेजें। क्रेडेंशियल को संग्रहण के समय AES-256-GCM से एन्क्रिप्ट किया जाता है और पढ़ते समय मास्क किया जाता है।

4. सामान्य समस्याएँ

समस्याकारणसमाधान
API Key जारी नहीं होतीसेवा पंजीकरण या एक्सेस IP पंजीकरण पूरा नहीं हुआ[सेवा पंजीकरण·सत्यापन] और [Seller API जानकारी] में एक्सेस IP पंजीकरण पूरा करें।
कॉल करते समय प्रमाणीकरण त्रुटिअनुमत सर्वर IP पंजीकृत नहीं[Seller API जानकारी] > [संपादित करें] में अनुमत सर्वर IP को फिर से जाँचें।

अंतिम जाँच सूची

  • 11Street विक्रेता खाते से Open API Center में साइन इन किया।
  • [सेवा पंजीकरण·सत्यापन] में विक्रेता विवरण दर्ज किए और शर्तों से सहमत हुए।
  • [Seller API जानकारी] में अनुमत सर्वर IP पंजीकृत किया।
  • जारी की गई Open API कुंजी और Seller ID की पुष्टि की।
  • GenieGo [पंजीकरण] में api_key·seller_id दर्ज करके सहेजा।

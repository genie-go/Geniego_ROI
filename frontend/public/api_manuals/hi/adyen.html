1. शुरू करने से पहले

Adyen एक एंटरप्राइज़ ग्लोबल भुगतान गेटवे है, इसलिए क्रेडेंशियल जारी करने से पहले मर्चेंट अनुबंध और समीक्षा स्वीकृति पहले से होनी चाहिए।

आवश्यक

मर्चेंट अनुबंध · समीक्षा स्वीकृति

Adyen एक एंटरप्राइज़ ग्लोबल PG है जिसके लिए मर्चेंट अनुबंध और समीक्षा स्वीकृति आवश्यक है। स्वीकृति के बाद ही आप क्रेडेंशियल जारी कर सकते हैं।

जारी की गई कुंजियाँ

X-API-Key · Merchant Account

स्वीकृति के बाद, Customer Area के [Developers] > [API credentials] में API कुंजी (X-API-Key) बनाएँ और अपने Merchant Account का नाम सत्यापित करें।

पूर्व-शर्त: मर्चेंट अनुबंध और समीक्षा स्वीकृति से पहले स्वयं जारी करना संभव नहीं है। आपको पहले Adyen के साथ मर्चेंट अनुबंध और समीक्षा पूरी करनी होगी।

2. जारी करने की आवेदन प्रक्रिया

समीक्षा स्वीकृति के बाद, नीचे दिए गए क्रम में क्रेडेंशियल जारी करें और पंजीकृत करें।

1

मर्चेंट अनुबंध · समीक्षा स्वीकृति

Adyen के साथ मर्चेंट अनुबंध करें और समीक्षा स्वीकृति प्राप्त करें।

Adyen सेल्स/अनुबंध
2

Customer Area में साइन इन करें

स्वीकृति के बाद Customer Area में साइन इन करें।

https://ca-live.adyen.com
3

API क्रेडेंशियल जारी करें

[Developers] > [API credentials] में API कुंजी (X-API-Key) बनाएँ और अपने Merchant Account का नाम सत्यापित करें।

Developers → API credentials
4

प्रारंभिक सेटलमेंट बैच संख्या सत्यापित करें

पहले सेटलमेंट संग्रह के लिए, Customer Area में सबसे हाल का settlement batch नंबर खोजें।

जब API कुंजी (X-API-Key) और Merchant Account सत्यापित हो जाते हैं, तो आपके क्रेडेंशियल तैयार हो जाते हैं।

3. GenieGo में पंजीकृत करने योग्य जानकारी

फ़ील्डविवरण
api_keyजारी की गई API कुंजी (X-API-Key, सावधानी से संभालें)
merchant_accountआपका सत्यापित Merchant Account नाम
batch_startपहले सेटलमेंट संग्रह के लिए प्रारंभिक सेटलमेंट बैच संख्या (Customer Area में सबसे हाल का settlement batch नंबर)
GenieGo Integration Hub के [पंजीकरण] में api_key·merchant_account·batch_start दर्ज करें और सहेजें। क्रेडेंशियल भंडारण के समय AES-256-GCM से एन्क्रिप्ट किए जाते हैं और पढ़ते समय मास्क किए जाते हैं।

4. आम समस्याएँ

समस्याकारणसमाधान
API credentials मेनू नहीं हैमर्चेंट अनुबंध · समीक्षा स्वीकृत नहींAdyen के साथ मर्चेंट अनुबंध · समीक्षा पूरी करें, फिर Customer Area में जाएँ।
प्रमाणीकरण विफलX-API-Key या Merchant Account गलत दर्ज किया गयाDevelopers > API credentials में जारी की गई API कुंजी और Merchant Account नाम फिर से जाँचें।

अंतिम चेकलिस्ट

  • Developers > API credentials में API कुंजी (X-API-Key) और Merchant Account सत्यापित किया।
  • सबसे हाल का settlement batch नंबर (batch_start) सत्यापित किया।
  • GenieGo [पंजीकरण] में api_key·merchant_account·batch_start दर्ज किया और सहेजा।

API Key जारी करने की नियमावली

Coupang WING विक्रेता कंसोल में Open API क्रेडेंशियल (Access Key·Secret Key) जारी करने और उन्हें GenieGo Integration Hub में पंजीकृत करने के लिए शुरुआती-अनुकूल, चरण-दर-चरण मार्गदर्शिका।

1. शुरू करने से पहले

Coupang Open API केवल विक्रेता खाते से ही जारी की जा सकती है।

आवश्यक

WING विक्रेता खाता

आपको व्यवसाय सत्यापन पूर्ण किए हुए एक Coupang WING विक्रेता खाते की आवश्यकता है। सामान्य खरीदार सदस्य खाते से इसे जारी नहीं किया जा सकता।

जारी की गई कुंजियाँ

Access Key · Secret Key

जारी करने की समाप्ति स्क्रीन पर Access Key और Secret Key खोजें, और vendor ID भी जाँचें।

नोट: जारी करते समय आपको अपनी कंपनी का नाम, होमपेज/सेवा URL (न हो तो wing.coupang.com) और एक्सेस IP (अधिकतम 10) दर्ज करना होगा।

2. Open API Key जारी करने के चरण

नीचे दिए गए क्रम में जारी करें और GenieGo में पंजीकृत करें।

1

WING में साइन इन करें

अपने विक्रेता खाते से साइन इन करें।

wing.coupang.com
2

Open API मेनू पर जाएँ

शीर्ष/सेटिंग्स क्षेत्र में [Open API] या [API Key प्रबंधन] मेनू पर जाएँ।

Open API → API Key प्रबंधन
3

Open API Key जारी करें

[Open API Key जारी करें] पर क्लिक करें और अपनी कंपनी का नाम, होमपेज URL और एक्सेस IP (अधिकतम 10) दर्ज करें।

Open API Key जारी करें
4

कुंजियाँ जाँचें

जारी करने की समाप्ति स्क्रीन पर Access Key·Secret Key और vendor ID खोजें।

जारी करने की समाप्ति स्क्रीन पर Access Key·Secret Key दिखाई देने के बाद, वे सही ढंग से जारी हो गए हैं।

3. GenieGo में पंजीकृत करने की जानकारी

फ़ील्डविवरण
access_keyजारी की गई Access Key
secret_keyजारी की गई Secret Key (सावधानी से संभालें)
vendor_idVendor ID
इन्हें GenieGo Integration Hub [पंजीकृत करें] में दर्ज करें और सहेजें। क्रेडेंशियल AES-256-GCM के साथ विश्राम अवस्था में एन्क्रिप्ट किए जाते हैं और पढ़ते समय मास्क किए जाते हैं। पंजीकरण के तुरंत बाद ऑर्डर सिंक्रनाइज़ेशन शुरू हो जाता है।

4. सामान्य समस्याएँ

समस्याकारणसमाधान
जारी करने का मेनू अनुपस्थित हैसामान्य सदस्य खाता, या व्यवसाय सत्यापन पूर्ण नहींऐसे WING विक्रेता खाते से साइन इन करें जिसका व्यवसाय सत्यापन पूर्ण हो।
कॉल पर 401 त्रुटिएक्सेस IP पंजीकृत नहीं, या Secret Key गलत दर्जजारी करते समय दर्ज किए गए एक्सेस IP और Secret Key को फिर से जाँचें।

अंतिम जाँच-सूची

  • व्यवसाय सत्यापन पूर्ण किए हुए WING विक्रेता खाते से साइन इन किया।
  • Open API Key जारी की और एक्सेस IP पंजीकृत किए।
  • Access Key·Secret Key·Vendor ID की जाँच की।
  • GenieGo [पंजीकृत करें] में access_key·secret_key·vendor_id दर्ज कर सहेजा।

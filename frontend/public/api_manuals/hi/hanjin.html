1. शुरू करने से पहले

Hanjin Logistics का समर्पित API एक अनुबंध-आधारित चैनल है जो सेल्फ-सर्विस कंसोल से तुरंत जारी नहीं किया जाता।

आवश्यक

Hanjin Logistics अनुबंध (ग्राहक) कोड

आपको Hanjin Logistics के साथ अनुबंध के माध्यम से जारी किया गया अनुबंध (ग्राहक) कोड चाहिए। यह कुछ ऐसा नहीं है जिसे आप किसी अलग कंसोल से सीधे जारी करते हैं।

अनुशंसित

Smart Delivery (एकीकृत ट्रैकिंग)

यदि आपको केवल डिलीवरी ट्रैकिंग चाहिए, तो आप [Smart Delivery (एकीकृत ट्रैकिंग)] चैनल पर एक ही t_key से सभी कूरियर को जोड़ सकते हैं।

नोट: यदि आपका लक्ष्य डिलीवरी ट्रैकिंग है, तो हम समर्पित API अनुबंध के बिना Smart Delivery एकीकृत कुंजी के माध्यम से जुड़ने की सलाह देते हैं।

2. जारी करने हेतु आवेदन प्रक्रिया

नीचे दिए गए क्रम में आवेदन करें और पंजीकृत करें।

1

अनुबंध (ग्राहक) कोड तैयार करें

अपना Hanjin Logistics अनुबंध (ग्राहक) कोड तैयार करें। समर्पित API अनुबंध आधार पर प्रदान किया जाता है।

2

तय करें कि एकीकृत ट्रैकिंग पर्याप्त है या नहीं

यदि आपको केवल डिलीवरी ट्रैकिंग चाहिए, तो आप Hanjin Logistics के समर्पित चैनल के बजाय [Smart Delivery (एकीकृत ट्रैकिंग)] चैनल पर एक ही t_key से सभी कूरियर को जोड़ सकते हैं।

3

समर्पित API के लिए आवेदन करें

यदि आपको समर्पित API चाहिए, तो अपने बिक्री प्रतिनिधि से अनुबंध (ग्राहक) कोड और API कुंजी प्राप्त करें।

4

GenieGo में पंजीकृत करें

GenieGo Integration Hub के [पंजीकरण] में api_key और cust_code दर्ज करें और सहेजें।

जैसे ही आप अपने बिक्री प्रतिनिधि से API कुंजी और अनुबंध (ग्राहक) कोड प्राप्त कर लेते हैं, आप पंजीकरण के लिए तैयार हैं।

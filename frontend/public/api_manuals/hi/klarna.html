1. शुरू करने से पहले

Klarna क्रेडेंशियल केवल आपके मर्चेंट अनुबंध के पूर्ण होने के बाद ही जारी किए जा सकते हैं।

आवश्यक

Klarna मर्चेंट खाता

सबसे पहले आपको Klarna (BNPL) मर्चेंट अनुबंध की आवश्यकता है। अनुबंध पूर्ण होने पर ही आप Merchant Portal में API क्रेडेंशियल जारी कर सकते हैं।

जारी की गई कुंजियाँ

API username · password · region

Merchant Portal में Settings > Klarna API credentials के अंतर्गत API username·password बनाएं और अपना region (eu/na/oc) पुष्टि करें।

नोट: region eu / na / oc में से एक होता है, और पंजीकरण के समय इसे साथ में दर्ज करना आवश्यक है।

2. जारी करने के चरण

अपने मर्चेंट अनुबंध के बाद, नीचे दिए गए क्रम में जारी करें और GenieGo में पंजीकृत करें।

1

मर्चेंट अनुबंध

सबसे पहले अपना Klarna (BNPL) मर्चेंट अनुबंध पूर्ण करें। अनुबंध पूर्ण होने पर ही आप अगले चरण पर जा सकते हैं।

2

Merchant Portal में साइन इन करें

Klarna Merchant Portal में साइन इन करें।

portal.klarna.com
3

API credentials पर जाएँ

[Settings] > [Klarna API credentials] मेनू पर जाएँ।

Settings → Klarna API credentials
4

क्रेडेंशियल बनाएं और पुष्टि करें

API username·password बनाएं और अपना region (eu/na/oc) पुष्टि करें।

यदि Settings > Klarna API credentials में API username·password बन गया है और आपने region की पुष्टि कर ली है, तो सब कुछ ठीक है।

3. GenieGo में पंजीकृत करने योग्य जानकारी

फ़ील्डविवरण
usernameAPI Username (MID/PID)
passwordAPI Password (सावधानी से संभालें)
regionRegion (eu / na / oc)
GenieGo Integration Hub के [पंजीकरण] में username·password·region दर्ज करके सहेजें। क्रेडेंशियल को संग्रहण के समय AES-256-GCM से एन्क्रिप्ट किया जाता है और पढ़ते समय मास्क किया जाता है।

4. सामान्य समस्याएँ

समस्याकारणसमाधान
API credentials मेनू में जारी करने का विकल्प नहीं हैमर्चेंट अनुबंध पूर्ण नहीं हुआसबसे पहले अपना Klarna मर्चेंट अनुबंध पूर्ण करें।
प्रमाणीकरण विफलregion (eu/na/oc) का बेमेल या password गलत दर्जMerchant Portal में पुष्टि किए गए region और username·password की पुनः जाँच करें।

अंतिम जाँच-सूची

  • Klarna (BNPL) मर्चेंट अनुबंध पूर्ण हो गया।
  • Merchant Portal में Settings > Klarna API credentials में क्रेडेंशियल बनाए।
  • API username·password और region (eu/na/oc) की पुष्टि की।
  • GenieGo [पंजीकरण] में username·password·region दर्ज करके सहेजा।

1. आरंभ करने से पहले

Kakao Moment Open API स्वयं जारी नहीं किया जाता; यह अनुमति आवेदन और समीक्षा प्रक्रिया से होकर गुजरता है।

महत्वपूर्ण: Kakao Moment Open API अनुमति केवल आधिकारिक Kakao विज्ञापन एजेंसियों को ही प्रदान की जाती है। सामान्य विज्ञापनदाता इसे स्वयं जारी नहीं कर सकते।

जारी की गई कुंजियाँ

Access token · Ad account ID

अनुमति स्वीकृत होने के बाद, आप OAuth प्राधिकरण के माध्यम से access token प्राप्त करते हैं और अपना ad account ID सत्यापित करते हैं।

token जारी करें

3. GenieGo में पंजीकृत करने योग्य जानकारी

फ़ील्डविवरण
access_tokenOAuth प्राधिकरण के माध्यम से जारी किया गया access token (सावधानी से संभालें)
ad_account_idKakao Moment विज्ञापन खाता ID
उन्हें GenieGo Integration Hub [Register] में दर्ज करें और सहेजें, और क्रेडेंशियल AES-256-GCM के साथ संग्रहण में एन्क्रिप्ट किए जाते हैं तथा पढ़ने पर मास्क किए जाते हैं।

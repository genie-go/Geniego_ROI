अपने WooCommerce एडमिन की REST API सेटिंग्स में Consumer Key·Consumer Secret जारी करने और उन्हें GenieGo Integration Hub में पंजीकृत करने के लिए शुरुआती-अनुकूल, चरण-दर-चरण मार्गदर्शिका।

1. शुरू करने से पहले

REST API कुंजियाँ केवल एडमिन अधिकारों के साथ ही जारी की जा सकती हैं।

आवश्यक

WooCommerce एडमिन अधिकार

आपको WordPress/WooCommerce एडमिन अधिकारों की आवश्यकता है। सामान्य उपयोगकर्ता अधिकारों के साथ REST API कुंजियाँ जारी नहीं की जा सकतीं।

जारी की गई कुंजियाँ

Consumer Key · Consumer Secret

REST API सेटिंग्स में एक कुंजी जोड़ने पर एक Consumer Key और एक Consumer Secret जारी होते हैं।

2. जारी करने के चरण

नीचे दिए गए क्रम में जारी करें और GenieGo में पंजीकृत करें।

1

REST API सेटिंग्स पर जाएँ

WooCommerce में, [सेटिंग्स] > [उन्नत] > [REST API] पर जाएँ।

WooCommerce → सेटिंग्स → उन्नत → REST API
2

कुंजी जोड़ें

[कुंजी जोड़ें] पर क्लिक करें, एक विवरण दर्ज करें और अनुमतियाँ (पढ़ना/लिखना) चुनें, फिर जनरेट करें।

कुंजी जोड़ें
3

कुंजियाँ कॉपी करें

जारी की गई Consumer Key और Consumer Secret को कॉपी करें।

4

जानकारी की पुष्टि करें

कॉपी की गई Consumer Key·Consumer Secret और अपनी store URL की पुष्टि करें।

यदि जारी करने वाली स्क्रीन पर Consumer Key और Consumer Secret दिखाई देते हैं, तो कुंजियाँ सही ढंग से जारी हो गई हैं।
सावधानी: Consumer Secret अक्सर जारी होने के तुरंत बाद ही पूरा दिखाया जाता है, इसलिए उसी समय इसे सुरक्षित रूप से कॉपी कर लें।

3. GenieGo में पंजीकृत करने योग्य जानकारी

फ़ील्डविवरण
site_urlstore URL
consumer_keyजारी की गई Consumer Key
consumer_secretजारी किया गया Consumer Secret (सावधानी से संभालें)
GenieGo Integration Hub [पंजीकृत करें] में site_url·consumer_key·consumer_secret दर्ज करें और सहेजें। क्रेडेंशियल को संग्रहण के समय AES-256-GCM से एन्क्रिप्ट किया जाता है और पढ़ते समय मास्क किया जाता है।

4. सामान्य समस्याएँ

समस्याकारणसमाधान
REST API मेनू नहीं हैअपर्याप्त एडमिन अधिकारएडमिन अधिकार वाले खाते से लॉगिन करें और सेटिंग्स > उन्नत > REST API जाँचें।
लिखने की क्रियाएँ विफलकुंजी बनाते समय केवल-पढ़ने की अनुमति चुनी गई[कुंजी जोड़ें] करते समय पढ़ना/लिखना अनुमतियों के साथ फिर से जारी करें।

अंतिम जाँच सूची

  • WordPress/WooCommerce एडमिन अधिकारों के साथ लॉगिन किया।
  • सेटिंग्स > उन्नत > REST API में एक कुंजी जोड़ी।
  • अनुमतियाँ (पढ़ना/लिखना) चुनीं और Consumer Key·Consumer Secret कॉपी किया।
  • GenieGo [पंजीकृत करें] में site_url·consumer_key·consumer_secret दर्ज किया और सहेजा।

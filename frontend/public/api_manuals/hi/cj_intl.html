1. शुरू करने से पहले

CJ Logistics International Express एक अनुबंध-आधारित चैनल है, जहाँ बिक्री अनुबंध के बाद ग्राहक (अनुबंध) कोड जारी किया जाता है।

आवश्यक

International Express अनुबंध

CJ Logistics International Express का उपयोग करने के लिए पहले बिक्री प्रतिनिधि के साथ अनुबंध करना आवश्यक है। अनुबंध के बिना ग्राहक (अनुबंध) कोड जारी नहीं किया जा सकता।

जारी की गई कुंजियाँ

API कुंजी · ग्राहक (अनुबंध) कोड

अनुबंध पूरा होने के बाद आपको ग्राहक (अनुबंध) कोड और API कुंजी प्राप्त होती है।

नोट: यह चैनल स्व-सेवा कंसोल से सीधे जारी करने का तरीका नहीं है, बल्कि बिक्री अनुबंध के माध्यम से क्रेडेंशियल प्राप्त करने का तरीका है।

नीचे दिए गए क्रम में आवेदन करें और प्राप्त करें, फिर GenieGo में पंजीकृत करें।

3

4

GenieGo में पंजीकृत करें

जारी की गई api_key, cust_code को GenieGo [पंजीकरण] में दर्ज करें और सहेजें।

यदि आपको बिक्री प्रतिनिधि से ग्राहक (अनुबंध) कोड और API कुंजी दोनों प्राप्त हो गए हैं, तो वे सही ढंग से जारी किए गए हैं।

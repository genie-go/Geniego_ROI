1. शुरू करने से पहले

टोकन तभी जारी होता है जब आप Partner Center में एक ऐप बनाते हैं और किसी शॉप को प्राधिकृत करते हैं।

आवश्यक

TikTok Shop विक्रेता खाता

आपके पास TikTok Shop विक्रेता खाता होना चाहिए और partner.tiktokshop.com (Partner Center) पर डेवलपर के रूप में पंजीकरण करना होगा।

जारी की गई कुंजियाँ

App Key · App Secret · Access Token

ऐप बनाने पर आपको App Key·App Secret·Service ID मिलते हैं, और आप शॉप प्राधिकरण के माध्यम से access_token प्राप्त करते हैं।

नोट: प्रोडक्शन में स्विच करते समय ऐप समीक्षा आवश्यक है।

2. जारी करने के चरण

नीचे दिए गए क्रम में जारी करें और GenieGo में पंजीकृत करें।

1

Partner Center में डेवलपर के रूप में पंजीकरण करें

partner.tiktokshop.com (Partner Center) पर डेवलपर के रूप में पंजीकरण करें।

https://partner.tiktokshop.com
2

ऐप बनाएँ

[App & Service] > [Create App] के माध्यम से एक ऐप बनाएँ और App Key, App Secret, Service ID जाँचें।

App & Service → Create App
3

API अनुमतियाँ सक्षम करें

ऐप सेटिंग्स में, आवश्यक API अनुमतियाँ (Shop Authorized Information, Product, Order) सक्षम करें।

4

शॉप प्राधिकरण → access_token जारी करें

[Get shop authorization] में, विक्रेता शॉप को प्राधिकृत करने के लिए क्षेत्र (Region) चुनें, और access_token जारी हो जाता है।

Get shop authorization
5

कुंजियाँ सत्यापित करें

App Key·App Secret और जारी किया गया access_token सत्यापित करें। यदि आवश्यक हो, तो Shop Cipher भी जाँचें।

शॉप प्राधिकरण के बाद access_token जारी हो जाने पर, यह सही ढंग से संसाधित हो गया है।

3. GenieGo में पंजीकृत करने की जानकारी

फ़ील्डविवरण
app_keyजारी किया गया App Key
app_secretजारी किया गया App Secret (सावधानी से संभालें)
access_tokenशॉप प्राधिकरण के माध्यम से जारी किया गया एक्सेस टोकन
shop_cipherShop Cipher (वैकल्पिक)
GenieGo Integration Hub के [पंजीकरण] में app_key·app_secret·access_token दर्ज करें और सहेजें। क्रेडेंशियल को संग्रहण के समय AES-256-GCM से एन्क्रिप्ट किया जाता है और पढ़ते समय मास्क किया जाता है।

4. सामान्य समस्याएँ

समस्याकारणसमाधान
access_token जारी नहीं होताशॉप प्राधिकरण (Get shop authorization) पूरा नहीं हुआ, या क्षेत्र (Region) नहीं चुना गया[Get shop authorization] में, क्षेत्र चुनें और विक्रेता शॉप को प्राधिकृत करें।
अनुमति त्रुटिआवश्यक API अनुमतियाँ सक्षम नहीं हैंऐप सेटिंग्स में, Shop Authorized Information·Product·Order अनुमतियाँ सक्षम करें।
प्रोडक्शन कॉल अस्वीकृतप्रोडक्शन स्विच समीक्षा पास नहीं हुईप्रोडक्शन में स्विच करने के लिए आवश्यक ऐप समीक्षा पूरी करें।

अंतिम जाँच सूची

  • partner.tiktokshop.com पर डेवलपर के रूप में पंजीकरण किया।
  • App & Service > Create App के माध्यम से App Key·App Secret·Service ID जाँचा।
  • आवश्यक API अनुमतियाँ (Shop/Product/Order) सक्षम कीं।
  • शॉप प्राधिकरण (Get shop authorization) के माध्यम से access_token जारी किया।
  • GenieGo [पंजीकरण] में app_key·app_secret·access_token दर्ज किया और सहेजा।

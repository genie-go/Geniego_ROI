1. शुरू करने से पहले

Lotte Global Logistics की समर्पित API एक अनुबंध-आधारित चैनल है, यह सेल्फ-सर्विस कंसोल से तुरंत जारी नहीं होती।

आवश्यक

Lotte Global Logistics अनुबंध (ग्राहक) कोड

आपको Lotte Global Logistics के साथ अनुबंध के माध्यम से जारी किया गया अनुबंध (ग्राहक) कोड चाहिए। यह किसी अलग कंसोल से सीधे जारी नहीं किया जाता।

अनुशंसित

Smart Parcel (एकीकृत ट्रैकिंग)

यदि आपको केवल डिलीवरी ट्रैकिंग की आवश्यकता है, तो आप [Smart Parcel (एकीकृत ट्रैकिंग)] चैनल पर एक ही t_key से सभी कूरियर कंपनियों को जोड़ सकते हैं।

नोट: यदि आपका उद्देश्य डिलीवरी ट्रैकिंग है, तो हम समर्पित API अनुबंध के बिना Smart Parcel एकीकृत कुंजी के माध्यम से जुड़ने की अनुशंसा करते हैं।

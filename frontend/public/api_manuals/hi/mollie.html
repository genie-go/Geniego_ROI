1. शुरू करने से पहले

आपकी Mollie API कुंजी आपके Mollie खाते के डैशबोर्ड में मिलती है।

आवश्यक

Mollie खाता

आपको एक Mollie खाता (यूरोप-केंद्रित) चाहिए। आप उससे डैशबोर्ड में साइन इन कर सकने में सक्षम होने चाहिए।

जारी कुंजी

Live API कुंजी

डैशबोर्ड में, Developers > API keys पर जाएँ और live_ से शुरू होने वाली Live API कुंजी ढूँढें।

ध्यान दें: सुनिश्चित करें कि आप live_ से शुरू होने वाली Live API कुंजी ढूँढें। इसे Test कुंजी के साथ भ्रमित न करने का ध्यान रखें।

2. जारी करने के चरण

नीचे दिए क्रम में जारी करें और GenieGo में पंजीकृत करें।

1

डैशबोर्ड तक पहुँचें

Mollie डैशबोर्ड में साइन इन करें।

my.mollie.com
2

API keys पर जाएँ

[Developers] > [API keys] पर नेविगेट करें।

https://my.mollie.com/dashboard/developers/api-keys
3

Live API कुंजी ढूँढें

live_ से शुरू होने वाली Live API कुंजी ढूँढें।

यदि आपको Developers > API keys के अंतर्गत live_ से शुरू होने वाली कुंजी दिखती है, तो यह सही है।

3. GenieGo में पंजीकृत करने योग्य जानकारी

फ़ील्डविवरण
api_keyLive API कुंजी (live_ से शुरू; सावधानी से संभालें)
GenieGo Integration Hub [पंजीकृत करें] में, अपनी api_key दर्ज करें और सहेजें। क्रेडेंशियल को संग्रहण के समय AES-256-GCM से एन्क्रिप्ट किया जाता है और पढ़ने पर मास्क किया जाता है।

4. आम समस्याएँ

समस्याकारणसमाधान
प्रमाणीकरण विफलTest कुंजी का उपयोगदोबारा जाँचें कि यह live_ से शुरू होने वाली Live API कुंजी है।
API keys मेनू में कोई कुंजी नहींडैशबोर्ड साइन-इन खाते की पुष्टि करना आवश्यकपुष्टि करें कि आपने अपने Mollie खाते से my.mollie.com डैशबोर्ड में साइन इन किया है।

अंतिम जाँच सूची

  • अपने Mollie खाते से my.mollie.com डैशबोर्ड में साइन इन किया।
  • Developers > API keys में Live API कुंजी ढूँढी।
  • live_ से शुरू होने वाली Live API कुंजी ढूँढी।
  • GenieGo [पंजीकृत करें] में api_key दर्ज किया और सहेजा।

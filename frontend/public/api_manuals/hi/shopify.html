API Key जारी करने की मैनुअल

अपने Shopify Admin में एक कस्टम ऐप बनाकर Admin API access token जारी करने और अपने Shopify स्टोर को कनेक्ट करने के लिए इसे शॉप डोमेन के साथ GenieGo Integration Hub में पंजीकृत करने की चरण-दर-चरण मार्गदर्शिका।

1. शुरू करने से पहले

Shopify कस्टम ऐप केवल स्टोर एडमिन अनुमति के साथ ही बनाया जा सकता है।

आवश्यक

स्टोर एडमिन अनुमति

कस्टम ऐप बनाने और token जारी करने के लिए आपको Shopify स्टोर की एडमिन (Admin) अनुमति की आवश्यकता है।

जारी की गई कुंजियाँ

शॉप डोमेन · Admin API token

आपको xxx.myshopify.com प्रारूप में एक शॉप डोमेन और ऐप इंस्टॉल करने के बाद दिखाया गया Admin API access token मिलेगा।

नोट: Admin API access token इंस्टॉलेशन के तुरंत बाद केवल एक बार दिखाया जाता है, इसलिए जब यह दिखे तो आपको इसे तुरंत कॉपी कर लेना चाहिए।

2. जारी करने के चरण

नीचे दिए गए क्रम में जारी करें और GenieGo में पंजीकृत करें।

1

ऐप्स और बिक्री चैनलों पर जाएँ

अपने Shopify Admin में, [सेटिंग्स] > [ऐप्स और बिक्री चैनल] पर जाएँ।

https://www.shopify.com/admin/settings/apps
2

एक कस्टम ऐप बनाएँ

कस्टम ऐप बनाने के लिए [ऐप्स विकसित करें] > [ऐप बनाएँ] पर क्लिक करें।

ऐप्स विकसित करें → ऐप बनाएँ
3

scopes चुनें · ऐप इंस्टॉल करें

Admin API scopes चुनें और ऐप इंस्टॉल करें।

4

access token कॉपी करें

इंस्टॉलेशन के बाद दिखाया गया Admin API access token तुरंत कॉपी करें (केवल एक बार दिखाया जाता है)।

ऐप इंस्टॉल करने के बाद जब Admin API access token दिखाई दे, तो इसका मतलब है कि यह सही ढंग से जारी हो गया है। यह केवल एक बार दिखाया जाता है, इसलिए इसे तुरंत कॉपी करें।

3. GenieGo में पंजीकृत करने योग्य जानकारी

फ़ील्डविवरण
shop_domainxxx.myshopify.com प्रारूप में शॉप डोमेन
access_tokenजारी किया गया Admin API access token (सावधानी से संभालें)
GenieGo [पंजीकरण] में, अपना shop_domain (xxx.myshopify.com) और access_token दर्ज करें और सहेजें। क्रेडेंशियल्स को AES-256-GCM के साथ संग्रहण के समय एन्क्रिप्ट किया जाता है और पढ़ते समय मास्क किया जाता है।

4. आम समस्याएँ

समस्याकारणसमाधान
token दोबारा नहीं देख पा रहेAdmin API access token केवल एक बार दिखाया जाता हैयदि आपका token खो गया है, तो उसे कस्टम ऐप में फिर से जारी करें और तुरंत कॉपी करें।
अपर्याप्त अनुमति त्रुटिAdmin API scopes चयनित नहींआवश्यक Admin API scopes चुनें, फिर ऐप को फिर से इंस्टॉल करें।
डोमेन त्रुटिशॉप डोमेन प्रारूप गलत दर्ज किया गयाशॉप डोमेन को xxx.myshopify.com प्रारूप में सही ढंग से दर्ज करें।

अंतिम चेकलिस्ट

  • Shopify स्टोर एडमिन अनुमति के साथ साइन इन किया।
  • सेटिंग्स > ऐप्स और बिक्री चैनल > ऐप्स विकसित करें के अंतर्गत एक कस्टम ऐप बनाया।
  • Admin API scopes चुने और ऐप इंस्टॉल किया।
  • इंस्टॉलेशन के बाद दिखाया गया Admin API access token तुरंत कॉपी किया।
  • GenieGo [पंजीकरण] में shop_domain·access_token दर्ज किया और सहेजा।

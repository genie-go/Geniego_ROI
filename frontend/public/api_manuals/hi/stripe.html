API कुंजी जारी करने की मैनुअल

Stripe डैशबोर्ड के Developers मेनू से API कुंजियाँ (Publishable Key·Secret Key) जारी करने और उन्हें GenieGo Integration Hub में पंजीकृत करने के लिए शुरुआती-अनुकूल, चरण-दर-चरण मार्गदर्शिका।

परिणामAPI क्रेडेंशियल

1. शुरू करने से पहले

Stripe API कुंजियाँ आपके Stripe खाते के डैशबोर्ड से जारी की जाती हैं।

आवश्यक

Stripe खाता

आपको अपने Stripe खाते से dashboard.stripe.com पर साइन इन करने में सक्षम होना चाहिए।

जारी की गई कुंजियाँ

Publishable Key · Secret Key

Developers > API keys स्क्रीन पर अपनी Publishable key और Secret key खोजें।

नोट: वास्तविक भुगतान एकीकरण के लिए, परीक्षण कुंजियों के बजाय लाइव कुंजियों का उपयोग करें।

2. जारी करने के चरण

नीचे दिए गए क्रम में जारी करें और GenieGo में पंजीकृत करें।

1

Stripe में साइन इन करें

अपने तैयार किए गए Stripe खाते से डैशबोर्ड में साइन इन करें।

dashboard.stripe.com
2

API keys मेनू पर जाएँ

[Developers] > [API keys] मेनू पर जाएँ।

Developers → API keys
3

कुंजियाँ जाँचें

अपनी Publishable key और Secret key खोजें। लाइव कुंजियों का उपयोग करें।

https://dashboard.stripe.com/apikeys
जब Publishable key और Secret key दोनों की पुष्टि हो जाए, तो वे सही ढंग से जारी हो चुकी हैं।

3. GenieGo में पंजीकृत करने योग्य जानकारी

फ़ील्डविवरण
publishable_keyजारी की गई Publishable Key
secret_keyजारी की गई Secret Key (सावधानी से संभालें)
GenieGo Integration Hub [पंजीकरण] में publishable_key·secret_key दर्ज करें और सहेजें। क्रेडेंशियल AES-256-GCM के साथ संग्रहण में एन्क्रिप्ट किए जाते हैं और पढ़ने पर मास्क किए जाते हैं।

4. सामान्य समस्याएँ

समस्याकारणसमाधान
प्रमाणीकरण विफलपरीक्षण कुंजी दर्ज की गईजाँचें कि क्या आप लाइव कुंजियों (Publishable·Secret) का उपयोग कर रहे हैं।
Secret key दिखाई नहीं दे रहीकुंजी केवल एक बार दिखती है या छिपी हुई हैDevelopers > API keys स्क्रीन पर फिर से जाँचें या नई जारी करें।

अंतिम चेकलिस्ट

  • अपने Stripe खाते से dashboard.stripe.com पर साइन इन किया।
  • Developers > API keys पर गए।
  • लाइव Publishable Key·Secret Key की पुष्टि की।
  • GenieGo [पंजीकरण] में publishable_key·secret_key दर्ज किया और सहेजा।

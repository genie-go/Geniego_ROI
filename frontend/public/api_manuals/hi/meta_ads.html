1. शुरू करने से पहले

Meta Ads एकीकरण के लिए एक बिज़नेस खाता और एक ऐसा विज्ञापन खाता चाहिए जो सक्रिय रूप से विज्ञापन चला रहा हो।

आवश्यक

Facebook व्यक्तिगत खाता · विज्ञापन खाता

आपको एक Facebook व्यक्तिगत खाता और एक Meta विज्ञापन खाता चाहिए जो सक्रिय रूप से विज्ञापन चला रहा हो।

जारी की गई कुंजियाँ

एक्सेस टोकन · विज्ञापन खाता ID (act_)

आप एक सिस्टम यूज़र टोकन (एक्सेस टोकन) और act_ से शुरू होने वाला एक संख्यात्मक विज्ञापन खाता ID प्राप्त करेंगे।

नोट: अपने स्वयं के बिज़नेस के बाहर के डेटा तक पहुँचने के लिए ऐप समीक्षा (App Review) आवश्यक है। यदि आप केवल अपने स्वयं के बिज़नेस के भीतर के विज्ञापन खाते जोड़ते हैं, तो आप समीक्षा के बिना आगे बढ़ सकते हैं।

2. जारी करने के चरण

नीचे दिए गए क्रम में जारी करें और GenieGo में पंजीकृत करें।

1

बिज़नेस खाता बनाएँ और सत्यापित करें

अपने ब्राउज़र में साइट खोलें, साइन इन करें, फिर [बिज़नेस बनाएँ] के माध्यम से एक बिज़नेस खाता बनाएँ और ईमेल सत्यापन पूरा करें।

business.facebook.com
2

एक ऐप बनाएँ

ऊपर दाईं ओर [मेरे ऐप्स] > [ऐप बनाएँ] पर क्लिक करें, "बिज़नेस" प्रकार चुनें और ऐप बनाएँ।

developers.facebook.com → मेरे ऐप्स → ऐप बनाएँ
3

Marketing API जोड़ें

ऐप डैशबोर्ड पर, "Marketing API" कार्ड खोजें और इसे उत्पाद के रूप में जोड़ने के लिए [सेट अप] पर क्लिक करें।

4

सिस्टम यूज़र बनाएँ और अनुमतियाँ निर्दिष्ट करें

बिज़नेस सेटिंग्स > [उपयोगकर्ता] > [सिस्टम उपयोगकर्ता] > [जोड़ें] पर जाकर एक सिस्टम यूज़र बनाएँ, फिर [संपत्तियाँ निर्दिष्ट करें] के तहत विज्ञापन-खाता अनुमतियाँ प्रदान करें।

business.facebook.com/settings → उपयोगकर्ता → सिस्टम उपयोगकर्ता
5

एक एक्सेस टोकन जनरेट करें

उसी स्क्रीन पर, [टोकन जनरेट करें] पर क्लिक करें, अपना ऐप चुनें, अनुमतियाँ (ads_read, ads_management) चेक करें, टोकन जनरेट करें और इसका मान कॉपी करें।

6

अपना विज्ञापन खाता ID खोजें

Ads Manager में, act_ से शुरू होने वाला संख्यात्मक विज्ञापन खाता ID खोजें।

adsmanager.facebook.com
एक बार जब टोकन ads_read और ads_management अनुमतियों के चेक किए हुए साथ जनरेट हो जाता है, तो इसे सही ढंग से जारी कर दिया गया है।

3. GenieGo में पंजीकृत करने योग्य जानकारी

फ़ील्डविवरण
access_tokenजारी किया गया सिस्टम यूज़र एक्सेस टोकन (सावधानी से संभालें)
ad_account_idact_ से शुरू होने वाला विज्ञापन खाता ID
GenieGo Integration Hub > संबंधित चैनल के [पंजीकृत करें] में, अपना एक्सेस टोकन और विज्ञापन खाता ID (act_) पेस्ट करें और सहेजें। क्रेडेंशियल को संग्रहण के समय AES-256-GCM से एन्क्रिप्ट किया जाता है और पढ़ते समय मास्क किया जाता है।

4. आम समस्याएँ

समस्याकारणसमाधान
अनुमति त्रुटियों के कारण डेटा क्वेरी नहीं किया जा सकताटोकन अनुमतियाँ (ads_read·ads_management) चेक नहीं की गईं, या विज्ञापन खाता अनुमति निर्दिष्ट नहीं की गईसिस्टम यूज़र को विज्ञापन-खाता अनुमति निर्दिष्ट करें, और टोकन जनरेट करते समय ads_read·ads_management अनुमतियाँ चेक करें।
किसी अन्य बिज़नेस के डेटा तक पहुँच अस्वीकृतऐप समीक्षा (App Review) पूरी नहीं हुईअपने स्वयं के बिज़नेस के बाहर के डेटा तक पहुँचने के लिए, ऐप समीक्षा के लिए आवेदन करें और इसे पूरा करें।
विज्ञापन खाता ID प्रारूप त्रुटिact_ उपसर्ग गायब, या केवल संख्याएँAds Manager में, act_ से शुरू होने वाला पूरा ID खोजें और दर्ज करें।

अंतिम चेकलिस्ट

  • business.facebook.com पर एक बिज़नेस खाता बनाया और ईमेल सत्यापन पूरा किया।
  • developers.facebook.com पर एक बिज़नेस-प्रकार का ऐप बनाया और Marketing API जोड़ा।
  • एक सिस्टम यूज़र बनाया और विज्ञापन-खाता अनुमति निर्दिष्ट की।
  • ads_read·ads_management अनुमतियों के साथ एक एक्सेस टोकन जनरेट किया और कॉपी किया।
  • act_ से शुरू होने वाला विज्ञापन खाता ID खोजा।
  • GenieGo [पंजीकृत करें] में access_token·ad_account_id दर्ज किया और सहेजा।

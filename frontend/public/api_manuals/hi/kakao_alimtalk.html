1. Kakao Alimtalk जारी करने की अवधारणा

Alimtalk एक व्यावसायिक संदेश सेवा है जो ऑर्डर, आरक्षण, शिपिंग और भुगतान जैसे सूचनात्मक संदेश Kakao Talk के माध्यम से भेजती है। यह ऐसी संरचना नहीं है जिसमें केवल एक सामान्य Kakao डेवलपर ऐप कुंजी से सीधे भेजा जा सके; इसके लिए व्यवसाय चैनल सत्यापन और भेजने वाली एजेंसी/प्लेटफ़ॉर्म सेटअप आवश्यक है।

महत्वपूर्ण: Alimtalk विज्ञापन संदेशों के बजाय सूचनात्मक संदेशों पर केंद्रित है, और टेम्पलेट के लिए पूर्व समीक्षा आवश्यक है।

2. आवश्यक सामग्री

  • Kakao Business खाता
  • Kakao Talk चैनल
  • व्यवसाय पंजीकरण प्रमाणपत्र जैसे व्यवसाय सत्यापन दस्तावेज़
  • भेजे जाने वाले Alimtalk टेम्पलेट का पाठ
  • भेजने वाली एजेंसी या क्लाउड मैसेजिंग सेवा का खाता
  • प्राप्तकर्ता फ़ोन नंबर संग्रह और सहमति प्रबंधन नीति

3. Kakao Talk चैनल तैयार करना

1

Kakao Business तक पहुँचें

Kakao Business व्यवस्थापक केंद्र में लॉग इन करें।

business.kakao.com
2

Kakao Talk चैनल बनाएँ

अपने ब्रांड या सेवा के नाम से एक चैनल बनाएँ।

Kakao Talk चैनल व्यवस्थापक केंद्र
3

व्यवसाय सत्यापन करें

व्यवसाय चैनल को सत्यापित करने के लिए अपनी व्यावसायिक जानकारी और सहायक दस्तावेज़ जमा करें।

व्यवसाय सत्यापन
4

चैनल खोज ID जाँचें

Alimtalk भेजने की सेटिंग के लिए आवश्यक चैनल ID या खोज ID जाँचें।

चैनल प्रोफ़ाइल → खोज ID
Alimtalk प्रेषक प्रोफ़ाइल पंजीकृत करने से पहले व्यवसाय सत्यापन पूरा होना आवश्यक है।

4. प्रेषक प्रोफ़ाइल / Sender Profile पंजीकरण

1

भेजने का प्लेटफ़ॉर्म चुनें

NHN Cloud, Naver Cloud SENS, Infobank, Bizgo जैसे Alimtalk प्रदाता का चयन करें।

Messaging Provider
2

प्रोजेक्ट बनाएँ

आपके द्वारा चुने गए प्लेटफ़ॉर्म पर एक Alimtalk प्रोजेक्ट बनाएँ।

Project / Service
3

Kakao Talk चैनल कनेक्ट करें

व्यवसाय-सत्यापित Kakao Talk चैनल की जानकारी पंजीकृत करें।

Sender Profile
4

प्रेषक प्रोफ़ाइल समीक्षा

चैनल स्वामित्व पुष्टि और प्रेषक प्रोफ़ाइल समीक्षा करें।

Profile Review
5

API क्रेडेंशियल की पुष्टि करें

प्लेटफ़ॉर्म द्वारा प्रदान किए गए App Key, Secret Key, API Key आदि की जाँच करें।

API Key / Secret

5. टेम्पलेट पंजीकरण और अनुमोदन

1

टेम्पलेट लिखें

ऑर्डर पुष्टि, शिपिंग सूचना जैसे सूचनात्मक पाठ लिखें।

Template Message
2

वेरिएबल सेट करें

नाम, ऑर्डर नंबर, शिपिंग नंबर जैसे गतिशील रूप से बदलने वाले मानों को वेरिएबल के रूप में सेट करें।

#{वेरिएबल}
3

समीक्षा का अनुरोध करें

टेम्पलेट सहेजें और Kakao समीक्षा अनुरोध करें।

Request Approval
4

अनुमोदन की पुष्टि करें

अनुमोदित टेम्पलेट कोड या ID की पुष्टि करें।

Template Code
यदि समीक्षा से पहले का पाठ वास्तव में भेजे गए पाठ से भिन्न है, तो भेजना विफल हो सकता है या प्रतिबंध लग सकते हैं।

6. API भेजने के प्रवाह का उदाहरण

1) अपने सर्वर से Alimtalk API Endpoint को कॉल करें
2) API Key या Secret से प्रमाणित करें
3) प्राप्तकर्ता फ़ोन नंबर, टेम्पलेट कोड और वेरिएबल मान भेजें
4) भेजने का परिणाम प्राप्त करें
5) विफलता पर SMS विकल्प भेजना है या नहीं, इसे संभालें

7. अंतिम जाँच सूची

  • Kakao Talk चैनल बनाया।
  • व्यवसाय सत्यापन पूरा किया।
  • Alimtalk भेजने का प्लेटफ़ॉर्म चुना।
  • Sender Profile पंजीकृत किया।
  • API Key या Secret की पुष्टि की।
  • टेम्पलेट पंजीकृत किया और अनुमोदन प्राप्त किया।
  • भेजने की विफलता और विकल्प SMS नीति तैयार की।

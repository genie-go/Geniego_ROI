1. शुरू करने से पहले

DHL Express क्रेडेंशियल डेवलपर पोर्टल पर अनुरोध किए जाते हैं, और एक ग्राहक खाता संख्या आवश्यक है।

तैयारी

DHL Express ग्राहक खाता संख्या

DHL Express ग्राहक खाता संख्या होने से क्रेडेंशियल अनुरोध आसान हो जाता है। GenieGo में पंजीकरण करते समय भी खाता संख्या दर्ज की जाती है।

जारी की गई कुंजियाँ

API Key · Secret

आप पोर्टल पर अपने स्वयं के ऐप (कंपनी का नाम) के माध्यम से API Key और Secret प्राप्त करेंगे।

नोट: जब आप [Get Access] के माध्यम से अपनी ग्राहक जानकारी पंजीकृत करते हैं और क्रेडेंशियल का अनुरोध करते हैं, तो आपको आमतौर पर अगले कार्यदिवस पर Test/Production अनुमोदन ईमेल प्राप्त होता है।

2. जारी करने के चरण

नीचे दिए गए क्रम में जारी करें और GenieGo में पंजीकृत करें।

1

डेवलपर पोर्टल पर साइन अप करें

DHL Express डेवलपर पोर्टल पर जाएँ, अपने ईमेल से साइन अप करें और इसे सत्यापित करें।

https://developer.dhl.com
2

क्रेडेंशियल का अनुरोध करें

साइन इन करने के बाद, [Get Access] अनुभाग में अपनी ग्राहक जानकारी पंजीकृत करें और क्रेडेंशियल का अनुरोध करें।

Get Access
3

अनुमोदन ईमेल प्राप्त करें

आपको आमतौर पर अगले कार्यदिवस पर Test/Production अनुमोदन ईमेल प्राप्त होता है।

4

API Key·Secret खोजें

पोर्टल पर, API Key और Secret खोजने के लिए अपने स्वयं के ऐप (कंपनी का नाम) > [Show Key] पर जाएँ।

आपका ऐप (कंपनी का नाम) → Show Key
एक बार जब आप अनुमोदन ईमेल प्राप्त कर लेते हैं और पोर्टल पर API Key·Secret दिखाई देते हैं, तो वे सही ढंग से जारी हो गए हैं।

3. GenieGo में पंजीकृत करने की जानकारी

फ़ील्डविवरण
api_keyपोर्टल पर मिली DHL API Key
account_numberDHL Express ग्राहक खाता संख्या
एक बार जब आप उन्हें GenieGo Integration Hub [पंजीकरण] में दर्ज कर सहेजते हैं, तो क्रेडेंशियल को AES-256-GCM के साथ स्थिर अवस्था में एन्क्रिप्ट किया जाता है और पढ़ते समय मास्क किया जाता है।

4. सामान्य समस्याएँ

समस्याकारणसमाधान
अनुमोदन ईमेल नहीं आताअगले कार्यदिवस के अनुमोदन प्रसंस्करण की प्रतीक्षा, या ग्राहक जानकारी पंजीकृत नहीं[Get Access] में जाँचें कि क्या ग्राहक पंजीकरण और क्रेडेंशियल अनुरोध पूर्ण हैं, और कार्यदिवसों के आधार पर प्रतीक्षा करें।
कॉल करते समय प्रमाणीकरण त्रुटिTest/Production वातावरण के बीच भ्रम, या खाता संख्या गायबअनुमोदित वातावरण की API Key·Secret और खाता संख्या (account_number) को फिर से जाँचें।

  • अगले कार्यदिवस पर Test/Production अनुमोदन ईमेल प्राप्त किया।

  • GenieGo [पंजीकरण] में api_key·account_number दर्ज कर सहेजा।

कुंजियाँ जाँचें

जारी की गई QSM API कुंजी और अपना Seller ID जाँचें।

जब API प्रमाणीकरण कुंजी API प्रबंधन स्क्रीन पर दिखाई दे, तो इसका अर्थ है कि यह सही ढंग से जारी हो गई है।

3. GenieGo में पंजीकृत करने योग्य जानकारी

फ़ील्डविवरण
api_keyजारी की गई QSM API कुंजी (सावधानी से संभालें)
seller_idआपका अपना Seller ID
GenieGo Integration Hub [पंजीकरण] में api_key·seller_id दर्ज करें और सहेजें। क्रेडेंशियल को संग्रहण के समय AES-256-GCM से एन्क्रिप्ट किया जाता है और पढ़ते समय मास्क किया जाता है।

4. सामान्य समस्याएँ

API प्रबंधन मेनू मौजूद नहीं हैविक्रेता (QSM) खाता नहीं है, या अपर्याप्त अनुमतियाँQoo10 विक्रेता (QSM) खाते से साइन इन करें और API प्रबंधन मेनू जाँचें।

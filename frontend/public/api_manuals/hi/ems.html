1. आरंभ करने से पहले

EMS कंसोल के माध्यम से स्वयं-सेवा जारी करने वाली सेवा नहीं है, बल्कि अनुबंध (ग्राहक) कोड पर निर्भर एक अनुबंध-आधारित सेवा है।

सूचना: Korea Post International Express (EMS) अनुबंध (ग्राहक) कोड पर आधारित है। यह वह तरीका नहीं है जिसमें आप स्वयं क्रेडेंशियल जारी करते हैं, बल्कि एक प्रक्रिया है जिसमें आप EMS अनुबंध करते हैं और फिर ग्राहक (अनुबंध) कोड तथा Korea Post API कुंजी प्राप्त करते हैं।
आवश्यकता

Korea Post EMS अनुबंध (ग्राहक कोड)

आपको Korea Post International Express (EMS) का अनुबंध (ग्राहक) कोड चाहिए। यह केवल अनुबंध के माध्यम से ही जारी किया जाता है।

अनुबंध पूरा होने के बाद, आपको ग्राहक (अनुबंध) कोड और Korea Post API कुंजी प्राप्त होती है।

2. जारी करने के लिए आवेदन प्रक्रिया

चूंकि यह एक अनुबंध-आधारित सेवा है, अनुबंध करें, अपने क्रेडेंशियल प्राप्त करें, और उन्हें नीचे दिए गए क्रम में पंजीकृत करें।

1

EMS अनुबंध करें

इंटरनेट डाकघर या अनुबंध काउंटर के माध्यम से EMS अनुबंध करें।

2

ग्राहक कोड और API कुंजी प्राप्त करें

अनुबंध पूरा होने के बाद, आपको ग्राहक (अनुबंध) कोड और Korea Post API कुंजी प्राप्त होती है।

3

GenieGo में पंजीकृत करें

प्राप्त क्रेडेंशियल को GenieGo [पंजीकरण] में दर्ज करें और सहेजें।

चूंकि यह एक अनुबंध-आधारित सेवा है, अनुबंध प्रसंस्करण स्थिति के अनुसार आपके क्रेडेंशियल प्राप्त करने में कुछ समय लग सकता है।

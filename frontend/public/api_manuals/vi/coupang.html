Hướng dẫn cấp API Key

Hướng dẫn từng bước thân thiện với người mới, về cách cấp thông tin xác thực Open API (Access Key·Secret Key) trong bảng điều khiển người bán Coupang WING và đăng ký vào Trung tâm tích hợp GenieGo.

1. Trước khi bắt đầu

Coupang Open API chỉ có thể được cấp từ tài khoản người bán.

Bắt buộc

Tài khoản người bán WING

Bạn cần một tài khoản người bán Coupang WING đã hoàn tất xác minh doanh nghiệp. Không thể cấp bằng tài khoản thành viên người mua thông thường.

Khóa được cấp

Access Key · Secret Key

Trên màn hình hoàn tất cấp, hãy tìm Access Key và Secret Key, đồng thời kiểm tra vendor ID.

Lưu ý: Khi cấp, bạn phải nhập tên công ty, URL trang chủ/dịch vụ (nếu không có thì dùng wing.coupang.com) và IP truy cập (tối đa 10).

2. Các bước cấp Open API Key

Cấp theo thứ tự bên dưới và đăng ký trong GenieGo.

1

Đăng nhập WING

Đăng nhập bằng tài khoản người bán của bạn.

wing.coupang.com
2

Đi tới menu Open API

Đi tới menu [Open API] hoặc [Quản lý API Key] ở khu vực trên cùng/cài đặt.

Open API → Quản lý API Key
3

Cấp Open API Key

Nhấp [Cấp Open API Key] và nhập tên công ty, URL trang chủ và IP truy cập (tối đa 10).

Cấp Open API Key
4

Kiểm tra khóa

Trên màn hình hoàn tất cấp, hãy tìm Access Key·Secret Key và vendor ID.

Khi Access Key·Secret Key xuất hiện trên màn hình hoàn tất cấp, nghĩa là chúng đã được cấp đúng cách.

3. Thông tin cần đăng ký trong GenieGo

TrườngMô tả
access_keyAccess Key đã cấp
secret_keySecret Key đã cấp (xử lý cẩn thận)
vendor_idVendor ID
Nhập chúng vào [Đăng ký] trong Trung tâm tích hợp GenieGo và lưu lại. Thông tin xác thực được mã hóa khi lưu trữ bằng AES-256-GCM và được che khi đọc. Việc đồng bộ đơn hàng bắt đầu ngay sau khi đăng ký.

Danh sách kiểm tra cuối cùng

  • Đã đăng nhập bằng tài khoản người bán WING đã hoàn tất xác minh doanh nghiệp.
  • Đã cấp Open API Key và đăng ký các IP truy cập.
  • Đã kiểm tra Access Key·Secret Key·Vendor ID.
  • Đã nhập access_key·secret_key·vendor_id vào GenieGo [Đăng ký] và lưu lại.

Hướng dẫn thực hành thân thiện với người mới dành cho doanh nghiệp, thương hiệu, live-commerce và nhà vận hành nền tảng IT/logistics để tạo kênh Twitch chính thức và hoàn tất thiết lập phát sóng, kết nối OBS và cấp Twitch Developer API.

1. Hiểu cấu trúc tài khoản Twitch

Khác với Instagram, Twitch không có cấu trúc chọn riêng tài khoản doanh nghiệp/nhà sáng tạo.

Kênh chính thức của doanh nghiệp

Để vận hành thương hiệu

Tạo một tài khoản Twitch thường bằng tên công ty hoặc tên thương hiệu, sau đó vận hành như một kênh chính thức. Phù hợp cho giới thiệu sản phẩm, live-commerce và phát sóng sự kiện.

Cấp độ phát triển streamer

Affiliate / Partner

Khi bạn đáp ứng các điều kiện về thành tích phát sóng, người theo dõi và thời lượng xem, bạn có thể tham gia chương trình Affiliate hoặc Partner.

Điểm cốt lõi: Doanh nghiệp cũng nên tạo trước một tài khoản Twitch thường, sau đó tiến hành xây dựng thương hiệu kênh, xác thực hai bước, kết nối OBS và đăng ký ứng dụng trong Developer Console.

2. Những thứ cần chuẩn bị

  • Địa chỉ email doanh nghiệp
  • Tên thương hiệu hoặc tên công ty
  • Tên người dùng Twitch dự kiến
  • Logo công ty và hình banner kênh
  • Văn bản giới thiệu công ty và URL website
  • PC hoặc laptop để phát sóng
  • OBS Studio hoặc phần mềm phát trực tuyến
  • OAuth Redirect URL để tích hợp API

6. Thiết lập xác thực hai bước

Việc thiết lập 2FA rất quan trọng để sử dụng ổn định các thiết lập phát sóng và bảo mật trên Twitch.

1

Vào menu bảo mật

Mở menu Security and Privacy trong Cài đặt.

Settings → Security and Privacy
2

Bật 2FA

Bật Two-Factor Authentication.

Set Up Two-Factor Authentication
3

Kết nối điện thoại hoặc ứng dụng xác thực

Kết nối xác minh SMS hoặc ứng dụng xác thực như Google Authenticator, Authy.

Authenticator App
Lưu ý: Tài khoản doanh nghiệp có thể được nhiều người sử dụng, vì vậy hãy chắc chắn quản lý quyền quản trị viên và email khôi phục.

7. Kiểm tra Stream Key & kết nối OBS

1

Mở Creator Dashboard

Vào Creator Dashboard từ menu hồ sơ.

Creator Dashboard
2

Vào menu Stream

Chọn mục Stream trong menu cài đặt.

Settings → Stream
3

Kiểm tra Primary Stream Key

Sao chép khóa stream chính. Khóa này phải được bảo quản như mật khẩu.

Primary Stream Key
4

Kết nối trong OBS

Trong OBS Studio, đặt dịch vụ là Twitch và nhập giá trị đã sao chép làm khóa stream.

OBS → Settings → Stream → Twitch
Nếu khóa stream bị lộ, người khác có thể phát sóng trên kênh của bạn. Tuyệt đối không công khai.

11. Kiểm tra Client ID / Client Secret

1

Chọn ứng dụng đã đăng ký

Nhấp vào ứng dụng bạn vừa tạo trong danh sách Applications của Developer Console.

Applications → chọn ứng dụng
2

Sao chép Client ID

Sao chép Client ID hiển thị trên màn hình và lưu vào cài đặt máy chủ.

Client ID
3

Tạo Client Secret

Nhấp nút New Secret để tạo Client Secret và lưu trữ an toàn.

New Secret
Client Secret thường chỉ có thể xem một lần. Hãy lưu vào nơi an toàn ngay sau khi tạo.

12. Cấu trúc xác thực Twitch API

Twitch Channel │ ▼ Developer Console │ ▼ Application │ ▼ Client ID / Client Secret │ ▼ OAuth Token │ ▼ Twitch Helix API │ ▼ Thông tin kênh, trạng thái phát sóng, chat bot, phân tích người xem

13. Quy trình cấp token OAuth

1) Tạo URL ủy quyền Twitch OAuth 2) Người dùng đăng nhập bằng tài khoản Twitch 3) Nhận authorization code tại Redirect URL 4) Yêu cầu Access Token bằng authorization code + Client Secret 5) Gọi Twitch API bằng Access Token 6) Làm mới bằng Refresh Token khi cần
Việc truy vấn dữ liệu công khai đơn giản có thể thực hiện bằng App Access Token, trong khi các thao tác cần quyền người dùng thì cần User Access Token.

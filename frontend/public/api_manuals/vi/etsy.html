Hướng dẫn cấp API Key

Hướng dẫn từng bước thân thiện với người mới về việc đăng ký ứng dụng trong cổng nhà phát triển Etsy để cấp API Key (keystring), nhận access token qua ủy quyền OAuth và đăng ký trong GenieGo Integration Hub.

1. Trước khi bắt đầu

API Key của Etsy được cấp bằng cách đăng ký một ứng dụng dưới tài khoản người bán của bạn.

Bắt buộc

Tài khoản người bán Etsy

Để tiến hành cấp, bạn cần một tài khoản người bán Etsy. Bạn sẽ dùng tài khoản này để truy cập cổng nhà phát triển Etsy.

Khóa được cấp

API Key (keystring)

Sau khi bạn nhập thông tin ứng dụng, một API Key (keystring) sẽ được cấp. Sau đó bạn nhận access token qua ủy quyền OAuth.

Lưu ý: Việc đăng ký GenieGo yêu cầu api_key đã cấp cùng với shop_id, định danh cửa hàng của bạn.

2. Các bước cấp

Cấp theo thứ tự dưới đây và đăng ký trong GenieGo.

1

Tạo ứng dụng mới

Truy cập etsy.com/developers và nhấp vào [Create a New App].

etsy.com/developers
2

Nhập thông tin ứng dụng và cấp khóa

Sau khi bạn nhập thông tin ứng dụng, một API Key (keystring) sẽ được cấp.

https://www.etsy.com/developers/your-apps
3

Ủy quyền OAuth

Nhận access token qua ủy quyền OAuth.

4

Xác nhận thông tin đăng ký

Kiểm tra api_key đã cấp và shop_id của cửa hàng bạn.

Khi API Key (keystring) được cấp và bạn đã nhận access token qua ủy quyền OAuth, nghĩa là nó đã được cấp đúng cách.

3. Thông tin cần đăng ký trong GenieGo

TrườngMô tả
api_keyAPI Key (keystring) đã được cấp
shop_idShop ID của cửa hàng Etsy bạn muốn kết nối
Trong GenieGo Integration Hub [Đăng ký], nhập api_key và shop_id của bạn rồi lưu. Thông tin xác thực được mã hóa khi lưu trữ bằng AES-256-GCM và được che khi đọc.

Danh sách kiểm tra cuối cùng

  • Đã truy cập etsy.com/developers bằng tài khoản người bán Etsy.
  • Đã cấp API Key (keystring) qua [Create a New App].
  • Đã nhận access token qua ủy quyền OAuth.
  • Đã nhập api_key và shop_id vào GenieGo [Đăng ký] và lưu.

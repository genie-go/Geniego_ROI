Hướng dẫn cấp API Key

Hướng dẫn từng bước thân thiện với người mới để cấp API key (Publishable Key·Secret Key) từ menu Developers của bảng điều khiển Stripe và đăng ký vào GenieGo Integration Hub.

1. Trước khi bắt đầu

API key của Stripe được cấp từ bảng điều khiển tài khoản Stripe của bạn.

Bắt buộc

Tài khoản Stripe

Bạn phải đăng nhập được vào dashboard.stripe.com bằng tài khoản Stripe của mình.

Khóa được cấp

Publishable Key · Secret Key

Tìm Publishable key và Secret key của bạn trên màn hình Developers > API keys.

Lưu ý: Đối với tích hợp thanh toán thực tế, hãy dùng khóa live thay vì khóa thử nghiệm.

2. Các bước cấp khóa

Cấp theo thứ tự bên dưới và đăng ký trong GenieGo.

1

Đăng nhập Stripe

Đăng nhập vào bảng điều khiển bằng tài khoản Stripe bạn đã chuẩn bị.

dashboard.stripe.com
2

Vào menu API keys

Vào menu [Developers] > [API keys].

Developers → API keys
3

Kiểm tra khóa

Tìm Publishable key và Secret key của bạn. Dùng các khóa live.

https://dashboard.stripe.com/apikeys
Khi cả Publishable key và Secret key đều được xác nhận, nghĩa là chúng đã được cấp chính xác.

3. Thông tin cần đăng ký trong GenieGo

TrườngMô tả
publishable_keyPublishable Key đã được cấp
secret_keySecret Key đã được cấp (xử lý cẩn thận)
Trong GenieGo Integration Hub [Đăng ký], nhập publishable_key·secret_key và lưu. Thông tin xác thực được mã hóa khi lưu trữ bằng AES-256-GCM và được che khi đọc.

Danh sách kiểm tra cuối cùng

  • Đã đăng nhập dashboard.stripe.com bằng tài khoản Stripe của bạn.
  • Đã vào Developers > API keys.
  • Đã xác nhận Publishable Key·Secret Key live.
  • Đã nhập publishable_key·secret_key vào GenieGo [Đăng ký] và lưu.

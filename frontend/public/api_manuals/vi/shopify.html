Hướng dẫn cấp API Key

Hướng dẫn từng bước để tạo ứng dụng tùy chỉnh trong trang Quản trị Shopify (Admin) nhằm cấp Admin API access token và đăng ký nó cùng với tên miền cửa hàng trong GenieGo Integration Hub để kết nối cửa hàng Shopify của bạn.

1. Trước khi bắt đầu

Chỉ có thể tạo ứng dụng tùy chỉnh Shopify khi có quyền quản trị cửa hàng.

Bắt buộc

Quyền quản trị cửa hàng

Bạn cần quyền quản trị (Admin) cho cửa hàng Shopify để tạo ứng dụng tùy chỉnh và cấp token.

Khóa được cấp

Tên miền cửa hàng · Admin API token

Bạn sẽ nhận được tên miền cửa hàng theo định dạng xxx.myshopify.com và Admin API access token hiển thị sau khi cài đặt ứng dụng.

Lưu ý: Admin API access token chỉ hiển thị một lần ngay sau khi cài đặt, vì vậy bạn phải sao chép ngay khi nó xuất hiện.

2. Các bước cấp

Cấp theo thứ tự bên dưới và đăng ký trong GenieGo.

1

Đi tới Ứng dụng và kênh bán hàng

Trong trang Quản trị Shopify (Admin), đi tới [Cài đặt] > [Ứng dụng và kênh bán hàng].

https://www.shopify.com/admin/settings/apps
2

Tạo ứng dụng tùy chỉnh

Nhấp [Phát triển ứng dụng] > [Tạo ứng dụng] để tạo ứng dụng tùy chỉnh.

Phát triển ứng dụng → Tạo ứng dụng
3

Chọn phạm vi · Cài đặt ứng dụng

Chọn các Admin API scopes và cài đặt ứng dụng.

4

Sao chép access token

Sao chép ngay Admin API access token hiển thị sau khi cài đặt (chỉ hiển thị một lần).

Khi Admin API access token xuất hiện sau khi cài đặt ứng dụng, nghĩa là đã được cấp thành công. Vì chỉ hiển thị một lần, hãy sao chép ngay.

3. Thông tin cần đăng ký trong GenieGo

TrườngMô tả
shop_domainTên miền cửa hàng theo định dạng xxx.myshopify.com
access_tokenAdmin API access token đã được cấp (xử lý cẩn thận)
Trong GenieGo [Đăng ký], hãy nhập shop_domain (xxx.myshopify.com) và access_token của bạn rồi lưu. Thông tin xác thực được mã hóa khi lưu trữ bằng AES-256-GCM và được che dấu khi đọc.

4. Các vấn đề thường gặp

Lỗi không đủ quyềnChưa chọn Admin API scopesChọn các Admin API scopes cần thiết, sau đó cài đặt lại ứng dụng.

Danh sách kiểm tra cuối cùng

  • Đã đăng nhập bằng quyền quản trị cửa hàng Shopify.
  • Đã tạo ứng dụng tùy chỉnh trong Cài đặt > Ứng dụng và kênh bán hàng > Phát triển ứng dụng.
  • Đã chọn các Admin API scopes và cài đặt ứng dụng.
  • Đã sao chép ngay Admin API access token hiển thị sau khi cài đặt.
  • Đã nhập shop_domain·access_token trong GenieGo [Đăng ký] và lưu.

1. Sự khác biệt giữa API Key và OAuth

Đây là phần mà mọi người dễ nhầm lẫn nhất khi mới bắt đầu.

API Key

Dùng để truy vấn dữ liệu công khai

Được dùng để lấy dữ liệu YouTube công khai như tìm kiếm video, tra cứu thông tin kênh, lấy danh sách phát và đọc bình luận.

OAuth 2.0

Dùng cho các tác vụ cần quyền của tài khoản bạn

Được dùng cho các tác vụ cần sự cho phép của người dùng, như tải video lên, quản lý kênh của bạn và truy cập dữ liệu riêng tư.

Quan trọng: Để tải video lên tự động, chỉ API Key là không đủ; cần có xác thực OAuth 2.0.

2. Bật YouTube Data API v3

Trước khi tạo API Key, trước tiên bạn phải bật API muốn dùng trong dự án của mình.

1

Truy cập Google Cloud Console

Đăng nhập vào Google Cloud Console.

console.cloud.google.com
2

Chọn hoặc tạo dự án

Trong trình chọn dự án ở trên cùng, chọn một dự án hiện có hoặc tạo dự án mới.

ví dụ: Geniego ROI
3

Đi tới Thư viện API

Trong menu bên trái, đi tới API và dịch vụ → Thư viện.

API và dịch vụ → Thư viện
4

Tìm YouTube Data API v3 và bật nó

Nhập YouTube Data API v3 vào ô tìm kiếm, chọn API đó và nhấp nút Bật.

YouTube Data API v3 → Bật
Nếu bạn thấy Trạng thái: Đã bật trên màn hình, nghĩa là nó đã được bật đúng cách.

3. Tạo API Key

Sau khi YouTube Data API v3 được bật, bạn có thể tạo API Key trên màn hình Thông tin xác thực.

1

Đi tới menu Thông tin xác thực

Trong menu bên trái, đi tới API và dịch vụ → Thông tin xác thực.

API và dịch vụ → Thông tin xác thực
2

Nhấp Tạo thông tin xác thực

Nhấp nút + Tạo thông tin xác thực ở trên cùng.

+ Tạo thông tin xác thực
3

Chọn Khóa API

Trong menu hiện ra, chọn Khóa API.

Khóa API
4

Chọn API có thể truy cập

Trong phần Hạn chế API, xác nhận rằng YouTube Data API v3 đã được chọn.

API đã chọn: YouTube Data API v3
5

Nhấp nút Tạo

Nhấp nút Tạo ở dưới cùng để tạo API Key.

Tạo
Khóa được tạo thường bắt đầu bằng AIzaSy.... Chuỗi này chính là YouTube API Key của bạn.

5. Ví dụ sử dụng API Key

Bạn có thể kiểm tra bằng cách đặt khóa đã tạo vào phần YOUR_API_KEY.

Ví dụ tìm kiếm videohttps://www.googleapis.com/youtube/v3/search?part=snippet&q=korea&type=video&key=YOUR_API_KEY
Ví dụ tra cứu thông tin kênhhttps://www.googleapis.com/youtube/v3/channels?part=snippet&id=CHANNEL_ID&key=YOUR_API_KEY
Nếu kết quả dạng JSON xuất hiện khi bạn nhập URL trên vào thanh địa chỉ của trình duyệt, nghĩa là API Key đang hoạt động đúng.

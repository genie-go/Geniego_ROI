Hướng dẫn cấp API Key

Hướng dẫn cấp API key dùng chung cho Gmarket·Auction từ ESM Trading API (etapi.gmarket.com) và đăng ký nó trong GenieGo Integration Hub. Gmarket và Auction được vận hành chung thông qua ESM Plus, vì vậy một API key duy nhất kết nối cả hai kênh cùng lúc.

1. Trước khi bắt đầu

Bạn phải được đăng ký làm người bán trên cả Gmarket và Auction và có ESM Plus master ID.

Bắt buộc

Tài khoản người bán + ESM master ID

Bạn cần tài khoản người bán cho cả Auction và Gmarket, cùng với một master ID được tạo trong ESM Plus (esmplus.com).

Khóa được cấp

API key · Seller ID

Sau khi ESM Trading API được phê duyệt, hãy kiểm tra API key và seller ID được cấp cùng nhau.

Lưu ý: ESM Trading API cần được phê duyệt, và đơn đăng ký của bạn có thể bị từ chối. Khi đăng ký, bạn phải gửi ba mục: Auction seller ID, Gmarket seller ID và ESM master ID của bạn.

2. Các bước cấp khóa

Đăng ký và cấp khóa theo thứ tự dưới đây, sau đó đăng ký trong GenieGo.

1

Đăng ký làm người bán và tạo ESM master ID

esmplus.com

Đăng ký quyền truy cập ESM Trading API

Truy cập ESM Trading API và đăng ký quyền truy cập API. Vì cần được phê duyệt nên có thể bị từ chối.

https://etapi.gmarket.com
3

Gửi thông tin nhận dạng

Khi đăng ký, hãy gửi ba mục: Auction seller ID, Gmarket seller ID và ESM master ID của bạn.

4

Nhận khóa

Sau khi được phê duyệt, một API key sẽ được cấp. Hãy kiểm tra API key và seller ID được cấp.

Khi đơn đăng ký của bạn được phê duyệt và API key được cấp, nghĩa là đã xử lý đúng cách. Đối với các thắc mắc liên quan đến phê duyệt, hãy liên hệ etapihelp@gmail.com.

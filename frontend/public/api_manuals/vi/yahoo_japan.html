1. Khái niệm cấp API Yahoo! Japan

Tùy theo loại dịch vụ, các API thuộc Yahoo! Japan sẽ cấp Client ID, Client Secret và thông tin xác thực OAuth từ Yahoo! JAPAN Developers hoặc LY/Yahoo!広告 Developer Center.

Quan trọng: "Yahoo! Japan" có trang cấp và quy trình xét duyệt khác nhau tùy theo API bạn muốn dùng. Bạn cần phân biệt các API Mua sắm, Quảng cáo, Đăng nhập và Tìm kiếm.

2. Vật chuẩn bị

  • Yahoo! JAPAN ID hoặc tài khoản doanh nghiệp
  • Quyết định loại API sẽ sử dụng
  • Tên ứng dụng và mô tả dịch vụ
  • Redirect URI
  • URL chính sách quyền riêng tư
  • Tên miền vận hành hoặc tên miền thử nghiệm

3. Trình tự cơ bản đăng ký ứng dụng

1

Truy cập trang nhà phát triển

Đăng nhập vào Yahoo! JAPAN Developers hoặc Developer Center của API tương ứng.

developer.yahoo.co.jp / ads-developers.yahoo.co.jp
2

Tạo ứng dụng

Nhấp vào menu thêm ứng dụng mới.

Create Application / Add Application
3

Nhập thông tin ứng dụng

Nhập tên ứng dụng, mô tả, thông tin người phụ trách và URL trang web.

Application Information
4

Nhập Redirect URI

Nhập chính xác URL sẽ quay về sau khi xác thực OAuth.

Redirect URI
5

Chọn quyền hoặc API

Chọn quyền API và phạm vi (scope) bạn muốn sử dụng.

Scope / API Permission
6

Hoàn tất đăng ký

Sau khi đăng ký, kiểm tra Client ID và Client Secret.

Client ID / Client Secret

5. Luồng cấp OAuth Token

1) Tạo URL xác thực người dùng bằng Client ID 2) Người dùng đăng nhập bằng tài khoản Yahoo! 3) Nhận authorization code tại Redirect URI 4) Yêu cầu Access Token bằng authorization code + Client Secret 5) Gọi API bằng Access Token 6) Làm mới bằng Refresh Token khi cần

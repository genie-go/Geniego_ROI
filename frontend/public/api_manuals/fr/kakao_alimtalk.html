1. Concept d'émission de Kakao Alimtalk

Alimtalk est un service de messagerie professionnelle qui envoie des messages informatifs tels que des commandes, des réservations, des expéditions et des paiements via Kakao Talk. Ce n'est pas une structure permettant d'envoyer directement avec une simple clé d'application développeur Kakao ; elle nécessite la vérification commerciale du canal et la configuration d'une agence d'envoi/plateforme.

Important : Alimtalk est axé sur les messages informatifs plutôt que promotionnels, et les modèles nécessitent un examen préalable.

2. Prérequis

  • Compte Kakao Business
  • Canal Kakao Talk
  • Documents de vérification commerciale tels qu'un certificat d'immatriculation d'entreprise
  • Le texte du modèle Alimtalk à envoyer
  • Un compte auprès d'une agence d'envoi ou d'un service de messagerie cloud
  • Une politique de collecte des numéros de téléphone des destinataires et de gestion du consentement

3. Préparation du canal Kakao Talk

1

Accéder à Kakao Business

Connectez-vous au centre d'administration Kakao Business.

business.kakao.com
2

Créer un canal Kakao Talk

Créez un canal au nom de votre marque ou de votre service.

Centre d'administration des canaux Kakao Talk
3

Effectuer la vérification commerciale

Soumettez les informations de votre entreprise et les documents justificatifs pour vérifier le canal commercial.

Vérification commerciale
4

Vérifier l'ID de recherche du canal

Vérifiez l'ID du canal ou l'ID de recherche requis pour les paramètres d'envoi Alimtalk.

Profil du canal → ID de recherche
La vérification commerciale doit être terminée avant de pouvoir enregistrer un profil d'expéditeur Alimtalk.

4. Enregistrement du profil d'expéditeur / Sender Profile

1

Choisir une plateforme d'envoi

Sélectionnez un fournisseur Alimtalk tel que NHN Cloud, Naver Cloud SENS, Infobank ou Bizgo.

Messaging Provider
2

Créer un projet

Créez un projet Alimtalk sur la plateforme que vous avez choisie.

Project / Service
3

Connecter le canal Kakao Talk

Enregistrez les informations de votre canal Kakao Talk vérifié commercialement.

Sender Profile
4

Examen du profil d'expéditeur

Effectuez la confirmation de la propriété du canal et l'examen du profil d'expéditeur.

Profile Review
5

Confirmer les identifiants d'API

Vérifiez l'App Key, le Secret Key, l'API Key, etc. fournis par la plateforme.

API Key / Secret

5. Enregistrement et approbation des modèles

1

Rédiger le modèle

Rédigez des textes informatifs tels que des confirmations de commande et des avis d'expédition.

Template Message
2

Définir les variables

Définissez comme variables les valeurs changeant dynamiquement, telles que le nom, le numéro de commande et le numéro de suivi.

#{variable}
3

Demander l'examen

Enregistrez le modèle et soumettez la demande d'examen Kakao.

Request Approval
4

Confirmer l'approbation

Confirmez le code de modèle ou l'ID approuvé.

Template Code
Si le texte avant examen diffère du texte réellement envoyé, l'envoi peut échouer ou des sanctions peuvent s'appliquer.

6. Exemple de flux d'envoi via API

1) Appelez l'Alimtalk API Endpoint depuis votre serveur
2) Authentifiez-vous avec l'API Key ou le Secret
3) Transmettez le numéro de téléphone du destinataire, le code de modèle et les valeurs des variables
4) Recevez le résultat de l'envoi
5) Gérez l'envoi ou non d'un SMS de secours en cas d'échec

7. Liste de contrôle finale

  • J'ai créé un canal Kakao Talk.
  • J'ai terminé la vérification commerciale.
  • J'ai sélectionné une plateforme d'envoi Alimtalk.
  • J'ai enregistré un Sender Profile.
  • J'ai confirmé l'API Key ou le Secret.
  • J'ai enregistré le modèle et obtenu son approbation.
  • J'ai préparé une politique d'échec d'envoi et de SMS de secours.

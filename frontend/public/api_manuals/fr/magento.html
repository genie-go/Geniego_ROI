🟧 Guide du débutant

Magento (Adobe Commerce)
Manuel d'émission de clé API

Un guide pas à pas convivial pour les débutants permettant de créer une intégration (Integration) dans l'administration Magento (Adobe Commerce) afin d'émettre un jeton d'accès et de l'enregistrer dans le GenieGo Integration Hub.

Temps estiméEnviron 5–15 min
Compte nécessaireDroits d'administrateur Magento (Adobe Commerce)
Clés émisesJeton d'accès Integration
RésultatIdentifiants REST API

1. Avant de commencer

Une intégration ne peut être créée qu'avec des droits d'administrateur.

Requis

Droits d'administrateur Magento

Vous avez besoin de droits d'administrateur Magento (Adobe Commerce). Une intégration ne peut pas être créée avec des droits d'utilisateur ordinaire.

Clés émises

Jeton d'accès Integration

Lorsque vous créez une intégration et cliquez sur Activate, un jeton d'accès est émis.

2. Étapes d'émission

Émettez dans l'ordre ci-dessous et enregistrez dans GenieGo.

1

Accéder au menu Integration

Dans l'administration Magento, allez dans [Système] > [Integrations].

Administration → Système → Integrations
2

Créer une intégration

Créez une intégration avec [Add New Integration] et configurez les autorisations de ressources.

Add New Integration
3

Émettre le jeton via Activate

Cliquez sur [Activate] pour obtenir le jeton d'accès.

Activate
4

Vérifier les informations

Confirmez le jeton d'accès émis et l'URL de votre boutique (base_url).

Une fois que le jeton d'accès s'affiche après Activate, il a été émis correctement.

3. Informations à enregistrer dans GenieGo

ChampDescription
base_urlURL de la boutique
access_tokenLe jeton d'accès Integration émis (à manipuler avec soin)
Saisissez base_url·access_token dans [Enregistrer] du GenieGo Integration Hub et enregistrez. Les identifiants sont chiffrés au repos avec AES-256-GCM et masqués à la lecture.

4. Problèmes fréquents

ProblèmeCauseSolution
Le menu Integrations est absentDroits d'administrateur insuffisantsConnectez-vous avec un compte administrateur disposant de l'autorisation de créer des intégrations.
Erreur d'autorisation lors des appelsAutorisations de ressources de l'intégration non définies, ou Activate non terminéDéfinissez les autorisations de ressources de l'intégration et terminez Activate pour émettre le jeton.

Liste de vérification finale

  • Connecté avec des droits d'administrateur Magento (Adobe Commerce).
  • Créé une intégration via Add New Integration dans Système > Integrations.
  • Défini les autorisations de ressources et émis le jeton d'accès via Activate.
  • Saisi base_url·access_token dans GenieGo [Enregistrer] et enregistré.

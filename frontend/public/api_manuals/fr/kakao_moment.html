Kakao Moment
Manuel de délivrance de clé API

Un guide pour demander l'autorisation Kakao Moment Open API, obtenir votre jeton d'accès et votre ID de compte publicitaire, puis les enregistrer dans le Hub d'intégration GenieGo. L'autorisation Open API n'est accordée qu'aux agences de publicité officielles de Kakao.

1. Avant de commencer

Kakao Moment Open API n'est pas auto-délivré ; il suit un processus de demande et d'examen d'autorisation.

Important : L'autorisation Kakao Moment Open API n'est accordée qu'aux agences de publicité officielles de Kakao. Les annonceurs ordinaires ne peuvent pas l'auto-délivrer.
Requis

Compte publicitaire + Business Channel

Vous avez besoin d'un compte publicitaire Kakao Moment et d'un Kakao Business Channel.

Clés délivrées

Jeton d'accès · ID de compte publicitaire

Une fois l'autorisation approuvée, vous obtenez un jeton d'accès via l'autorisation OAuth et confirmez votre ID de compte publicitaire.

2. Processus de demande de délivrance

Procédez dans cet ordre : demande d'autorisation Open API → examen/approbation → délivrance du jeton → enregistrement.

1

Créez une application et basculez vers une application Business

Créez une application sur developers.kakao.com et basculez-la vers une [Business App].

developers.kakao.com
2

Demandez l'autorisation Open API

Soumettez la Kakao Moment Open API [Additional Feature Application]. L'autorisation est accordée après examen de votre éligibilité en tant qu'agence officielle.

Additional Feature Application
3

Délivrez le jeton

Une fois l'autorisation approuvée, obtenez un jeton d'accès via l'autorisation OAuth et confirmez votre ID de compte publicitaire. Vous pouvez utiliser l'OAuth en un clic dans le Hub d'intégration GenieGo.

Une fois l'autorisation approuvée et votre jeton d'accès ainsi que votre ID de compte publicitaire confirmés, vous pouvez procéder à l'enregistrement.

3. Informations à enregistrer dans GenieGo

ChampDescription
access_tokenLe jeton d'accès délivré via l'autorisation OAuth (à manipuler avec précaution)
ad_account_idID de compte publicitaire Kakao Moment
Saisissez-les et enregistrez-les dans le Hub d'intégration GenieGo [Register] ; les identifiants sont chiffrés au repos avec AES-256-GCM et masqués à la lecture.

4. Problèmes fréquents

ProblèmeCauseSolution
Impossible d'obtenir l'autorisation Open APICe n'est pas une agence de publicité officielleL'autorisation Open API n'est accordée qu'aux agences de publicité officielles de Kakao. Faites la demande via une agence.
Impossible de poursuivre [Additional Feature Application]L'application n'a pas été basculée vers une application BusinessBasculez d'abord l'application vers une [Business App], puis demandez la fonctionnalité supplémentaire.

Liste de contrôle finale

  • Préparé un compte publicitaire Kakao Moment et un Kakao Business Channel.
  • Basculé l'application vers une [Business App] et soumis l'Open API [Additional Feature Application].
  • Après l'approbation de l'autorisation, confirmé le jeton d'accès et l'ID de compte publicitaire via l'autorisation OAuth.
  • Saisi access_token·ad_account_id dans GenieGo [Register] et enregistré.

🔷 Guide du débutant

Razorpay
Manuel d'émission de clé API

Un guide pas à pas, adapté aux débutants, pour créer et vérifier des clés d'API en mode Live (Key ID·Key Secret) dans le Razorpay Dashboard et les enregistrer dans le GenieGo Integration Hub.

Temps estiméEnviron 5–10 min
Compte requisCompte Razorpay (axé sur l'Inde)
Clés émisesKey ID (rzp_live_) · Key Secret
RésultatIdentifiants Live API

1. Avant de commencer

Les clés d'API Razorpay sont créées dans le mode Live du Dashboard.

Requis

Compte Razorpay

Vous avez besoin d'un compte Razorpay (axé sur l'Inde). Vous devez pouvoir vous connecter au Dashboard avec ce compte.

Clés émises

Key ID · Key Secret

Créez et vérifiez le Key ID et le Key Secret en mode Live dans le Dashboard, sous Settings > API Keys.

Remarque : Veillez à créer les clés en mode Live. Le Key ID commence par rzp_live_.

2. Étapes d'émission

Émettez dans l'ordre ci-dessous et enregistrez dans GenieGo.

1

Ouvrir le Dashboard

Connectez-vous au Razorpay Dashboard.

dashboard.razorpay.com
2

Aller dans API Keys

Allez dans [Settings] > [API Keys].

Settings → API Keys
3

Créer et vérifier les clés

En mode Live, créez et vérifiez le Key ID et le Key Secret.

Si vous voyez le Key ID de mode Live (rzp_live_) et le Key Secret sous Settings > API Keys, ils ont été créés correctement.

3. Informations à enregistrer dans GenieGo

ChampDescription
key_idKey ID de mode Live (commence par rzp_live_)
key_secretKey Secret de mode Live (à manipuler avec précaution)
Dans le GenieGo Integration Hub [Enregistrer], saisissez votre key_id·key_secret et enregistrez. Les identifiants sont chiffrés au repos avec AES-256-GCM et masqués à la lecture.

4. Problèmes fréquents

ProblèmeCauseSolution
Échec de l'authentificationUtilisation d'une clé de mode TestVérifiez que vous utilisez un Key ID (rzp_live_)·Key Secret créé en mode Live.
Impossible de revoir le Key SecretLe Key Secret n'est affiché qu'une seule fois à la créationConservez le Key Secret en lieu sûr lors de la création ; en cas de perte, créez-en un nouveau.

Liste de contrôle finale

  • Connecté à dashboard.razorpay.com avec un compte Razorpay.
  • Créé des clés de mode Live sous Settings > API Keys.
  • Vérifié le Key ID (rzp_live_)·Key Secret.
  • Saisi key_id·key_secret dans GenieGo [Enregistrer] et enregistré.

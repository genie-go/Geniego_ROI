Facebook Live
Manuel d'Émission d'API

Un guide adapté aux débutants pour créer une application sur Meta for Developers et préparer l'App ID, l'App Secret et le jeton d'accès à la Page (Page Access Token) nécessaires pour connecter la diffusion en direct d'une page Facebook.

Concept cléPas une API Key, mais un Access Token
Site développeursdevelopers.facebook.com
API utiliséeLive Video API (Graph)
Autorisations requisesAdministrateur de page + Page Token

1. L'essentiel à connaître d'abord

Sur YouTube, une seule API Key suffit, mais Facebook (Meta) fonctionne avec une structure App ID · App Secret · Access Token.

Important : Avec Meta, il ne suffit pas de "copier une seule API Key". Vous devez généralement suivre l'ordre suivant : créer une application → ajouter un produit → configurer les autorisations → émettre un Page Access Token → examen de l'application.

2. Prérequis

  • Un compte Meta/Facebook
  • Un accès à Meta for Developers
  • Une page Facebook sur laquelle vous avez des droits d'administrateur
  • Une URL de politique de confidentialité · une URL de conditions d'utilisation pour le service réel

3. Créer une application Meta Developer

Toute intégration de l'API Meta commence par la création d'une application.

1

Accéder à Meta for Developers

Connectez-vous à Meta for Developers dans votre navigateur.

developers.facebook.com
2

Mes applications → Créer une application

Cliquez sur My Apps → Create App en haut.

My Apps → Create App
3

Choisir le type d'application

L'intégration de plateformes/le live commerce se rapproche davantage d'un objectif professionnel.

Sélectionner le type Business
4

Saisir le nom de l'application·l'e-mail → créer

Saisissez le nom de l'application et un e-mail de contact, puis créez l'application.

App Name / Contact Email
Après avoir créé l'application, vérifiez l'App ID et l'App Secret dans Tableau de bord → Paramètres de l'application → Paramètres de base. (N'exposez jamais l'App Secret dans le frontend ou un dépôt public.)

4. Flux de configuration de l'API Facebook Live

1

Ajouter un produit

Sur le tableau de bord de l'application, préparez les paramètres liés à Facebook Login / Graph API.

Add Product → Facebook Login
2

Connecter une page + jeton de page

Pour diffuser un direct sur une page, vous avez besoin de droits d'administrateur de page et d'un Page Access Token.

Page Access Token
3

Demander les autorisations requises (App Review)

Demandez des autorisations destinées à publier du contenu·gérer des pages·créer des vidéos en direct (p. ex. pages_read_engagement, pages_manage_posts).

Permissions → App Review
4

Émettre·tester le jeton avec Graph API Explorer

Émettez un jeton incluant les autorisations et testez les appels d'API.

Tools → Graph API Explorer
Exemple d'appel de création Facebook Live
POST /{page-id}/live_videos
access_token={PAGE_ACCESS_TOKEN}
En cas de création réussie, les informations de diffusion en direct telles que l'ID de diffusion·l'URL du flux·la clé de flux sont renvoyées. Consultez l'ID de page (page_id) dans les informations de la page.

5. Enregistrer dans le GenieGo Integration Hub

1

Copier le Page Access Token

Copiez le Page Access Token que vous avez émis (un jeton de longue durée est recommandé).

access_token (Page token)
2

Confirmer l'ID de page

Confirmez l'ID (page_id) de la page Facebook cible.

page_id
3

Enregistrer dans l'Integration Hub

Dans GenieGo Integration Hub → Facebook Live → [Enregistrer], saisissez les valeurs ci-dessous et enregistrez.

access_token · page_id
Les clés enregistrées sont chiffrées au repos avec AES-256-GCM et masquées à la lecture.

6. Types d'Access Token

Type de jetonUsageRemarques
User Access TokenAppels d'API basés sur la connexion de l'utilisateurLes données accessibles varient selon l'étendue des autorisations accordées.
Page Access TokenPublication sur une page · création de directs de pageDes droits d'administrateur de page et les autorisations associées sont requis.
Long-Lived TokenUn jeton de longue durée avec une expiration prolongéeEn production, une logique d'expiration·de renouvellement est nécessaire.

7. Liste de contrôle

  • Créé une application Meta Developer.
  • Confirmé l'App ID et l'App Secret.
  • Connecté une page Facebook et émis un Page Access Token.
  • Organisé·demandé les autorisations requises (App Review).
  • Saisi access_token·page_id dans GenieGo [Enregistrer] et enregistré.
  • Stocké le jeton en toute sécurité dans les variables d'environnement du serveur.

8. Problèmes fréquents

ProblèmeCauseSolution
Aucun menu API KeyMeta n'utilise pas une structure d'API Key uniqueProcédez avec la structure App ID·App Secret·Access Token.
Erreur d'autorisationLe jeton ne dispose pas des autorisations requisesAjoutez les autorisations dans Graph API Explorer puis réémettez le jeton.
Échec de la création du direct de pagePage Access Token/droits d'administrateur insuffisantsVérifiez les droits d'administrateur de page et les autorisations du jeton.

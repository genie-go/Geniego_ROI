🔶 Guide du débutant

11Street
Manuel de délivrance de la clé API

Un guide pas à pas, adapté aux débutants, pour délivrer une clé Open API depuis le centre Open API de 11Street (openapi.11st.co.kr) et l'enregistrer dans le GenieGo Integration Hub.

Temps estiméEnviron 5–15 min
Compte requisCompte vendeur 11Street
Clés délivréesClé Open API · Seller ID
RésultatIdentifiants Open API

1. Avant de commencer

Les clés Open API de 11Street ne peuvent être délivrées que depuis un compte vendeur.

Requis

Compte vendeur 11Street

Vous avez besoin d'un compte vendeur 11Street. Connectez-vous au centre Open API avec votre Seller ID.

Clés délivrées

Clé Open API · Seller ID

Une fois l'enregistrement du service et l'enregistrement de l'IP d'accès terminés, vérifiez la clé Open API délivrée ainsi que votre Seller ID.

Remarque : Vous devez enregistrer une IP de serveur autorisée à accéder à l'API avant de pouvoir utiliser la clé.

2. Étapes de délivrance

Délivrez dans l'ordre ci-dessous et enregistrez dans GenieGo.

1

Connectez-vous au centre Open API

Connectez-vous au centre Open API de 11Street avec votre Seller ID.

https://openapi.11st.co.kr/openapi/OpenApiFrontMain.tmall
2

Enregistrez le service

Dans le menu, cliquez sur [Enregistrement·Vérification du service], saisissez les informations du vendeur, acceptez les conditions et enregistrez.

Enregistrement·Vérification du service
3

Enregistrez l'IP d'accès

Dans [Informations Seller API] > [Modifier], enregistrez l'IP du serveur autorisée à accéder à l'API.

Informations Seller API → Modifier
4

Copiez la clé API

Copiez la clé API délivrée.

Une fois l'enregistrement du service et l'enregistrement de l'IP d'accès terminés, la clé API est délivrée.

3. Informations à enregistrer dans GenieGo

ChampDescription
api_keyLa clé Open API délivrée
seller_idSeller ID
Saisissez-la et enregistrez-la dans le GenieGo Integration Hub [Enregistrer]. Les identifiants sont chiffrés au repos avec AES-256-GCM et masqués à la lecture.

4. Problèmes fréquents

ProblèmeCauseSolution
La clé API n'est pas délivréeEnregistrement du service ou enregistrement de l'IP d'accès non terminéTerminez [Enregistrement·Vérification du service] et l'enregistrement de l'IP d'accès dans [Informations Seller API].
Erreur d'authentification lors des appelsIP de serveur autorisée non enregistréeVérifiez à nouveau l'IP du serveur autorisée dans [Informations Seller API] > [Modifier].

5. Avant d'enregistrer des produits (important)

Même avec une clé API valide, 11Street refuse l'enregistrement si ces trois éléments manquent.

  • Catégorie d'affichage (dispCtgrNo) — à définir dans GenieGo [Synchronisation du catalogue] > [Mappage des catégories] ou dans le panneau « Catégorie et marque à définir »
  • Marque — à créer dans GenieGo [Synchronisation du catalogue] > [Gestion des marques] puis à assigner au produit (le nom suffit, aucun code requis)
  • Mode de vente — GenieGo envoie automatiquement le prix fixe (01) ; aucune saisie nécessaire
Message d'erreurSignification réelleSolution
-997 Aucune information d'API enregistréeCe n'est PAS un échec d'authentification. L'URL d'API appelée (chemin + méthode) n'existe pas chez 11Street.Ne régénérez pas la clé. Faites vérifier l'URL appelée.
Code de marque manquantAucune marque n'est assignée au produit.Créez la marque dans [Synchronisation du catalogue] > [Gestion des marques] et assignez-la.
Mode de vente indisponible via APILe code de mode de vente n'est pas autorisé par l'API.GenieGo envoie automatiquement le prix fixe (01). Aucune action requise.
Vérifiez ceci avant de régénérer la clé. L'erreur -997 n'est pas un problème de clé, d'IP ou de demande d'API. Même si les trois sont valides, une URL erronée renvoie -997.

Liste de contrôle finale

  • Connecté au centre Open API avec un compte vendeur 11Street.
  • Saisi les informations du vendeur et accepté les conditions dans [Enregistrement·Vérification du service].
  • Enregistré l'IP du serveur autorisée dans [Informations Seller API].
  • Confirmé la clé Open API délivrée et le Seller ID.
  • Saisi api_key·seller_id dans GenieGo [Enregistrer] et enregistré.

📮 Guide pour débutants

FedEx
Manuel d'émission de la clé API

Un guide qui résume le processus de création d'un projet sur le portail développeur FedEx (developer.fedex.com) pour émettre une API Key et une Secret Key, puis de les enregistrer avec votre numéro de compte FedEx dans le GenieGo Integration Hub.

Temps estiméEnviron 5–15 min
Compte nécessaireCompte FedEx (numéro de compte)
Clés émisesAPI Key · Secret Key
RésultatIdentifiants API · numéro de compte

1. Avant de commencer

Les identifiants FedEx sont émis en créant un projet sur le portail développeur, et un numéro de compte est requis.

Prérequis

Numéro de compte FedEx

Vous avez besoin d'un numéro de compte FedEx. Le numéro de compte est saisi en même temps lors de l'enregistrement dans GenieGo.

Clés émises

API Key · Secret Key

Lorsque vous créez un projet sur le portail développeur, une API Key et une Secret Key sont émises.

2. Étapes d'émission

Émettez dans l'ordre ci-dessous et enregistrez dans GenieGo.

1

S'inscrire et se connecter au portail développeur

Inscrivez-vous au portail développeur FedEx et connectez-vous.

https://developer.fedex.com
2

Créer un projet

Lorsque vous créez un projet, une API Key et une Secret Key sont émises.

Créer un projet
3

Vérifier les clés et le numéro de compte

Vérifiez l'API Key et la Secret Key émises ainsi que votre numéro de compte FedEx.

Une fois l'API Key et la Secret Key émises après la création du projet, tout fonctionne correctement.

3. Informations à enregistrer dans GenieGo

ChampDescription
api_keyL'API Key émise
api_secretLa Secret Key émise (à manipuler avec précaution)
account_numberNuméro de compte FedEx
Saisissez-les et enregistrez-les dans le GenieGo Integration Hub [Enregistrer] ; les identifiants sont chiffrés au repos avec AES-256-GCM et masqués à la lecture.

4. Problèmes fréquents

ProblèmeCauseSolution
Les clés ne sont pas visibles dans le projetCréation du projet non terminéeCréez un projet sur developer.fedex.com et l'API Key et la Secret Key seront émises.
Erreur d'authentification lors de l'appelSecret Key mal saisie ou numéro de compte manquantVérifiez à nouveau api_secret et le numéro de compte (account_number).

Liste de contrôle finale

  • Vérifié le numéro de compte FedEx.
  • Inscrit sur developer.fedex.com et créé un projet.
  • Émis l'API Key et la Secret Key.
  • Saisi api_key·api_secret·account_number dans GenieGo [Enregistrer] et enregistré.

🌳 Guide du débutant

Braintree
Manuel d'émission de clé API

Un guide pas à pas adapté aux débutants pour trouver vos identifiants Production API (Merchant ID, Public Key, Private Key) dans le Braintree (PayPal) Control Panel et les enregistrer dans le GenieGo Integration Hub.

Temps estiméEnviron 5–10 min
Compte requisCompte marchand Braintree (PayPal) (approbation requise)
Clés émisesMerchant ID · Public Key · Private Key
RésultatIdentifiants Production API

1. Avant de commencer

Les identifiants Braintree (PayPal) ne sont disponibles qu'après l'approbation de votre compte marchand.

Requis

Compte marchand Braintree (PayPal)

Vous avez d'abord besoin d'un compte marchand Braintree (PayPal) approuvé. Les identifiants Production ne sont disponibles qu'une fois votre compte marchand approuvé.

Clés émises

Merchant ID · Public/Private Key

Dans le Control Panel, allez dans Settings > API Keys pour trouver le Merchant ID, la Public Key et la Private Key de l'environnement Production.

Remarque : Veillez à utiliser les identifiants de l'environnement Production. Prenez soin de ne pas les confondre avec les clés Sandbox.

2. Procédure d'émission

Après l'approbation de votre compte marchand, trouvez les identifiants dans l'ordre ci-dessous et enregistrez-les dans GenieGo.

1

Approbation du compte marchand

Terminez d'abord l'approbation du compte marchand Braintree (PayPal). Vous ne pouvez passer à l'étape suivante qu'une fois l'approbation terminée.

2

Se connecter au Control Panel

Connectez-vous au Braintree Control Panel.

braintreegateway.com
3

Aller dans API Keys

Accédez au menu [Settings] > [API Keys].

Settings → API Keys
4

Trouver les identifiants

Trouvez le Merchant ID, la Public Key et la Private Key de l'environnement Production.

Si le Merchant ID, la Public Key et la Private Key de Production apparaissent dans Settings > API Keys, tout est en ordre.

3. Informations à enregistrer dans GenieGo

ChampDescription
merchant_idProduction Merchant ID
public_keyProduction Public Key
private_keyProduction Private Key (à manipuler avec précaution)
Saisissez merchant_id, public_key et private_key dans [Enregistrer] du GenieGo Integration Hub et enregistrez. Les identifiants sont chiffrés au repos avec AES-256-GCM et masqués à la lecture.

4. Problèmes courants

ProblèmeCauseSolution
Aucun identifiant dans le menu API KeysApprobation du compte marchand non terminéeTerminez d'abord l'approbation du compte marchand Braintree (PayPal).
Échec de l'authentificationClés Sandbox utilisées, ou Private Key mal saisieVérifiez à nouveau le Merchant ID, la Public Key et la Private Key de l'environnement Production.

Liste de contrôle finale

  • Le compte marchand Braintree (PayPal) a été approuvé.
  • Identifiants Production trouvés dans Settings > API Keys du Control Panel.
  • Merchant ID, Public Key et Private Key confirmés.
  • merchant_id, public_key et private_key saisis dans GenieGo [Enregistrer] et enregistrés.

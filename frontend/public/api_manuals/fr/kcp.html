💳 Guide pour débutants

NHN KCP
Manuel d'émission de la clé API

Un guide qui vous accompagne dans l'émission et la vérification de votre code de site (site_cd) et de votre clé de site dans l'administration marchand NHN KCP, puis leur enregistrement dans le GenieGo Integration Hub.

Temps estiméEnviron 5–10 min
Compte requisContrat marchand NHN KCP · compte administrateur
Clés émisesCode de site · Clé de site
RésultatIdentifiants d'intégration de paiement

1. Avant de commencer

Avec NHN KCP, vous émettez les informations de votre site dans l'administration après la conclusion d'un contrat marchand.

Requis

Contrat marchand · Compte administrateur

Votre contrat marchand NHN KCP doit être conclu et vous devez pouvoir vous connecter à l'administration marchand.

Clés émises

Code de site · Clé de site

Dans l'administration marchand, émettez ou vérifiez votre code de site (site_cd) et votre clé de site.

Remarque : L'enregistrement nécessite deux valeurs : le code de site (site_cd) et la clé de site.

2. Étapes d'émission

Émettez dans l'ordre ci-dessous et enregistrez dans GenieGo.

1

Se connecter à l'administration marchand

Connectez-vous à l'administration marchand NHN KCP.

https://admin8.kcp.co.kr
2

Émettre le code de site et la clé de site

Émettez ou vérifiez votre code de site (site_cd) et votre clé de site.

Une fois votre code de site et votre clé de site confirmés, vous êtes prêt à vous enregistrer.

3. Informations à enregistrer dans GenieGo

ChampDescription
site_cdLe code de site émis
site_keyLa clé de site émise (à manipuler avec précaution)
Saisissez les valeurs dans le GenieGo Integration Hub sous [Enregistrer] et enregistrez. Les identifiants sont chiffrés au repos avec AES-256-GCM et masqués à la lecture.

4. Problèmes courants

ProblèmeCauseSolution
Impossible de se connecter à l'administrationContrat marchand non concluConnectez-vous avec un compte administrateur dont le contrat marchand est conclu.
Échec de l'authentification du paiementCode de site ou clé de site mal saisisVérifiez à nouveau le site_cd·site_key émis dans l'administration marchand.

Liste de contrôle finale

  • J'ai conclu le contrat marchand NHN KCP et me suis connecté à l'administration marchand.
  • J'ai émis et vérifié le code de site (site_cd) et la clé de site.
  • J'ai saisi site_cd·site_key dans GenieGo [Enregistrer] et enregistré.
  • J'ai confirmé que les identifiants s'affichent masqués après l'enregistrement.

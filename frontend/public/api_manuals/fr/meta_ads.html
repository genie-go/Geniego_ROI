📘 Guide pour débutants

Meta Ads
Manuel d'émission de la clé API

Un guide pas à pas adapté aux débutants pour émettre un jeton d'accès d'utilisateur système depuis votre application professionnelle Meta et l'enregistrer avec votre ID de compte publicitaire (act_) dans le GenieGo Integration Hub afin de connecter vos données publicitaires Meta Ads.

Durée estiméeEnviron 10–20 min
Compte requisCompte Facebook personnel · Compte publicitaire Meta
Clés émisesJeton d'accès · ID de compte publicitaire (act_)
RésultatIdentifiants Marketing API

1. Avant de commencer

L'intégration de Meta Ads nécessite un compte professionnel et un compte publicitaire diffusant activement des publicités.

Requis

Compte Facebook personnel · Compte publicitaire

Vous avez besoin d'un compte Facebook personnel et d'un compte publicitaire Meta diffusant activement des publicités.

Clés émises

Jeton d'accès · ID de compte publicitaire (act_)

Vous obtiendrez un jeton d'utilisateur système (jeton d'accès) et un ID numérique de compte publicitaire commençant par act_.

Remarque : L'accès aux données en dehors de votre propre entreprise nécessite un examen de l'application (App Review). Si vous ne connectez que des comptes publicitaires au sein de votre propre entreprise, vous pouvez continuer sans examen.

2. Étapes d'émission

Émettez dans l'ordre ci-dessous et enregistrez dans GenieGo.

1

Créer et vérifier un compte professionnel

Ouvrez le site dans votre navigateur, connectez-vous, puis créez un compte professionnel via [Créer une entreprise] et terminez la vérification par e-mail.

business.facebook.com
2

Créer une application

Cliquez sur [Mes applications] > [Créer une application] en haut à droite, choisissez le type "Entreprise" et créez l'application.

developers.facebook.com → Mes applications → Créer une application
3

Ajouter la Marketing API

Sur le tableau de bord de l'application, trouvez la carte "Marketing API" et cliquez sur [Configurer] pour l'ajouter en tant que produit.

4

Créer un utilisateur système et attribuer des autorisations

Accédez à Paramètres de l'entreprise > [Utilisateurs] > [Utilisateurs système] > [Ajouter] pour créer un utilisateur système, puis accordez les autorisations de compte publicitaire sous [Attribuer des actifs].

business.facebook.com/settings → Utilisateurs → Utilisateurs système
5

Générer un jeton d'accès

Sur le même écran, cliquez sur [Générer un jeton], sélectionnez votre application, cochez les autorisations (ads_read, ads_management), générez le jeton et copiez sa valeur.

6

Trouver votre ID de compte publicitaire

Dans le Gestionnaire de publicités, trouvez l'ID numérique de compte publicitaire qui commence par act_.

adsmanager.facebook.com
Une fois le jeton généré avec les autorisations ads_read et ads_management cochées, il a été émis correctement.

3. Informations à enregistrer dans GenieGo

ChampDescription
access_tokenLe jeton d'accès d'utilisateur système émis (à manipuler avec précaution)
ad_account_idID de compte publicitaire commençant par act_
Dans le GenieGo Integration Hub > le [Enregistrer] du canal, collez votre jeton d'accès et votre ID de compte publicitaire (act_), puis enregistrez. Les identifiants sont chiffrés au repos avec AES-256-GCM et masqués à la lecture.

4. Problèmes courants

ProblèmeCauseSolution
Impossible d'interroger les données en raison d'erreurs d'autorisationAutorisations du jeton (ads_read·ads_management) non cochées, ou autorisation de compte publicitaire non attribuéeAttribuez l'autorisation de compte publicitaire à l'utilisateur système et cochez les autorisations ads_read·ads_management lors de la génération du jeton.
L'accès aux données d'une autre entreprise est refuséExamen de l'application (App Review) non terminéPour accéder aux données en dehors de votre propre entreprise, demandez et terminez l'examen de l'application.
Erreur de format de l'ID de compte publicitairePréfixe act_ manquant, ou chiffres uniquementDans le Gestionnaire de publicités, trouvez l'ID complet qui commence par act_ et saisissez-le.

Liste de vérification finale

  • Créé un compte professionnel sur business.facebook.com et terminé la vérification par e-mail.
  • Créé une application de type Entreprise sur developers.facebook.com et ajouté la Marketing API.
  • Créé un utilisateur système et attribué l'autorisation de compte publicitaire.
  • Généré et copié un jeton d'accès avec les autorisations ads_read·ads_management.
  • Trouvé l'ID de compte publicitaire qui commence par act_.
  • Saisi access_token·ad_account_id dans GenieGo [Enregistrer] et enregistré.

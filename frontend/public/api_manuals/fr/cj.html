CJ Logistics
Manuel d'émission de la clé API

L'API dédiée de CJ Logistics est émise sur la base d'un code de contrat (client). Ce manuel vous guide pour obtenir le code de contrat (client) et la clé API auprès de votre représentant commercial, puis les enregistrer dans le GenieGo Integration Hub.

Temps estiméDe plusieurs jours à plusieurs semaines, selon l'avancement du contrat
Compte requisContrat CJ Logistics (client entreprise)
Clés émisesClé API · Code client (contrat)
RésultatIdentifiants d'API

1. Avant de commencer

L'API dédiée de CJ Logistics est un canal basé sur un contrat et n'est pas émise instantanément depuis une console en libre-service.

Requis

Code de contrat (client) CJ Logistics

Vous devez disposer d'un code de contrat (client) émis dans le cadre d'un contrat avec CJ Logistics. Il n'est pas émis directement depuis une console distincte.

Recommandé

Smart Parcel (suivi unifié)

Si vous n'avez besoin que du suivi des livraisons, vous pouvez connecter tous les transporteurs avec une seule t_key via le canal [Smart Parcel (suivi unifié)].

Remarque : Si votre objectif est le suivi des livraisons, nous recommandons de vous connecter via la clé unifiée Smart Parcel (SweetTracker) sans contrat d'API dédié.

2. Procédure de demande d'émission

Faites la demande et enregistrez dans l'ordre ci-dessous.

1

Préparer le code de contrat (client)

Préparez votre code de contrat (client) CJ Logistics. L'API dédiée est fournie sur une base contractuelle.

2

Décider si vous avez besoin du suivi unifié

Si vous n'avez besoin que du suivi des livraisons, vous pouvez connecter tous les transporteurs avec une seule t_key via le canal [Smart Parcel (suivi unifié)] plutôt que le canal dédié de CJ Logistics.

3

Demander l'API dédiée

Si vous avez besoin de l'API dédiée de CJ Logistics, obtenez le code de contrat (client) et la clé API auprès de votre représentant commercial.

4

Enregistrer dans GenieGo

Saisissez api_key et cust_code dans [Enregistrer] du GenieGo Integration Hub et enregistrez.

Une fois que vous avez reçu la clé API et le code de contrat (client) de votre représentant commercial, vous êtes prêt à enregistrer.

3. Informations à enregistrer dans GenieGo

ChampDescription
api_keyLa clé API dédiée de CJ Logistics émise
cust_codeCode de contrat (client) CJ Logistics
Lorsque vous la saisissez et l'enregistrez dans [Enregistrer] du GenieGo Integration Hub, les identifiants sont chiffrés au repos avec AES-256-GCM et masqués à la lecture.

🛒 Beginner Guide

Coupang Wing
Manuel d'émission de clé API

Un guide pas à pas adapté aux débutants pour émettre des identifiants Open API (Access Key·Secret Key) dans la console vendeur Coupang WING et les enregistrer dans le GenieGo Integration Hub.

Durée estiméeEnviron 5–10 min
Compte requisCompte vendeur Coupang WING
Clés émisesAccess Key · Secret Key
RésultatIdentifiants Open API

1. Avant de commencer

L'Open API de Coupang ne peut être émise qu'à partir d'un compte vendeur.

Requis

Compte vendeur WING

Vous avez besoin d'un compte vendeur Coupang WING dont la vérification d'entreprise est terminée. L'émission est impossible avec un compte membre acheteur ordinaire.

Clés émises

Access Key · Secret Key

Sur l'écran de fin d'émission, repérez l'Access Key et la Secret Key, et vérifiez également le vendor ID.

Remarque : Lors de l'émission, vous devez saisir le nom de votre entreprise, l'URL de la page d'accueil/du service (utilisez wing.coupang.com à défaut) et les IP d'accès (jusqu'à 10).

2. Étapes d'émission de la clé Open API

Émettez dans l'ordre ci-dessous et enregistrez dans GenieGo.

1

Connectez-vous à WING

Connectez-vous avec votre compte vendeur.

wing.coupang.com
2

Accédez au menu Open API

Accédez au menu [Open API] ou [Gestion des clés API] dans la zone supérieure/des paramètres.

Open API → Gestion des clés API
3

Émettez une clé Open API

Cliquez sur [Émettre une clé Open API] et saisissez le nom de votre entreprise, l'URL de la page d'accueil et les IP d'accès (jusqu'à 10).

Émettre une clé Open API
4

Vérifiez les clés

Sur l'écran de fin d'émission, repérez l'Access Key·Secret Key et le vendor ID.

Dès que l'Access Key·Secret Key apparaissent sur l'écran de fin d'émission, c'est qu'elles ont été émises correctement.

3. Informations à enregistrer dans GenieGo

ChampDescription
access_keyL'Access Key émise
secret_keyLa Secret Key émise (à manipuler avec précaution)
vendor_idVendor ID
Saisissez-les dans le GenieGo Integration Hub [Enregistrer] et enregistrez. Les identifiants sont chiffrés au repos avec AES-256-GCM et masqués à la lecture. La synchronisation des commandes démarre immédiatement après l'enregistrement.

4. Problèmes courants

ProblèmeCauseSolution
Le menu d'émission est absentCompte membre ordinaire, ou vérification d'entreprise non terminéeConnectez-vous avec un compte vendeur WING dont la vérification d'entreprise est terminée.
Erreur 401 lors de l'appelIP d'accès non enregistrée, ou Secret Key mal saisieVérifiez à nouveau les IP d'accès saisies lors de l'émission et la Secret Key.

Liste de contrôle finale

  • Connecté avec un compte vendeur WING dont la vérification d'entreprise est terminée.
  • Émis une clé Open API et enregistré les IP d'accès.
  • Vérifié l'Access Key·Secret Key·Vendor ID.
  • Saisi access_key·secret_key·vendor_id dans GenieGo [Enregistrer] et enregistré.

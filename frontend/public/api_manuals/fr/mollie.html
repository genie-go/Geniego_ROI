💙 Guide pour débutants

Mollie
Manuel d'émission de la clé API

Un guide pas à pas adapté aux débutants pour trouver votre clé Live API (commençant par live_) dans le tableau de bord Mollie et l'enregistrer dans le GenieGo Integration Hub.

Temps estiméEnviron 5 à 10 min
Compte requisCompte Mollie (axé sur l'Europe)
Clé émiseClé Live API (live_)
RésultatIdentifiants Live API

1. Avant de commencer

Votre clé API Mollie se trouve dans le tableau de bord de votre compte Mollie.

Requis

Compte Mollie

Vous avez besoin d'un compte Mollie (axé sur l'Europe). Vous devez pouvoir vous connecter au tableau de bord avec celui-ci.

Clé émise

Clé Live API

Dans le tableau de bord, allez dans Developers > API keys et trouvez la clé Live API qui commence par live_.

Remarque : Veillez à trouver la clé Live API qui commence par live_. Faites attention à ne pas la confondre avec la clé Test.

2. Étapes d'émission

Émettez dans l'ordre ci-dessous et enregistrez dans GenieGo.

1

Accéder au tableau de bord

Connectez-vous au tableau de bord Mollie.

my.mollie.com
2

Aller dans API keys

Accédez à [Developers] > [API keys].

https://my.mollie.com/dashboard/developers/api-keys
3

Trouver la clé Live API

Trouvez la clé Live API qui commence par live_.

Si vous voyez une clé qui commence par live_ sous Developers > API keys, c'est correct.

3. Informations à enregistrer dans GenieGo

ChampDescription
api_keyClé Live API (commence par live_ ; à manipuler avec précaution)
Dans le GenieGo Integration Hub [Enregistrer], saisissez votre api_key et enregistrez. Les identifiants sont chiffrés au repos avec AES-256-GCM et masqués à la lecture.

4. Problèmes fréquents

ProblèmeCauseSolution
L'authentification échoueUtilisation de la clé TestVérifiez à nouveau qu'il s'agit bien de la clé Live API qui commence par live_.
Aucune clé dans le menu API keysLe compte de connexion au tableau de bord doit être vérifiéVérifiez que vous vous êtes connecté au tableau de bord my.mollie.com avec votre compte Mollie.

Liste de contrôle finale

  • Connecté au tableau de bord my.mollie.com avec mon compte Mollie.
  • Trouvé la clé Live API dans Developers > API keys.
  • Trouvé la clé Live API qui commence par live_.
  • Saisi l'api_key dans GenieGo [Enregistrer] et enregistré.

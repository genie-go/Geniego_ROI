YouTube API Key
Manuel de délivrance

Un guide pratique étape par étape que même les débutants peuvent suivre pour activer la YouTube Data API v3 dans la Google Cloud Console et créer une API Key.

Temps estiméEnviron 5 à 10 min
Compte nécessaireCompte Google
API utiliséeYouTube Data API v3
RésultatUne API Key au format AIzaSy...

1. API Key ou OAuth

C'est la partie qui prête le plus à confusion lors des premiers pas.

API Key

Pour interroger des données publiques

Utilisée pour récupérer des données YouTube publiques, comme rechercher des vidéos, consulter les informations d'une chaîne, récupérer des playlists et lire des commentaires.

OAuth 2.0

Pour les tâches nécessitant les autorisations de votre compte

Utilisée pour les tâches nécessitant l'approbation de l'utilisateur, comme l'envoi de vidéos, la gestion de votre propre chaîne et l'accès à des données privées.

Important : Pour envoyer des vidéos automatiquement, une API Key seule ne suffit pas ; une authentification OAuth 2.0 est requise.

2. Activer la YouTube Data API v3

Avant de créer une API Key, vous devez d'abord activer dans votre projet l'API que vous souhaitez utiliser.

1

Accéder à la Google Cloud Console

Connectez-vous à la Google Cloud Console.

console.cloud.google.com
2

Sélectionner ou créer un projet

Dans le sélecteur de projet en haut, choisissez un projet existant ou créez-en un nouveau.

ex. : Geniego ROI
3

Aller à la bibliothèque d'API

Dans le menu de gauche, allez dans API et services → Bibliothèque.

API et services → Bibliothèque
4

Rechercher la YouTube Data API v3 et l'activer

Saisissez YouTube Data API v3 dans le champ de recherche, sélectionnez cette API, puis cliquez sur le bouton Activer.

YouTube Data API v3 → Activer
Si vous voyez État : Activée à l'écran, c'est qu'elle a été activée correctement.

3. Créer une API Key

Une fois la YouTube Data API v3 activée, vous pouvez créer une API Key sur l'écran Identifiants.

1

Aller au menu Identifiants

Dans le menu de gauche, allez dans API et services → Identifiants.

API et services → Identifiants
2

Cliquer sur Créer des identifiants

Cliquez sur le bouton + Créer des identifiants en haut.

+ Créer des identifiants
3

Sélectionner Clé API

Dans le menu qui apparaît, sélectionnez Clé API.

Clé API
4

Sélectionner l'API accessible

Sous Restrictions des API, vérifiez que la YouTube Data API v3 est sélectionnée.

API sélectionnée : YouTube Data API v3
5

Cliquer sur le bouton Créer

Cliquez sur le bouton Créer en bas pour générer l'API Key.

Créer
La clé générée commence généralement par AIzaSy.... Cette chaîne est votre YouTube API Key.

4. Quelles options choisir sur l'écran de création ?

ÉlémentRéglage recommandé pour les débutantsDescription
NomYouTube API KeySaisissez un nom facile à reconnaître par la suite.
Restrictions des APIYouTube Data API v3Restreignez cette clé pour qu'elle ne soit utilisée qu'avec la YouTube API.
Authentification par compte de serviceLaisser décochéGénéralement pas nécessaire pour créer une YouTube Data API Key.
Restrictions d'applicationAucune pour le premier testAprès avoir confirmé que cela fonctionne correctement, nous recommandons de définir des restrictions par site web ou adresse IP.
Lors du déploiement vers un service réel, veillez à définir des restrictions de clé. Si elle est exposée sans restriction, d'autres personnes peuvent utiliser votre clé.

5. Exemples d'utilisation de l'API Key

Vous pouvez tester en plaçant votre clé générée dans la partie YOUR_API_KEY.

Exemple de recherche de vidéoshttps://www.googleapis.com/youtube/v3/search?part=snippet&q=korea&type=video&key=YOUR_API_KEY
Exemple de consultation des infos d'une chaînehttps://www.googleapis.com/youtube/v3/channels?part=snippet&id=CHANNEL_ID&key=YOUR_API_KEY
Si un résultat au format JSON apparaît lorsque vous saisissez l'URL ci-dessus dans la barre d'adresse de votre navigateur, l'API Key fonctionne correctement.

6. Résolution des problèmes courants

ProblèmeCauseSolution
Aucune API sélectionnéeLa YouTube Data API v3 n'a pas encore été activée, ou elle n'a pas été sélectionnée sous Restrictions des APIActivez la YouTube Data API v3 dans la bibliothèque d'API, puis sélectionnez-la à nouveau.
Aucune clé API à afficherVous n'avez pas encore créé d'API KeyCliquez sur Créer des identifiants → Clé API.
Erreur 403Restrictions des API, quota dépassé ou problèmes d'autorisationVérifiez les restrictions des API, le quota et l'état d'activation.
Les envois ne fonctionnent pasUtilisation d'une API Key uniquementL'envoi de vidéos doit utiliser l'authentification OAuth 2.0.

7. Liste de vérification finale

  • Sélectionné un projet Google Cloud.
  • Activé la YouTube Data API v3.
  • Créé une API Key dans Identifiants.
  • Vérifié que l'API Key générée commence par AIzaSy.
  • Défini les restrictions des API sur la YouTube Data API v3.
  • Définir des restrictions par site web ou IP avant le déploiement vers un service réel.

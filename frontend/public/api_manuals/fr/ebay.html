🛍️ Guide pour débutants

eBay Seller Hub
Manuel d'émission de la clé API

Un guide pas à pas adapté aux débutants pour émettre un jeu de clés de production (Production) (DevID·AppID·CertID) dans le Programme pour développeurs eBay, générer un jeton utilisateur OAuth et l'enregistrer dans le GenieGo Integration Hub.

Temps estiméEnviron 10–20 min
Compte requisCompte eBay
Clés émisesDevID · AppID · CertID + jeton OAuth
RésultatJeton d'accès OAuth

1. Avant de commencer

L'émission d'un jeu de clés eBay nécessite l'inscription au Programme pour développeurs.

Requis

Compte eBay

Pour procéder à l'émission, vous avez besoin d'un compte eBay. Utilisez ce compte pour vous inscrire au Programme pour développeurs sur developer.ebay.com.

Clés émises

DevID · AppID · CertID

La création d'un jeu de clés de production (Production) émet un DevID, un AppID et un CertID. Vous devez ensuite aussi générer un jeton utilisateur OAuth pour l'enregistrer auprès de GenieGo.

Remarque : Un jeu de clés de production peut afficher l'avis "Your Keyset is currently disabled". Dans ce cas, il n'est activé qu'après avoir terminé la procédure d'abonnement/de refus de la notification de suppression de compte de la marketplace.

2. Étapes d'émission

Émettez dans l'ordre ci-dessous et enregistrez dans GenieGo.

1

S'inscrire au Programme pour développeurs

Allez sur developer.ebay.com et inscrivez-vous au Programme pour développeurs avec votre compte eBay.

developer.ebay.com
2

Créer un jeu de clés de production

Sur la page [Application Keys], cliquez sur [Create a keyset] dans la section Production.

https://developer.ebay.com/my/keys
3

Procédure d'activation du jeu de clés

Si l'avis "Your Keyset is currently disabled" s'affiche, cliquez sur le lien pour terminer la procédure d'abonnement/de refus de la notification de suppression de compte de la marketplace.

Application Keys → Production
4

Vérifier les clés et générer un jeton OAuth

Vérifiez le DevID, l'AppID et le CertID émis, puis générez un jeton utilisateur OAuth.

Une fois le DevID·AppID·CertID émis et un jeton utilisateur OAuth généré, l'émission est terminée.

3. Informations à enregistrer dans GenieGo

ChampDescription
Jeton d'accès OAuthLe jeton utilisateur OAuth que vous avez généré (à manipuler avec précaution)
Dans le GenieGo Integration Hub [Enregistrer], saisissez votre jeton d'accès OAuth et enregistrez. Les identifiants sont chiffrés au repos avec AES-256-GCM et masqués à la lecture.

4. Problèmes fréquents

ProblèmeCauseSolution
Le jeu de clés est à l'état désactivéLa procédure d'abonnement/de refus de la notification de suppression de compte de la marketplace n'est pas terminéeCliquez sur le lien de l'avis "Your Keyset is currently disabled" et terminez cette procédure.
Impossible d'émettre le jeton OAuthLe jeu de clés de production (DevID·AppID·CertID) n'a pas encore été émisDans [Application Keys], créez d'abord le jeu de clés de production, puis générez le jeton OAuth.

Liste de vérification finale

  • Inscrit au Programme pour développeurs sur developer.ebay.com avec votre compte eBay.
  • Émis le DevID·AppID·CertID via Production [Create a keyset].
  • Lorsque le jeu de clés était désactivé, terminé la procédure d'abonnement/de refus de la notification de suppression de compte de la marketplace.
  • Généré un jeton utilisateur OAuth et saisi le jeton d'accès OAuth dans GenieGo [Enregistrer], puis enregistré.

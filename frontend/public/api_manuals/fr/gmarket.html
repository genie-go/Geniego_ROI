Gmarket
Manuel d'émission de la clé API

Un guide pour émettre la clé API partagée Gmarket·Auction depuis l'ESM Trading API (etapi.gmarket.com) et l'enregistrer dans le GenieGo Integration Hub. Gmarket et Auction sont exploités conjointement via ESM Plus, de sorte qu'une seule clé API connecte les deux canaux à la fois.

Temps estiméPlusieurs jours, y compris l'attente d'approbation (soumis à approbation)
Compte requisComptes vendeur Gmarket·Auction + ID maître ESM Plus
Clés émisesClé API · ID vendeur
RésultatIdentifiants ESM Trading API

1. Avant de commencer

Vous devez être inscrit en tant que vendeur sur Gmarket et Auction et disposer d'un ID maître ESM Plus.

Requis

Comptes vendeur + ID maître ESM

Vous avez besoin de comptes vendeur pour Auction et Gmarket, ainsi que d'un ID maître créé dans ESM Plus (esmplus.com).

Clés émises

Clé API · ID vendeur

Après l'approbation de l'ESM Trading API, vérifiez ensemble la clé API émise et l'ID vendeur.

Remarque : L'ESM Trading API est soumise à approbation, et votre demande peut être refusée. Lors de la demande, vous devez fournir trois éléments : votre ID vendeur Auction, votre ID vendeur Gmarket et votre ID maître ESM.

2. Étapes d'émission

Faites la demande et émettez la clé dans l'ordre ci-dessous, puis enregistrez-la dans GenieGo.

1

Inscrivez-vous en tant que vendeur et créez un ID maître ESM

Inscrivez-vous en tant que vendeur sur Auction et Gmarket, puis connectez-vous à ESM Plus et créez un ID maître.

esmplus.com
2

Demandez l'accès à l'ESM Trading API

Accédez à l'ESM Trading API et demandez l'accès à l'API. Elle est soumise à approbation, elle peut donc être refusée.

https://etapi.gmarket.com
3

Soumettez les informations d'identification

Lors de la demande, soumettez trois éléments : votre ID vendeur Auction, votre ID vendeur Gmarket et votre ID maître ESM.

4

Recevez la clé

Une fois approuvée, une clé API est émise. Vérifiez la clé API émise et l'ID vendeur.

Une fois votre demande approuvée et la clé API émise, le traitement est correct. Pour les questions liées à l'approbation, contactez etapihelp@gmail.com.

3. Informations à enregistrer dans GenieGo

ChampDescription
api_keyLa clé API émise
seller_idID vendeur
Saisissez-les et enregistrez-les dans le GenieGo Integration Hub [Enregistrer]. Les identifiants sont chiffrés au repos avec AES-256-GCM et masqués à la lecture.

Liste de vérification finale

  • Inscrit en tant que vendeur sur Auction et Gmarket.
  • Créé un ID maître dans ESM Plus.
  • Demandé et obtenu l'approbation pour l'accès à l'ESM Trading API sur etapi.gmarket.com.
  • Vérifié l'api_key et le seller_id émis.
  • Saisi api_key·seller_id dans GenieGo [Enregistrer] et enregistré.

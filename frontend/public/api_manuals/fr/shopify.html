Manuel d'émission de la API Key

Un guide étape par étape pour créer une application personnalisée dans votre interface d'administration Shopify afin d'émettre un Admin API access token et de l'enregistrer avec le domaine de votre boutique dans le GenieGo Integration Hub pour connecter votre boutique Shopify.

Temps estiméEnviron 5–15 min
Compte requisPermission d'administrateur de la boutique Shopify
Clés émisesDomaine de la boutique · Admin API access token
RésultatIdentifiants Admin API

1. Avant de commencer

Une application personnalisée Shopify ne peut être créée qu'avec une permission d'administrateur de la boutique.

Requis

Permission d'administrateur de la boutique

Vous avez besoin d'une permission d'administrateur de la boutique Shopify pour créer une application personnalisée et émettre un token.

Clés émises

Domaine de la boutique · Admin API token

Vous obtiendrez un domaine de boutique au format xxx.myshopify.com et l'Admin API access token affiché après l'installation de l'application.

Remarque : L'Admin API access token n'est affiché qu'une seule fois juste après l'installation, vous devez donc le copier immédiatement dès qu'il apparaît.

2. Étapes d'émission

Émettez dans l'ordre ci-dessous et enregistrez dans GenieGo.

1

Accéder à Applications et canaux de vente

Dans votre interface d'administration Shopify, allez dans [Paramètres] > [Applications et canaux de vente].

https://www.shopify.com/admin/settings/apps
2

Créer une application personnalisée

Cliquez sur [Développer des applications] > [Créer une application] pour créer une application personnalisée.

Développer des applications → Créer une application
3

Sélectionner les scopes · Installer l'application

Sélectionnez les Admin API scopes et installez l'application.

4

Copier l'access token

Copiez immédiatement l'Admin API access token affiché après l'installation (affiché une seule fois).

Dès que l'Admin API access token apparaît après l'installation de l'application, il a été émis correctement. Il n'est affiché qu'une seule fois, copiez-le donc immédiatement.

3. Informations à enregistrer dans GenieGo

ChampDescription
shop_domainDomaine de la boutique au format xxx.myshopify.com
access_tokenL'Admin API access token émis (à manipuler avec précaution)
Dans GenieGo [Enregistrer], saisissez votre shop_domain (xxx.myshopify.com) et access_token, puis enregistrez. Les identifiants sont chiffrés au repos avec AES-256-GCM et masqués à la lecture.

4. Problèmes courants

ProblèmeCauseSolution
Impossible de revoir le tokenL'Admin API access token n'est affiché qu'une seule foisSi vous avez perdu le token, ré-émettez-le dans l'application personnalisée et copiez-le immédiatement.
Erreur de permission insuffisanteAdmin API scopes non sélectionnésSélectionnez les Admin API scopes requis, puis réinstallez l'application.
Erreur de domaineFormat du domaine de la boutique saisi incorrectementSaisissez le domaine de la boutique avec précision au format xxx.myshopify.com.

Liste de contrôle finale

  • Connecté avec une permission d'administrateur de la boutique Shopify.
  • Créé une application personnalisée dans Paramètres > Applications et canaux de vente > Développer des applications.
  • Sélectionné les Admin API scopes et installé l'application.
  • Copié immédiatement l'Admin API access token affiché après l'installation.
  • Saisi shop_domain·access_token dans GenieGo [Enregistrer] et enregistré.

Auction
Manuel d'émission de clé API

Un guide expliquant comment obtenir une clé API partagée pour Auction·Gmarket via l'ESM Trading API (etapi.gmarket.com) et l'enregistrer dans le GenieGo Integration Hub. Comme Auction et Gmarket sont exploités conjointement via ESM Plus, une seule clé API connecte les deux canaux.

Temps estiméPlusieurs jours, attente d'approbation incluse (approbation requise)
Compte requisCompte vendeur Auction·Gmarket + ID maître ESM Plus
Clés émisesClé API · ID vendeur
RésultatIdentifiants ESM Trading API

1. Avant de commencer

Vous devez être enregistré en tant que vendeur à la fois sur Auction et Gmarket et disposer d'un ID maître ESM Plus.

Requis

Compte vendeur + ID maître ESM

Vous avez besoin d'un compte vendeur pour Auction et pour Gmarket, ainsi que d'un ID maître créé sur ESM Plus (esmplus.com).

Clés émises

Clé API · ID vendeur

Après l'approbation de l'ESM Trading API, confirmez la clé API émise ainsi que l'ID vendeur.

Remarque : L'ESM Trading API est soumise à approbation et les demandes peuvent être rejetées. Lors de la demande, vous devez soumettre trois éléments : votre ID vendeur Auction, votre ID vendeur Gmarket et votre ID maître ESM.

2. Étapes d'émission

Faites la demande et obtenez la clé dans l'ordre ci-dessous, puis enregistrez-la dans GenieGo.

1

S'inscrire en tant que vendeur et créer un ID maître ESM

Inscrivez-vous en tant que vendeur sur Auction et Gmarket, puis connectez-vous à ESM Plus et créez un ID maître.

esmplus.com
2

Demander l'utilisation de l'ESM Trading API

Accédez à l'ESM Trading API et demandez l'accès à l'API. Comme elle est soumise à approbation, la demande peut être rejetée.

https://etapi.gmarket.com
3

Soumettre les informations d'identification

Lors de la demande, soumettez trois éléments : votre ID vendeur Auction, votre ID vendeur Gmarket et votre ID maître ESM.

4

Émission de la clé

Une fois approuvée, une clé API est émise. Confirmez la clé API émise et l'ID vendeur.

Une fois la clé API émise après l'approbation, le processus a réussi. Pour les questions relatives à l'approbation, contactez etapihelp@gmail.com.

3. Informations à enregistrer dans GenieGo

ChampDescription
api_keyLa clé API émise
seller_idID vendeur
Saisissez les valeurs dans le GenieGo Integration Hub [Enregistrer] et enregistrez. Les identifiants sont chiffrés au repos avec AES-256-GCM et masqués à la lecture.

  • Confirmé l'api_key et le seller_id émis.
  • Saisi api_key·seller_id dans GenieGo [Enregistrer] et enregistré.

API Key Issuance Manual

A beginner-friendly, step-by-step guide to finding your Production API credentials (Merchant ID, Public Key, Private Key) in the Braintree (PayPal) Control Panel and registering them in the GenieGo Integration Hub.

Est. timeAbout 5–10 min

ResultProduction API credentials

1. Before You Begin

Braintree (PayPal) credentials become available only after your merchant account is approved.

Required

Braintree (PayPal) merchant account

You first need an approved Braintree (PayPal) merchant account. Production credentials are available only once your merchant account has been approved.

Keys issued

Merchant ID · Public/Private Key

In the Control Panel, go to Settings > API Keys to find the Production Merchant ID, Public Key, and Private Key.

Note: Be sure to use the credentials from the Production environment. Take care not to confuse them with Sandbox keys.

2. Issuance Procedure

After your merchant account is approved, find the credentials in the order below and register them in GenieGo.

1

Get your merchant account approved

Complete the Braintree (PayPal) merchant account approval first. You can only proceed to the next step once approval is complete.

2

Sign in to the Control Panel

Sign in to the Braintree Control Panel.

braintreegateway.com
3

Go to API Keys

Navigate to the [Settings] > [API Keys] menu.

Settings → API Keys
4

Find the credentials

Find the Merchant ID, Public Key, and Private Key for the Production environment.

If the Production Merchant ID, Public Key, and Private Key appear under Settings > API Keys, everything is in order.

3. Information to Register in GenieGo

FieldDescription
merchant_idProduction Merchant ID
public_keyProduction Public Key
private_keyProduction Private Key (handle with care)
Enter merchant_id, public_key, and private_key under [Register] in the GenieGo Integration Hub and save. Credentials are encrypted at rest with AES-256-GCM and masked on read.

4. Common Issues

IssueCauseSolution
No credentials under the API Keys menuMerchant account approval not completedComplete the Braintree (PayPal) merchant account approval first.
Authentication failureSandbox keys used, or the Private Key was entered incorrectlyRe-check the Production Merchant ID, Public Key, and Private Key.

Final Checklist

  • The Braintree (PayPal) merchant account has been approved.
  • Found the Production credentials under Settings > API Keys in the Control Panel.
  • Confirmed the Merchant ID, Public Key, and Private Key.
  • Entered merchant_id, public_key, and private_key in GenieGo [Register] and saved.

Magento (Adobe Commerce)
API Key Issuance Manual

A beginner-friendly, step-by-step guide to creating an Integration in the Magento (Adobe Commerce) admin to issue an access token and registering it in the GenieGo Integration Hub.

Est. timeAbout 5–15 min
Account neededMagento (Adobe Commerce) admin privileges
Keys issuedIntegration access token
ResultREST API credentials

1. Before You Begin

An Integration can only be created with admin privileges.

Required

Magento admin privileges

You need Magento (Adobe Commerce) admin privileges. An Integration cannot be created with regular user privileges.

Keys issued

Integration access token

When you create an Integration and click Activate, an access token is issued.

2. Issuance Steps

Issue in the order below and register in GenieGo.

1

Go to the Integration menu

In the Magento admin, go to [System] > [Integrations].

Admin → System → Integrations
2

Create an Integration

Create an Integration with [Add New Integration] and configure resource permissions.

Add New Integration
3

Issue the token via Activate

Click [Activate] to obtain the access token.

Activate
4

Verify the information

Confirm the issued access token and your store URL (base_url).

Once the access token is shown after Activate, it has been issued correctly.

3. Information to Register in GenieGo

FieldDescription
base_urlStore URL
access_tokenThe issued Integration access token (handle with care)
Enter base_url·access_token in the GenieGo Integration Hub [Register] and save. Credentials are encrypted at rest with AES-256-GCM and masked on read.

4. Common Issues

IssueCauseSolution
The Integrations menu is missingInsufficient admin privilegesSign in with an admin account that has permission to create Integrations.
Permission error on API callsIntegration resource permissions not set, or Activate not completedSet the Integration's resource permissions and complete Activate to issue the token.

Final Checklist

  • Signed in with Magento (Adobe Commerce) admin privileges.
  • Created an Integration via Add New Integration under System > Integrations.
  • Set resource permissions and issued the access token via Activate.
  • Entered base_url·access_token in GenieGo [Register] and saved.

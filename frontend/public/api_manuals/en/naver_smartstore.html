Naver Smart Store
API Key Issuance Manual

A step-by-step guide to registering an application in the Naver Commerce API Center (apicenter.commerce.naver.com) to obtain a Client ID and Client Secret, then registering them in the GenieGo Integration Hub.

Est. timeAbout 10–15 min
Account neededSmart Store account with "Integrated Manager" permission
Keys issuedClient ID · Client Secret
ResultCommerce API credentials

1. Before You Begin

Issuance is only possible with the Integrated Manager permission on a Smart Store seller account.

Required

Integrated Manager permission

You must have the "Integrated Manager" permission on a Smart Store seller account. Only an Integrated Manager can issue the keys.

Keys issued

Client ID · Client Secret

When you register an application, an application ID (Client ID) and a Secret (Client Secret) are issued.

Note: Registering the server IP that will call the API and periodically renewing the application authentication are required.

2. Issuance Steps

Issue in the order below and register in GenieGo.

1

Access the Commerce API Center

Sign in to the Naver Commerce API Center with your Integrated Manager account.

https://apicenter.commerce.naver.com
2

Create an account

Click [Create Account] in the top right, enter the developer company account name and an incident-response contact, agree to the terms, and sign up.

Create Account
3

Register an application

Under [My Store Applications], register an application; an application ID and Secret are then issued.

My Store Applications
4

Register IP and renew authentication

Register the server IP that will call the API, and periodically renew the application authentication as guided.

Once the application is registered and the application ID and Secret are issued, it has been processed correctly.

3. Information to Register in GenieGo

FieldDescription
client_idThe issued application ID (Client ID)
client_secretThe issued Secret (Client Secret, handle with care)
Enter them in the GenieGo Integration Hub [Register] and save; credentials are encrypted at rest with AES-256-GCM and masked on read.

4. Common Issues

IssueCauseSolution
Cannot register an applicationAccount does not have Integrated Manager permissionAccess apicenter.commerce.naver.com with an account that has Integrated Manager permission.
Authentication error on callCaller IP not registered or authentication renewal missedRegister the caller server IP and periodically renew the application authentication.

Final Checklist

  • Accessed the Commerce API Center with an Integrated Manager account.
  • Entered developer company info and an incident-response contact via [Create Account] and agreed to the terms.
  • Registered an application under [My Store Applications] and obtained the ID/Secret.
  • Registered the caller server IP and set up periodic authentication.
  • Entered client_id·client_secret in GenieGo [Register] and saved.

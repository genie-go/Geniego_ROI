Razorpay
API Key Issuance Manual

A beginner-friendly, step-by-step guide to creating and confirming Live mode API keys (Key ID·Key Secret) in the Razorpay Dashboard and registering them in the GenieGo Integration Hub.

Est. timeAbout 5–10 min
Account neededRazorpay account (India-focused)
Keys issuedKey ID (rzp_live_) · Key Secret
ResultLive API credentials

1. Before You Begin

Razorpay API keys are created in the Dashboard's Live mode.

Required

Razorpay account

You need a Razorpay account (India-focused). You must be able to sign in to the Dashboard with it.

Keys issued

Key ID · Key Secret

Create and confirm the Live mode Key ID and Key Secret in the Dashboard under Settings > API Keys.

Note: Be sure to create the keys in Live mode. The Key ID starts with rzp_live_.

2. Issuance Steps

Issue in the order below and register in GenieGo.

1

Open the Dashboard

Sign in to the Razorpay Dashboard.

dashboard.razorpay.com
2

Go to API Keys

Go to [Settings] > [API Keys].

Settings → API Keys
3

Create and confirm keys

In Live mode, create and confirm the Key ID and Key Secret.

If you can see the Live mode Key ID (rzp_live_) and Key Secret under Settings > API Keys, they have been created correctly.

3. Information to Register in GenieGo

FieldDescription
key_idLive mode Key ID (starts with rzp_live_)
key_secretLive mode Key Secret (handle with care)
In the GenieGo Integration Hub [Register], enter your key_id·key_secret and save. Credentials are encrypted at rest with AES-256-GCM and masked on read.

4. Common Issues

IssueCauseSolution
Authentication failureUsing a Test mode keyCheck that you are using a Key ID (rzp_live_)·Key Secret created in Live mode.
Cannot view the Key Secret againThe Key Secret is shown only once at creationStore the Key Secret safely at creation; if lost, create a new one.

Final Checklist

  • Signed in to dashboard.razorpay.com with a Razorpay account.
  • Created Live mode keys under Settings > API Keys.
  • Confirmed the Key ID (rzp_live_)·Key Secret.
  • Entered key_id·key_secret in GenieGo [Register] and saved.

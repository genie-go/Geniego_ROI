API Key Issuance Manual

A beginner-friendly, step-by-step guide to creating an app in the Square Developer Portal to issue an Access Token and Location ID, and registering them in the GenieGo Integration Hub.

Est. timeAbout 5–10 min
Account neededSquare account
Keys issuedAccess Token · Location ID
ResultAPI credentials

1. Before You Begin

Square API credentials are issued by creating an app in the Developer Portal.

Required

Square account

You need a Square account that can create an app at developer.squareup.com/apps.

Keys issued

Access Token · Location ID

Find your Access Token on the Production tab and your Location ID under [Locations].

Note: For real integration, use the Production token, not the Sandbox one.

2. Issuance Steps

Issue in the order below and register in GenieGo.

Once you have confirmed the Production Access Token and Location ID, they have been issued correctly.

3. Information to Register in GenieGo

FieldDescription
access_tokenThe Access Token found on the Production tab (handle with care)
location_idThe Location ID found under [Locations]
Enter access_token·location_id in the GenieGo Integration Hub [Register] and save. Credentials are encrypted at rest with AES-256-GCM and masked on read.

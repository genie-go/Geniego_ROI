1. Key Things to Know First

YouTube is done with a single API Key, but Facebook (Meta) works with an App ID · App Secret · Access Token structure.

Important: With Meta, it does not end at "copying a single API Key." You typically need the following order: create an app → add a product → set permissions → issue a Page Access Token → app review.

2. What You Need

  • A Meta/Facebook account
  • Access to Meta for Developers
  • A Facebook page on which you have admin rights
  • A live-service privacy policy URL · terms of service URL

3. Creating a Meta Developer App

Every Meta API integration starts with creating an app.

1

Open Meta for Developers

Sign in to Meta for Developers in your browser.

developers.facebook.com
2

My Apps → Create App

Click My Apps → Create App at the top.

My Apps → Create App
3

Choose the app type

Platform integration / live commerce is closer to a business purpose.

Select the Business type
4

Enter app name and email → create

Enter the app name and a contact email, then create the app.

App Name / Contact Email
After creating the app, check the App ID and App Secret under Dashboard → App Settings → Basic. (Never expose the App Secret in the frontend or a public repository.)

4. Facebook Live API Setup Flow

1

Add a product

On the app dashboard, prepare the Facebook Login / Graph API related settings.

Add Product → Facebook Login
2

Connect a page + page token

To broadcast a live stream to a page, you need page admin rights and a Page Access Token.

Page Access Token
3

Apply for required permissions (App Review)

Apply for permissions intended for publishing content, managing pages, and creating live videos (e.g., pages_read_engagement, pages_manage_posts).

Permissions → App Review
4

Issue and test a token with Graph API Explorer

Issue a token that includes the permissions and test the API calls.

Tools → Graph API Explorer
Example Facebook Live creation call
POST /{page-id}/live_videos
access_token={PAGE_ACCESS_TOKEN}
On successful creation, live broadcast information such as the broadcast ID, stream URL, and stream key is returned. Find the page ID (page_id) in the page information.

5. Register in the GenieGo Integration Hub

1

Copy the Page Access Token

Copy the Page Access Token you issued (a long-lived token is recommended).

access_token (Page token)
2

Confirm the page ID

Confirm the ID (page_id) of the target Facebook page.

page_id
3

Register in the Integration Hub

In GenieGo Integration Hub → Facebook Live → [Register], enter the values below and save.

access_token · page_id
Registered keys are encrypted at rest with AES-256-GCM and masked on read.

6. Access Token Types

Token typeUseNotes
User Access TokenAPI calls based on user loginThe data you can access varies depending on the scope of granted permissions.
Page Access TokenPage publishing · page live creationPage admin rights and the relevant permissions are required.
Long-Lived TokenA long-lived token with an extended expiryIn production, you need expiry/refresh logic.

7. Checklist

  • Created a Meta Developer app.
  • Confirmed the App ID and App Secret.
  • Connected a Facebook page and issued a Page Access Token.
  • Organized and applied for the required permissions (App Review).
  • Entered access_token·page_id in GenieGo [Register] and saved.
  • Stored the token securely in server environment variables.

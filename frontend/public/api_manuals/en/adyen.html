API Key Issuance Manual

A guide covering how to issue an API key (X-API-Key) and Merchant Account in the Customer Area after your Adyen merchant contract and screening are approved, and register them in the GenieGo Integration Hub.

Est. timeAbout 5–15 min after approval
Eligibility requiredAdyen merchant contract · screening approval
Keys issuedAPI key (X-API-Key) · Merchant Account
ResultAPI credentials

1. Before You Begin

Adyen is an enterprise global payment gateway, so a merchant contract and screening approval must come first before credentials can be issued.

Required

Merchant contract · screening approval

Adyen is an enterprise global PG that requires a merchant contract and screening approval. Only after approval can you issue credentials.

Keys issued

X-API-Key · Merchant Account

After approval, generate an API key (X-API-Key) under [Developers] > [API credentials] in the Customer Area and confirm your Merchant Account name.

Prerequisite: Self-issuance is not possible before the merchant contract and screening are approved. You must first complete the merchant contract and screening with Adyen.

2. Issuance Application Procedure

After screening approval, issue and register credentials in the order below.

1

Merchant contract · screening approval

Sign a merchant contract with Adyen and obtain screening approval.

Adyen sales/contract
2

Sign in to the Customer Area

After approval, sign in to the Customer Area.

https://ca-live.adyen.com
3

Issue API credentials

Under [Developers] > [API credentials], generate an API key (X-API-Key) and confirm your Merchant Account name.

Developers → API credentials
4

Confirm the starting settlement batch number

For the first settlement collection, find the most recent settlement batch number in the Customer Area.

Once the API key (X-API-Key) and Merchant Account are confirmed, your credentials are ready.

3. Information to Register in GenieGo

FieldDescription
api_keyThe issued API key (X-API-Key, handle with care)
merchant_accountYour confirmed Merchant Account name
batch_startStarting settlement batch number for the first settlement collection (the most recent settlement batch number in the Customer Area)
Enter api_key·merchant_account·batch_start in the GenieGo Integration Hub [Register] and save. Credentials are encrypted at rest with AES-256-GCM and masked on read.

4. Common Issues

IssueCauseSolution
No API credentials menuMerchant contract · screening not approvedComplete the merchant contract and screening with Adyen, then access the Customer Area.
Authentication failureX-API-Key or Merchant Account entered incorrectlyRecheck the API key you issued and the Merchant Account name under Developers > API credentials.

Final Checklist

  • Obtained Adyen merchant contract and screening approval.
  • Signed in to the Customer Area (ca-live.adyen.com).
  • Confirmed the API key (X-API-Key) and Merchant Account under Developers > API credentials.
  • Confirmed the most recent settlement batch number (batch_start).
  • Entered api_key·merchant_account·batch_start in GenieGo [Register] and saved.

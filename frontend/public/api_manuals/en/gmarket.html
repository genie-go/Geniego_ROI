Gmarket
API Key Issuance Manual

A guide to issuing the shared Gmarket·Auction API key from the ESM Trading API (etapi.gmarket.com) and registering it in the GenieGo Integration Hub. Gmarket and Auction are operated together through ESM Plus, so a single API key connects both channels at once.

Est. timeSeveral days incl. approval wait (approval-based)

Keys issuedAPI key · Seller ID
ResultESM Trading API credentials

1. Before You Begin

You must be registered as a seller on both Gmarket and Auction and have an ESM Plus master ID.

Required

Seller accounts + ESM master ID

You need seller accounts for both Auction and Gmarket, plus a master ID created in ESM Plus (esmplus.com).

Keys issued

API key · Seller ID

After ESM Trading API approval, check the issued API key and seller ID together.

Note: The ESM Trading API is approval-based, and your application may be rejected. When applying, you must submit three items: your Auction seller ID, Gmarket seller ID, and ESM master ID.

2. Issuance Steps

Apply and issue in the order below, then register in GenieGo.

1

Register as a seller and create an ESM master ID

Register as a seller on both Auction and Gmarket, then sign in to ESM Plus and create a master ID.

esmplus.com
2

Apply for ESM Trading API access

Go to the ESM Trading API and apply for API access. It is approval-based, so it may be rejected.

https://etapi.gmarket.com
3

Submit identification details

When applying, submit three items: your Auction seller ID, Gmarket seller ID, and ESM master ID.

4

Receive the key

Once approved, an API key is issued. Check the issued API key and seller ID.

Once your application is approved and the API key is issued, it has been processed correctly. For approval-related inquiries, contact etapihelp@gmail.com.

3. Information to Register in GenieGo

FieldDescription
api_keyThe issued API key
seller_idSeller ID
Enter and save them in the GenieGo Integration Hub [Register]. Credentials are encrypted at rest with AES-256-GCM and masked on read.

Final Checklist

  • Registered as a seller on both Auction and Gmarket.
  • Created a master ID in ESM Plus.
  • Applied for and received approval for ESM Trading API access at etapi.gmarket.com.
  • Checked the issued api_key and seller_id.
  • Entered api_key·seller_id in GenieGo [Register] and saved.

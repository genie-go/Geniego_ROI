Issuance Manual

A hands-on, step-by-step guide that even beginners can follow to enable the YouTube Data API v3 in the Google Cloud Console and create an API Key.

Est. timeAbout 5–10 min
Account neededGoogle account
API usedYouTube Data API v3
ResultAn API Key in the AIzaSy... format

1. API Key vs OAuth

This is the part people are most confused about when getting started.

API Key

For querying public data

Used to fetch public YouTube data such as searching videos, looking up channel information, retrieving playlists, and reading comments.

OAuth 2.0

For tasks that need your account's permissions

Used for tasks that require the user's approval, such as uploading videos, managing your own channel, and accessing private data.

Important: To upload videos automatically, an API Key alone is not enough; OAuth 2.0 authentication is required.

2. Enable the YouTube Data API v3

Before creating an API Key, you must first turn on the API you want to use in your project.

1

Access the Google Cloud Console

Sign in to the Google Cloud Console.

console.cloud.google.com
2

Select or create a project

In the project selector at the top, choose an existing project or create a new one.

e.g., Geniego ROI
3

Go to the API Library

In the left menu, go to APIs & Services → Library.

APIs & Services → Library
4

Search for the YouTube Data API v3 and enable it

Type YouTube Data API v3 in the search box, select that API, and click the Enable button.

YouTube Data API v3 → Enable
If you see Status: Enabled on the screen, it has been enabled correctly.

3. Create an API Key

Once the YouTube Data API v3 is enabled, you can create an API Key on the Credentials screen.

1

Go to the Credentials menu

In the left menu, go to APIs & Services → Credentials.

APIs & Services → Credentials
2

Click Create Credentials

Click the + Create Credentials button at the top.

+ Create Credentials
3

Select API key

In the menu that appears, select API key.

API key
4

Select the accessible API

Under API restrictions, confirm that the YouTube Data API v3 is selected.

Selected API: YouTube Data API v3
5

Click the Create button

Click the Create button at the bottom to generate the API Key.

Create
The generated key usually starts with AIzaSy.... This string is your YouTube API Key.

4. Which Options Should You Choose on the Creation Screen?

ItemRecommended setting for beginnersDescription
NameYouTube API KeyEnter a name that is easy to recognize later.
API restrictionsYouTube Data API v3Restrict this key so it is used only with the YouTube API.
Service account authenticationLeave uncheckedGenerally not required for creating a YouTube Data API Key.
Application restrictionsNone for initial testingAfter confirming it works correctly, we recommend setting website or IP address restrictions.
When deploying to a real service, be sure to set key restrictions. If it is exposed without restrictions, others can use your key.

5. API Key Usage Examples

You can test by placing your generated key in the YOUR_API_KEY part.

Video search examplehttps://www.googleapis.com/youtube/v3/search?part=snippet&q=korea&type=video&key=YOUR_API_KEY
Channel info lookup examplehttps://www.googleapis.com/youtube/v3/channels?part=snippet&id=CHANNEL_ID&key=YOUR_API_KEY
If a JSON result appears when you enter the URL above in your browser's address bar, the API Key is working correctly.

7. Final Checklist

  • Selected a Google Cloud project.
  • Enabled the YouTube Data API v3.
  • Created an API Key under Credentials.
  • Confirmed that the generated API Key starts with AIzaSy.
  • Set the API restrictions to the YouTube Data API v3.
  • Set website or IP restrictions before deploying to a real service.

📦 Guia para iniciantes

Smart Tracker (Unified Tracking)
Manual de emissão de API Key

Um guia passo a passo amigável para iniciantes para se cadastrar no SweetTracker (tracking.sweettracker.co.kr), emitir a chave de rastreamento unificado para todas as transportadoras (t_key) e registrá-la no GenieGo Integration Hub. Uma única t_key conecta o rastreamento de entregas de todas as transportadoras.

Tempo estimadoCerca de 5–15 min
Conta necessáriaConta SweetTracker
Chaves emitidast_key (chave de rastreamento unificado para todas as transportadoras)
ResultadoCredenciais da API de rastreamento de entregas unificado

1. Antes de começar

O Smart Tracker (Unified Tracking) é um canal de autoemissão que você pode usar apenas se cadastrando, sem contrato separado.

Obrigatório

Conta SweetTracker

Você só precisa de uma conta registrada em tracking.sweettracker.co.kr. Você pode emitir a chave apenas se cadastrando, sem contrato separado.

Chaves emitidas

t_key (chave de rastreamento unificado)

Emita uma única t_key, a chave de rastreamento unificado para todas as transportadoras, e o rastreamento de entregas de todas as transportadoras fica conectado.

Observação: Insira a t_key no campo api_key ao registrar no GenieGo.

2. Etapas de emissão da t_key

Emita na ordem abaixo e registre no GenieGo.

1

Cadastre-se e faça login no SweetTracker

Acesse tracking.sweettracker.co.kr, cadastre-se e faça login.

https://tracking.sweettracker.co.kr
2

Solicite a emissão de uma t_key

Solicite a emissão de uma t_key (a chave de rastreamento unificado para todas as transportadoras).

3

Verifique a t_key

Verifique a t_key emitida.

4

Registre no GenieGo

Insira a t_key (api_key) no GenieGo [Registrar], e o rastreamento de entregas de todas as transportadoras fica conectado.

Uma vez emitida a t_key, ela foi emitida corretamente, e o rastreamento de entregas de todas as transportadoras fica conectado imediatamente após o registro.

3. Informações a registrar no GenieGo

CampoDescrição
api_keyA t_key emitida (chave de rastreamento unificado para todas as transportadoras)
Insira-a e salve-a no GenieGo Integration Hub [Registrar]. As credenciais são criptografadas em repouso com AES-256-GCM e mascaradas na leitura. O rastreamento de entregas de todas as transportadoras fica conectado imediatamente após o registro.

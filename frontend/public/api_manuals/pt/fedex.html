FedEx
Manual de emissão de API

Um guia passo a passo para emitir credenciais FedEx (chave/token de API) e registrá-las no GenieGo Integration Hub.

Tempo estimadoCerca de 5–15 min
Conta necessáriaConta FedEx
Passos4 passos
ResultadoCredenciais de API

Passos de emissão

Siga os passos abaixo para emitir e registrar suas credenciais.

1

Pré-requisitos: você precisa de um número de conta da FedEx.

2

Cadastre-se em developer.fedex.com e faça login.

3

Ao criar um projeto, a API Key e a Secret Key serão emitidas.

4

No GenieGo [Registrar], insira api_key, api_secret e account_number, e salve.

Checklist final

  • Pré-requisitos: você precisa de um número de conta da FedEx.
  • Cadastre-se em developer.fedex.com e faça login.
  • Ao criar um projeto, a API Key e a Secret Key serão emitidas.
  • No GenieGo [Registrar], insira api_key, api_secret e account_number, e salve.
As chaves inseridas no GenieGo Integration Hub [Register] são armazenadas criptografadas com AES-256-GCM e mascaradas na leitura.
Mantenha suas chaves/segredos em segurança e nunca os exponha publicamente. Se suspeitar de vazamento, reemita imediatamente.

YouTube API Key
Manual de emissão

Um guia prático, passo a passo, que até iniciantes podem seguir para ativar a YouTube Data API v3 no Google Cloud Console e criar uma API Key.

Tempo estimadoCerca de 5–10 min
Conta necessáriaConta Google
API utilizadaYouTube Data API v3
ResultadoUma API Key no formato AIzaSy...

1. API Key vs OAuth

Esta é a parte que mais confunde as pessoas ao começar.

API Key

Para consultar dados públicos

Usada para obter dados públicos do YouTube, como pesquisar vídeos, consultar informações de canal, recuperar playlists e ler comentários.

OAuth 2.0

Para tarefas que exigem as permissões da sua conta

Usada para tarefas que exigem a aprovação do usuário, como enviar vídeos, gerenciar o seu próprio canal e acessar dados privados.

Importante: Para enviar vídeos automaticamente, uma API Key sozinha não basta; é necessária a autenticação OAuth 2.0.

2. Ativar a YouTube Data API v3

Antes de criar uma API Key, primeiro você precisa ativar no seu projeto a API que deseja usar.

1

Acessar o Google Cloud Console

Faça login no Google Cloud Console.

console.cloud.google.com
2

Selecionar ou criar um projeto

No seletor de projeto na parte superior, escolha um projeto existente ou crie um novo.

ex.: Geniego ROI
3

Ir para a Biblioteca de APIs

No menu à esquerda, vá em APIs e serviços → Biblioteca.

APIs e serviços → Biblioteca
4

Pesquisar a YouTube Data API v3 e ativá-la

Digite YouTube Data API v3 na caixa de pesquisa, selecione essa API e clique no botão Ativar.

YouTube Data API v3 → Ativar
Se você vir Status: Ativada na tela, ela foi ativada corretamente.

3. Criar uma API Key

Depois que a YouTube Data API v3 estiver ativada, você pode criar uma API Key na tela de Credenciais.

1

Ir para o menu Credenciais

No menu à esquerda, vá em APIs e serviços → Credenciais.

APIs e serviços → Credenciais
2

Clicar em Criar credenciais

Clique no botão + Criar credenciais na parte superior.

+ Criar credenciais
3

Selecionar Chave de API

No menu que aparece, selecione Chave de API.

Chave de API
4

Selecionar a API acessível

Em Restrições de API, confirme que a YouTube Data API v3 está selecionada.

API selecionada: YouTube Data API v3
5

Clicar no botão Criar

Clique no botão Criar na parte inferior para gerar a API Key.

Criar
A chave gerada normalmente começa com AIzaSy.... Essa sequência é a sua YouTube API Key.

4. Quais opções você deve escolher na tela de criação?

ItemConfiguração recomendada para iniciantesDescrição
NomeYouTube API KeyInsira um nome que seja fácil de reconhecer depois.
Restrições de APIYouTube Data API v3Restrinja esta chave para que seja usada apenas com a YouTube API.
Autenticação de conta de serviçoDeixar desmarcadaGeralmente não é necessária para criar uma YouTube Data API Key.
Restrições de aplicativoNenhuma para o teste inicialApós confirmar que funciona corretamente, recomendamos definir restrições por site ou endereço IP.
Ao implantar em um serviço real, certifique-se de definir restrições de chave. Se for exposta sem restrições, outras pessoas poderão usar a sua chave.

5. Exemplos de uso da API Key

Você pode testar colocando a sua chave gerada na parte YOUR_API_KEY.

Exemplo de pesquisa de vídeohttps://www.googleapis.com/youtube/v3/search?part=snippet&q=korea&type=video&key=YOUR_API_KEY
Exemplo de consulta de informações de canalhttps://www.googleapis.com/youtube/v3/channels?part=snippet&id=CHANNEL_ID&key=YOUR_API_KEY
Se um resultado no formato JSON aparecer ao inserir a URL acima na barra de endereço do seu navegador, a API Key está funcionando corretamente.

6. Solução de problemas comuns

ProblemaCausaSolução
Nenhuma API selecionadaA YouTube Data API v3 ainda não foi ativada, ou não foi selecionada em Restrições de APIAtive a YouTube Data API v3 na Biblioteca de APIs e selecione-a novamente.
Nenhuma chave de API para exibirVocê ainda não criou uma API KeyClique em Criar credenciais → Chave de API.
Erro 403Restrições de API, cota excedida ou problemas de permissãoVerifique as restrições de API, a cota e o status de ativação.
Os envios não funcionamUso apenas de uma API KeyOs envios de vídeo devem usar a autenticação OAuth 2.0.

7. Lista de verificação final

  • Selecionou um projeto Google Cloud.
  • Ativou a YouTube Data API v3.
  • Criou uma API Key em Credenciais.
  • Confirmou que a API Key gerada começa com AIzaSy.
  • Definiu as restrições de API para a YouTube Data API v3.
  • Defina restrições por site ou IP antes de implantar em um serviço real.

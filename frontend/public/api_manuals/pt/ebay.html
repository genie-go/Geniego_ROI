Manual de emissão de API Key

Um guia passo a passo e amigável para iniciantes para emitir um conjunto de chaves de produção (Production) (DevID·AppID·CertID) no Programa de Desenvolvedores do eBay, gerar um token de usuário OAuth e registrá-lo no GenieGo Integration Hub.

Tempo estimadoCerca de 10–20 min
Conta necessáriaConta do eBay
Chaves emitidasDevID · AppID · CertID + token OAuth
ResultadoToken de acesso OAuth

1. Antes de começar

Emitir um conjunto de chaves do eBay exige a inscrição no Programa de Desenvolvedores.

Obrigatório

Conta do eBay

Para prosseguir com a emissão, você precisa de uma conta do eBay. Use esta conta para se inscrever no Programa de Desenvolvedores em developer.ebay.com.

Chaves emitidas

DevID · AppID · CertID

Criar um conjunto de chaves de produção (Production) emite um DevID, um AppID e um CertID. Em seguida, você também deve gerar um token de usuário OAuth para registrá-lo no GenieGo.

Nota: Um conjunto de chaves de produção pode exibir o aviso "Your Keyset is currently disabled". Nesse caso, ele só é ativado depois que você concluir o procedimento de assinatura/recusa da notificação de exclusão de contas do marketplace.

2. Etapas de emissão

Emita na ordem abaixo e registre no GenieGo.

1

Inscreva-se no Programa de Desenvolvedores

Acesse developer.ebay.com e inscreva-se no Programa de Desenvolvedores com sua conta do eBay.

developer.ebay.com
2

Crie um conjunto de chaves de produção

Na página [Application Keys], clique em [Create a keyset] na seção Production.

https://developer.ebay.com/my/keys
3

Procedimento de ativação do conjunto de chaves

Se você vir o aviso "Your Keyset is currently disabled", clique no link para concluir o procedimento de assinatura/recusa da notificação de exclusão de contas do marketplace.

Application Keys → Production
4

Verifique as chaves e gere um token OAuth

Verifique o DevID, o AppID e o CertID emitidos e gere um token de usuário OAuth.

Depois que o DevID·AppID·CertID forem emitidos e um token de usuário OAuth for gerado, a emissão está concluída.

3. Informações para registrar no GenieGo

CampoDescrição
Token de acesso OAuthO token de usuário OAuth que você gerou (manuseie com cuidado)
No GenieGo Integration Hub [Registrar], insira seu token de acesso OAuth e salve. As credenciais são criptografadas em repouso com AES-256-GCM e mascaradas na leitura.

4. Problemas comuns

ProblemaCausaSolução
O conjunto de chaves está no estado desativadoProcedimento de assinatura/recusa da notificação de exclusão de contas do marketplace não concluídoClique no link do aviso "Your Keyset is currently disabled" e conclua esse procedimento.
Não é possível emitir o token OAuthO conjunto de chaves de produção (DevID·AppID·CertID) ainda não foi emitidoEm [Application Keys], crie primeiro o conjunto de chaves de produção e, em seguida, gere o token OAuth.

Lista de verificação final

  • Inscreveu-se no Programa de Desenvolvedores em developer.ebay.com com sua conta do eBay.
  • Emitiu o DevID·AppID·CertID via Production [Create a keyset].
  • Quando o conjunto de chaves estava desativado, concluiu o procedimento de assinatura/recusa da notificação de exclusão de contas do marketplace.
  • Gerou um token de usuário OAuth e inseriu o token de acesso OAuth no GenieGo [Registrar] e salvou.

🟧 Guia para iniciantes

Magento (Adobe Commerce)
Manual de emissão de API Key

Um guia passo a passo amigável para iniciantes para criar uma Integração (Integration) no administrador do Magento (Adobe Commerce) a fim de emitir um token de acesso e registrá-lo no GenieGo Integration Hub.

Tempo estimadoCerca de 5–15 min
Conta necessáriaPrivilégios de administrador do Magento (Adobe Commerce)
Chaves emitidasToken de acesso de Integration
ResultadoCredenciais de REST API

1. Antes de começar

Uma Integração só pode ser criada com privilégios de administrador.

Obrigatório

Privilégios de administrador do Magento

Você precisa de privilégios de administrador do Magento (Adobe Commerce). Uma Integração não pode ser criada com privilégios de usuário comum.

Chaves emitidas

Token de acesso de Integration

Quando você cria uma Integração e clica em Activate, um token de acesso é emitido.

2. Etapas de emissão

Emita na ordem abaixo e registre no GenieGo.

1

Ir ao menu Integration

No administrador do Magento, vá em [Sistema] > [Integrations].

Administrador → Sistema → Integrations
2

Criar uma Integração

Crie uma Integração com [Add New Integration] e configure as permissões de recursos.

Add New Integration
3

Emitir o token via Activate

Clique em [Activate] para obter o token de acesso.

Activate
4

Verificar as informações

Confirme o token de acesso emitido e o URL da sua loja (base_url).

Assim que o token de acesso for exibido após Activate, ele foi emitido corretamente.

3. Informações para registrar no GenieGo

CampoDescrição
base_urlURL da loja
access_tokenO token de acesso de Integration emitido (manuseie com cuidado)
Insira base_url·access_token em [Registrar] no GenieGo Integration Hub e salve. As credenciais são criptografadas em repouso com AES-256-GCM e mascaradas na leitura.

4. Problemas comuns

ProblemaCausaSolução
O menu Integrations está ausentePrivilégios de administrador insuficientesFaça login com uma conta de administrador que tenha permissão para criar Integrações.
Erro de permissão nas chamadasPermissões de recursos da Integração não definidas, ou Activate não concluídoDefina as permissões de recursos da Integração e conclua o Activate para emitir o token.

Lista de verificação final

  • Fez login com privilégios de administrador do Magento (Adobe Commerce).
  • Criou uma Integração via Add New Integration em Sistema > Integrations.
  • Definiu as permissões de recursos e emitiu o token de acesso via Activate.
  • Inseriu base_url·access_token em GenieGo [Registrar] e salvou.

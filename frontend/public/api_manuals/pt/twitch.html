Manual de abertura de canal empresarial e emissão de API

Um guia prático e acessível a iniciantes voltado a empresas, marcas, live-commerce e operadores de plataformas de TI/logística para criar um canal oficial na Twitch e concluir a configuração de transmissão, a conexão com o OBS e a emissão da Twitch Developer API.

1. Entendendo a estrutura da conta Twitch

Diferente do Instagram, a Twitch não tem uma estrutura para escolher separadamente uma conta empresarial/criador.

Canal oficial da empresa

Para operação da marca

Crie uma conta comum na Twitch com o nome da sua empresa ou marca e opere-a como um canal oficial. É adequado para apresentações de produtos, live-commerce e transmissões de eventos.

Nível de crescimento do streamer

Affiliate / Partner

Assim que você cumprir os requisitos de desempenho de transmissão, seguidores e tempo de exibição, poderá ingressar no programa Affiliate ou Partner.

Ponto-chave: As empresas também devem primeiro criar uma conta comum na Twitch e, em seguida, cuidar da identidade de marca do canal, da autenticação de dois fatores, da conexão com o OBS e do registro do aplicativo no Developer Console.

2. O que você precisa

  • Endereço de e-mail empresarial
  • Nome da marca ou da empresa
  • Nome de usuário candidato na Twitch
  • Logotipo da empresa e imagem de banner do canal
  • Texto de apresentação da empresa e URL do site
  • PC ou notebook para transmitir
  • OBS Studio ou software de streaming
  • OAuth Redirect URL para a integração da API

3. Criar uma conta Twitch

1

Abrir a Twitch

Abra o site oficial da Twitch no seu navegador.

https://www.twitch.tv
2

Clicar em Cadastrar

Clique no botão Sign Up ou Cadastrar no canto superior direito.

Sign Up
3

Cadastrar com os dados da empresa

Insira o e-mail da empresa, o nome de usuário, a senha e a data de nascimento.

Email / Username / Password
4

Verificar o e-mail

Abra o e-mail de verificação enviado pela Twitch e conclua o Verify Email.

Verify Email

4. Escolher o nome da conta da marca

O nome de usuário da Twitch se torna o endereço do canal, por isso um nome curto e fácil de lembrar é o ideal.

5. Configurar o perfil do canal

1

Ir ao menu de configurações

Clique no ícone de perfil no canto superior direito e depois vá para Settings.

Profile → Settings
2

Enviar uma imagem de perfil

Defina o logotipo da empresa ou o ícone da marca como imagem de perfil.

Profile Picture
3

Enviar um banner do canal

Envie uma imagem de banner com as cores da sua marca e a mensagem do serviço.

Profile Banner
4

Inserir o texto da bio

Insira a apresentação da empresa, a descrição do serviço, o e-mail de contato e o site.

Bio / Links

6. Configurar a autenticação de dois fatores

A configuração da 2FA é muito importante para usar de forma confiável as configurações de transmissão e segurança na Twitch.

1

Ir ao menu de segurança

Abra o menu Security and Privacy nas configurações.

Settings → Security and Privacy
2

Ativar a 2FA

Ative a Two-Factor Authentication.

Set Up Two-Factor Authentication
3

Conectar um telefone ou aplicativo autenticador

Conecte a verificação por SMS ou um aplicativo autenticador como o Google Authenticator ou o Authy.

Authenticator App
Atenção: Contas empresariais podem ser usadas por várias pessoas, portanto, certifique-se de gerenciar as permissões de administrador e o e-mail de recuperação.

7. Verificar a Stream Key e conectar o OBS

1

Abrir o Creator Dashboard

Acesse o Creator Dashboard pelo menu de perfil.

Creator Dashboard
2

Ir ao menu Stream

Selecione o item Stream no menu de configurações.

Settings → Stream
3

Verificar a Primary Stream Key

Copie a chave de transmissão principal. Essa chave deve ser guardada como uma senha.

Primary Stream Key
4

Conectar no OBS

No OBS Studio, defina o serviço como Twitch e insira o valor copiado como chave de transmissão.

OBS → Settings → Stream → Twitch
Se a sua chave de transmissão for exposta, outra pessoa poderá transmitir no seu canal. Nunca a torne pública.

9. Registrar-se no Twitch Developer Console

Para criar chat bots, análises de espectadores, verificação de status da transmissão e funções de automação, você deve registrar um aplicativo Twitch Developer.

1

Abrir o Twitch Developer

Abra o site de desenvolvedores da Twitch.

https://dev.twitch.tv
2

Ir ao Console

Clique em Console ou Developer Console no menu superior.

Developer Console
3

Selecionar o menu Applications

Selecione Applications no menu à esquerda ou superior.

Applications
4

Clicar em Register Your Application

Clique no botão para registrar um novo aplicativo.

Register Your Application

10. Inserir os dados de registro do aplicativo Twitch API

Campo de entradaExemplo de entradaDescrição

OAuth Redirect URLshttps://yourdomain.com/auth/twitch/callbackO endereço do servidor para o qual retornar após o login OAuth.

Quando o registro for concluído, o Client ID será exibido e você poderá gerar um Client Secret com o botão New Secret.

11. Verificar o Client ID / Client Secret

1

Selecionar o aplicativo registrado

Clique no aplicativo que você acabou de criar na lista Applications do Developer Console.

Applications → selecionar app
2

Copiar o Client ID

Copie o Client ID exibido na tela e salve-o nas configurações do seu servidor.

Client ID
3

Gerar o Client Secret

Clique no botão New Secret para gerar um Client Secret e armazene-o com segurança.

New Secret
O Client Secret muitas vezes só pode ser visualizado uma vez. Salve-o em um local seguro imediatamente após gerá-lo.

12. Estrutura de autenticação da Twitch API

Twitch Channel │ ▼ Developer Console │ ▼ Application │ ▼ Client ID / Client Secret │ ▼ OAuth Token │ ▼ Twitch Helix API │ ▼ Informações do canal, status da transmissão, chat bot, análise de espectadores

13. Fluxo de emissão do token OAuth

1) Gerar a URL de autorização do Twitch OAuth 2) O usuário faz login com a conta Twitch 3) Receber o authorization code na Redirect URL 4) Solicitar um Access Token com o authorization code + Client Secret 5) Chamar a Twitch API com o Access Token 6) Renovar com o Refresh Token quando necessário
Consultas simples de dados públicos são possíveis com um App Access Token, enquanto operações que exigem permissões de usuário precisam de um User Access Token.

15. Lista de verificação final

  • Criei uma conta Twitch com um e-mail empresarial.
  • Defini um nome de usuário que corresponde ao nome da marca.
  • Concluí a verificação do e-mail.
  • Configurei a imagem de perfil, o banner, o texto da bio e os links.
  • Ativei a autenticação de dois fatores.
  • Verifiquei a Stream Key no Creator Dashboard.
  • Conectei o OBS Studio ao canal Twitch.
  • Registrei o aplicativo no Twitch Developer Console.
  • Obtive o Client ID e o Client Secret.
  • Preparei a OAuth Redirect URL e o fluxo de emissão do token.

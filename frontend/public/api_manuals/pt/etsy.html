Manual de Emissão de API Key

Um guia passo a passo amigável para iniciantes sobre como registrar um app no portal de desenvolvedores da Etsy para emitir uma API Key (keystring), obter um access token via autorização OAuth e registrá-lo no GenieGo Integration Hub.

Tempo estimadoCerca de 5–15 min
Conta necessáriaConta de vendedor da Etsy
Chaves emitidasAPI Key (keystring) · Shop ID
ResultadoCredenciais de API

1. Antes de começar

A API Key da Etsy é emitida registrando um app sob a sua conta de vendedor.

Obrigatório

Conta de vendedor da Etsy

Para prosseguir com a emissão, você precisa de uma conta de vendedor da Etsy. Você usará esta conta para acessar o portal de desenvolvedores da Etsy.

Chaves emitidas

API Key (keystring)

Assim que você inserir as informações do seu app, uma API Key (keystring) é emitida. Em seguida, você obtém um access token via autorização OAuth.

Nota: O registro no GenieGo exige a api_key emitida juntamente com o shop_id, que identifica a sua loja.

2. Etapas de emissão

Emita na ordem abaixo e registre no GenieGo.

1

Criar um novo app

Acesse etsy.com/developers e clique em [Create a New App].

etsy.com/developers
2

Inserir informações do app e emitir a chave

Assim que você inserir as informações do seu app, uma API Key (keystring) é emitida.

https://www.etsy.com/developers/your-apps
3

Autorização OAuth

Obtenha um access token via autorização OAuth.

4

Confirmar informações de registro

Verifique a api_key emitida e o shop_id da sua loja.

Assim que a API Key (keystring) for emitida e você obtiver um access token via autorização OAuth, ela foi emitida corretamente.

3. Informações para registrar no GenieGo

CampoDescrição
api_keyA API Key (keystring) emitida
shop_idO Shop ID da loja Etsy que você deseja conectar
No GenieGo Integration Hub [Registrar], insira sua api_key e shop_id e salve. As credenciais são criptografadas em repouso com AES-256-GCM e mascaradas na leitura.

4. Problemas comuns

ProblemaCausaSolução
Não é possível criar o appNão é uma conta de vendedor da Etsy, ou o portal de desenvolvedores não foi acessadoAcesse etsy.com/developers com sua conta de vendedor da Etsy e prossiga com [Create a New App].
Falha de autenticaçãoAutorização OAuth não concluída, ou access token não emitidoApós emitir a API Key (keystring), obtenha um access token via autorização OAuth.

Lista de verificação final

  • Acessei etsy.com/developers com minha conta de vendedor da Etsy.
  • Emiti uma API Key (keystring) via [Create a New App].
  • Obtive um access token via autorização OAuth.
  • Inseri api_key e shop_id no GenieGo [Registrar] e salvei.

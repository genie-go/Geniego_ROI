Manual de emissão de API Key

A API dedicada da Lotte Global Logistics é emitida com base em um código de contrato (cliente). Este manual descreve o processo de obter o código de contrato (cliente) e a chave API junto a um representante de vendas e registrá-los no GenieGo Integration Hub.

Tempo estimadoDe vários dias a semanas, dependendo do contrato
Conta necessáriaContrato (cliente) da Lotte Global Logistics
Chaves emitidasChave API · Código de cliente (contrato)
ResultadoCredenciais da API

1. Antes de começar

A API dedicada da Lotte Global Logistics é um canal baseado em contrato, não um canal emitido instantaneamente a partir de um console de autoatendimento.

Obrigatório

Código de contrato (cliente) da Lotte Global Logistics

Você precisa de um código de contrato (cliente) emitido por meio de um contrato com a Lotte Global Logistics. Ele não é emitido diretamente a partir de um console separado.

Recomendado

Smart Parcel (rastreamento unificado)

Se você precisar apenas do rastreamento de entregas, pode conectar todas as transportadoras com uma única t_key no canal [Smart Parcel (rastreamento unificado)].

Nota: Se o seu objetivo for o rastreamento de entregas, recomendamos conectar por meio da chave unificada do Smart Parcel sem um contrato de API dedicada.

2. Procedimento de solicitação de emissão

Solicite e registre na ordem abaixo.

1

Preparar o código de contrato (cliente)

Prepare o seu código de contrato (cliente) da Lotte Global Logistics. A API dedicada é fornecida sob contrato.

2

Decidir se vai usar o rastreamento unificado

Se você precisar apenas do rastreamento de entregas, em vez do canal dedicado da Lotte Global Logistics, pode conectar todas as transportadoras com uma única t_key no canal [Smart Parcel (rastreamento unificado)].

3

Solicitar a emissão da API dedicada

Se você precisar da API dedicada, obtenha o código de contrato (cliente) e a chave API junto a um representante de vendas.

4

Registrar no GenieGo

Insira o seu api_key e cust_code em [Registrar] no GenieGo Integration Hub e salve.

Assim que receber a chave API e o código de contrato (cliente) de um representante de vendas, você estará pronto para registrar.

3. Informações a registrar no GenieGo

CampoDescrição
api_keyA chave API dedicada da Lotte Global Logistics emitida
cust_codeCódigo de contrato (cliente) da Lotte Global Logistics
Insira os valores em [Registrar] no GenieGo Integration Hub e salve. As credenciais são criptografadas em repouso com AES-256-GCM e mascaradas na leitura.

Lista de verificação final

  • Avaliei a integração com o Smart Parcel (rastreamento unificado) se precisar apenas do rastreamento de entregas.
  • Obtive a chave API e o código de contrato (cliente) junto a um representante de vendas.
  • Inseri api_key·cust_code em GenieGo [Registrar] e salvei.

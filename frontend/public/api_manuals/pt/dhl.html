DHL Express
Manual de emissão de API Key

Um guia passo a passo, amigável para iniciantes, para emitir uma API Key e um Secret no portal de desenvolvedores da DHL Express (developer.dhl.com) e registrá-los junto com o seu número de conta de cliente DHL Express no GenieGo Integration Hub.

Tempo estimadoCerca de 10–15 min (espera de aprovação à parte)
Conta necessáriaConta de cliente DHL Express (número de conta)
Chaves emitidasAPI Key · Secret
ResultadoCredenciais de API · número de conta

1. Antes de começar

As credenciais da DHL Express são solicitadas no portal de desenvolvedores, e é necessário um número de conta de cliente.

Preparação

Número de conta de cliente DHL Express

Ter um número de conta de cliente DHL Express torna a solicitação de credenciais mais tranquila. O número de conta também é inserido ao registrar no GenieGo.

Chaves emitidas

API Key · Secret

Você obterá a API Key e o Secret por meio do seu próprio app (nome da empresa) no portal.

Observação: Quando você registra suas informações de cliente via [Get Access] e solicita credenciais, geralmente recebe o e-mail de aprovação Test/Production no próximo dia útil.

2. Etapas de emissão

Emita na ordem abaixo e registre no GenieGo.

1

Cadastrar-se no portal de desenvolvedores

Acesse o portal de desenvolvedores da DHL Express, cadastre-se com o seu e-mail e verifique-o.

https://developer.dhl.com
2

Solicitar credenciais

Após fazer login, registre suas informações de cliente na seção [Get Access] e solicite credenciais.

Get Access
3

Receber o e-mail de aprovação

Você geralmente recebe o e-mail de aprovação Test/Production no próximo dia útil.

4

Encontrar a API Key·Secret

No portal, acesse o seu próprio app (nome da empresa) > [Show Key] para encontrar a API Key e o Secret.

Seu app (nome da empresa) → Show Key
Assim que você receber o e-mail de aprovação e a API Key·Secret estiverem visíveis no portal, eles foram emitidos corretamente.

3. Informações a registrar no GenieGo

CampoDescrição
api_keyA API Key DHL encontrada no portal
account_numberNúmero de conta de cliente DHL Express
Assim que você inseri-las no GenieGo Integration Hub [Registrar] e salvar, as credenciais são criptografadas em repouso com AES-256-GCM e mascaradas na leitura.

4. Problemas comuns

ProblemaCausaSolução
O e-mail de aprovação não chegaAguardando o processamento da aprovação do próximo dia útil, ou informações de cliente não registradasVerifique em [Get Access] se o registro do cliente e a solicitação de credenciais estão concluídos, e aguarde com base nos dias úteis.
Erro de autenticação ao chamarConfusão entre os ambientes Test/Production, ou número de conta ausenteVerifique novamente a API Key·Secret do ambiente aprovado e o número de conta (account_number).

Lista de verificação final

  • Confirmou o número de conta de cliente DHL Express.
  • Cadastrou-se em developer.dhl.com e solicitou credenciais via [Get Access].
  • Recebeu o e-mail de aprovação Test/Production no próximo dia útil.
  • Encontrou a API Key·Secret no portal.
  • Inseriu api_key·account_number no GenieGo [Registrar] e salvou.

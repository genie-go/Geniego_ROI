🔷 Guia para iniciantes

Razorpay
Manual de emissão de API Key

Um guia passo a passo, adequado para iniciantes, para criar e verificar chaves de API no modo Live (Key ID·Key Secret) no Razorpay Dashboard e registrá-las no GenieGo Integration Hub.

Tempo estimadoCerca de 5–10 min
Conta necessáriaConta Razorpay (focada na Índia)
Chaves emitidasKey ID (rzp_live_) · Key Secret
ResultadoCredenciais da Live API

1. Antes de começar

As chaves de API do Razorpay são criadas no modo Live do Dashboard.

Necessário

Conta Razorpay

Você precisa de uma conta Razorpay (focada na Índia). Você deve conseguir fazer login no Dashboard com ela.

Chaves emitidas

Key ID · Key Secret

Crie e verifique o Key ID e o Key Secret do modo Live no Dashboard, em Settings > API Keys.

Nota: Certifique-se de criar as chaves no modo Live. O Key ID começa com rzp_live_.

2. Etapas de emissão

Emita na ordem abaixo e registre no GenieGo.

1

Abrir o Dashboard

Faça login no Razorpay Dashboard.

dashboard.razorpay.com
2

Ir para API Keys

Vá para [Settings] > [API Keys].

Settings → API Keys
3

Criar e verificar as chaves

No modo Live, crie e verifique o Key ID e o Key Secret.

Se você conseguir ver o Key ID do modo Live (rzp_live_) e o Key Secret em Settings > API Keys, eles foram criados corretamente.

3. Informações a registrar no GenieGo

CampoDescrição
key_idKey ID do modo Live (começa com rzp_live_)
key_secretKey Secret do modo Live (manuseie com cuidado)
No GenieGo Integration Hub [Registrar], insira seu key_id·key_secret e salve. As credenciais são criptografadas em repouso com AES-256-GCM e mascaradas na leitura.

4. Problemas comuns

ProblemaCausaSolução
Falha de autenticaçãoUso de uma chave do modo TestVerifique se você está usando um Key ID (rzp_live_)·Key Secret criado no modo Live.
Não é possível ver o Key Secret novamenteO Key Secret é exibido apenas uma vez na criaçãoGuarde o Key Secret com segurança na criação; se perdê-lo, crie um novo.

Lista de verificação final

  • Fiz login em dashboard.razorpay.com com uma conta Razorpay.
  • Criei chaves do modo Live em Settings > API Keys.
  • Verifiquei o Key ID (rzp_live_)·Key Secret.
  • Inseri key_id·key_secret no GenieGo [Registrar] e salvei.

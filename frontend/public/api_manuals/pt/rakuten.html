🛒 Guia para iniciantes

Rakuten
Manual de emissão de API Key

Um guia passo a passo, fácil para iniciantes, sobre como emitir as credenciais do RMS Web Service (Service Secret e License Key) a partir do console do vendedor Rakuten RMS e registrá-las no GenieGo Integration Hub.

Tempo estimadoCerca de 5–15 min
Conta necessáriaConta de vendedor Rakuten (Ichiba)
Chaves emitidasService Secret · License Key
ResultadoCredenciais do RMS Web Service

1. Antes de começar

As chaves de vendedor (Ichiba) são emitidas pelo RMS Web Service.

Obrigatório

Conta de vendedor Rakuten

Você precisa de uma conta de vendedor Rakuten (Ichiba). Você precisa de uma conta que possa fazer login via RMS R-Login (glogin.rms.rakuten.co.jp).

Chaves emitidas

Service Secret · License Key

Você obterá um serviceSecret e uma licenseKey do RMS Web Service. A License Key expira após 90 dias e pode ser reemitida.

Atenção: O RMS Web Service (glogin.rms.rakuten.co.jp) que emite as chaves de vendedor (Ichiba) é diferente do webservice.rakuten.co.jp, que é para afiliados/busca de produtos. Certifique-se de emitir suas chaves no lado do RMS.

2. Etapas de emissão

Emita na ordem abaixo e registre no GenieGo.

1

Faça login no portal do desenvolvedor RMS

Faça login no portal do desenvolvedor Rakuten (RMS/Webservice) com sua conta de vendedor.

https://glogin.rms.rakuten.co.jp
2

Vá para o RMS Web Service

Após fazer login via RMS R-Login, navegue até a área do RMS Web Service.

RMS R-Login → RMS Web Service
3

Emita o Service Secret · License Key

No RMS Web Service, emita o Service Secret e a License Key. A License Key expira após 90 dias, portanto reemita-a quando expirar.

4

Verifique as chaves

Verifique o Service Secret e a License Key emitidos, juntamente com a Shop URL da sua loja.

Assim que o Service Secret e a License Key aparecerem na tela de emissão, eles foram emitidos corretamente.

3. Informações para registrar no GenieGo

CampoDescrição
service_secretO Service Secret emitido
license_keyA License Key emitida (expira em 90 dias·reemitível, manuseie com cuidado)
shop_urlA Shop URL da sua loja
No GenieGo Integration Hub [Registrar], insira service_secret·license_key·shop_url e salve. As credenciais são criptografadas em repouso com AES-256-GCM e mascaradas na leitura.

4. Problemas comuns

ProblemaCausaSolução
Chaves erradas foram emitidasEmitidas pelo webservice.rakuten.co.jp (para afiliados/busca de produtos)Reemita pelo RMS Web Service em RMS R-Login (glogin.rms.rakuten.co.jp).
A autenticação falha após algum tempoA License Key expirou após 90 diasReemita a License Key pelo RMS Web Service e insira-a novamente em [Registrar].

Lista de verificação final

  • Fiz login no RMS R-Login com uma conta de vendedor Rakuten (Ichiba).
  • Emiti o Service Secret·License Key pelo RMS Web Service.
  • Verifiquei a Shop URL da minha loja.
  • Inseri service_secret·license_key·shop_url no GenieGo [Registrar] e salvei.

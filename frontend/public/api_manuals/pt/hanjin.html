Hanjin Logistics
Manual de emissão de API Key

A API dedicada da Hanjin Logistics é emitida com base em um código de contrato (cliente). Este manual explica como obter o código de contrato (cliente) e a chave API com o seu representante de vendas e registrá-los no GenieGo Integration Hub.

Tempo estimadoDe dias a semanas, conforme o andamento do contrato
Conta necessáriaContrato da Hanjin Logistics (cliente corporativo)
Chaves emitidasChave API · Código do cliente (contrato)
ResultadoCredenciais de API

1. Antes de começar

A API dedicada da Hanjin Logistics é um canal baseado em contrato que não é emitido instantaneamente em um console de autoatendimento.

Obrigatório

Código de contrato (cliente) da Hanjin Logistics

Você precisa de um código de contrato (cliente) emitido por meio de um contrato com a Hanjin Logistics. Não é algo que você emite diretamente em um console separado.

Recomendado

Smart Delivery (rastreamento unificado)

Se você só precisa de rastreamento de entregas, pode conectar todas as transportadoras com uma única t_key no canal [Smart Delivery (rastreamento unificado)].

Nota: Se o seu objetivo é o rastreamento de entregas, recomendamos conectar pela chave unificada do Smart Delivery sem um contrato de API dedicada.

2. Processo de solicitação de emissão

Solicite e registre na ordem abaixo.

1

Preparar o código de contrato (cliente)

Prepare o seu código de contrato (cliente) da Hanjin Logistics. A API dedicada é fornecida em regime de contrato.

2

Decidir se o rastreamento unificado é suficiente

Se você só precisa de rastreamento de entregas, pode conectar todas as transportadoras com uma única t_key no canal [Smart Delivery (rastreamento unificado)] em vez do canal dedicado da Hanjin Logistics.

3

Solicitar a API dedicada

Se você precisa da API dedicada, obtenha o código de contrato (cliente) e a chave API com o seu representante de vendas.

4

Registrar no GenieGo

Insira api_key e cust_code em [Registrar] do GenieGo Integration Hub e salve.

Assim que receber a chave API e o código de contrato (cliente) do seu representante de vendas, você estará pronto para registrar.

3. Informações para registrar no GenieGo

CampoDescrição
api_keyA chave API dedicada da Hanjin Logistics emitida
cust_codeCódigo de contrato (cliente) da Hanjin Logistics
Insira os valores em [Registrar] do GenieGo Integration Hub e salve. As credenciais são criptografadas em repouso com AES-256-GCM e mascaradas na leitura.

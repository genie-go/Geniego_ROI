Manual de emissão de API Key

A API dedicada da CJ Logistics é emitida com base em um código de contrato (cliente). Este manual orienta você a obter o código de contrato (cliente) e a chave de API junto ao seu representante de vendas e registrá-los no GenieGo Integration Hub.

Tempo estimadoDe vários dias a semanas, conforme o andamento do contrato
Conta necessáriaContrato da CJ Logistics (cliente corporativo)
Chaves emitidasChave de API · Código de cliente (contrato)
ResultadoCredenciais de API

1. Antes de começar

A API dedicada da CJ Logistics é um canal baseado em contrato e não é emitida instantaneamente a partir de um console de autoatendimento.

Obrigatório

Código de contrato (cliente) da CJ Logistics

Você precisa ter um código de contrato (cliente) emitido por meio de um contrato com a CJ Logistics. Ele não é emitido diretamente em um console separado.

Recomendado

Smart Parcel (rastreamento unificado)

Se você só precisa de rastreamento de entregas, pode conectar todas as transportadoras com uma única t_key através do canal [Smart Parcel (rastreamento unificado)].

Nota: Se o seu objetivo é o rastreamento de entregas, recomendamos conectar-se através da chave unificada Smart Parcel (SweetTracker) sem um contrato de API dedicado.

2. Procedimento de solicitação de emissão

Solicite e registre na ordem abaixo.

1

Prepare o código de contrato (cliente)

Prepare o seu código de contrato (cliente) da CJ Logistics. A API dedicada é fornecida em regime de contrato.

2

Decida se você precisa de rastreamento unificado

Se você só precisa de rastreamento de entregas, pode conectar todas as transportadoras com uma única t_key através do canal [Smart Parcel (rastreamento unificado)] em vez do canal dedicado da CJ Logistics.

3

Solicite a API dedicada

Se você precisa da API dedicada da CJ Logistics, obtenha o código de contrato (cliente) e a chave de API junto ao seu representante de vendas.

4

Registre no GenieGo

Insira api_key e cust_code em [Registrar] do GenieGo Integration Hub e salve.

Assim que você receber a chave de API e o código de contrato (cliente) do seu representante de vendas, estará pronto para registrar.

3. Informações para registrar no GenieGo

CampoDescrição
api_keyA chave de API dedicada da CJ Logistics emitida
cust_codeCódigo de contrato (cliente) da CJ Logistics
Quando você inserir e salvar em [Registrar] do GenieGo Integration Hub, as credenciais são criptografadas em repouso com AES-256-GCM e mascaradas na leitura.

Gmarket
Manual de emissão da API Key

Um guia para emitir a chave de API compartilhada do Gmarket·Auction a partir da ESM Trading API (etapi.gmarket.com) e registrá-la no GenieGo Integration Hub. O Gmarket e o Auction são operados em conjunto por meio do ESM Plus, portanto uma única chave de API conecta ambos os canais de uma só vez.

Tempo estimadoVários dias, incluindo a espera pela aprovação (baseado em aprovação)
Conta necessáriaContas de vendedor do Gmarket·Auction + ID mestre do ESM Plus
Chaves emitidasChave de API · ID do vendedor
ResultadoCredenciais da ESM Trading API

1. Antes de começar

Você deve estar registrado como vendedor tanto no Gmarket quanto no Auction e ter um ID mestre do ESM Plus.

Obrigatório

Contas de vendedor + ID mestre do ESM

Você precisa de contas de vendedor tanto no Auction quanto no Gmarket, além de um ID mestre criado no ESM Plus (esmplus.com).

Chaves emitidas

Chave de API · ID do vendedor

Após a aprovação da ESM Trading API, verifique juntos a chave de API emitida e o ID do vendedor.

Nota: A ESM Trading API é baseada em aprovação, e sua solicitação pode ser rejeitada. Ao se inscrever, você deve enviar três itens: seu ID de vendedor do Auction, ID de vendedor do Gmarket e ID mestre do ESM.

2. Etapas de emissão

Inscreva-se e emita na ordem abaixo e, em seguida, registre no GenieGo.

1

Registre-se como vendedor e crie um ID mestre do ESM

Registre-se como vendedor tanto no Auction quanto no Gmarket, depois faça login no ESM Plus e crie um ID mestre.

esmplus.com
2

Solicite acesso à ESM Trading API

Acesse a ESM Trading API e solicite acesso à API. É baseada em aprovação, portanto pode ser rejeitada.

https://etapi.gmarket.com
3

Envie os dados de identificação

Ao se inscrever, envie três itens: seu ID de vendedor do Auction, ID de vendedor do Gmarket e ID mestre do ESM.

4

Receba a chave

Após a aprovação, uma chave de API é emitida. Verifique a chave de API emitida e o ID do vendedor.

Depois que sua solicitação for aprovada e a chave de API for emitida, ela foi processada corretamente. Para dúvidas relacionadas à aprovação, entre em contato com etapihelp@gmail.com.

3. Informações para registrar no GenieGo

CampoDescrição
api_keyA chave de API emitida
seller_idID do vendedor
Insira e salve no GenieGo Integration Hub [Register]. As credenciais são criptografadas em repouso com AES-256-GCM e mascaradas na leitura.

Lista de verificação final

  • Criei um ID mestre no ESM Plus.

  • Verifiquei a api_key e o seller_id emitidos.
  • Inseri api_key·seller_id no GenieGo [Register] e salvei.

Manual de emissão de chave de API

Um guia para solicitar a permissão da Kakao Moment Open API, obter seu token de acesso e o ID da conta de anúncios, e registrá-los no Hub de Integração da GenieGo. A permissão da Open API é concedida apenas às agências de publicidade oficiais da Kakao.

1. Antes de começar

A Kakao Moment Open API não é autoemitida; ela passa por um processo de solicitação e análise de permissão.

Importante: A permissão da Kakao Moment Open API é concedida apenas às agências de publicidade oficiais da Kakao. Anunciantes comuns não podem autoemiti-la.
Obrigatório

Conta de anúncios + Business Channel

Você precisa de uma conta de anúncios Kakao Moment e de um Kakao Business Channel.

Chaves emitidas

Token de acesso · ID da conta de anúncios

Após a permissão ser aprovada, você obtém um token de acesso por meio da autorização OAuth e confirma o ID da sua conta de anúncios.

2. Processo de solicitação de emissão

Prossiga nesta ordem: solicitação de permissão da Open API → análise/aprovação → emissão do token → registro.

1

Crie um aplicativo e mude para um aplicativo Business

Crie um aplicativo em developers.kakao.com e mude-o para um [Business App].

developers.kakao.com
2

Solicite a permissão da Open API

Envie a Kakao Moment Open API [Additional Feature Application]. A permissão é concedida após a análise da sua elegibilidade como agência oficial.

Additional Feature Application
3

Emita o token

Após a permissão ser aprovada, obtenha um token de acesso por meio da autorização OAuth e confirme o ID da sua conta de anúncios. Você pode usar o OAuth de um clique no Hub de Integração da GenieGo.

Assim que a permissão for aprovada e seu token de acesso e ID da conta de anúncios forem confirmados, você poderá prosseguir com o registro.

3. Informações para registrar na GenieGo

CampoDescrição
access_tokenO token de acesso emitido por meio da autorização OAuth (manuseie com cuidado)
ad_account_idID da conta de anúncios Kakao Moment
Insira-os e salve-os no Hub de Integração da GenieGo [Register], e as credenciais são criptografadas em repouso com AES-256-GCM e mascaradas na leitura.

4. Problemas comuns

ProblemaCausaSolução
Não é possível obter a permissão da Open APINão é uma agência de publicidade oficialA permissão da Open API é concedida apenas às agências de publicidade oficiais da Kakao. Solicite por meio de uma agência.
Não é possível prosseguir com [Additional Feature Application]O aplicativo não foi alterado para um aplicativo BusinessMude o aplicativo para um [Business App] primeiro e, em seguida, solicite o recurso adicional.

Lista de verificação final

  • Preparei uma conta de anúncios Kakao Moment e um Kakao Business Channel.
  • Alterei o aplicativo para um [Business App] e enviei a Open API [Additional Feature Application].
  • Após a aprovação da permissão, confirmei o token de acesso e o ID da conta de anúncios por meio da autorização OAuth.
  • Inseri access_token·ad_account_id no GenieGo [Register] e salvei.

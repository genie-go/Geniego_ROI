Braintree
Manual de emissão de API Key

Um guia passo a passo amigável para iniciantes para encontrar suas credenciais de Production API (Merchant ID·Public Key·Private Key) no Braintree (PayPal) Control Panel e registrá-las no GenieGo Integration Hub.

Tempo estimadoCerca de 5–10 min
Conta necessáriaConta de comerciante Braintree (PayPal) (requer aprovação)
Chaves emitidasMerchant ID · Public Key · Private Key
ResultadoCredenciais de Production API

1. Antes de começar

As credenciais do Braintree (PayPal) só ficam disponíveis depois que sua conta de comerciante for aprovada.

Obrigatório

Conta de comerciante Braintree (PayPal)

Primeiro você precisa de uma conta de comerciante Braintree (PayPal) aprovada. As credenciais de Production só ficam disponíveis depois que sua conta de comerciante for aprovada.

Chaves emitidas

Merchant ID · Public/Private Key

No Control Panel, vá em Settings > API Keys para encontrar o Merchant ID, a Public Key e a Private Key do ambiente Production.

Observação: Certifique-se de usar as credenciais do ambiente Production. Tome cuidado para não confundi-las com as chaves de Sandbox.

2. Procedimento de emissão

Após a aprovação da sua conta de comerciante, encontre as credenciais na ordem abaixo e registre-as no GenieGo.

1

Aprovação da conta de comerciante

Conclua primeiro a aprovação da conta de comerciante Braintree (PayPal). Você só pode avançar para a próxima etapa depois que a aprovação for concluída.

2

Entrar no Control Panel

Entre no Braintree Control Panel.

braintreegateway.com
3

Ir para API Keys

Navegue até o menu [Settings] > [API Keys].

Settings → API Keys
4

Encontrar as credenciais

Encontre o Merchant ID, a Public Key e a Private Key do ambiente Production.

Se o Merchant ID, a Public Key e a Private Key de Production aparecerem em Settings > API Keys, está tudo certo.

3. Informações para registrar no GenieGo

CampoDescrição
merchant_idProduction Merchant ID
public_keyProduction Public Key
private_keyProduction Private Key (manuseie com cuidado)
Insira merchant_id·public_key·private_key em [Registrar] no GenieGo Integration Hub e salve. As credenciais são criptografadas em repouso com AES-256-GCM e mascaradas na leitura.

4. Problemas comuns

ProblemaCausaSolução
Nenhuma credencial no menu API KeysAprovação da conta de comerciante não concluídaConclua primeiro a aprovação da conta de comerciante Braintree (PayPal).
Falha na autenticaçãoChaves de Sandbox usadas, ou Private Key inserida incorretamenteVerifique novamente o Merchant ID, a Public Key e a Private Key do ambiente Production.

Lista de verificação final

  • A conta de comerciante Braintree (PayPal) foi aprovada.
  • Credenciais de Production encontradas em Settings > API Keys do Control Panel.
  • Merchant ID, Public Key e Private Key confirmados.
  • merchant_id·public_key·private_key inseridos em GenieGo [Registrar] e salvos.

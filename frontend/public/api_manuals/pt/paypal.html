PayPal
Manual de emissão de API Key

Um guia passo a passo para iniciantes sobre como criar um app no PayPal Developer Portal para emitir um Client ID e um Secret, e registrá-los no GenieGo Integration Hub.

Tempo estimadoCerca de 5–10 min
Conta necessáriaConta empresarial do PayPal
Chaves emitidasClient ID · Secret
ResultadoCredenciais de API

1. Antes de começar

As credenciais da API do PayPal são emitidas criando um app no Developer Portal com uma conta empresarial.

Obrigatório

Conta empresarial do PayPal

Você precisa de uma conta empresarial do PayPal que possa fazer login em developer.paypal.com.

Chaves emitidas

Client ID · Secret

Criar um app Live emite um Client ID e um Secret.

Observação: Para a integração real, crie o app na aba [Live], não na Sandbox.

2. Etapas de emissão

Emita na ordem abaixo e registre no GenieGo.

1

Faça login no Developer Portal

Faça login com sua conta empresarial do PayPal e depois vá para [Apps & Credentials].

developer.paypal.com → Apps & Credentials
2

Crie um app Live

Na aba [Live], clique em [Create App] para criar o app.

Live → Create App
3

Verifique suas credenciais

Verifique o Client ID e o Secret emitidos.

Assim que o Client ID e o Secret forem exibidos, eles foram emitidos corretamente.

3. Informações para registrar no GenieGo

CampoDescrição
client_idO Client ID emitido
client_secretO Secret emitido (manuseie com cuidado)
No GenieGo Integration Hub [Registrar], insira seu client_id·client_secret e salve. As credenciais são criptografadas em repouso com AES-256-GCM e mascaradas na leitura.

4. Problemas comuns

ProblemaCausaSolução
Falha na autenticaçãoUso de credenciais de um app SandboxUse o Client ID·Secret de um app criado na aba [Live].
Não é possível criar um appLogin com uma conta comum (pessoal)Faça login com uma conta empresarial do PayPal.

Lista de verificação final

  • Fiz login em developer.paypal.com com uma conta empresarial do PayPal.
  • Fui para a aba [Live] em [Apps & Credentials].
  • Criei um app via [Create App] e verifiquei o Client ID·Secret.
  • Inseri client_id·client_secret no GenieGo [Registrar] e salvei.

Coupang Wing
Manual de emissão de API Key

Um guia passo a passo, ideal para iniciantes, sobre como emitir credenciais de Open API (Access Key·Secret Key) no console de vendedor da Coupang WING e registá-las no GenieGo Integration Hub.

Tempo estimadoCerca de 5–10 min
Conta necessáriaConta de vendedor da Coupang WING
Chaves emitidasAccess Key · Secret Key
ResultadoCredenciais de Open API

1. Antes de começar

A Open API da Coupang só pode ser emitida a partir de uma conta de vendedor.

Obrigatório

Conta de vendedor WING

Precisa de uma conta de vendedor da Coupang WING com a verificação de empresa concluída. Não é possível emitir com uma conta de membro comprador comum.

Chaves emitidas

Access Key · Secret Key

No ecrã de conclusão da emissão, localize a Access Key e a Secret Key e verifique também o vendor ID.

Nota: Ao emitir, deve introduzir o nome da sua empresa, o URL da página inicial/serviço (use wing.coupang.com se não tiver) e os IPs de acesso (até 10).

2. Passos para emitir a Open API Key

Emita pela ordem abaixo e registe no GenieGo.

1

Inicie sessão no WING

Inicie sessão com a sua conta de vendedor.

wing.coupang.com
2

Vá ao menu Open API

Vá ao menu [Open API] ou [Gestão de API Key] na área superior/de definições.

Open API → Gestão de API Key
3

Emita uma Open API Key

Clique em [Emitir Open API Key] e introduza o nome da sua empresa, o URL da página inicial e os IPs de acesso (até 10).

Emitir Open API Key
4

Verifique as chaves

No ecrã de conclusão da emissão, localize a Access Key·Secret Key e o vendor ID.

Assim que a Access Key·Secret Key aparecerem no ecrã de conclusão da emissão, foram emitidas corretamente.

3. Informações a registar no GenieGo

CampoDescrição
access_keyA Access Key emitida
secret_keyA Secret Key emitida (manuseie com cuidado)
vendor_idVendor ID
Introduza-as no GenieGo Integration Hub [Registar] e guarde. As credenciais são encriptadas em repouso com AES-256-GCM e mascaradas na leitura. A sincronização de encomendas inicia imediatamente após o registo.

4. Problemas comuns

ProblemaCausaSolução
O menu de emissão está em faltaConta de membro comum, ou verificação de empresa não concluídaInicie sessão com uma conta de vendedor WING com a verificação de empresa concluída.
Erro 401 na chamadaIP de acesso não registado, ou Secret Key introduzida incorretamenteVerifique novamente os IPs de acesso que introduziu na emissão e a Secret Key.

Lista de verificação final

  • Iniciei sessão com uma conta de vendedor WING com a verificação de empresa concluída.
  • Emiti uma Open API Key e registei os IPs de acesso.
  • Verifiquei a Access Key·Secret Key·Vendor ID.
  • Introduzi access_key·secret_key·vendor_id no GenieGo [Registar] e guardei.

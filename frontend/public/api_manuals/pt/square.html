Manual de emissão da API Key

Um guia passo a passo e amigável para iniciantes sobre como criar um aplicativo no Square Developer Portal para emitir um Access Token e um Location ID, e registrá-los no GenieGo Integration Hub.

Tempo estimadoCerca de 5–10 min
Conta necessáriaConta Square
Chaves emitidasAccess Token · Location ID
ResultadoCredenciais de API

1. Antes de começar

As credenciais da API do Square são emitidas criando um aplicativo no Developer Portal.

Obrigatório

Conta Square

Você precisa de uma conta Square capaz de criar um aplicativo em developer.squareup.com/apps.

Chaves emitidas

Access Token · Location ID

Encontre o Access Token na aba Production e o Location ID em [Locations].

Observação: Para a integração real, use o token de Production, não o de Sandbox.

2. Etapas de emissão

Emita na ordem abaixo e registre no GenieGo.

1

Criar um aplicativo

Crie um aplicativo no Square Developer Portal.

developer.squareup.com/apps
2

Encontrar o Access Token

Encontre o Access Token na aba Production.

Production
3

Encontrar o Location ID

Encontre o Location ID em [Locations].

Locations
Depois de confirmar o Production Access Token e o Location ID, eles foram emitidos corretamente.

3. Informações para registrar no GenieGo

CampoDescrição
access_tokenO Access Token encontrado na aba Production (manuseie com cuidado)
location_idO Location ID encontrado em [Locations]
Insira access_token·location_id em [Registrar] do GenieGo Integration Hub e salve. As credenciais são criptografadas em repouso com AES-256-GCM e mascaradas na leitura.

4. Problemas comuns

ProblemaCausaSolução
Falha na autenticaçãoEstá sendo usado um token de SandboxUse o Access Token encontrado na aba Production.
Os pedidos não sincronizamLocation ID inserido incorretamenteVerifique novamente o Location ID encontrado em [Locations].

Lista de verificação final

  • Criei um aplicativo em developer.squareup.com/apps.
  • Encontrei o Access Token na aba Production.
  • Encontrei o Location ID em [Locations].
  • Inseri access_token·location_id no GenieGo [Registrar] e salvei.

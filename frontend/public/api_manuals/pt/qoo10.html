🟡 Guia para Iniciantes

Qoo10
Manual de Emissão de API Key

Um guia passo a passo, ideal para iniciantes, para emitir uma chave de autenticação de API no Qoo10 QSM (o console do vendedor) e registrá-la no GenieGo Integration Hub.

Tempo estimadoCerca de 5–10 min
Conta necessáriaConta de vendedor Qoo10 (QSM)
Chaves emitidasChave API do QSM
ResultadoCredenciais de API

1. Antes de Começar

Uma chave API do Qoo10 só pode ser emitida a partir de uma conta de vendedor (QSM).

Obrigatório

Conta de vendedor Qoo10 (QSM)

Você precisa de uma conta de vendedor Qoo10 (QSM). Uma conta de membro comprador comum não pode emitir uma chave API.

Chaves emitidas

Chave API do QSM

Emita uma chave de autenticação de API no menu de gerenciamento de API do QSM e verifique ao mesmo tempo o seu próprio Seller ID.

2. Etapas de Emissão

Emita na ordem abaixo e registre no GenieGo.

1

Fazer login no QSM

Faça login com a sua conta de vendedor Qoo10 (QSM).

https://qsm.qoo10.jp
2

Ir ao menu de gerenciamento de API

No QSM, vá até o menu de gerenciamento de API.

Gerenciamento de API
3

Emitir a chave de autenticação de API

Emita uma chave de autenticação de API no menu de gerenciamento de API.

4

Verificar as chaves

Verifique a chave API do QSM emitida e o seu próprio Seller ID.

Assim que a chave de autenticação de API aparecer na tela de gerenciamento de API, ela foi emitida corretamente.

3. Informações a Registrar no GenieGo

CampoDescrição
api_keyA chave API do QSM emitida (manuseie com cuidado)
seller_idO seu próprio Seller ID
No GenieGo Integration Hub [Registrar], insira api_key·seller_id e salve. As credenciais são criptografadas em repouso com AES-256-GCM e mascaradas na leitura.

4. Problemas Comuns

ProblemaCausaSolução
O menu de gerenciamento de API está ausenteNão é uma conta de vendedor (QSM), ou permissões insuficientesFaça login com uma conta de vendedor Qoo10 (QSM) e verifique o menu de gerenciamento de API.
Falha na autenticaçãoChave API ou Seller ID inseridos incorretamenteVerifique novamente a chave API e o Seller ID emitidos no QSM e insira-os.

Lista de Verificação Final

  • Fiz login com uma conta de vendedor Qoo10 (QSM).
  • Emiti uma chave de autenticação de API no menu de gerenciamento de API.
  • Verifiquei o meu próprio Seller ID.
  • Inseri api_key·seller_id no GenieGo [Registrar] e salvei.

📘 Guia para iniciantes

Meta Ads
Manual de emissão da API Key

Um guia passo a passo voltado para iniciantes sobre como emitir um token de acesso de usuário do sistema a partir do seu app empresarial da Meta e registrá-lo junto com o ID da conta de anúncios (act_) no GenieGo Integration Hub para conectar seus dados de publicidade do Meta Ads.

Tempo estimadoCerca de 10–20 min
Conta necessáriaConta pessoal do Facebook · Conta de anúncios da Meta
Chaves emitidasToken de acesso · ID da conta de anúncios (act_)
ResultadoCredenciais da Marketing API

1. Antes de começar

A integração do Meta Ads requer uma conta empresarial e uma conta de anúncios que esteja veiculando anúncios ativamente.

Obrigatório

Conta pessoal do Facebook · Conta de anúncios

Você precisa de uma conta pessoal do Facebook e de uma conta de anúncios da Meta que esteja veiculando anúncios ativamente.

Chaves emitidas

Token de acesso · ID da conta de anúncios (act_)

Você obterá um token de usuário do sistema (token de acesso) e um ID numérico de conta de anúncios que começa com act_.

Observação: Acessar dados fora do seu próprio negócio requer Análise do App (App Review). Se você conectar apenas contas de anúncios dentro do seu próprio negócio, poderá prosseguir sem análise.

2. Etapas de emissão

Emita na ordem abaixo e registre no GenieGo.

1

Criar e verificar uma conta empresarial

Abra o site no seu navegador, faça login e, em seguida, crie uma conta empresarial por meio de [Criar empresa] e conclua a verificação por e-mail.

business.facebook.com
2

Criar um app

Clique em [Meus apps] > [Criar app] no canto superior direito, escolha o tipo "Empresa" e crie o app.

developers.facebook.com → Meus apps → Criar app
3

Adicionar a Marketing API

No painel do app, encontre o card "Marketing API" e clique em [Configurar] para adicioná-la como produto.

4

Criar um usuário do sistema e atribuir permissões

Vá em Configurações da empresa > [Usuários] > [Usuários do sistema] > [Adicionar] para criar um usuário do sistema e, em seguida, conceda permissões de conta de anúncios em [Atribuir ativos].

business.facebook.com/settings → Usuários → Usuários do sistema
5

Gerar um token de acesso

Na mesma tela, clique em [Gerar token], selecione seu app, marque as permissões (ads_read, ads_management), gere o token e copie seu valor.

6

Encontrar o ID da sua conta de anúncios

No Gerenciador de Anúncios, encontre o ID numérico de conta de anúncios que começa com act_.

adsmanager.facebook.com
Depois que o token for gerado com as permissões ads_read e ads_management marcadas, ele terá sido emitido corretamente.

3. Informações para registrar no GenieGo

CampoDescrição
access_tokenO token de acesso de usuário do sistema emitido (manuseie com cuidado)
ad_account_idID da conta de anúncios que começa com act_
No GenieGo Integration Hub > o [Registrar] do canal, cole seu token de acesso e o ID da conta de anúncios (act_) e salve. As credenciais são criptografadas em repouso com AES-256-GCM e mascaradas na leitura.

4. Problemas comuns

ProblemaCausaSolução
Os dados não podem ser consultados devido a erros de permissãoPermissões do token (ads_read·ads_management) não marcadas, ou permissão da conta de anúncios não atribuídaAtribua a permissão de conta de anúncios ao usuário do sistema e marque as permissões ads_read·ads_management ao gerar o token.
O acesso aos dados de outra empresa é negadoAnálise do App (App Review) não concluídaPara acessar dados fora do seu próprio negócio, solicite e conclua a Análise do App.
Erro de formato do ID da conta de anúnciosPrefixo act_ ausente, ou apenas númerosNo Gerenciador de Anúncios, encontre o ID completo que começa com act_ e insira-o.

Lista de verificação final

  • Criei uma conta empresarial em business.facebook.com e concluí a verificação por e-mail.
  • Criei um app do tipo Empresa em developers.facebook.com e adicionei a Marketing API.
  • Criei um usuário do sistema e atribuí a permissão de conta de anúncios.
  • Gerei e copiei um token de acesso com as permissões ads_read·ads_management.
  • Encontrei o ID da conta de anúncios que começa com act_.
  • Inseri access_token·ad_account_id no GenieGo [Registrar] e salvei.

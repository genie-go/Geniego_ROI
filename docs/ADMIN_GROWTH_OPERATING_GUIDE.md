# ADMIN GROWTH — OPERATING GUIDE (운영 가이드)

## 1. 접근
1. Admin 계정 로그인(로고 숨김 진입 + GENIEGO-ADMIN, MFA 통과).
2. 사이드바 **시스템 관리 → 🚀 그로스 센터** (`/admin/growth`). 일반 구독 사용자에게는 보이지 않음.

## 2. 첫 설정
1. **세그먼트 탭** → "기본 타겟 17종 시드" 클릭(이커머스셀러~총판). 필요 시 편집/추가.
2. **설정 탭** → 모드는 기본 **TEST**. 안전하게 시뮬레이션부터 시작.

## 3. 리드 운영
- **리드 탭** → "+ 리드"로 수집(또는 향후 랜딩/이벤트 연동).
- 각 리드 "이벤트" 버튼 → 방문/데모/체험/유료 등 기록 시 **점수·등급·단계 자동 갱신**.
- 등급 Hot/SQL = 영업 우선. 필터로 추출.

## 4. 캠페인 운영
1. **캠페인 탭** → "+ 캠페인"(이름/목표/세그먼트/채널/예산).
2. "AI 생성" → 세그먼트 맞춤 카피 12종 생성 → **승인 대기**.
3. **승인 탭**에서 콘텐츠 승인.
4. 실행:
   - **TEST 모드**: "시뮬레이션" → 예상 리드/유료/CAC/회수기간 산출(실집행 0).
   - **LIVE 모드**: "실행" → ① Live 전환 승인 → ② 실행 승인 → ③ 채널 자격증명 등록 확인 → ④ 실제 집행+감사.

## 5. Live 전환 절차 (안전)
1. 설정 탭 → "LIVE 모드" 클릭 → **승인 요청** 생성(즉시 전환 안 됨).
2. 승인 탭에서 `live_mode` 승인 → 모드 LIVE.
3. 채널별 자격증명을 **연동 허브**에서 `platform_growth` 컨텍스트로 등록(관리자 별도 저장).
4. 캠페인 "실행" → 자격증명 충족 시 running.

## 6. 자격증명 (관리자 별도 등록)
- 광고/메시징 채널 키는 기존 **연동 허브/채널 자격증명** 인프라(`channel_credential`)에 등록.
- Growth 용은 테넌트 `platform_growth` 로 분리 저장 → 등록 즉시 Live 실행 가능.
- AI 디자인/영상은 AI 디자인 엔진에서 생성하거나 수동 업로드한 소재를 캠페인에 연결.

## 7. 성과 검증
- **대시보드 탭**: 오늘 리드·데모·체험·유료·MRR·CAC·LTV·ROAS·Trial-to-Paid·Payback·순이익ROI + 퍼널 차트.
- TEST 예상치 vs LIVE 실측치 비교로 자동화 전/후 성과 입증.

## 8. 감사
- 설정 탭 하단 **감사 로그**: 생성/승인/실행/모드전환 전 이력(행위자·시각·상세).

## 9. 모바일
- 카드 flex-wrap, 테이블 가로 스크롤로 모바일/Capacitor에서 동작.

## 주의
- 배포는 사용자 승인 필수(운영=master push 시 자동 배포). 운영/데모 동반 swap.
- Live 실집행은 `AdAdapters` 월 예산 cap·kill-switch와 함께 운영.

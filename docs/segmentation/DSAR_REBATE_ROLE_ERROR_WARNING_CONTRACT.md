## 🔴 Error 와 Warning 의 경계
**Error = 차단** · **Warning = 통과시키되 기록**.
**Dormant·Orphan·Drift 가 Warning 인 이유**: 탐지에 오탐이 있고, **오탐 차단은 정상 사용자를 끊는다**(1-9).

## 실측·규칙
**Rebate Resource 실체는 전부 부재**(rebate grep 0·1-1~1-4 확정) → **등록 대상**. 기존 `acl_permission` 매트릭스에 **Resource×Action 으로 등록**(중복 Permission Store 신설 금지). **REBATE_CREDENTIAL_REFERENCE 는 Secret 원문이 아니라 Reference** — 조회 권한 생성 금지(§39). REBATE_FEATURE_FLAG / REBATE_INCIDENT 는 **Registry 자체가 부재**(1-4 실측) → Reference 만.

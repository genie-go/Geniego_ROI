## GeniegoROI 강점 (경쟁사 초과 6개 도메인)

3. **글로벌 i18n (+7)** — 15개국+한국 국내마켓+글로벌 마켓 단일플랫폼. 서구 경쟁사는 한국 커버리지 취약, 한국 솔루션은 글로벌 취약.
4. **정산 P&L (+1)** · **조직/구독 (+1)** · **★통합도/폐루프 (+21)**.

## "코드 아님"(외부의존·재플래그 금지) — 감점 성격 아님
- **SOC2 Type II·ISO 27001 인증** = 외부 감사 프로세스(준비도 대시보드 완비). 코드로 해결 불가.
- **실 광고계정 쓰기 OAuth·CTV/DSP/PMax·text-to-video** = 실키 등록 후 라이브 검증.
- **발송 DNS(SPF/DKIM/DMARC)·라이브 SFU 미디어서버·geo-ID맵·DW 크리덴셜·IdP** = 인프라/외부자산.
- **원시 국내마켓 채널수(사방넷 650)** = 벤더별 실 API·크리덴셜(genericFetch 정직 pending 커버).

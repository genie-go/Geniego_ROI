# FeatureRegistry — 기능 레지스트리 (포인터)

> **정본(SSOT)**: **`docs/IMPLEMENTATION_STATUS.md`** (기능 구현 정본 이력·file:line 실증·✅/🔧/⏳ 상태).
> 서브도메인 기능맵: `docs/ADMIN_GROWTH_FEATURE_MAP.md`·`EXISTING_PERMISSION_FEATURE_MAP.md`.
> 메모리 인덱스: `MEMORY.md` → `project_n*`(차수별).

## 원칙 (사용자 지시·중복금지)
- 신규 기능 착수 전 **여기+IMPLEMENTATION_STATUS에서 기구현 확인**. 이미 "✅ 구현됨"이면 재구현 금지·심화만.
- 진짜 부재만 신설(도메인 구분 명시). 신규/확인분 즉시 IMPLEMENTATION_STATUS에 기록.

## 갱신 규칙
신규/확장/확인 기능은 `docs/IMPLEMENTATION_STATUS.md`에 추가(정본). FeatureStatusMatrix.md 와 연동.

# FeatureStatusMatrix — 기능 상태 매트릭스 (포인터)

> **정본(SSOT)**: **`docs/IMPLEMENTATION_STATUS.md`** — 도메인별 ✅구현·배포 / 🔧진행중 / ⏳보류(외부 크리덴셜/인프라)로 상태 표기 + "🔧진행중/⏳보류" 갭 표.
> 경쟁 재평가: `docs/COMPETITIVE_REVALIDATION_*.md`(도메인 점수).

## 상태 표기 규약
- ✅ 구현·배포됨(file:line 실증) · 🔧 진행중 · ⏳ 보류(외부의존).
- ⏳ 보류 = 외부 크리덴셜/인프라 필요(실광고키·검색API·SFU·SOC2인증·영상DCO)—투기적 구현 금지.

## 갱신 규칙
기능 상태 변경(구현완료·보류해제) 시 IMPLEMENTATION_STATUS 상태표 갱신.

## 1. 평가 방법론 & 전제

### 1.1 점수 체계 (기능 도메인별 100점 만점)
- **경쟁사 점수** = 해당 도메인 **best-in-class 글로벌 경쟁사**가 제공하는 기능 성숙도(시장 표준 100 기준).
- **GeniegoROI 점수** = 동일 도메인에서 GeniegoROI가 구현한 기능 성숙도.
- 종합 점수 = 도메인 가중 평균(가중치는 e커머스 ROI 플랫폼 사용가치 기준).

### 1.2 핵심 전제 (★중요)
> **"자격증명만 등록하면 즉시 실행" 구현 완료 = 점수 인정.**
> GeniegoROI는 다수 채널/PG/미디어서버 연동이 **코드 완비**(어댑터·인증·멱등·라우팅) 상태이며, 라이브 OAuth 키·PG 라이브키·외부 미디어 infra만 미등록입니다. 평가는 **코드 완비 기준**으로 점수를 부여하고, "라이브 검증 대기"는 별도 명시합니다. (근거: 핸들러 85종·라우트 v377~v431·어댑터 다수 — §본문 각주)

## 2. 기능 도메인별 평가표

### 2.1 어트리뷰션 (멀티터치·Shapley·Markov)
**Northbeam/Triple Whale 95 → GeniegoROI 93** =
- **강점**: 6모델(last/first/linear/time-decay/position/**Markov removal-effect 데이터기반**) + **Exact Shapley 2^n** + Pairwise 시너지 + 부트스트랩 신뢰구간 + **순이익(net-profit) 기여도**까지 산출(경쟁사는 매출 기여 중심). 서버측 어트리뷰션 엔진(`AttributionEngine`·`Attribution`·`AttributionMetrics`) 테넌트 격리, 여정×전환 데이터 기반.
- **약점**: 쿠키리스 환경의 **결정론적 ID 그래프**(고객 cross-device stitching)는 경쟁사(Northbeam) 대비 단순. 픽셀 수집은 구현(`PixelTracking` HMAC·dedup)되어 있으나 라이브 트래픽 검증 표본이 제한적.
- 보강: ID-resolution(이메일/주문 기반) 고도화 + 라이브 픽셀 표본 확대.

## 4. GeniegoROI 결정적 우위 (Moat)

1. **순이익 P&L 통합(+12)** — 매출 ROAS가 아닌 COGS·물류·반품·정산·수수료·다통화를 반영한 **실제 순이익 ROI**를 광고~커머스~물류 전 경로에서 단일 SSOT로 산출. 경쟁사는 광고 또는 커머스 한쪽만.
2. **통합 범위** — 어트리뷰션+MMM+자동화+CRM+OMS+WMS+정산+라이브+AI를 멀티테넌트 단일 제품으로. 사방넷(커머스/물류)·Triple Whale(광고)·Klaviyo(CRM)를 따로 쓰던 것을 하나로.
3. **AI 통합(+2)** — 측정→실행→학습 OS + 전체 용어 15개국 챗봇.
4. **글로벌 현지화(+2)** — 15개국 현지 자연어(본 차수 712키 누출 종결).
5. **"자격증명만 등록하면 즉시 실행"** — 채널/PG/미디어 어댑터 코드 완비 → 도입 즉시 활성.
